# 現場スタッフ向け 操作マニュアル

> **対象ユーザー**: 現場で働くスタッフの方
> **使用デバイス**: スマートフォン（推奨）

---

## ⚠️ リーダー・一般スタッフの権限制限について

このシステムでは、**4つの権限レベル**があります：
- **admin（管理者）**: すべての操作が可能
- **manager（マネージャー）**: 現場運営の中心的役割（道具・消耗品の登録編集、在庫調整など）
- **leader（リーダー）**: 現場での日常操作（一般スタッフと同等）
- **staff（一般スタッフ）**: 現場での日常操作

**注意**: leader（リーダー）とstaff（一般スタッフ）の権限は同じです。以下の制限が適用されます。

### 🚫 リーダー・一般スタッフができないこと
1. **道具セットの登録・編集・削除**
   - 道具セットの閲覧と移動は可能です
   - 新しいセットの作成は管理者またはマネージャーに依頼してください

2. **CSV一括登録**
   - すべてのページ（道具、消耗品、重機など）でCSV一括登録は管理者のみ利用可能です

3. **道具・消耗品・現場・重機の登録・編集**
   - 閲覧と移動は可能です
   - 登録・編集はマネージャー以上に依頼してください

### ⚖️ 制限付きでできること
1. **消耗品の在庫調整**
   - **±100個までの調整が可能**です
   - 100個を超える大幅な調整が必要な場合は、管理者またはマネージャーに依頼してください
   - 例：棚卸しで10個の差異 → ✅ 調整可能
   - 例：大量入荷で200個追加 → ❌ マネージャー以上に依頼

### ✅ 通常通りできること
- 道具の移動（QRスキャン、一括移動）
- 道具セットの移動
- 在庫確認
- 移動履歴の閲覧
- その他日常的な操作

## 🔥 最も使う機能：一括移動

### 使用シーン
- **朝、現場へ道具を持ち出すとき**
- **夕方、倉庫へ道具を返却するとき**
- **道具を別の現場へ移動するとき**

### 操作手順

#### **1. 移動先を選択**

画面下部の「📦 一括移動」ボタンをタップ。

3つのボタンから移動先を選びます：

```
┌─────────────────────────────────────┐
│  🏢 倉庫  │  🏗️ 現場  │  🔧 修理  │
└─────────────────────────────────────┘
```

- **🏢 倉庫**: 道具を倉庫に返却する
- **🏗️ 現場**: 道具を現場へ持ち出す
- **🔧 修理**: 道具を修理に出す

「現場」を選んだ場合、次に具体的な現場名を選択します：

```
現場を選択 *
┌─────────────────────────────────┐
│ ▼ 現場を選択してください        │
│   ・◯◯町マンション現場          │
│   ・△△ビル塗装現場             │
│   ・□□橋補修工事               │
└─────────────────────────────────┘
```

#### **2. 道具をスキャン**

移動先を選んだら、次に道具を選択します。

**方法A: カメラでQRコードをスキャン（推奨）**

1. 「📷 カメラでスキャン」ボタンをタップ
2. カメラが起動するので、道具に貼られたQRコードを画面中央の枠に合わせる
3. 自動で読み取られ、「✓ 道具を追加しました！」と表示される
4. 次の道具のQRコードをスキャン（繰り返し）
5. すべてスキャンしたら「×」で閉じる

```
┌─────────────────────────────────┐
│  QRコードをスキャン       ×    │
├─────────────────────────────────┤
│                                 │
│         ┌─────────┐             │
│         │  ░░░░░  │             │
│         │  ░QR░░  │ ← カメラ映像│
│         │  ░░░░░  │             │
│         └─────────┘             │
│                                 │
│  QRコードをカメラに向けてください│
└─────────────────────────────────┘
```

**方法B: 手動入力モード**

QRコードリーダー（ハンディスキャナー）を使う場合：

1. 「⌨️ 手動入力」ボタンをタップ（緑色に変わる）
2. QRコードリーダーでスキャン
3. 自動的に入力欄にコードが入力される
4. **Enterキー**を押す
5. 「✓ 道具を追加しました！」と表示される
6. 次の道具をスキャン（繰り返し）

```
┌─────────────────────────────────┐
│ QRコードをスキャン（Enterで確定）│
│ ┌─────────────────────────────┐ │
│ │ [入力されたコード]    ← ここ│ │
│ └─────────────────────────────┘ │
└─────────────────────────────────┘
```

#### **3. 選択した道具を確認**

スキャンした道具が一覧で表示されます：

```
┌─────────────────────────────────────┐
│ 3. 選択中の道具（5件）  すべてクリア │
├─────────────────────────────────────┤
│ ✓ 電動ドリル               削除 ×  │
│   #001 • Model-XYZ                  │
│   現在位置: 倉庫                    │
├─────────────────────────────────────┤
│ ✓ サンダー                 削除 ×  │
│   #002 • Model-ABC                  │
│   現在位置: 倉庫                    │
├─────────────────────────────────────┤
│ ...                                 │
└─────────────────────────────────────┘
```

間違って追加した場合は、右側の「削除 ×」をタップして削除できます。

#### **4. 一括登録ボタンを押す**

画面下部の青いボタンをタップして完了：

```
┌─────────────────────────────────┐
│   5件を一括登録   │  キャンセル  │
└─────────────────────────────────┘
```

登録中は進捗が表示されます：

```
┌─────────────────────────────────┐
│ 登録中... (3 / 5)          60%  │
│ ▓▓▓▓▓▓▓▓▓▓▓▓░░░░░░░░░         │
└─────────────────────────────────┘
```

完了すると、自動で移動履歴ページに遷移します。

## 🧰 消耗品の操作

### 在庫を確認する

1. ホーム画面の「🧰 消耗品管理」をタップ
2. 一覧が表示される：

```
┌─────────────────────────────────────┐
│ 消耗品名      倉庫  現場  合計  操作 │
├─────────────────────────────────────┤
│ 軍手         100個  20個 120個      │
│              在庫調整 | 移動         │
├─────────────────────────────────────┤
│ ⚠️ 養生テープ    5個   0個   5個    │
│ (在庫不足)   在庫調整 | 移動         │
└─────────────────────────────────────┘
```

⚠️マークは、最小在庫を下回っている警告です。

### 在庫を調整する（入庫・出庫）

> ⚠️ **一般スタッフの制限**: ±100個までの調整が可能です。100個を超える調整は管理者またはリーダーに依頼してください。

**入庫（購入して倉庫に追加する場合）**

1. 該当の消耗品の「在庫調整」をタップ
2. 「在庫を追加」を選択
3. 数量を入力（例：50） ※一般スタッフは100個まで
4. 理由を入力（例：棚卸による修正）
5. 「在庫を調整」ボタンをタップ

**出庫（廃棄・消費した場合）**

1. 該当の消耗品の「在庫調整」をタップ
2. 「在庫を減らす」を選択
3. 数量を入力（例：10） ※一般スタッフは100個まで
4. 理由を入力（例：紛失・破損）
5. 「在庫を調整」ボタンをタップ

**100個を超える調整が必要な場合**

大量入荷などで100個を超える調整が必要な場合：
1. 管理者またはリーダーに連絡
2. 調整内容（消耗品名、数量、理由）を伝える
3. 管理者が代わりに調整を実施

### 消耗品を現場へ移動する

1. 該当の消耗品の「移動」をタップ
2. 移動元を選択（倉庫 or 現場）
3. 移動先を選択（倉庫 or 現場）
4. 数量を入力
5. 「登録」ボタンをタップ

## 💡 よくある質問（FAQ）

### Q1. QRコードが読み取れない

**A.** 以下を試してください：
1. 明るい場所で試す
2. QRコードを画面中央の枠にしっかり合わせる
3. カメラのピントが合うまで少し待つ
4. QRコードが汚れていないか確認
5. 手動入力モードでQRコードリーダーを使う

### Q2. 間違って道具を選択してしまった

**A.** 選択済みリストの右側にある「削除 ×」ボタンをタップすれば削除できます。

### Q3. 一括移動の登録中にエラーが出た

**A.** 一部の道具だけ失敗した場合、エラーメッセージに失敗した道具名が表示されます。失敗した道具だけがリストに残るので、もう一度「一括登録」ボタンを押してください。

### Q4. 移動先の現場が見つからない

**A.** 現場は管理者が登録する必要があります。管理者に連絡して現場を追加してもらってください。

### Q5. 道具を持ち出したのに、まだ倉庫にあると表示される

**A.** 移動登録をしていない可能性があります。「一括移動」機能で正しく登録してください。リアルタイムで位置が更新されます。

### Q6. カメラが起動しない

**A.** ブラウザにカメラの許可を与える必要があります：
1. ブラウザの設定を開く
2. 「サイトの設定」または「権限」を選択
3. カメラを「許可」に設定
4. ページを再読み込み

### Q7. スマホでログインできない

**A.** 以下を確認してください：
1. メールアドレスとパスワードが正しいか
2. インターネットに接続されているか
3. 会社から指定された正しいURLにアクセスしているか

問題が解決しない場合は、管理者に連絡してください。

## 📝 作業報告書

### 作業報告書とは

現場での作業内容を記録・報告するための機能です。以下の情報を記録できます：
- 作業日、現場、天気
- 作業内容の説明
- 作業時間と作業員
- 作業場所、進捗率
- 使用した材料・道具
- 安全・品質に関する事項
- 次回作業予定

### 作業報告書の作成

#### 1. 新規作成画面へ移動
1. メニューから「作業報告書」をタップ
2. 画面右上の「+ 新規作成」ボタンをタップ

#### 2. 基本情報を入力

**必須項目**：
- **現場**: プルダウンから選択
- **作業日**: カレンダーから選択（初期値：今日）
- **作業内容**: 今日行った作業の詳細を記入

**その他の項目**：
- **天気**: 晴れ/曇り/雨/雪から選択
- **作業時間**: 開始時刻、終了時刻、休憩時間
- **作業員**: 一緒に作業した人を選択（複数選択可能）

#### 3. 詳細情報を入力（任意）

**作業場所**：
- 現場内での具体的な作業場所（例：2階リビング、外壁東側）

**進捗率**：
- 作業の進み具合を％で入力

**使用した材料・道具**：
- 作業で使用した材料や道具を記入

**安全・品質関連**：
- 安全上の問題や品質上の問題があれば記録
- 問題があった場合は詳細を記入

**施主対応**：
- 施主との打ち合わせや確認があれば記録

**次回作業予定**：
- 次回行う予定の作業内容を記入

#### 4. 写真・資料の添付（任意）

**写真の追加**：
1. 「写真を追加」ボタンをタップ
2. カメラで撮影、またはギャラリーから選択
3. 複数枚の写真を追加できます

**資料の添付**：
1. 「資料を添付」ボタンをタップ
2. PDFや画像ファイルを選択

#### 5. 保存・提出

2つの保存方法があります：

**下書き保存**：
- 「下書き保存」ボタンをタップ
- 後で編集・提出できます
- 自分だけが閲覧できる状態

**提出**：
- 「提出」ボタンをタップ
- リーダー・管理者に報告が通知されます
- 提出後は編集できません（承認フローがある場合）

### 作業報告書の一覧・確認

#### 一覧画面の見方

作業報告書一覧では以下が表示されます：
- 作業日
- 現場名
- 作業内容（一部）
- ステータス（下書き/提出済/承認済/差戻）
- 作成者

#### フィルター機能

以下の条件で絞り込みができます：
- **ステータス**: 全て/下書き/提出済/承認済/差戻
- **現場**: プルダウンから選択
- **作成者**: 作成したスタッフで絞り込み
- **日付範囲**: 期間を指定
- **キーワード**: 作業内容から検索

#### 報告書の詳細確認

1. 一覧から報告書をタップ
2. 詳細画面で以下を確認できます：
   - 全ての記入内容
   - 添付された写真・資料
   - 承認履歴（承認フローがある場合）

### 承認フロー（組織の設定による）

組織で承認フローが有効になっている場合：

#### スタッフ側の流れ

1. **作業報告書を作成・提出**
   - 提出すると「提出済」ステータスになります
   - リーダー・管理者に通知が届きます

2. **承認・差戻の結果を確認**
   - **承認された場合**:
     - ステータスが「承認済」になります
     - 通知で結果を確認できます

   - **差戻された場合**:
     - ステータスが「差戻」になります
     - 差戻理由を確認できます
     - 報告書を編集して再提出できます

#### 再提出の方法（差戻された場合）

1. 差戻された報告書の詳細画面を開く
2. 「編集」ボタンをタップ
3. 指摘事項を修正
4. 「提出」ボタンで再提出

### リーダー・管理者の承認作業

#### 承認待ち一覧の確認

1. 作業報告書一覧画面の「承認待ち一覧を見る →」をタップ
2. 提出済みの報告書が表示されます

#### 個別承認

1. 報告書をタップして内容を確認
2. 画面上部の承認ボタンをタップ：
   - **承認**: 内容に問題がない場合
   - **却下**: 修正が必要な場合

3. コメントを入力（任意）
   - 承認時: 良かった点などを記入
   - 却下時: 修正してほしい点を記入

#### 一括承認機能

複数の報告書をまとめて承認できます：

1. 承認待ち一覧で報告書の左側のチェックボックスを選択
2. 「全選択」で全ての報告書を選択可能
3. 「一括承認」ボタンをタップ
4. コメントを入力（全ての報告書に適用されます）
5. 「一括承認する」で確定

#### 一括PDF出力

複数の報告書をまとめてPDF出力できます：

1. 承認待ち一覧で報告書を複数選択
2. 「一括PDF出力」ボタンをタップ
3. 選択した報告書のPDFが連続でダウンロードされます

### PDF出力

作業報告書をPDF形式で出力できます：

#### 個別PDF出力

1. 報告書の詳細画面を開く
2. 画面上部の「PDF出力」ボタンをタップ
3. PDFファイルがダウンロードされます

PDFには以下が含まれます：
- 全ての記入内容
- 添付された写真（複数ページ）
- 添付資料の一覧
- カスタムフィールド（組織で設定されている場合）

### 通知機能

作業報告書に関する通知が届きます：

#### スタッフが受け取る通知

- **承認済**: 報告書が承認されました
- **差戻**: 報告書が差戻されました（修正が必要です）

#### リーダー・管理者が受け取る通知

- **提出済**: スタッフが報告書を提出しました
- **再提出**: 差戻された報告書が再提出されました

通知は画面右上の🔔アイコンから確認できます。

### よくある質問（FAQ）

**Q: 提出後に間違いに気づきました。どうすればいいですか？**
A: 承認前であればリーダー・管理者に連絡して差戻してもらうか、新しい報告書を作成してください。承認後は新規報告書で補足してください。

**Q: 写真は何枚まで添付できますか？**
A: 制限はありませんが、通信量を考慮して必要な写真のみを添付してください。

**Q: 下書きはいつまで保存されますか？**
A: 削除しない限り保存されます。不要になったら削除してください。

**Q: 過去の報告書を確認したいのですが？**
A: 作業報告書一覧でフィルター機能を使って日付範囲や現場で絞り込んでください。

**Q: 複数の現場で作業した場合はどうすればいいですか？**
A: 現場ごとに別々の作業報告書を作成してください。
