# 現場スタッフ向け 操作マニュアル

> **対象ユーザー**: 現場で働くスタッフの方
> **使用デバイス**: スマートフォン（推奨）

---

## ⚠️ リーダー・一般スタッフの権限制限について

このシステムでは、**4つの権限レベル**があります：
- **admin（管理者）**: すべての操作が可能
- **manager（マネージャー）**: 現場運営の中心的役割（道具・消耗品の登録編集、在庫調整など）
- **leader（リーダー）**: 現場での日常操作（一般スタッフと同等）
- **staff（一般スタッフ）**: 現場での日常操作

**注意**: leader（リーダー）とstaff（一般スタッフ）の権限は同じです。以下の制限が適用されます。

### 🚫 リーダー・一般スタッフができないこと
1. **道具セットの登録・編集・削除**
   - 道具セットの閲覧と移動は可能です
   - 新しいセットの作成は管理者またはマネージャーに依頼してください

2. **CSV一括登録**
   - すべてのページ（道具、消耗品、重機など）でCSV一括登録は管理者のみ利用可能です

3. **道具・消耗品・現場・重機の登録・編集**
   - 閲覧と移動は可能です
   - 登録・編集はマネージャー以上に依頼してください

### ⚖️ 制限付きでできること
1. **消耗品の在庫調整**
   - **±100個までの調整が可能**です
   - 100個を超える大幅な調整が必要な場合は、管理者またはマネージャーに依頼してください
   - 例：棚卸しで10個の差異 → ✅ 調整可能
   - 例：大量入荷で200個追加 → ❌ マネージャー以上に依頼

### ✅ 通常通りできること
- 道具の移動（QRスキャン、一括移動）
- 道具セットの移動
- 在庫確認
- 移動履歴の閲覧
- その他日常的な操作

## 🔥 最も使う機能：一括移動

### 使用シーン
- **朝、現場へ道具を持ち出すとき**
- **夕方、倉庫へ道具を返却するとき**
- **道具を別の現場へ移動するとき**

### 操作手順

#### **1. 移動先を選択**

画面下部の「📦 一括移動」ボタンをタップ。

3つのボタンから移動先を選びます：

```
┌─────────────────────────────────────┐
│  🏢 倉庫  │  🏗️ 現場  │  🔧 修理  │
└─────────────────────────────────────┘
```

- **🏢 倉庫**: 道具を倉庫に返却する
- **🏗️ 現場**: 道具を現場へ持ち出す
- **🔧 修理**: 道具を修理に出す

「現場」を選んだ場合、次に具体的な現場名を選択します：

```
現場を選択 *
┌─────────────────────────────────┐
│ ▼ 現場を選択してください        │
│   ・◯◯町マンション現場          │
│   ・△△ビル塗装現場             │
│   ・□□橋補修工事               │
└─────────────────────────────────┘
```

#### **2. 道具をスキャン**

移動先を選んだら、次に道具を選択します。

**方法A: カメラでQRコードをスキャン（推奨）**

1. 「📷 カメラでスキャン」ボタンをタップ
2. カメラが起動するので、道具に貼られたQRコードを画面中央の枠に合わせる
3. 自動で読み取られ、「✓ 道具を追加しました！」と表示される
4. 次の道具のQRコードをスキャン（繰り返し）
5. すべてスキャンしたら「×」で閉じる

```
┌─────────────────────────────────┐
│  QRコードをスキャン       ×    │
├─────────────────────────────────┤
│                                 │
│         ┌─────────┐             │
│         │  ░░░░░  │             │
│         │  ░QR░░  │ ← カメラ映像│
│         │  ░░░░░  │             │
│         └─────────┘             │
│                                 │
│  QRコードをカメラに向けてください│
└─────────────────────────────────┘
```

**方法B: 手動入力モード**

QRコードリーダー（ハンディスキャナー）を使う場合：

1. 「⌨️ 手動入力」ボタンをタップ（緑色に変わる）
2. QRコードリーダーでスキャン
3. 自動的に入力欄にコードが入力される
4. **Enterキー**を押す
5. 「✓ 道具を追加しました！」と表示される
6. 次の道具をスキャン（繰り返し）

```
┌─────────────────────────────────┐
│ QRコードをスキャン（Enterで確定）│
│ ┌─────────────────────────────┐ │
│ │ [入力されたコード]    ← ここ│ │
│ └─────────────────────────────┘ │
└─────────────────────────────────┘
```

#### **3. 選択した道具を確認**

スキャンした道具が一覧で表示されます：

```
┌─────────────────────────────────────┐
│ 3. 選択中の道具（5件）  すべてクリア │
├─────────────────────────────────────┤
│ ✓ 電動ドリル               削除 ×  │
│   #001 • Model-XYZ                  │
│   現在位置: 倉庫                    │
├─────────────────────────────────────┤
│ ✓ サンダー                 削除 ×  │
│   #002 • Model-ABC                  │
│   現在位置: 倉庫                    │
├─────────────────────────────────────┤
│ ...                                 │
└─────────────────────────────────────┘
```

間違って追加した場合は、右側の「削除 ×」をタップして削除できます。

#### **4. 一括登録ボタンを押す**

画面下部の青いボタンをタップして完了：

```
┌─────────────────────────────────┐
│   5件を一括登録   │  キャンセル  │
└─────────────────────────────────┘
```

登録中は進捗が表示されます：

```
┌─────────────────────────────────┐
│ 登録中... (3 / 5)          60%  │
│ ▓▓▓▓▓▓▓▓▓▓▓▓░░░░░░░░░         │
└─────────────────────────────────┘
```

完了すると、自動で移動履歴ページに遷移します。

## 📱 QRスキャン機能（個別確認）

1つの道具の情報を確認したいときに使います。

### 使用シーン
- この道具は今どこにあるか確認したい
- この道具のステータスを知りたい
- この道具の詳細情報を見たい

### 操作手順

1. 画面下部の「QR」ボタンをタップ
2. カメラが起動
3. 道具のQRコードをスキャン
4. 自動で道具の詳細画面に移動

詳細画面では以下が確認できます：
- 道具名、型番
- 現在の位置（倉庫 or 現場名）
- ステータス（使用中、利用可能など）
- シリアル番号
- 写真（登録されている場合）
- 過去の移動履歴

## 🧰 消耗品の操作

### 在庫を確認する

1. ホーム画面の「🧰 消耗品管理」をタップ
2. 一覧が表示される：

```
┌─────────────────────────────────────┐
│ 消耗品名      倉庫  現場  合計  操作 │
├─────────────────────────────────────┤
│ 軍手         100個  20個 120個      │
│              在庫調整 | 移動         │
├─────────────────────────────────────┤
│ ⚠️ 養生テープ    5個   0個   5個    │
│ (在庫不足)   在庫調整 | 移動         │
└─────────────────────────────────────┘
```

⚠️マークは、最小在庫を下回っている警告です。

### 在庫を調整する（入庫・出庫）

> ⚠️ **一般スタッフの制限**: ±100個までの調整が可能です。100個を超える調整は管理者またはリーダーに依頼してください。

**入庫（購入して倉庫に追加する場合）**

1. 該当の消耗品の「在庫調整」をタップ
2. 「在庫を追加」を選択
3. 数量を入力（例：50） ※一般スタッフは100個まで
4. 理由を入力（例：棚卸による修正）
5. 「在庫を調整」ボタンをタップ

**出庫（廃棄・消費した場合）**

1. 該当の消耗品の「在庫調整」をタップ
2. 「在庫を減らす」を選択
3. 数量を入力（例：10） ※一般スタッフは100個まで
4. 理由を入力（例：紛失・破損）
5. 「在庫を調整」ボタンをタップ

**100個を超える調整が必要な場合**

大量入荷などで100個を超える調整が必要な場合：
1. 管理者またはリーダーに連絡
2. 調整内容（消耗品名、数量、理由）を伝える
3. 管理者が代わりに調整を実施

### 消耗品を現場へ移動する

1. 該当の消耗品の「移動」をタップ
2. 移動元を選択（倉庫 or 現場）
3. 移動先を選択（倉庫 or 現場）
4. 数量を入力
5. 「登録」ボタンをタップ

## 🔔 通知の確認

画面下部の「🔔 通知」ボタンをタップすると、システムからの通知を確認できます。

### 通知の種類
- **在庫アラート**: 消耗品の在庫が不足しています
- **保証期限**: 道具の保証期限が近づいています
- **システム通知**: 重要なお知らせ

未読の通知がある場合、ベルアイコンに赤い丸（数字）が表示されます：

```
🔔 (3)  ← 未読3件
```

通知をタップすると既読になります。

## 💡 よくある質問（FAQ）

### Q1. QRコードが読み取れない

**A.** 以下を試してください：
1. 明るい場所で試す
2. QRコードを画面中央の枠にしっかり合わせる
3. カメラのピントが合うまで少し待つ
4. QRコードが汚れていないか確認
5. 手動入力モードでQRコードリーダーを使う

### Q2. 間違って道具を選択してしまった

**A.** 選択済みリストの右側にある「削除 ×」ボタンをタップすれば削除できます。

### Q3. 一括移動の登録中にエラーが出た

**A.** 一部の道具だけ失敗した場合、エラーメッセージに失敗した道具名が表示されます。失敗した道具だけがリストに残るので、もう一度「一括登録」ボタンを押してください。

### Q4. 移動先の現場が見つからない

**A.** 現場は管理者が登録する必要があります。管理者に連絡して現場を追加してもらってください。

### Q5. 道具を持ち出したのに、まだ倉庫にあると表示される

**A.** 移動登録をしていない可能性があります。「一括移動」機能で正しく登録してください。リアルタイムで位置が更新されます。

### Q6. カメラが起動しない

**A.** ブラウザにカメラの許可を与える必要があります：
1. ブラウザの設定を開く
2. 「サイトの設定」または「権限」を選択
3. カメラを「許可」に設定
4. ページを再読み込み

### Q7. スマホでログインできない

**A.** 以下を確認してください：
1. メールアドレスとパスワードが正しいか
2. インターネットに接続されているか
3. 会社から指定された正しいURLにアクセスしているか

問題が解決しない場合は、管理者に連絡してください。
