# Field Tool Manager 料金戦略・収益モデル設計書【決定版】

## 1. エグゼクティブサマリー

### 1.1 サービス概要
- **サービス名**: Field Tool Manager
- **カテゴリ**: 建設・現場業向けSaaS（現場資産管理・業務効率化プラットフォーム）
- **技術スタック**: Next.js + Supabase + PostgreSQL
- **ターゲット**: 建設業、塗装業、電気工事業等の現場系事業者（従業員10〜100名規模）

### 1.2 料金モデル概要
- **基本構造**: ベース料金（人数ベース）+ 機能パック料金 + 初期費用
- **課金方式**: 月額サブスクリプション（年契約で10%割引）
- **請求方法**: 請求書払い（初期）→ クレジットカード対応（30社到達後）
- **最終更新**: 2025年12月（決定版）

## 10. 戦略的提言

### 10.1 価格戦略
1. **初期**: 市場平均の80%で参入（シェア獲得重視）
2. **成長期**: 機能追加による単価向上
3. **成熟期**: 価格を市場平均に近づける

### 10.2 営業戦略
1. **フリーミアム導入**: 5名まで無料（3ヶ月間）
2. **紹介プログラム**: 紹介成約で2ヶ月無料
3. **年契約促進**: 15%割引 + 初期設定サポート無料

### 10.3 収益最大化施策
1. **アップセル**: 機能パックの段階的提案
2. **クロスセル**: 初期設定・研修サービスの追加
3. **LTV向上**: カスタマーサクセス投資
4. **初期費用収益**: データ登録・QR印刷サービスの推進

### 10.4 営業パートナーとの協業
1. **インセンティブ設計**:
   - 累積顧客数に応じたボーナス
   - 解約率改善ボーナス
2. **役割分担の明確化**:
   - パートナー: 営業・1次サポート
   - 自社: 開発・技術サポート
3. **定期的な戦略会議**: 月次での情報共有
