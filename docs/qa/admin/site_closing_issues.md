# 現場完了・片付けのトラブルシューティング

## Q1. 現場を完了にする手順を教えてください。

**A.** 現場詳細画面の「完了にする」ボタンから完了処理できます。

**手順:**
1. **現場管理** を開く
2. 完了する現場をクリック
3. **「完了にする」** ボタンをクリック
4. 確認ダイアログが表示される
5. **「OK」** をクリック

**完了処理で実行されること:**
- 現場のステータスが「完了」になる
- 完了日が自動記録される
- 出退勤打刻ができなくなる
- 道具の持ち出し先として選択できなくなる

**注意:**
- 完了処理は取り消せません（再度有効化するには管理者に依頼が必要）
- 道具の返却は完了前に完了してください

**関連**: [現場完了・片付けマニュアル - 現場の完了処理](../../manual/scenarios/daily/site_closing.md#現場の完了処理)

## Q2. 現場を完了にしたいのですが、道具がまだ現場にあります。どうすればいいですか?

**A.** 道具をすべて返却してから完了処理してください。

**手順:**
1. 現場にある道具の一覧を確認
2. 道具をすべて倉庫に返却登録
3. 返却完了を確認
4. 現場完了処理

**現場にある道具の確認方法:**
1. **道具管理** を開く
2. フィルターで対象の現場を選択
3. 現場にある道具の一覧が表示される

**道具が返却できない場合:**
- 紛失した道具 → 紛失登録
- 破損した道具 → 破損登録

**関連**: [現場完了・片付けマニュアル - 道具の回収確認](../../manual/scenarios/daily/site_closing.md#道具の回収確認)

## Q3. 道具を紛失しました。どう処理すればいいですか?

**A.** 道具詳細画面から「紛失登録」を行ってください。

**手順:**
1. **道具管理** を開く
2. 紛失した道具を検索
3. 道具詳細画面で **「紛失登録」** をクリック
4. 紛失日と状況を入力
5. **「登録」** をクリック

**紛失登録の効果:**
- 道具のステータスが「紛失」になる
- 在庫から除外される
- 紛失記録として保管される

**保険適用について:**
- 保険適用可能な場合は、保険会社に連絡
- 紛失記録をエクスポートして提出

**関連**:
- [現場完了・片付けマニュアル - 紛失・破損道具の処理](../../manual/scenarios/daily/site_closing.md#紛失破損道具の処理)

## Q4. 道具が破損しました。廃棄処理はどうすればいいですか?

**A.** 道具詳細画面から「破損登録」を行い、廃棄を選択してください。

**手順:**
1. **道具管理** を開く
2. 破損した道具を検索
3. 道具詳細画面で **「破損登録」** をクリック
4. 破損日と状況を入力
5. **「廃棄」** を選択
6. **「登録」** をクリック

**修理する場合:**
- 「修理可能」を選択
- 修理完了後、ステータスを「利用可能」に変更

**廃棄処理の効果:**
- 道具のステータスが「廃棄済み」になる
- 在庫から除外される
- 持ち出し・返却登録ができなくなる

**関連**:
- [現場完了・片付けマニュアル - 紛失・破損道具の処理](../../manual/scenarios/daily/site_closing.md#紛失破損道具の処理)

## Q5. 作業報告書に未提出や承認待ちがあります。完了処理できますか?

**A.** はい、完了処理できます。ただし、完了前にすべて承認することを推奨します。

**推奨手順:**
1. 未提出報告書をスタッフに提出してもらう
2. 承認待ちの報告書をすべて承認
3. すべて承認済みになってから完了処理

**完了後も作業報告書は:**
- 参照可能
- 承認可能
- PDF出力可能

**未提出・承認待ちが残っている場合:**
- スタッフに提出を催促
- 承認者に承認を依頼

**関連**: [現場完了・片付けマニュアル - 作業報告書の確認](../../manual/scenarios/daily/site_closing.md#作業報告書の確認)

## Q6. 現場QRコードは完了後に撤去する必要がありますか?

**A.** はい、完了処理前または直後に撤去してください。

**撤去手順:**
1. 現場に設置したQRコードを取り外す
2. ラミネートやケースを回収
3. QRコードは廃棄またはファイリング

**QRコードの扱い:**
- QRコード自体に個人情報は含まれていません
- 同じQRコードは再利用できません（現場ごとに固有）
- 記念に保管する場合もあります

**撤去しないとどうなる?**
- スタッフがスキャンしても打刻できない（完了済み現場のため）
- セキュリティ上の問題はありませんが、撤去推奨

**関連**: [現場完了・片付けマニュアル - 現場QRコードの撤去](../../manual/scenarios/daily/site_closing.md#現場qrコードの撤去)

## Q7. 完了処理を取り消すことはできますか?

**A.** 管理者権限でも取り消せません。現場を再度有効化する必要があります。

**再度有効化する方法:**
1. **現場管理** を開く
2. 完了した現場をクリック
3. **「編集」** をクリック
4. ステータスを **「稼働中」** に変更
5. 完了日を削除
6. **「更新」** をクリック

**注意:**
- 完了日が削除されます
- 出退勤打刻・道具管理が再び可能になります

**推奨:**
- 完了処理前に道具の返却、作業報告書の確認を済ませてください

**関連**: [現場管理マニュアル](../../manual/sites.md)

## Q8. 工事情報も完了にする必要がありますか?

**A.** はい、現場完了と合わせて工事情報も完了にしてください。

**工事完了処理の手順:**
1. **工事管理** を開く
2. 対象の工事をクリック
3. **「編集」** をクリック
4. ステータスを **「完了」** に変更
5. 実際の完了日を入力
6. **「更新」** をクリック

**現場と工事の違い:**
- **現場**: 物理的な作業場所
- **工事**: プロジェクト全体

**工事情報がない場合:**
- 現場だけ完了処理すればOK

**関連**:
- [現場完了・片付けマニュアル - 工事情報の完了処理](../../manual/scenarios/daily/site_closing.md#工事情報の完了処理)

## Q9. 消耗品が余っています。どう処理すればいいですか?

**A.** 使用可能な消耗品は倉庫に返却し、使用不可能なものは廃棄してください。

**使用可能な消耗品:**
- 倉庫に返却
- 在庫数の更新
- 次の現場で使用可能

**使用不可能な消耗品:**
- 廃棄処理
- 在庫から削除

**在庫数の確認・調整:**
1. **消耗品管理** を開く
2. 現場で使用した消耗品を検索
3. 在庫数が実数と一致しているか確認
4. 差異がある場合は **「在庫調整」** で修正

**関連**:
- [現場完了・片付けマニュアル - 消耗品・在庫の整理](../../manual/scenarios/daily/site_closing.md#消耗品在庫の整理)

- [ ] 鍵の返却

### 道具・在庫
- [ ] 道具の返却完了（全点）
- [ ] 紛失・破損道具の処理完了
- [ ] 消耗品の整理完了

### 書類・システム

- [ ] 作業報告書すべて承認済み
- [ ] 発注書すべて精算済み

- [ ] システムで現場完了処理済み
- [ ] システムで工事完了処理済み

### 取引先対応
- [ ] 完了報告送付

## Q11. 請求書を発行していません。現場を完了にできますか?

**A.** はい、できます。請求書は現場完了後でも発行可能です。

**推奨手順:**
1. 請求書を発行
2. 取引先に送付
3. 現場完了処理

**完了後に請求書を発行する場合:**
1. **請求書管理** を開く
2. **「+ 新規作成」** をクリック
3. 完了済みの現場を選択
4. 請求書を作成

**関連**:
- [請求書管理マニュアル](../../manual/invoices.md)

## Q12. 発注書に未精算のものがあります。完了処理できますか?

**A.** はい、完了処理できます。ただし、精算してから完了することを推奨します。

**推奨手順:**
1. 未精算の発注書を確認
2. 支払い処理を完了
3. 発注書のステータスを「精算済み」に変更
4. 現場完了処理

**完了後も発注書は:**
- 参照可能
- 精算処理可能
- PDF出力可能

**未精算のまま完了すると:**
- 原価計算が不正確になる可能性があります

**関連**: [発注書管理マニュアル](../../manual/purchase_orders.md)

## Q13. 原価計算はどこで確認できますか?

**A.** 工事管理画面の「原価」タブで確認できます。

**手順:**
1. **工事管理** を開く
2. 完了した工事をクリック
3. **「原価」タブ** をクリック
4. 発注書・人件費の集計を確認

**確認項目:**
- 契約金額（売上）
- 発注額（原価）
- 人件費（労務費）
- 粗利益・利益率

**原価計算の条件:**
- 工事情報が登録されている
- 発注書が工事に紐づいている
- 勤怠データが記録されている

**関連**:
- [現場完了・片付けマニュアル - 原価計算](../../manual/scenarios/daily/site_closing.md#原価計算)

## Q14. 完了報告書を取引先に送付したいです。どうすればいいですか?

**A.** 作業報告書をPDF出力して送付してください。

**手順:**
1. **作業報告書** を開く
2. フィルターで対象の現場を選択
3. 必要な報告書を選択
4. **「PDFエクスポート」** をクリック
5. PDFファイルをダウンロード
6. メールまたは郵送で送付

**複数の報告書をまとめる場合:**
- 各報告書を個別にPDF出力
- PDF結合ソフトでまとめる

**関連**: [作業報告書マニュアル - PDF出力](../../manual/work_reports.md)

## Q15. 完了済みの現場データは削除されますか?

**A.** いいえ、削除されません。完了後も参照可能です。

**完了後も可能なこと:**
- 現場情報の参照
- 作業報告書の参照・PDF出力
- 道具の移動履歴の確認
- 発注書・請求書の参照

**完了後にできないこと:**
- 出退勤打刻
- 道具の持ち出し先として選択

**データの保存期間:**
- 無期限（削除しない限り）
- バックアップも定期的に取得されます

**関連**: [現場管理マニュアル](../../manual/sites.md)

## Q16. 道具の一括返却はできますか?

**A.** 道具セット機能を使えば一括返却できます。

**手順:**
1. **道具セット** を開く
2. 返却するセットを選択
3. **「一括返却」** をクリック
4. 返却先で倉庫の保管場所を選択
5. **「返却する」** をクリック

**個別に返却する場合:**
1. **道具管理** を開く
2. 返却する道具を検索
3. 道具詳細画面で **「移動」** をクリック
4. 移動先で「倉庫」を選択
5. 保管場所を指定
6. **「持ち出す」** をクリック

**関連**:
- [現場完了・片付けマニュアル - 道具の返却登録](../../manual/scenarios/daily/site_closing.md#道具の返却登録)

## Q17. 完了日はいつに設定されますか? 変更できますか?

**A.** 「完了にする」ボタンを押した日時が完了日として記録されます。変更可能です。

**完了日の記録:**
- 「完了にする」ボタンを押した日時が自動記録されます

**完了日を変更する場合:**
1. 現場詳細画面で **「編集」** をクリック
2. 完了日を編集
3. **「更新」** をクリック

**注意:**
- 完了日は手動で変更可能です
- 実際の完了日と異なる場合は修正してください

**関連**: [現場管理マニュアル](../../manual/sites.md)

## Q18. データをエクスポートしてバックアップしたいです。

**A.** 各管理画面からCSVエクスポートが可能です。

**エクスポート可能なデータ:**
- 道具一覧
- 作業報告書
- 発注書
- 請求書
- 勤怠データ

**手順（例: 道具一覧）:**
1. **道具管理** を開く
2. フィルターで対象の現場を選択
3. **「CSVエクスポート」** をクリック
4. CSVファイルがダウンロードされます

**保存場所:**
- クラウドストレージ（Google Drive、Dropboxなど）
- 社内ファイルサーバ
- 外付けHDD

**関連**: [現場完了・片付けマニュアル - データの整理・保管](../../manual/scenarios/daily/site_closing.md#データの整理保管)

## Q19. 現場完了後、スタッフは現場情報を見られますか?

**A.** はい、完了後も現場情報は参照可能です。

**スタッフができること:**
- 現場情報の参照
- 自分が提出した作業報告書の参照
- 自分の勤怠記録の参照

**スタッフができないこと:**
- 出退勤打刻
- 道具の持ち出し・返却
- 作業報告書の新規作成

**管理者ができること:**
- すべての情報の参照
- データのエクスポート
- 原価計算

**関連**: [現場管理マニュアル](../../manual/sites.md)

## Q20. 完了した現場を再開することはできますか?

**A.** はい、ステータスを「稼働中」に変更することで再開できます。

**再開手順:**
1. **現場管理** を開く
2. 完了した現場をクリック
3. **「編集」** をクリック
4. ステータスを **「稼働中」** に変更
5. 完了日を削除（または空欄に）
6. **「更新」** をクリック

**再開後にできること:**
- 出退勤打刻
- 道具の持ち出し・返却
- 作業報告書の提出

**注意:**
- 完了日が削除されます
- 原価計算に影響する可能性があります

**関連**: [現場管理マニュアル](../../manual/sites.md)
