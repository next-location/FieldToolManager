# 新規現場立ち上げのトラブルシューティング

## Q1. 現場を登録したいのですが、取引先が選択肢に表示されません。

**A.** 取引先の「顧客タイプ」が「仕入先のみ」になっている可能性があります。

**原因:**
- 現場登録では「顧客」または「顧客兼仕入先」の取引先のみ表示されます
- 「仕入先のみ」の取引先は表示されません

**対処法:**
1. **取引先管理** を開く
2. 該当する取引先を検索
3. 「編集」をクリック
4. 顧客タイプを **「顧客」** または **「顧客兼仕入先」** に変更
5. 「更新」をクリック

**取引先が未登録の場合:**
- 先に取引先マスタから登録してください

**関連**: [取引先管理マニュアル](../../manual/clients.md)

## Q2. 現場QRコードはどこで発行できますか?

**A.** 現場詳細画面の「QRコード」タブから発行できます。

**手順:**
1. **現場管理** を開く
2. 登録した現場をクリック
3. **「QRコード」タブ** をクリック
4. **「QRコードを表示」** をクリック
5. QRコードが表示されます
6. **「印刷」** ボタンで印刷

**印刷設定:**
- 推奨サイズ: 10cm × 10cm 以上
- カラー・白黒どちらでも可
- ラミネート加工推奨（屋外使用のため）

**関連**: [新規現場立ち上げマニュアル - 現場QRコード発行](../../manual/scenarios/daily/site_opening.md#現場qrコード発行)

## Q3. 現場QRコードの設置場所のおすすめはありますか?

**A.** 以下の条件を満たす場所に設置してください。

**おすすめの設置場所:**
- 現場の入口付近
- スタッフが毎日通る場所
- 雨や直射日光が当たりにくい場所
- 道具置き場の近く

**避けるべき場所:**
- 雨が直接かかる場所
- 直射日光が当たる場所（色褪せの原因）
- スタッフが通らない場所

**設置方法:**
- ラミネート加工して掲示
- 防水ケースに入れて設置
- 壁やボードに貼り付け

**注意事項:**
- 定期的に読み取りテストを行う
- 破損した場合は再発行（いつでも可能）

**関連**: [新規現場立ち上げマニュアル - QRコード設置](../../manual/scenarios/daily/site_opening.md#qrコード設置)

## Q4. 現場責任者を設定したいのですが、選択肢に表示されません。

**A.** そのユーザーが「アクティブ」かつ「削除されていない」ことを確認してください。

**確認方法:**
1. **スタッフ管理** を開く
2. 該当するスタッフを検索
3. ステータスを確認

**表示されない原因:**
- スタッフが **無効化** されている
- スタッフが **削除** されている
- スタッフが **他の組織** に所属している

**対処法:**
- スタッフのステータスを「有効」に変更
- 削除されている場合は、再度追加

**関連**: [スタッフ管理マニュアル](../../manual/staff_management.md)

## Q5. 工事情報と現場情報の違いは何ですか?

**A.** 現場は「物理的な場所」、工事は「プロジェクト全体」を表します。

| 項目 | 現場 | 工事 |
|------|------|------|
| **意味** | 物理的な作業場所 | プロジェクト全体 |
| **例** | ◯◯ビル建設現場 | ◯◯ビル建設工事 |
| **用途** | 出退勤打刻、道具管理 | 予算管理、原価計算 |
| **登録必須** | はい（道具管理に必要） | 任意（予算管理したい場合） |

**関係:**
- 1つの工事に複数の現場が紐づくこともあります
- 現場だけ登録して、工事を登録しないことも可能

**関連**: [工事管理マニュアル](../../manual/projects.md)

## Q6. 道具セットを使って現場に道具を配置したいです。手順を教えてください。

**A.** 道具セット画面から「一括持ち出し」を使用してください。

**手順:**
1. **道具セット** を開く
2. 現場に適した道具セットを選択（例: 基本工具セットA）
3. **「一括持ち出し」** をクリック
4. 移動先で登録した現場を選択
5. **「持ち出す」** をクリック

**道具セットがない場合:**
- 先に道具セットを作成してください
- または、個別に道具を持ち出し登録してください

**道具セットのメリット:**
- 一度に複数の道具を持ち出し登録できる
- 定型パターンを使い回せる

**関連**:
- [新規現場立ち上げマニュアル - 道具配置](../../manual/scenarios/daily/site_opening.md#道具配置)
- [道具セットマニュアル](../../manual/tool_sets.md)

## Q7. 現場コードは自動生成されますか? 変更できますか?

**A.** はい、自動生成されますが、登録時または編集時に変更できます。

**自動生成形式:**
- **S-YYYY-NNNN** 形式（例: S-2026-0001）
- S = Site（現場）
- YYYY = 年
- NNNN = 連番

**変更方法:**
1. 現場登録フォームで「現場コード」欄を編集
2. または、登録後に「編集」画面で変更

**注意事項:**
- 組織内で重複しない番号にしてください
- 半角英数字とハイフンのみ使用可能

**関連**: [現場管理マニュアル](../../manual/sites.md)

## Q8. 現場を登録したのに、スタッフの出勤打刻で現場が選択できません。

**A.** 現場のステータスが「停止中」になっている可能性があります。

**確認方法:**
1. **現場管理** を開く
2. 該当する現場をクリック
3. ステータスを確認

**ステータスの種類:**
- **稼働中**: 出勤打刻・道具管理で使用可能
- **停止中**: 使用不可
- **完了**: 使用不可

**対処法:**
1. 現場詳細画面で **「編集」** をクリック
2. ステータスを **「稼働中」** に変更
3. **「更新」** をクリック

**関連**: [現場管理マニュアル](../../manual/sites.md)

---

## Q9. 現場の工期はどのように設定すればいいですか?

**A.** 実際の工事開始日と完了予定日を設定してください。

**入力項目:**
- **開始日**: 工事開始日（作業開始日）
- **完了予定日**: 工事完了予定日

**任意項目です:**
- 未定の場合は空欄でもOK
- 後から編集で追加可能

**工期の活用:**
- 現場一覧で期間で絞り込み可能
- 原価計算時の期間指定に使用

**関連**: [新規現場立ち上げマニュアル - 入力項目](../../manual/scenarios/daily/site_opening.md#入力項目)

## Q10. スタッフへの周知事項は何を伝えればいいですか?

**A.** 以下の情報を初日までに伝えてください。

**必須情報:**
- 現場名・住所
- 集合時間・集合場所
- 現場責任者
- 駐車場の有無
- 作業内容の概要
- 必要な装備・持ち物

**システム関連:**
- 出退勤打刻の方法（QRコードの場所）
- 現場QRコードの場所
- 道具管理の方法

**初日の流れ:**
- キックオフミーティングの有無
- 作業開始時刻
- 昼休憩の時間

**関連**: [新規現場立ち上げマニュアル - スタッフアサイン](../../manual/scenarios/daily/site_opening.md#スタッフアサイン)

## Q11. 現場QRコードが汚れて読み取れなくなりました。再発行できますか?

**A.** はい、いつでも再発行できます。QRコードIDは変わりません。

**再発行手順:**
1. **現場管理** を開く
2. 該当する現場をクリック
3. **「QRコード」タブ** をクリック
4. **「QRコードを表示」** をクリック
5. **「印刷」** ボタンで再印刷

**QRコードIDは同じです:**
- 再発行しても同じQRコードが発行されます
- スタッフは同じQRコードを使い続けられます

**予防策:**
- ラミネート加工を施す
- 防水ケースに入れる
- 定期的に清掃する

**関連**: [新規現場立ち上げマニュアル - QRコード発行](../../manual/scenarios/daily/site_opening.md#現場qrコード発行)

## Q12. 現場に配置した道具の一覧はどこで確認できますか?

**A.** 道具管理画面でフィルターを使用して確認できます。

**手順:**
1. **道具管理** を開く
2. フィルターで **「現在地」** を選択
3. 対象の現場を選択
4. その現場にある道具の一覧が表示されます

**CSV出力も可能:**
- **「CSVエクスポート」** ボタンでダウンロード
- Excelで管理できます

**関連**: [道具管理マニュアル](../../manual/tool_management.md)

## Q13. 初日のキックオフミーティングで説明すべきことは何ですか?

**A.** 以下の内容を説明してください。

**現場について:**
- 工事内容・工期
- 役割分担
- 注意事項・ルール
- 安全確認

**システム操作:**
- 出退勤打刻の方法
  - QRコードの場所
  - 打刻の手順
- 道具管理の方法
  - 持ち出し・返却の手順
  - QRコードの読み方
- 作業報告書の書き方
  - 提出期限
  - 記載項目

**質疑応答:**
- スタッフからの質問を受け付ける

**関連**: [新規現場立ち上げマニュアル - 初日の準備](../../manual/scenarios/daily/site_opening.md#初日の準備)

## Q14. 道具セットがありません。個別に持ち出し登録する方法は?

**A.** 道具管理画面から個別に持ち出し登録してください。

**手順:**
1. **道具管理** を開く
2. 持ち出す道具を検索
3. 道具詳細画面で **「移動」** をクリック
4. 移動先で登録した現場を選択
5. **「持ち出す」** をクリック

**複数の道具を登録する場合:**
- 上記の手順を繰り返してください
- 時間がかかる場合は、道具セットの作成を検討してください

**道具セットの作成:**
- よく使う道具の組み合わせを登録しておくと便利
- 次回から一括持ち出しが可能になります

**関連**:
- [新規現場立ち上げマニュアル - 道具配置](../../manual/scenarios/daily/site_opening.md#道具配置)
- [道具管理マニュアル](../../manual/tool_management.md)

## Q15. 現場登録時に契約金額や予算は必須ですか?

**A.** いいえ、任意項目です。後から追加できます。

**入力項目:**
- **契約金額**: 取引先との契約金額
- **予算金額**: 社内の予算管理用

**入力するメリット:**
- 原価計算時に利益率を算出できる
- 予算超過アラートを設定できる

**工事情報で管理する場合:**
- 現場登録時に入力しなくてもOK
- 工事情報で契約金額・予算金額を管理

**関連**: [工事管理マニュアル](../../manual/projects.md)

## Q16. リーダー用QRとは何ですか? 現場QRとの違いは?

**A.** リーダー用QRは、リーダーが発行する一時的な出勤打刻用QRコードです。

| 項目 | 現場QR | リーダー用QR |
|------|--------|------------|
| **用途** | 出勤打刻、道具管理 | 出勤打刻のみ |
| **発行者** | 管理者（現場登録時） | リーダー（毎日） |
| **有効期限** | 永続的 | 24時間 |
| **設置場所** | 現場に固定設置 | リーダーのスマホ画面 |

**使用場面:**
- 現場に複数のスタッフが一緒に到着した場合
- リーダーがQRコードを発行し、スタッフ全員がスキャンして出勤打刻

**どちらを使うか:**
- 現場QRがあればそちらを推奨
- 現場QRがない場合や、リーダーがまとめて管理したい場合はリーダー用QR

**関連**: [勤怠管理マニュアル - リーダー用QR発行](../../manual/attendance.md#リーダー用qr発行leaderマネージャーadmin限定)

## Q17. 現場を登録したのに、道具の持ち出し先に表示されません。

**A.** 現場のステータスを確認してください。「稼働中」になっている必要があります。

**確認方法:**
1. **現場管理** を開く
2. 該当する現場をクリック
3. ステータスを確認

**対処法:**
- ステータスを **「稼働中」** に変更

**その他の原因:**
- 現場が削除されている（削除済みフィルターで確認）
- ブラウザのキャッシュが古い（リロードしてください）

**関連**: [現場管理マニュアル](../../manual/sites.md)

## Q18. 取引先が未登録です。先に登録する必要がありますか?

**A.** いいえ、現場登録時は取引先を選択しなくてもOKです。後から編集で追加できます。

**取引先を選択しない場合:**
- 現場名だけで登録可能
- 後から編集で取引先を追加

**取引先を登録するメリット:**
- 見積書・請求書の作成時に自動入力される
- 取引先別の売上集計が可能

**取引先の登録方法:**
1. **取引先管理** を開く
2. **「+ 新規登録」** をクリック
3. 取引先情報を入力
4. **「登録」** をクリック

**関連**:
- [新規現場立ち上げマニュアル - 取引先の確認](../../manual/scenarios/daily/site_opening.md#取引先の確認)
- [取引先管理マニュアル](../../manual/clients.md)

## Q20. 道具を現場に配置した後、在庫を確認する方法は?

**A.** 道具管理画面でフィルターを使用して確認できます。

**手順:**
1. **道具管理** を開く
2. フィルターで **「現在地」** を選択
3. 「現場名」で絞り込み
4. その現場にある道具の一覧が表示されます

**CSV出力:**
- **「CSVエクスポート」** ボタンでダウンロード可能

**リアルタイム確認:**
- 道具の移動はリアルタイムで反映されます
- スタッフが持ち出し・返却登録すると即座に反映

**関連**: [道具管理マニュアル](../../manual/tool_management.md)
