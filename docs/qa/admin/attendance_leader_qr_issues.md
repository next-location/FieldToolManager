## 基本的な質問

### Q1. リーダー用QR発行とは何ですか？

**A.** リーダー用QR発行は、スタッフが出退勤時にスマートフォンでスキャンするQRコードを発行する機能です。以下の2種類があります：

- **会社QR**: 会社での出退勤用（Manager/Admin のみ発行可能）
- **現場QR**: 各現場での出退勤用（Leader/Manager/Admin が発行可能）

QRコードをスキャンすると、位置情報の選択なしで自動的に打刻されるため、スタッフの打刻がスムーズになります。

### Q2. 誰がQRコードを発行できますか？

**A.** 権限によって発行できるQRコードが異なります：

| 権限 | 会社QR | 現場QR |
|------|--------|--------|
| **Staff** | ❌ | ❌ |
| **Leader** | ❌ | ✅ |
| **Manager** | ✅ | ✅ |
| **Admin** | ✅ | ✅ |

会社QRは重要度が高いため、Manager以上の権限が必要です。

### Q3. QRコードの有効期限はどのくらいですか？

**A.** QRコードの有効期限は、**出退勤設定の「QR更新頻度」**で設定されます。デフォルトは**30日**です。

選択肢:
- 1日（毎日再発行）
- 3日
- 7日（1週間）
- 30日（推奨）

設定変更は、「出退勤 > 出退勤設定 > QR更新頻度」から行えます。

### Q4. 会社QRと現場QRの違いは何ですか？

**A.** 主な違いは以下の通りです：

| 項目 | 会社QR | 現場QR |
|------|--------|--------|
| **用途** | 会社での出退勤 | 現場での出退勤 |
| **発行数** | 1つ（組織全体で1つ） | 現場ごとに1つ |
| **発行権限** | Manager / Admin のみ | Leader / Manager / Admin |
| **設置場所** | 会社の入口・受付 | 各現場の入口 |
| **打刻時の位置** | 会社 | 該当する現場 |

スタッフが会社で打刻する場合は会社QR、現場で打刻する場合は現場QRをスキャンします。

### Q5. QRコード発行画面へのアクセス方法は？

**A.** 以下の手順でアクセスできます：

```
1. サイドメニューから「出退勤」をクリック
2. 「出退勤QR発行」をクリック
```

または、ダッシュボードから直接「出退勤QR発行」のリンクをクリックします。

Leader権限の場合は現場QRのみ、Manager/Admin権限の場合は会社QRと現場QRの両方が表示されます。

## 会社QR発行に関する質問

### Q6. 会社QRが表示されません。なぜですか？

**A.** 以下の理由が考えられます：

**1. 権限不足**
- Leader権限では会社QRを発行できません
- Manager または Admin 権限が必要です

**2. 出退勤QR機能が無効**
- 出退勤設定で「QRスキャン許可」がOFFになっている可能性があります
- Admin権限のユーザーに設定変更を依頼してください

権限を確認するには、画面右上のユーザーメニューで自分の権限を確認してください。

### Q7. 会社QRは何個まで発行できますか？

**A.** 会社QRは**組織全体で1つのみ**です。複数の会社QRを同時に有効にすることはできません。

再発行すると、古い会社QRは自動的に無効化されます。複数の設置場所がある場合は、同じQRコードを印刷して複数箇所に設置してください。

### Q8. 会社QRを再発行すると、古いQRコードはどうなりますか？

**A.** 再発行すると、**古い会社QRコードは即座に無効化**されます。

**影響**:
- 古いQRコードをスキャンしても打刻できなくなります
- 既に印刷・掲示している場合は、新しいQRコードに貼り替える必要があります

**推奨対応**:
1. 再発行前にスタッフに周知
2. 再発行後、すぐに新しいQRコードを印刷・設置
3. 古いQRコードを取り外す

### Q9. 会社QRの有効期限を確認する方法は？

**A.** 「出退勤QR発行」画面の「会社QR発行」セクションで確認できます。

表示内容:
```
会社QRコード
有効期限: 2026-02-20 23:59:59
残り 30 日
```

有効期限が近づいたら（例: 残り3日）、早めに再発行することを推奨します。

### Q10. 会社QRはどこに設置すればよいですか？

**A.** 以下の場所への設置を推奨します：

**推奨設置場所**:
- ✅ 会社の正面玄関（入口）
- ✅ 受付カウンター
- ✅ タイムカード機の近く
- ✅ 社員食堂入口

**設置時の注意**:
- スマートフォンのカメラで撮影しやすい高さ（1.2m〜1.5m）
- 雨風に濡れない場所
- 直射日光が当たらない場所
- ラミネート加工を推奨

複数の入口がある場合は、同じQRコードを印刷して全ての入口に設置してください。

## 現場QR発行に関する質問

### Q11. 現場QRは何個まで発行できますか？

**A.** 現場QRは**現場ごとに1つ**発行できます。登録されている現場の数だけQRコードを発行できます。

例:
```
現場A → 現場AのQRコード
現場B → 現場BのQRコード
現場C → 現場CのQRコード
```

各現場で異なるQRコードが発行されるため、スタッフがスキャンすると自動的にどの現場で打刻したかが記録されます。

### Q12. 現場がドロップダウンに表示されません

**A.** 以下の理由が考えられます：

**1. 現場が登録されていない**
- 現場管理で現場を登録してください
- 登録後、すぐにドロップダウンに表示されます

**2. 現場が削除されている**
- 削除された現場は表示されません
- 必要であれば現場を再登録してください

**3. ページの再読み込みが必要**
- 最近登録した現場が表示されない場合、ページを再読み込みしてください

現場の登録は、「現場 > 現場管理 > 新規登録」から行えます。

### Q13. 発行済みの現場QRコードはどこで確認できますか？

**A.** 「出退勤QR発行」画面の下部、**「有効な現場QRコード」**セクションで確認できます。

表示内容:
```
A現場
発行日時: 2026-01-21 10:30:00
有効期限: 2026-02-20 23:59:59

[表示] [保存] [印刷]
```

有効な現場QRコードのみが表示されます。期限切れのQRコードは自動的に非表示になります。

### Q14. 現場QRを再発行するタイミングは？

**A.** 以下の場合に再発行してください：

**必須の場合**:
- ✅ 有効期限が切れた
- ✅ QRコードを紛失した

**推奨する場合**:
- 印刷物が破損・汚損した
- QRコードが読み取りにくくなった
- セキュリティ上、定期的に更新したい

**再発行の手順**:
1. 現場選択ドロップダウンで該当する現場を選択
2. 「QRコード再発行」ボタンをクリック
3. 新しいQRコードを印刷・設置

### Q15. 1つの現場に複数のQRコードを設置できますか？

**A.** はい、できます。**同じQRコード**を複数印刷して、現場の複数箇所に設置できます。

**設置例**:
```
A現場:
- 現場入口ゲート → 同じQRコードを印刷
- 現場事務所入口 → 同じQRコードを印刷
- 休憩所入口 → 同じQRコードを印刷
```

スタッフはどのQRコードをスキャンしても、「A現場」として打刻されます。

ただし、**現場ごとに異なるQRコード**が発行されるため、他の現場のQRコードと混同しないように注意してください。

## 有効期限に関する質問

### Q16. 有効期限が切れたQRコードはどうなりますか？

**A.** 有効期限が切れたQRコードは**スキャンしても打刻できません**。

スタッフがスキャンすると、以下のエラーメッセージが表示されます:
```
❌ このQRコードは期限切れです
新しいQRコードをスキャンしてください
```

**対応**:
1. リーダー・管理者が新しいQRコードを再発行
2. 新しいQRコードを印刷・設置
3. 古いQRコードを取り外す

**予防策**:
- 有効期限の1週間前にカレンダーにリマインダーを設定
- 有効期限を長めに設定（30日推奨）

### Q17. QR更新頻度を変更すると、既存のQRコードはどうなりますか？

**A.** **既存のQRコードの有効期限は変わりません**。次回発行から新しい設定が適用されます。

**例**:
```
【現在の状態】
QR更新頻度: 30日
既存の会社QR: 2026-02-20まで有効

↓ QR更新頻度を7日に変更

【変更後】
既存の会社QR: 引き続き2026-02-20まで有効（変わらない）
次回発行する会社QR: 7日後まで有効（新しい設定が適用）
```

既存のQRコードの有効期限を短くしたい場合は、手動で再発行してください。

### Q18. QR更新頻度は何日に設定すればよいですか？

**A.** 組織の運用に合わせて設定してください。以下は推奨設定です：

| 頻度 | 推奨シーン | メリット | デメリット |
|------|-----------|---------|----------|
| **30日** | 通常の現場 | 管理が簡単 | セキュリティは低め |
| **7日** | セキュリティ重視 | 週次で更新 | 週1回の再発行作業 |
| **3日** | 短期プロジェクト | こまめに更新 | 頻繁な再発行作業 |
| **1日** | 高セキュリティ | 毎日更新 | 毎日の再発行作業 |

**推奨**: **30日**（バランスが良く、管理が簡単）

### Q19. 有効期限を延長することはできますか？

**A.** いいえ、既に発行されたQRコードの有効期限を延長することはできません。

**対応方法**:
1. **QR更新頻度を長く設定**（例: 30日 → 90日）
2. **QRコードを再発行**（新しい有効期限が適用される）

ただし、QR更新頻度の最大は30日です。90日などの長期設定はできません。

### Q20. 期限が近づいたら通知は来ますか？

**A.** 現在のバージョンでは、**自動通知機能はありません**。

**推奨対応**:
1. **カレンダーにリマインダーを設定**
   - 有効期限の3日前にリマインダー
   - 例: 「会社QR再発行」

2. **定期的に確認**
   - 毎週月曜日に「出退勤QR発行」画面で有効期限を確認

3. **長めの有効期限を設定**
   - 30日に設定すれば、月1回の再発行で済む

## 印刷・保存に関する質問

### Q21. QRコードの保存形式は何ですか？

**A.** QRコードは**PNG形式**で保存されます。

**ファイル名**:
- 会社QR: `office-qr.png`
- 現場QR: `site-qr-現場名.png`（例: `site-qr-A現場.png`）

PNG形式は印刷に適しており、拡大しても画質が劣化しにくい形式です。

### Q22. 保存したQRコードを別の人に送ることはできますか？

**A.** はい、できます。保存したPNG画像をメールやチャットで送信できます。

**送信方法の例**:
- メール添付
- Slack / LINE / Teams で送信
- Googleドライブ / Dropbox で共有

**セキュリティ上の注意**:
- ⚠️ QRコードは出退勤に使用できるため、**社外には共有しないでください**
- ⚠️ 不正使用を防ぐため、信頼できる人のみに送信してください

### Q23. QRコードを印刷する際の推奨サイズは？

**A.** 以下のサイズを推奨します：

**印刷サイズ**:
- **最小**: 8cm × 8cm
- **推奨**: **10cm × 10cm**
- **最大**: 15cm × 15cm

小さすぎるとスマートフォンのカメラで読み取りにくくなります。大きすぎると印刷コストが高くなります。

**A4用紙の場合**:
- 1枚のA4用紙に1つのQRコードを印刷
- QRコードサイズ: 10cm × 10cm
- タイトル: 「出退勤QR - A現場」
- 有効期限を明記

### Q24. ラミネート加工は必要ですか？

**A.** **屋外や汚れやすい場所に設置する場合は、ラミネート加工を強く推奨します**。

**ラミネート加工のメリット**:
- ✅ 雨風に強い
- ✅ 汚れに強い
- ✅ 破れにくい
- ✅ 長期間使用できる

**ラミネート加工の方法**:
1. QRコードを印刷
2. ラミネーターで加工（推奨: A4サイズ、厚さ100μm）
3. 端をカット（必要に応じて）
4. 壁に貼り付け（両面テープやマグネット）

**屋内の場合**: ラミネート加工は必須ではありませんが、耐久性が向上します。

### Q25. タブレットでQRコードを表示することはできますか？

**A.** はい、できます。タブレット表示は**印刷不要**で、再発行時も画像を更新するだけなので便利です。

**タブレット表示の手順**:
1. QRコード画像を保存（「保存」ボタンをクリック）
2. タブレットに画像を転送（AirDrop、メール、クラウドストレージ）
3. 画像を全画面表示
4. タブレットを固定（スタンド使用）

**推奨タブレット**:
- iPad（第7世代以降、10.2インチ以上）
- Android タブレット（10インチ以上）

**メリット**:
- ✅ 印刷不要
- ✅ 再発行時も画像を更新するだけ
- ✅ 雨風に強い（屋外用ケース使用）

**デメリット**:
- ❌ バッテリー管理が必要
- ❌ 盗難リスク

## セキュリティに関する質問

### Q26. QRコードを不正使用されるリスクはありますか？

**A.** はい、QRコードの画像が外部に流出すると、不正に出退勤打刻される可能性があります。

**セキュリティ対策**:
1. **QRコードの画像を社外に共有しない**
2. **有効期限を短めに設定**（7日または30日）
3. **定期的に再発行**（月1回など）
4. **印刷物を紛失しないよう管理**
5. **打刻履歴を定期的に確認**（不正打刻がないかチェック）

**不正使用が疑われる場合**:
1. 即座にQRコードを再発行（古いQRを無効化）
2. 打刻履歴を確認
3. スタッフに注意喚起

### Q27. QRコードのスクリーンショットを撮られたら悪用されますか？

**A.** はい、QRコードのスクリーンショットがあれば、そのQRコードをスキャンして打刻できます。

**対策**:
1. **QRコードを画面表示する時間を最小限にする**
   - 表示後、すぐに「非表示」ボタンをクリック

2. **印刷・タブレット表示を推奨**
   - スタッフのスマートフォン画面でQRコードを表示しない

3. **定期的な再発行**
   - 月1回再発行すれば、古いスクリーンショットは無効化される

4. **打刻履歴の確認**
   - 不自然な打刻がないか定期的にチェック

### Q28. QRコードを紛失した場合はどうすればよいですか？

**A.** 以下の手順で対応してください：

**ステップ1: 即座に再発行**
1. 「出退勤QR発行」画面を開く
2. 該当する現場（または会社QR）を再発行
3. 古いQRコードが無効化される

**ステップ2: 打刻履歴を確認**
1. 出退勤 > 出退勤記録 で履歴を確認
2. 不正な打刻がないかチェック

**ステップ3: 新しいQRコードを設置**
1. 新しいQRコードを印刷
2. 適切な場所に設置

**予防策**:
- 印刷物をラミネート加工して破損を防ぐ
- 設置場所を記録（どこに何枚設置したか）
- 定期的に設置状況を確認

### Q29. QRコードの内容は暗号化されていますか？

**A.** はい、QRコードには**暗号化されたデータ**が含まれています。

**QRコードの内容**:
- 組織ID（暗号化）
- 現場ID（暗号化、現場QRの場合）
- 発行日時
- ランダムな認証トークン

QRコードを外部の人がスキャンしても、組織情報や現場情報は読み取れません。ただし、ザイロクアプリ内でスキャンすると打刻できるため、**QRコード画像の管理には注意が必要**です。

### Q30. 会社QRと現場QRは見た目で区別できますか？

**A.** いいえ、QRコード自体の見た目は同じです。区別するには、以下の方法があります：

**推奨方法**:
1. **印刷時にタイトルを追加**
   - 会社QR: 「会社 出退勤QR」
   - 現場QR: 「A現場 出退勤QR」

2. **色分け**（印刷時）
   - 会社QR: 青色の枠
   - 現場QR: 緑色の枠

3. **設置場所で区別**
   - 会社QR: 会社の入口のみ
   - 現場QR: 各現場の入口のみ

スタッフがスキャンすると、アプリ内で「会社」または「現場名」が表示されるため、スキャン後に確認できます。

## トラブルシューティング

### Q31. 「QRコード発行」ボタンをクリックしても何も起こりません

**A.** 以下を確認してください：

**1. 現場が選択されているか**（現場QRの場合）
- 現場選択ドロップダウンで現場を選択してください

**2. インターネット接続**
- Wi-Fiまたはモバイルデータ通信が接続されているか確認

**3. ブラウザのキャッシュ**
- ページを再読み込み（Ctrl+R / Cmd+R）
- キャッシュをクリアして再試行

**4. JavaScriptエラー**
- ブラウザの開発者ツール（F12）でエラーがないか確認
- エラーがある場合は管理者に連絡

### Q32. 「保存」ボタンを押してもダウンロードされません

**A.** 以下を確認してください：

**1. ブラウザのポップアップブロック**
- アドレスバーにブロックアイコンがないか確認
- ポップアップを許可

**2. ダウンロードフォルダ**
- ブラウザの設定でダウンロードフォルダを確認
- フォルダが存在するか、書き込み権限があるか確認

**3. 別の方法で保存**
- 「表示」ボタンでQRコードを表示
- QRコード画像を右クリック > 「名前を付けて画像を保存」

### Q33. 印刷するとQRコードが小さくなってしまいます

**A.** 以下を確認してください：

**1. 印刷プレビュー**
- ページサイズ: A4
- 余白: 2cm（標準）
- 拡大縮小: なし（100%）

**2. ブラウザの印刷設定**
- 「背景のグラフィック」をON
- 「ヘッダーとフッター」をOFF
- 向き: 縦

**3. PDFに保存してから印刷**
- 印刷画面で「PDFに保存」を選択
- 保存したPDFを開いて印刷

**4. 画像を保存してから印刷**
- 「保存」ボタンでPNG画像を保存
- 画像編集ソフトで開いてサイズを調整
- A4用紙に10cm × 10cmで印刷

### Q34. QRコードが期限切れと言われましたが、有効期限内です

**A.** 以下を確認してください：

**1. サーバーの時刻**
- サーバーの時刻がずれている可能性があります
- 管理者に連絡してサーバー時刻を確認

**2. ブラウザのキャッシュ**
- 古い情報がキャッシュされている可能性があります
- ページを再読み込み（Ctrl+Shift+R / Cmd+Shift+R）

**3. 再発行**
- 念のため、QRコードを再発行してください

**4. 有効期限の確認**
- 「出退勤QR発行」画面で実際の有効期限を確認
- スタッフに正しい有効期限を伝える

### Q35. QRコードを削除することはできますか？

**A.** QRコードを完全に削除する機能はありませんが、以下の方法で無効化できます：

**方法1: 再発行**
- QRコードを再発行すると、古いQRコードは自動的に無効化されます

**方法2: 有効期限切れを待つ**
- 有効期限が切れると、QRコードは自動的に使用できなくなります

**方法3: 現場を削除**（現場QRの場合）
- 現場を削除すると、その現場のQRコードも使用できなくなります
- ただし、現場の削除は慎重に行ってください（出退勤履歴も見えなくなります）

**推奨**: 再発行して古いQRコードを無効化する方法が最も簡単です。
