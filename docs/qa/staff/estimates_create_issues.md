## 基本的な質問

### Q1. 誰が見積書を作成できますか？

**A.** **Leader以上の権限**で見積書を作成できます。

| 権限 | 見積書作成 |
|------|-----------|
| **Staff** | ❌ |
| **Leader** | ✅ |
| **Manager** | ✅ |
| **Admin** | ✅ |

Staffは見積書を作成できません。見積書の作成・提出にはLeader以上の権限が必要です。

### Q2. 見積書作成画面へのアクセス方法は？

**A.** 以下の手順でアクセスできます：

```
1. サイドメニューから「見積管理」をクリック
2. 画面右上の「+ 新規見積書」ボタンをクリック
```

または、ダッシュボードから直接「新規見積書」のリンクをクリックします。

### Q3. 見積番号は手動で入力しますか？

**A.** いいえ、見積番号は**自動的に生成**されます。

**フォーマット**: `EST-年-連番`

**例**:
```
EST-2026-0001
EST-2026-0002
EST-2026-0003
```

見積番号フィールドは読み取り専用のため、変更できません。年ごとに連番がリセットされます。

### Q4. 見積書作成に必要な情報は何ですか？

**A.** 以下の情報が必須です：

**必須項目**:
- ✅ 取引先
- ✅ 見積日
- ✅ 件名
- ✅ 明細（最低1行）
  - 品目名
  - 数量
  - 単位
  - 単価

**任意項目**:
- 工事
- 有効期限（デフォルト: 30日後）
- 摘要・説明
- 備考（顧客向け）
- 社内メモ

### Q5. 見積書作成にかかる時間はどのくらいですか？

**A.** 見積書の複雑さによりますが、以下が目安です：

| 見積書の種類 | 所要時間 |
|------------|---------|
| **簡単な見積書**（工事一式1行） | 5分 |
| **通常の見積書**（明細3〜5行） | 10〜15分 |
| **複雑な見積書**（明細10行以上） | 20分〜 |

慣れてくると、より短時間で作成できるようになります。

## 取引先・工事選択に関する質問

### Q6. 取引先が選択肢に表示されません。どうすればよいですか？

**A.** 以下を確認してください：

**1. 取引先が登録されているか確認**
- 取引先管理で取引先を登録してください

**2. 取引先タイプが正しいか確認**
- 取引先タイプが「顧客」または「顧客兼仕入先」の取引先のみ選択可能
- 「仕入先」のみの取引先は表示されません

**3. 取引先が削除されていないか確認**
- 削除された取引先は表示されません

**4. 検索ボックスで検索**
- 取引先フィールドをクリックして検索ボックスを表示
- 取引先名やコードを入力して検索

### Q7. 取引先を検索する際、ひらがなでも検索できますか？

**A.** はい、**ひらがなでも検索できます**。

**検索対象**:
- 取引先名（漢字）
- 取引先名（カナ）
- 取引先コード

**検索の仕組み**:
- ひらがなを入力すると、自動的にカタカナに変換して検索します

**例**:
```
入力: 「やまだ」
↓ 自動変換
検索: 「ヤマダ」

検索結果:
- 山田建設株式会社（ヤマダケンセツ）
- 山田工業株式会社（ヤマダコウギョウ）
```

### Q8. 工事が選択肢に表示されません。どうすればよいですか？

**A.** 以下を確認してください：

**1. 取引先を先に選択しているか確認**
- 取引先を選択していないと、工事は選択できません
- 先に取引先を選択してください

**2. 選択した取引先に紐づいた工事があるか確認**
- 工事は選択した取引先に紐づいたもののみ表示されます
- 別の取引先の工事は表示されません

**3. 工事のステータスを確認**
- ステータスが「計画中」または「進行中」の工事のみ表示されます
- 「完了」「中止」の工事は表示されません

**4. 工事を登録**
- 工事が登録されていない場合は、プロジェクト管理から工事を登録してください

### Q9. 工事を選択すると、件名が自動的に設定されますが、変更できますか？

**A.** はい、**件名は自由に変更できます**。

**自動設定の仕組み**:
```
工事を選択すると、以下の形式で件名が自動設定されます:
「工事名 見積書」

例:
工事: 新宿ビル建設工事
↓
件名: 新宿ビル建設工事 見積書
```

**変更方法**:
- 件名フィールドに直接入力して変更
- 自動設定された内容をベースに修正も可能

### Q10. 工事を選択せずに見積書を作成できますか？

**A.** はい、**工事は任意項目**のため、選択しなくても見積書を作成できます。

**工事を選択しない場合の用途**:
- 単発の見積もり
- 小規模な案件
- 工事に紐づかない見積もり（資材販売など）

ただし、工事に紐づけることで、プロジェクトごとの見積履歴を管理できるため、**可能な限り工事を選択することを推奨**します。

## 明細入力に関する質問

### Q11. 明細は何行まで追加できますか？

**A.** 明細の行数に制限はありません。必要なだけ追加できます。

**明細を追加する方法**:
1. 明細セクションの下部にある「明細を追加」ボタンをクリック
2. 新しい明細行が追加される

**推奨**:
- 明細が多い場合（20行以上）は、見積書を分割することを検討してください
- 顧客が見やすい範囲（5〜10行程度）を推奨

### Q12. 全角で数字を入力してしまいましたが、大丈夫ですか？

**A.** はい、大丈夫です。**全角数字は自動的に半角に変換**されます。

**自動変換の仕組み**:
```
入力: ５００００（全角）
↓ 自動変換
保存: 50000（半角）
```

**対象フィールド**:
- 数量
- 単価

そのまま入力すれば、システムが自動的に半角に変換して保存します。

### Q13. 明細の種別「その他」はどのような時に使いますか？

**A.** 以下のような場合に「その他」を使用します：

**用途**:
- 工事、材料、経費のいずれにも当てはまらない品目
- カスタムの種別が必要な場合

**例**:
```
種別: その他
カスタム種別: 設計費
品目名: 設計監理費
```

**「その他」を選択した場合**:
- カスタム種別入力フィールドが表示されます
- 自由に種別名を入力できます

### Q14. 単位「その他」はどのような時に使いますか？

**A.** 以下のような場合に「その他」を使用します：

**用途**:
- 既存の単位選択肢にない単位を使用する場合

**例**:
```
単位: その他
カスタム単位: 台
品目名: エアコン
数量: 3
```

**よくあるカスタム単位**:
- 台
- セット
- 組
- 式（大規模工事）
- 棟

### Q15. 税率は変更できますか？

**A.** はい、明細ごとに税率を変更できます。

**選択肢**:
- **10%**（標準税率）← デフォルト
- **8%**（軽減税率）
- **0%**（非課税）

**使用例**:
```
10%: 通常の工事、材料
8%: 食品など軽減税率対象
0%: 非課税取引、海外取引
```

**注意**: 通常の工事では10%を使用します。

## 金額計算に関する質問

### Q16. 金額はどのように計算されますか？

**A.** 金額は以下の計算式で自動計算されます：

**明細の金額**:
```
金額 = 数量 × 単価
```

**小計**:
```
小計 = 各明細の金額の合計
```

**消費税**:
```
消費税 = 各明細の（金額 × 税率 / 100）の合計
```

**合計**:
```
合計 = 小計 + 消費税
```

**例**:
```
明細1: 数量 10、単価 50,000円、税率 10%
  → 金額 500,000円、消費税 50,000円

明細2: 数量 5、単価 20,000円、税率 10%
  → 金額 100,000円、消費税 10,000円

小計: 600,000円
消費税: 60,000円
合計: 660,000円
```

### Q17. 金額を手動で変更できますか？

**A.** いいえ、**金額は自動計算のみ**です。手動で変更することはできません。

**金額を変更したい場合**:
- 数量を変更
- または、単価を変更

数量または単価を変更すると、即座に金額が再計算されます。

### Q18. 消費税が正しく計算されているか確認する方法は？

**A.** 見積書の下部に表示される合計金額セクションで確認できます。

**確認項目**:
```
小計: ¥600,000
消費税（10%）: ¥60,000 ← ここを確認
合計: ¥660,000
```

**手動で確認する方法**:
```
小計 × 各明細の税率 / 100 = 消費税

例:
明細1: 500,000円 × 10% = 50,000円
明細2: 100,000円 × 10% = 10,000円
合計消費税: 60,000円
```

### Q19. 端数が出る場合、どのように処理されますか？

**A.** 端数は**小数点以下を四捨五入**して表示されます。

**例**:
```
数量: 1.5
単価: 10,000円
↓
金額: 15,000円

数量: 3
単価: 3,333円
↓
金額: 9,999円（3,333 × 3 = 9,999）
```

通常は整数で計算されるため、端数が出ることは少ないです。

### Q20. 合計金額が予算オーバーした場合、どうすればよいですか？

**A.** 以下の対応を検討してください：

**1. 明細を見直す**
- 不要な品目を削除
- 数量を減らす
- 単価を再交渉

**2. 明細を分割**
- 必須品目と任意品目に分ける
- 複数の見積書に分割

**3. 上司に相談**
- 予算オーバーの理由を説明
- 承認を依頼

**4. 顧客に確認**
- 予算の追加可能性を確認
- 見積書の内容を調整

## 保存・提出に関する質問

### Q21. 「下書き保存」と「提出する」の違いは何ですか？

**A.** 以下の違いがあります：

| 項目 | 下書き保存 | 提出する |
|------|-----------|---------|
| **ステータス** | 下書き（灰色） | 提出済み（オレンジ色） |
| **編集** | 可能 | 不可 |
| **承認** | 不要 | Manager/Adminが承認 |
| **用途** | 途中保存、後で修正 | 承認依頼 |

**使い分け**:
- 途中まで作成: 「下書き保存」
- 内容が確定: 「提出する」

### Q22. 下書き保存した見積書は後で編集できますか？

**A.** はい、**下書き保存した見積書は何度でも編集できます**。

**編集方法**:
1. 見積書一覧から見積書をクリック
2. 見積書詳細画面を開く
3. 「編集」ボタンをクリック
4. 見積書編集画面で修正
5. 「下書き保存」または「提出する」をクリック

### Q23. 提出する前に上司に確認してもらうことはできますか？

**A.** はい、以下の方法で確認してもらえます：

**方法1: 下書き保存して共有**
1. 「下書き保存」をクリック
2. 上司に見積書番号を伝える
3. 上司が見積書詳細画面で内容を確認
4. 問題なければ「提出する」をクリック

**方法2: 口頭で確認**
1. 見積書を作成
2. 画面を見せながら上司に確認
3. 問題なければ「提出する」をクリック

**注意**: 下書き状態では承認は不要なため、自由に作成・修正できます。

### Q24. 提出後、間違いに気づきました。どうすればよいですか？

**A.** 以下の対応を取ってください：

**ステップ1: Manager/Adminに連絡**
- すぐにManager/Adminに連絡
- 「見積書#EST-2026-0001に誤りがあります」と伝える
- 修正内容を説明

**ステップ2: Manager/Adminが差し戻し**
- Manager/Adminが「差し戻し」を実行
- ステータスが「下書き」に戻る

**ステップ3: 修正して再提出**
- 「編集」ボタンをクリック
- 誤りを修正
- 「提出する」をクリック

**予防策**: 提出前に内容を十分に確認してください。

### Q25. 見積書を提出しないで、下書きのままにしておくことはできますか？

**A.** はい、**下書きのまま保存しておくことができます**。

**用途**:
- 参考見積もりとして保存
- 後で修正する予定の見積書
- 顧客から正式な依頼があってから提出

**注意**: 下書きのままでは承認されないため、顧客に送付する場合は必ず提出してください。

## 編集・修正に関する質問

### Q26. 見積書を複製することはできますか？

**A.** 現在のバージョンでは、見積書の複製機能はありません。

**代替方法**:
1. 既存の見積書を参考に、新しい見積書を作成
2. 取引先、工事、明細などを手動でコピー

将来的に複製機能が追加される予定です。

### Q27. 見積書を削除することはできますか？

**A.** はい、**下書き状態の見積書のみ削除できます**。

**削除方法**:
1. 見積書詳細画面を開く
2. 「削除」ボタンをクリック
3. 確認ダイアログで「削除」をクリック

**削除できないケース**:
- 提出済みの見積書
- 承認済みの見積書
- 顧客送付済みの見積書

**注意**: 削除は取り消せません。慎重に実行してください。

### Q28. 明細の順番を変更できますか？

**A.** 現在のバージョンでは、明細の順番を変更する機能はありません。

**代替方法**:
1. 明細を削除
2. 正しい順番で明細を再度追加

**ヒント**: 最初から正しい順番で明細を入力することを推奨します。

### Q29. 見積書をExcelにエクスポートできますか？

**A.** 現在のバージョンでは、Excelエクスポート機能はありません。

**代替方法**:
1. 見積書をPDF出力（承認済みの場合）
2. または、見積書詳細画面をスクリーンショット

将来的にExcelエクスポート機能が追加される予定です。

### Q30. 過去の見積書を参考に新しい見積書を作成できますか？

**A.** はい、過去の見積書を参考にできます。

**方法**:
1. 見積書一覧から過去の見積書をクリック
2. 見積書詳細画面で内容を確認
3. 別のタブで新規見積書作成画面を開く
4. 過去の見積書を見ながら、同じ内容を入力

**ヒント**: 明細の品目名、単価などをメモしておくと便利です。

## トラブルシューティング

### Q31. 「必須項目を入力してください」というエラーが表示されます

**A.** 以下の必須項目が未入力の可能性があります：

**確認項目**:
- ✅ 取引先を選択したか
- ✅ 見積日を入力したか
- ✅ 件名を入力したか
- ✅ 明細の品目名を入力したか（各行）
- ✅ 明細の数量を入力したか（各行）
- ✅ 明細の単位を選択したか（各行）
- ✅ 明細の単価を入力したか（各行）

すべての必須項目を入力してから、再度保存・提出してください。

### Q32. 取引先を選択しても、工事が選択できません

**A.** 以下を確認してください：

**1. 選択した取引先に紐づいた工事があるか確認**
- 工事がない場合は、「この取引先に紐づいた工事がありません」と表示されます

**2. 工事のステータスを確認**
- 「計画中」または「進行中」の工事のみ選択可能
- 「完了」「中止」の工事は表示されません

**3. 工事を登録**
- 工事が登録されていない場合は、プロジェクト管理から工事を登録してください

**4. 取引先を変更**
- 別の取引先を選択すると、その取引先に紐づいた工事が表示されます

### Q33. 数量や単価を入力しても、金額が0円のままです

**A.** 以下を確認してください：

**1. 数値を正しく入力しているか確認**
- 数字以外の文字が含まれていないか
- 例: 「10個」→「10」に修正

**2. 数量と単価の両方を入力しているか確認**
- どちらか一方が0円の場合、金額も0円になります

**3. ページを再読み込み**
- 一時的なエラーの可能性
- ページを再読み込みして再度入力

### Q34. 「見積書の作成に失敗しました」というエラーが表示されます

**A.** 以下を確認してください：

**1. インターネット接続を確認**
- Wi-Fiまたはモバイルデータ通信が接続されているか確認

**2. ブラウザのキャッシュをクリア**
- ブラウザのキャッシュをクリアして再試行

**3. 入力内容を確認**
- 特殊文字や絵文字が含まれていないか確認
- 文字数制限を超えていないか確認

**4. 管理者に連絡**
- 上記を試しても解決しない場合は、管理者に連絡してください

### Q35. 保存ボタンをクリックしても反応がありません

**A.** 以下を確認してください：

**1. 必須項目を入力しているか確認**
- 未入力の必須項目があると、保存できません
- エラーメッセージが表示されていないか確認

**2. ボタンをダブルクリックしていないか確認**
- ボタンは1回だけクリックしてください
- 処理中は「保存中...」と表示されます

**3. ブラウザのJavaScriptを確認**
- JavaScriptが有効になっているか確認
- ブラウザの設定で確認

**4. 別のブラウザで試す**
- Chrome、Firefox、Edgeなど別のブラウザで試してください
