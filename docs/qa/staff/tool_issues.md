# 道具管理のトラブルシューティング

## Q1. 道具が一覧に表示されません。どうすれば見つけられますか?

**A.** 以下の方法で検索してください。

### 1. 検索ボックスを使う
- 画面上部の検索ボックスに道具名、型番、メーカー名を入力
- 部分一致で検索できます

### 2. フィルタ機能を使う
- **「フィルタ」** ボタンをクリック
- カテゴリ、メーカー、在庫状態、場所で絞り込み
- 複数条件を組み合わせて検索可能

### 3. 個別アイテムの状態を確認
- 道具詳細画面の **「個別アイテム一覧」** を開く
- 「紛失」状態になっていないかチェック

### 4. 削除されていないか確認
- 削除された道具は一覧に表示されません
- 管理者に確認してください

## Q2. QRコードが汚れて読めなくなりました。どうすればいいですか?

**A.** QRコードを再印刷して貼り直してください。

**手順:**
1. 道具詳細画面を開く
2. **「個別アイテム一覧」** セクションを確認
3. 対象アイテムの **「QR印刷」** をクリック
4. PDFをダウンロードして印刷
5. 古いQRコードを剥がし、新しいQRコードを貼る

**安心ポイント:**
- QRコード（UUID）自体は変わりません
- 既存の移動履歴はすべて保持されます
- 何度でも再印刷可能

**関連**: [QRコード印刷マニュアル](../../manual/qr_printing.md)

---

## Q3. 道具の個数を減らしたいです。どうすればいいですか?

**A.** 個別アイテムを削除できます（条件あり）。

**手順:**
1. 道具詳細画面の **「個別アイテム一覧」** を開く
2. 削除したいアイテムの **「削除」** をクリック
3. 確認ダイアログで **「削除」** をクリック

**削除できる条件:**
- ✅ 移動履歴がない個別アイテム（登録直後のもの）
- ❌ 移動履歴がある個別アイテム
- ❌ 現在使用中の個別アイテム

**削除できない場合:**
- 「廃棄」や「紛失」として移動登録してください
- 詳細は管理者に確認

---

## Q4. 道具の個数を増やしたいです。どうすればいいですか?

**A.** 既存の道具マスタに個別アイテムを追加できます。

**手順:**
1. 道具詳細画面を開く
2. **「個別アイテムを追加」** をクリック
3. 追加する個数を入力（例: 2）
4. **「追加」** をクリック

**結果:**
```
既存: 電動ドリル #001, #002, #003
追加後: 電動ドリル #001, #002, #003, #004, #005（2個追加した場合）
```

新しい個別アイテムには、自動的に連番が割り当てられます。

## Q5. 道具マスタと個別アイテムの違いは何ですか?

**A.** 道具の「種類」と「実際の物」の違いです。

| 項目 | 道具マスタ（Tool） | 個別アイテム（ToolItem） |
|------|------------------|------------------------|
| **意味** | 道具の種類 | 実際の物理的な道具 |
| **例** | 電動ドリル DRL-5000 | 電動ドリル #001、#002、#003 |
| **QRコード** | なし | 各個別アイテムに1つずつ |
| **登録数** | 1つ | 個数分（3個なら3つ） |
| **追跡** | 種類全体の情報 | 個体ごとの所在・状態 |

**イメージ:**
```
道具マスタ: 電動ドリル DRL-5000
├── 型番: DRL-5000
├── メーカー: マキタ
├── 全体の個数: 3個
│
└── 個別アイテム:
    ├── #001（QRコード: xxx-001、倉庫にある）
    ├── #002（QRコード: xxx-002、A現場で使用中）
    └── #003（QRコード: xxx-003、修理中）
```

**関連**: [道具管理マニュアル](../../manual/tool_management.md)

## Q6. 道具を削除できません。なぜですか?

**A.** 以下の条件に該当する道具は削除できません。

**削除できない条件:**
- ✅ 移動履歴がある道具
- ✅ 個別アイテムが1つでも存在する道具
- ✅ 現在使用中の道具

**削除するには:**
1. すべての個別アイテムを削除する
   - 移動履歴がない個別アイテムのみ削除可能
2. それでも削除できない場合は、管理者に確認

**注意:** 誤って登録した道具で、まだ使用していないもののみ削除可能です。

**代替案:**
- 削除せず、「廃棄済み」「紛失」として状態を変更することを推奨

## Q7. 道具と消耗品の違いは何ですか?

**A.** 管理方法が異なります。

| 項目 | 道具（Tool） | 消耗品（Consumable） |
|------|-------------|---------------------|
| **管理方法** | 個別管理（1個1個にQRコード） | 数量管理（まとめて在庫数で管理） |
| **例** | 電動ドリル、測定器、重機 | 軍手、ビス、養生テープ |
| **QRコード** | 個別アイテムごとに発行 | 種類ごとに1つ |
| **追跡** | どの個体がどこにあるか | 各場所に何個あるか |
| **単価** | 高価（数千円〜数百万円） | 安価（数十円〜数千円） |

**どちらで登録するべきか:**
- 高価で個別管理が必要 → **道具**（例: 電動ドリル、測定器、重機）
- 安価で数量だけ管理すればOK → **消耗品**（例: 軍手、ビス、テープ）

**関連**: [消耗品管理マニュアル](../../manual/consumable_management.md)

---

## Q8. 個別アイテムの状態（利用可能、使用中、修理中、紛失）は自動で変わりますか?

**A.** はい、QRスキャンや移動登録によって自動的に更新されます。

**状態の種類:**

| 状態 | 説明 | 自動更新のタイミング |
|------|------|---------------------|
| **利用可能** | 正常に使用できる状態 | 倉庫に返却したとき |
| **使用中** | 現在、誰かが使用中 | 現場に持ち出したとき |
| **修理中** | メンテナンス・修理中 | 修理に出したとき |
| **紛失** | 紛失した状態 | 紛失登録したとき |

**手動で変更することも可能:**
1. 道具詳細画面の **「個別アイテム一覧」** を開く
2. 対象アイテムの **「状態変更」** をクリック
3. 新しい状態を選択
4. 理由を入力（オプション）
5. **「変更」** をクリック

## Q9. 道具のカスタムフィールドは何に使いますか?

**A.** 業種や業務に応じて、独自の情報を記録できます。

**使用例:**

### 測定器の場合
- 校正日: 2026-01-15
- 校正期限: 2027-01-15
- 校正業者: ○○計測株式会社

### レンタル品の場合
- レンタル開始日: 2026-01-10
- レンタル返却期限: 2026-02-10
- レンタル業者: ○○レンタル

### 電気工事の場合
- 絶縁検査日: 2026-01-20
- 検査合格番号: INS-2026-001

**設定方法:**
1. 道具編集画面の **「カスタムフィールド」** セクションを開く
2. **「+ フィールドを追加」** をクリック
3. フィールド名（例: 校正日）と値（例: 2026-01-15）を入力
4. **「保存」** をクリック

**関連**: [道具管理マニュアル - カスタムフィールド](../../manual/tool_management.md#カスタムフィールド拡張機能)

## Q10. 道具の購入価格や購入日は後から変更できますか?

**A.** はい、道具編集画面で変更できます。

**手順:**
1. **道具管理 > 道具一覧** を開く
2. 編集したい道具をクリック
3. **「編集」** をクリック
4. 購入価格、購入日を修正
5. **「保存」** をクリック

**編集可能な項目:**
- 道具名、型番、メーカー
- カテゴリ
- 購入日、購入価格
- 最小在庫数
- 低在庫アラート設定
- カスタムフィールド

**編集できない項目:**
- 個数（個別アイテムの追加・削除で対応）

## Q11. 最小在庫数を設定すると何が起こりますか?

**A.** 在庫数が最小在庫数を下回ると、警告マークが表示されます。

**表示例:**
```
⚠️ 電動ドリル DRL-5000 (在庫: 1個 / 最小: 3個)
```

**活用方法:**
1. よく使う道具には最小在庫数を設定
2. 道具一覧で警告マークをチェック
3. 在庫が少ない道具を優先的に購入・補充

**推奨設定:**
- よく使う道具: 最小在庫数 = 3〜5個
- たまに使う道具: 最小在庫数 = 1〜2個
- ほとんど使わない道具: 設定不要

**設定方法:**
- 道具編集画面で **最小在庫数** を入力
- **低在庫アラートを有効にする** にチェック

## Q12. 他社のQRコードをスキャンしたらどうなりますか?

**A.** 「道具が見つかりません」とエラーが表示されます。

**理由:**
ザイロクはマルチテナント型のため、各組織のデータは完全に分離されています。

**エラーが出る原因:**
- 他の会社のQRコードをスキャンしている
- 削除された道具のQRコードをスキャンしている
- 古いQRコード（再登録前のもの）をスキャンしている

**対処法:**
- 自社で発行したQRコードか確認
- 最新のQRコードを使用
- 管理者に該当の道具を確認してもらう

**セキュリティ:**
他社のデータは一切見えないため、安心してご利用ください。

**関連**: [QRスキャンのトラブルシューティング](./qr_scan_issues.md)

## Q13. 道具を退職者から引き継ぐ方法は？

**A.** 退職者が保有している道具を倉庫に返却し、移動履歴を記録してください。

**手順:**
1. 退職者が保有する道具をリストアップ
   - 道具一覧画面でフィルタを使い、対象現場の道具を絞り込み
2. 一括移動機能で倉庫に返却
   - **移動登録 > 一括移動** を開く
   - **移動先**: 「倉庫」を選択
   - 退職者の道具をQRスキャンで選択
   - 備考に「退職に伴う返却（担当: ○○）」と記入
3. 実物を倉庫に搬入
4. 後任者に引き継ぎ完了を報告

**注意点:**
- 破損・紛失がある場合は先に対応してください
- 引き継ぎ日の記録を残すこと（移動履歴の備考欄）

**関連**: [道具の引き継ぎ手順マニュアル](../../manual/scenarios/tool_operations/tool_handover.md)

## Q14. 現場完了時に道具を一括で返却する方法は？

**A.** 一括移動機能を使えば、複数の道具を同時に倉庫に返却できます。

**手順:**
1. 現場で道具を集めて車に積む
2. 倉庫に到着後、**移動登録 > 一括移動** を開く
3. **移動先**: 「倉庫」を選択
4. **個別選択**タブで、QRスキャンで道具を1個ずつスキャン
5. すべて選択したら **「移動を実行（○個）」** をクリック
6. 完了！

**メリット:**
- 10個以上の道具でも5分程度で移動登録完了
- 1個ずつ移動登録する必要がない
- 返却漏れを防止できる

**制約:**
- 同じ場所の道具のみ一括移動可能
- 現場Aと現場Bの道具を同時に移動することはできません（2回に分ける）

**関連**: [道具の一括操作マニュアル](../../manual/scenarios/tool_operations/tool_bulk_operations.md)

## Q15. 道具が紛失しました。どうすればいいですか？

**A.** 以下の手順で対応してください。

### ステップ1: 捜索（発見から30分〜1時間）
1. 最後に使った場所を探す
2. システムで最終移動履歴を確認
3. チームメンバーに確認
4. 車両・トラックを確認

### ステップ2: 報告（1時間経過）
1. 上司・現場責任者に報告
2. 組織的な捜索を実施

### ステップ3: 紛失確定後の処理
1. 管理者に削除を依頼
   - 道具名、シリアル番号、紛失日時、捜索結果を伝える
2. 管理者が道具詳細画面で **「削除」** をクリック
3. 代替品を手配

**警察への届出:**
- 盗難の可能性がある場合は必ず警察に届け出てください
- 受理番号を控えておく（保険請求に必要）

**関連**: [道具の紛失・破損対応マニュアル](../../manual/scenarios/tool_operations/tool_lost_or_damaged.md)

## Q16. 道具が破損しました。修理に出す方法は？

**A.** 以下の手順で修理に出してください。

### ステップ1: 使用中止
1. 即座に使用を中止
2. 「故障中・使用禁止」の紙を貼る
3. 破損箇所の写真を撮影

### ステップ2: 修理見積
1. 修理業者に写真を送付
2. 見積（費用、納期）を取得
3. 上司に報告・承認を得る

### ステップ3: システム上で「修理中」に変更
1. 道具詳細画面の **「移動登録」** をクリック
2. **移動先**: **「🔧 修理」** を選択
3. **備考**: 修理内容と業者名を記入
   - 例: 「修理内容: バッテリー接続端子交換、業者: ○○工具サービス、返却予定: 1/27」
4. **「登録する」** をクリック

### ステップ4: 実物を業者に送付
- 修理依頼書を添付して発送または持ち込み

### ステップ5: 修理完了後
1. 動作確認を実施
2. **移動登録** で **「倉庫」** に戻す
3. 備考に「修理完了（費用: ○○円）」と記入

**判断基準:**
- 修理費 < 新品価格の50% → 修理推奨
- 修理費 ≥ 新品価格の50% → 新品購入を検討

**関連**: [道具の紛失・破損対応マニュアル](../../manual/scenarios/tool_operations/tool_lost_or_damaged.md)

## Q17. 道具の定期メンテナンスはどうすればいいですか？

**A.** 3段階のメンテナンスを実施してください。

### レベル1: 日常点検（毎日）
**使用前**:
- 外観に破損・ヒビがないか
- 電源コード・バッテリーは正常か
- 刃物に欠け・摩耗がないか

**使用後**:
- 汚れ・ホコリを拭き取る
- 水分を完全に除去
- 工具箱に保管

### レベル2: 定期点検（月1回）
**実施内容**:
- 分解清掃（可能な範囲で）
- 注油・グリスアップ
- 消耗品交換（ブラシ、ベルトなど）
- ネジ・ボルトの増し締め

**記録方法**:
- 道具の備考欄に点検内容を追記
- または、Excelで点検記録簿を作成

### レベル3: 専門メンテナンス（年1回）
**対象**:
- 異音・振動が直らない道具
- 測定器具の校正（年1回推奨）

**手順**:
1. メーカーサービスセンターに見積依頼
2. システム上で **「修理」** に変更
3. 業者に発送
4. 返却後、動作確認して **「倉庫」** に戻す

**注意**: 道具（tools）には個別のメンテナンス記録機能がありません。備考欄または外部記録簿で管理してください。

**関連**: [道具のメンテナンス・点検マニュアル](../../manual/scenarios/tool_operations/tool_maintenance.md)

## Q18. 一括移動で「現在地が異なるため選択できません」とエラーが出ます

**A.** 一括移動では、最初に選択した道具と同じ場所にある道具しか選択できません。

**原因:**
```
❌ NG: 現場Aの道具5個 + 現場Bの道具5個を同時に選択
✅ OK: 現場Aの道具10個のみ選択
```

**解決方法:**
1. **「すべてクリア」** ボタンで選択をリセット
2. 同じ場所の道具のみ選択し直す
3. または、場所ごとに2回に分けて移動

**仕組み:**
- 一括移動は「同じ場所にある複数の道具を、同じ移動先にまとめて移動」するための機能です
- 異なる場所の道具を混在させると、移動種別（持ち出し/返却/移動）が判定できないため制限されています

**関連**: [道具の一括操作マニュアル](../../manual/scenarios/tool_operations/tool_bulk_operations.md)

## Q19. QRスキャンが反応しません。どうすればいいですか？

**A.** 以下を確認してください。

### 確認1: カメラ権限
- ブラウザのカメラ権限を許可してください
- 許可ダイアログが表示されたら **「許可」** をクリック

### 確認2: QRコードの状態
- 汚れている → 清掃するか、再印刷してください
- 破れている → 再印刷してください
- 色褪せている → 再印刷してください

### 確認3: スキャン環境
- 照明が暗い → 明るい場所でスキャンしてください
- 距離が不適切 → 10〜30cmの距離でスキャンしてください
- 手ブレ → スマホを固定してスキャンしてください

### 代替手段: 手動入力
1. QRコードの下に印刷されているUUIDをメモ
2. 道具一覧画面の検索ボックスにUUIDを入力
3. 該当する道具を手動で選択

**関連**:
- [QRスキャンのトラブルシューティング](./qr_scan_issues.md)
- [道具の一括操作マニュアル - QRスキャンのコツ](../../manual/scenarios/tool_operations/tool_bulk_operations.md#ステップ3-移動する道具を選択)

## Q20. 複数の道具のQRコードをまとめて印刷する方法は？

**A.** 一括QR印刷機能を使ってください。

**手順:**
1. **道具一覧** 画面を開く
2. 画面右上の **「一括QR印刷」** ボタンをクリック
3. フィルタで印刷する道具を絞り込み
   - 例: カテゴリ「電動工具」のみ
4. 印刷する道具を選択
   - **「すべて選択」** または個別にチェック
5. レイアウトを選択（1列/2列/3列）
   - 推奨: **1列（大）** - スキャンしやすい
6. **「PDF生成」** ボタンをクリック
7. PDFをダウンロードして印刷

**メリット:**
- 50個の道具のQRコードを一度に印刷可能
- 新規登録時や再発行時に便利
- レイアウトを選択できる（大きさを調整）

**注意点:**
- QRコードは道具ごとに固有です
- 間違った道具に貼らないように、道具名・シリアル番号を照合してください
