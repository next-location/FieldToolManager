# 道具登録のよくある質問（Q&A）

## Q1. 共通マスタと自社マスタの違いは何ですか?

**A.** 共通マスタは業界標準の道具テンプレート、自社マスタは自社で登録した道具です。

| 項目 | 🏢 共通マスタ | 👥 自社マスタ |
|------|-------------|-------------|
| **意味** | 業界で一般的な道具のテンプレート | 過去に自社で登録した道具 |
| **例** | 電動ドリル、測定器、重機など | 過去に購入した電動ドリル DRL-5000 |
| **登録者** | ザイロク運営（共通） | 自社の管理者 |
| **変更** | 変更不可 | 自社マスタは編集可能 |
| **用途** | 新規登録時のテンプレート | 同じ道具の追加購入時に便利 |

**使い分け:**
- 初めて登録する道具 → **共通マスタ**から選択（推奨）
- 過去に登録した道具を追加購入 → **自社マスタ**から選択
- 共通マスタにない道具 → 道具・消耗品マスタページから新規作成

**メリット:**
- 共通マスタを使うと、道具名・型番・メーカーの入力が不要
- 登録作業が大幅に短縮される

**関連**: [道具管理マニュアル - 道具の新規登録](../../manual/tool_management.md#道具の新規登録)

---

## Q2. 共通マスタに欲しい道具がありません。どうすればいいですか?

**A.** 道具・消耗品マスタページから新規作成できます。

**手順:**
1. **道具・消耗品マスタ** ページを開く
2. **「+ 新規作成」** をクリック
3. 以下の情報を入力:
   - 道具名
   - 型番
   - メーカー
   - 管理タイプ（個別管理 / 数量管理）
4. **「登録」** をクリック

**登録後:**
- 自社マスタとして保存される
- 次回から道具登録時に選択できる
- 他の組織には表示されない（自社専用）

**共通マスタへの追加リクエスト:**
- よく使う道具を共通マスタに追加してほしい場合
- サポートに連絡してリクエストできます

## Q3. 道具を登録したのに、QRコードが表示されません。

**A.** 個別アイテム一覧から各アイテムのQRコードを確認できます。

**手順:**
1. **道具管理 > 道具一覧** を開く
2. 登録した道具をクリック
3. **「個別アイテム一覧」** セクションを確認
4. 各個別アイテムに **「QR印刷」** ボタンが表示されます

**QRコードについて:**
- 道具マスタには QRコードは発行されません
- 個別アイテムごとに固有のQRコードが発行されます
- 例: 電動ドリルを3個登録 → 3つの個別QRコードが発行される

**関連**: [道具管理マニュアル - QRコードの印刷](../../manual/tool_management.md#qrコードの印刷と貼り付け)

## Q4. 個数を入力し忘れました。後から変更できますか?

**A.** 個数フィールドは編集できませんが、個別アイテムを追加・削除できます。

**個数を増やしたい場合:**
1. 道具詳細画面を開く
2. **「個別アイテムを追加」** をクリック
3. 追加する個数を入力
4. **「追加」** をクリック

**個数を減らしたい場合:**
1. 道具詳細画面の **「個別アイテム一覧」** を開く
2. 削除したいアイテムの **「削除」** をクリック
3. 確認ダイアログで **「削除」** をクリック

**削除の条件:**
- 移動履歴がない個別アイテムのみ削除可能
- 現在使用中のアイテムは削除不可

**関連**: [道具管理のトラブルシューティング - 個数の変更](./tool_issues.md#q3-道具の個数を減らしたいですどうすればいいですか)

## Q5. 購入日や購入価格は必須ですか?

**A.** いいえ、任意項目です。ただし入力を推奨します。

**入力を推奨する理由:**

### 1. 資産管理
- 会社の資産として正確に記録
- 減価償却の計算に必要

### 2. 原価計算
- 工事の原価計算に使用
- 利益率の算出に必要

### 3. 購入計画
- 購入時期を把握
- 次回購入のタイミングを予測

**後から入力できます:**
1. 道具詳細画面で **「編集」** をクリック
2. 購入日、購入価格を入力
3. **「保存」** をクリック

---

## Q6. 自社マスタを編集できますか?

**A.** はい、自社で作成したマスタは編集できます。

**編集手順:**
1. **道具・消耗品マスタ** ページを開く
2. 編集したいマスタを検索
3. **「編集」** をクリック
4. 道具名、型番、メーカーなどを修正
5. **「保存」** をクリック

**編集の影響:**
- 既存の道具には影響しません
- 新規登録時のテンプレートのみ変更されます

**共通マスタは編集できません:**
- 共通マスタは全ユーザー共通のため、編集不可
- 自社専用のマスタを作成してください

---

## Q7. 登録時に道具名を間違えました。修正できますか?

**A.** はい、道具詳細画面から編集できます。

**手順:**
1. **道具管理 > 道具一覧** を開く
2. 修正したい道具をクリック
3. **「編集」** をクリック
4. 道具名を修正
5. **「保存」** をクリック

**注意:**
- 道具マスタ自体の名前が変更されます
- 個別アイテムの名前も自動的に更新されます
- QRコードは変わりません（再印刷不要）

---

## Q8. 共通マスタから登録した道具の型番を変更したいです。

**A.** 道具編集画面で型番を変更できます。

**手順:**
1. 道具詳細画面で **「編集」** をクリック
2. 型番フィールドを修正
3. **「保存」** をクリック

**変更の影響:**
- その道具のみ型番が変更されます
- 共通マスタ自体は変わりません
- 他の道具には影響しません

**例:**
```
共通マスタ: 電動ドリル DRL-5000
↓ 登録後に型番を変更
自社の道具: 電動ドリル DRL-5001（改良版）
```

---

## Q9. 登録した道具を削除したいです。削除できますか?

**A.** 移動履歴がない道具のみ削除できます。

**削除できる条件:**
- ✅ 誤って登録した直後
- ✅ まだ誰も使用していない
- ✅ 移動履歴がない

**削除できない条件:**
- ❌ 移動履歴がある
- ❌ 現在使用中
- ❌ 過去に貸出記録がある

**削除手順:**
1. 道具詳細画面を開く
2. **「削除」** をクリック
3. 確認ダイアログで **「削除」** をクリック

**削除できない場合:**
- 「廃棄」または「紛失」として状態を変更
- 論理削除（非表示）として処理

**関連**: [道具管理のトラブルシューティング - 道具の削除](./tool_issues.md#q6-道具を削除できませんなぜですか)

## Q10. 備考欄は何に使いますか?

**A.** 道具に関する特記事項やメモを記録できます。

**活用例:**

### 購入情報
- 「2026年1月購入、◯◯商事から」
- 「レンタル品、返却期限: 2026-02-28」

### 使用上の注意
- 「バッテリーは別売り」
- 「使用前に校正必須」

### メンテナンス情報
- 「年1回オーバーホール必要」
- 「絶縁検査済み（2026-01-15）」

### 保管場所
- 「倉庫A-3棚に保管」
- 「高温多湿を避ける」

**個別アイテムのメモとの違い:**
- 備考: 道具マスタ全体に関する情報
- 個別アイテムのメモ: 各個体ごとの情報（例: 「#001はグリップ破損」）

## Q11. プリセット（共通マスタ）はどのくらい登録されていますか?

**A.** 建設・工事業界でよく使われる道具が100種類以上登録されています。

**カテゴリ例:**
- 電動工具（ドリル、グラインダー、インパクトドライバーなど）
- 測定器（レーザー距離計、水平器、測量機など）
- 重機（ミニショベル、高所作業車など）
- 安全装備（ヘルメット、安全帯など）
- 切断・溶接（丸ノコ、溶接機など）

**追加リクエスト:**
- よく使う道具が共通マスタにない場合
- サポートに追加リクエストできます
- 次回のアップデートで追加される可能性があります

## Q12. 個別アイテムのシリアル番号は自動で割り当てられますか?

**A.** はい、自動で連番が割り当てられます。

**シリアル番号の形式:**
```
#001, #002, #003, #004, ...
```

**特徴:**
- 3桁の連番（001から開始）
- 削除された番号は再利用されません
- 追加登録時は最後の番号の続きから割り当て

**例:**
```
初回登録: #001, #002, #003
↓ #002を削除
残り: #001, #003
↓ 2個追加
結果: #001, #003, #004, #005（#002は欠番）
```

**カスタマイズ:**
- シリアル番号のフォーマットは変更できません
- 固有のシリアル番号が必要な場合は、個別アイテムのメモ欄に記録

## Q13. 同じ道具を複数回登録してしまいました。統合できますか?

**A.** 統合機能はありません。片方を削除して個数を調整してください。

**推奨手順:**

### 1. 統合先の道具を決める
- より詳細に登録された方を残す
- 購入日や購入価格が正確な方を残す

### 2. 統合先の道具に個別アイテムを追加
1. 統合先の道具詳細画面を開く
2. **「個別アイテムを追加」** をクリック
3. 統合元の道具の個数を入力
4. **「追加」** をクリック

### 3. 統合元の道具を削除
1. 統合元の道具詳細画面を開く
2. すべての個別アイテムを削除
3. 道具マスタを削除

**注意:**
- 移動履歴がある道具は削除できません
- その場合は、統合元の道具を「廃棄済み」として非表示にする

## Q14. 登録後にQRコードを印刷し忘れました。後から印刷できますか?

**A.** はい、いつでも印刷できます。

**手順:**
1. **道具管理 > 道具一覧** を開く
2. 対象の道具をクリック
3. **「個別アイテム一覧」** セクションを確認
4. 各アイテムの **「QR印刷」** をクリック
5. PDFをダウンロードして印刷

**一括印刷:**
- 複数の道具をまとめて印刷したい場合
- **道具管理 > QR一括印刷** から印刷可能

**関連**: [道具管理マニュアル - QRコードの印刷](../../manual/tool_management.md#qrコードの印刷と貼り付け)

---

## Q15. 道具を登録したら、すぐに使い始められますか?

**A.** QRコードを印刷して貼り付けた後、使い始められます。

**使用開始までの流れ:**

### 1. 道具登録（完了）
- 道具マスタ選択
- 個数入力
- 登録完了

### 2. QRコード印刷
- 個別アイテムのQRコードを印刷
- A4用紙またはラベル用紙に印刷

### 3. QRコード貼り付け
- 各道具に対応するQRコードを貼り付け
- 見やすい場所、汚れにくい場所を選ぶ

### 4. 使用開始
- QRスキャンで道具を持ち出し
- 移動履歴が記録される

**QRコードなしでも使えますか?**
- 道具一覧から手動で移動登録可能
- ただし、QRコードを使う方が圧倒的に早い

**関連**: [QRスキャンマニュアル](../../manual/qr_scan.md)
