# 倉庫位置管理のトラブルシューティング

## Q1. 倉庫位置管理とは何ですか?

**A.** 倉庫内のどこに道具があるかを正確に記録する機能です。

**課題:**
```
❌ 倉庫のどこかにあるが、見つからない
❌ 「Aエリアにある」までは分かるが、具体的な場所が不明
❌ 棚卸しに時間がかかる
```

**解決:**
```
✅ 「Aエリア 3番棚 上段」と具体的な位置を記録
✅ QRコードをスキャンして位置を確認
✅ 位置から道具を逆引き検索
```

**メリット:**
- 道具を素早く見つけられる
- 棚卸しが効率化される
- 倉庫内の整理整頓がしやすい

**関連**: [倉庫位置管理マニュアル](/manual/warehouse_locations)

---

## Q2. 階層構造とは何ですか?

**A.** 倉庫の構造を階層的に管理する仕組みです。

**例1: 3階層の倉庫**
```
レベル1: エリア（A、B、C）
  └── レベル2: 棚（1番、2番、3番）
      └── レベル3: 段（上段、中段、下段）

具体的な位置:
- Aエリア 1番棚 上段
- Aエリア 1番棚 中段
- Bエリア 2番棚 上段
```

**例2: 1階層の倉庫**
```
レベル1: 場所のみ
- 北側壁掛け1
- 南側壁掛け2
- 作業台横
```

**例3: 4階層の倉庫**
```
レベル1: フロア（1F、2F）
  └── レベル2: エリア（工具、消耗品、重機部品）
      └── レベル3: 保管方法（棚、壁掛け、床置き）
          └── レベル4: 番号（1、2、3...）
```

**メリット:**
- 倉庫の構造に合わせて柔軟に設定できる
- 階層的に整理できて分かりやすい
- 位置の親子関係が明確

## Q3. 倉庫位置を登録できるのは誰ですか?

**A.** Admin（管理者）のみが登録できます。

**権限別の操作:**

| 権限 | 階層設定 | 位置登録 | 位置編集 | 位置削除 | 閲覧 |
|------|---------|---------|---------|---------|-----|
| **Staff** | ❌ | ❌ | ❌ | ❌ | ✅ |
| **Leader** | ❌ | ❌ | ❌ | ❌ | ✅ |
| **Manager** | ❌ | ❌ | ❌ | ❌ | ✅ |
| **Admin** | ✅ | ✅ | ✅ | ✅ | ✅ |

**注意:**
- 倉庫位置管理のすべての操作はAdmin（管理者）専用機能です
- Staff、Leader、Managerは倉庫位置情報の閲覧のみ可能

---

## Q4. 階層設定を変更できますか?

**A.** はい、Admin（管理者）であれば変更できます。

**手順:**
1. **設定 > 倉庫位置設定 > 階層設定** を開く
2. 階層レベルまたは名称を変更
3. **「保存」** をクリック

**注意:**
- 階層設定を変更しても、既存の倉庫位置の階層レベルは自動では変わりません
- 必要に応じて、倉庫位置を個別に編集してください
- 倉庫位置を登録する前に階層設定を行うことを推奨します

**影響:**
- 既存の倉庫位置の表示が変わる可能性があります
- 新規登録時の入力フォームが変わります

## Q5. 位置コードと表示名の違いは何ですか?

**A.** 以下の違いがあります。

| 項目 | 位置コード | 表示名 |
|------|----------|--------|
| **用途** | システム内で使用 | ユーザーに表示 |
| **形式** | 短いコード | 分かりやすい名前 |
| **例** | A-1-上 | Aエリア 1番棚 上段 |
| **一意性** | 組織内で一意 | 重複可能 |

**位置コードの命名規則（推奨）:**
```
3階層の場合: [エリア]-[棚番号]-[段]
- A-1-上（Aエリア 1番棚 上段）
- A-1-中（Aエリア 1番棚 中段）
- B-2-下（Bエリア 2番棚 下段）

4階層の場合: [フロア]-[エリア]-[保管方法]-[番号]
- 1F-工具-棚-5（1F 工具エリア 棚 5番）
- 2F-消耗品-壁掛け-3（2F 消耗品エリア 壁掛け 3番）
```

**推奨:**
- 位置コードは短く、入力しやすいものにする
- 表示名は誰が見ても分かる名前にする

## Q6. QRコードはどこに貼ればいいですか?

**A.** 実際の倉庫のその位置に貼ってください。

**推奨貼付位置:**
- **棚の場合**: 棚の前面（見やすい高さ）
- **壁掛けの場合**: 壁掛けフックの近く
- **床置きエリアの場合**: 床または壁

**ラベルサイズ:** 5cm x 5cm 以上（見やすいサイズ）

**注意:**
- 汚れにくい場所に貼る
- スキャンしやすい高さに貼る
- ラベル用紙を使うと貼り付けが簡単

**活用:**
- QRコードをスキャンして、現在地を確認
- QRコードをスキャンして、その位置に道具を配置
- QRコードをスキャンして、その位置の道具一覧を表示

**関連**: [倉庫位置管理マニュアル - QRコードの印刷](../../manual/warehouse_locations.md#ステップ3-qrコードの印刷と貼り付け)

## Q7. 倉庫位置を削除できません。なぜですか?

**A.** 以下の条件に該当する倉庫位置は削除できません。

**削除できない条件:**
- ✅ 道具が配置されている位置
- ✅ 子位置がある位置（階層構造の場合）

**削除するには:**
1. その位置にある道具をすべて別の位置に移動
2. 子位置がある場合は、子位置を先に削除
3. その後、位置を削除

**注意:**
- 誤って登録した位置で、まだ使用していないもののみ削除可能
- 削除は論理削除（データは残るが非表示）

**代替案:**
- 削除せず、**無効化**することを推奨
- 無効化すると、新規登録時には表示されないが、既存のデータは保持される

## Q8. 道具を倉庫位置に配置するにはどうすればいいですか?

**A.** 以下の2つの方法があります。

### 方法1: QRスキャンで配置

1. スマートフォンで **QRスキャン** を開く
2. 倉庫位置のQRコードをスキャン
3. 配置したい道具のQRコードをスキャン
4. **「この位置に配置」** をクリック

**メリット:** スマホで素早く配置できる

### 方法2: 手動で配置

1. **道具管理 > 道具一覧** で対象の道具をクリック
2. **「編集」** をクリック
3. **倉庫位置**を選択（例: Aエリア 1番棚 上段）
4. **「保存」** をクリック

**メリット:** パソコンから一括で設定できる

**関連**: [倉庫位置管理マニュアル - 道具の配置](../../manual/warehouse_locations.md#倉庫位置を使った道具の配置)

## Q9. 特定の倉庫位置にある道具を確認できますか?

**A.** はい、確認できます。

**手順:**
1. **倉庫位置管理 > 位置マスタ** を開く
2. 対象の位置をクリック
3. **「この位置の道具」** タブを開く
4. 現在その位置にある道具の一覧が表示されます

**活用例:**
- 棚卸し時に、特定の位置の道具をチェック
- 位置別の在庫確認
- 空いている位置の確認

---

## Q10. 倉庫位置が見つかりません。どうすれば見つけられますか?

**A.** 以下の方法で検索してください。

### 1. 検索ボックスを使う
- 画面上部の検索ボックスに位置コードまたは表示名を入力
- 部分一致で検索できます

### 2. フィルタ機能を使う
- **「フィルタ」** ボタンをクリック
- 階層レベル、親位置で絞り込み

### 3. 削除されていないか確認
- 削除された倉庫位置は一覧に表示されません
- 管理者に確認してください

---

## Q11. QRコードをスキャンしても倉庫位置が表示されません

**A.** 以下を確認してください。

### 1. インターネット接続を確認
- Wi-Fiまたはモバイルデータ通信が有効か確認

### 2. QRコードが正しいか確認
- 倉庫位置のQRコードか確認（道具や消耗品のQRコードではないか）
- 自社で発行したQRコードか確認

### 3. 削除されていないか確認
- その倉庫位置が削除されている可能性があります
- 管理者に確認してください

### 4. ブラウザのエラーを確認
- ブラウザのコンソールにエラーが表示されていないか確認
- ページを再読み込みして再度試す

**それでも解決しない場合:**
- 管理者に連絡してください
- QRコードを再印刷して試してください

**関連**: QRスキャンのトラブルシューティング

## Q12. 階層構造なしで倉庫位置を管理できますか?

**A.** はい、できます。1階層のみで管理可能です。

**設定方法:**
1. **設定 > 倉庫位置設定 > 階層設定** を開く
2. **階層レベル**: 1階層のみを選択
3. **レベル1の名称**: 「場所」などと設定
4. **「保存」** をクリック

**登録例:**
```
- 北側壁掛け1
- 北側壁掛け2
- 南側壁掛け1
- 作業台横
- 入口付近
```

**メリット:**
- シンプルで分かりやすい
- 小規模な倉庫に最適

**デメリット:**
- 階層的な整理ができない
- 倉庫が大きい場合は管理が大変

## Q13. 倉庫位置の親子関係を変更できますか?

**A.** はい、倉庫位置編集画面で変更できます。

**手順:**
1. 倉庫位置詳細画面を開く
2. **「編集」** をクリック
3. **親位置**を変更
4. **「保存」** をクリック

**例:**
```
変更前: Aエリア 1番棚 上段（親: Aエリア 1番棚）
変更後: Bエリア 2番棚 上段（親: Bエリア 2番棚）
```

**注意:**
- 親子関係を変更すると、階層構造が変わります
- 位置コードや表示名も合わせて変更することを推奨

---

## Q14. 棚卸し時に倉庫位置管理をどう活用すればいいですか?

**A.** 位置別に道具をチェックできます。

**手順:**
1. **倉庫位置管理 > 位置マスタ** で位置を選択
2. **「この位置の道具」** タブで道具一覧を表示
3. 実際の倉庫でその位置の道具をチェック
4. 不足や過剰があれば在庫調整

**効率化のコツ:**
- QRコードをスキャンして位置を確認
- 位置ごとにチェックリストを作成
- チェック済みの位置にマークを付ける

**メリット:**
- 棚卸しが効率化される
- 見落としを防げる
- 位置別に進捗を管理できる

**関連**: [倉庫位置管理マニュアル - 活用例](../../manual/warehouse_locations.md#活用例)

---

## Q15. 倉庫位置の表示順を変更できますか?

**A.** はい、倉庫位置編集画面で変更できます。

**手順:**
1. 倉庫位置詳細画面を開く
2. **「編集」** をクリック
3. **表示順**を変更（数値が小さいほど上に表示）
4. **「保存」** をクリック

**例:**
```
表示順: 10 → Aエリア（最初に表示）
表示順: 20 → Bエリア
表示順: 30 → Cエリア（最後に表示）
```

**推奨:**
- 10刻みで設定すると、後から挿入しやすい
- よく使う位置を上位に設定
