# 現場スタッフの1日の流れのトラブルシューティング

## Q1. 初めてザイロクを使います。何から始めればいいですか?

**A.** まずはログインして出勤打刻から始めてください。

**初日の基本的な流れ:**
1. ログイン（管理者から送られたメールの初期パスワードを使用）
2. パスワード変更（初回ログイン時に促されます）
3. 出勤打刻（**勤怠管理 > 出退勤打刻**）
4. 道具の持ち出し（QRスキャンまたは道具一覧から）
5. 作業開始

**困ったときは:**
- 現場責任者やリーダーに操作方法を聞いてください
- マニュアルを参照してください

**関連**: [現場スタッフの1日の流れマニュアル](../../manual/scenarios/daily/daily_workflow.md)

---

## Q2. 朝、会社に着いたら最初に何をすればいいですか?

**A.** 出勤打刻をしてください。

**手順:**
1. スマートフォンでザイロクにアクセス
2. **勤怠管理 > 出退勤打刻** を開く
3. **「会社に出勤」** をタップ

**QRコードがある場合:**
- 会社の入口に設置されたQRコードをスキャン
- 自動的に出勤打刻されます

**注意:**
- 出勤打刻を忘れると、後で管理者に依頼して手動登録が必要になります
- スマホのリマインダーを設定すると便利です

**関連**: [勤怠管理マニュアル](../../manual/attendance.md#出勤打刻)

---

## Q3. 現場に直行する場合、出勤打刻はどうすればいいですか?

**A.** 「現場に直行」を選択して出勤打刻してください。

**手順:**
1. **勤怠管理 > 出退勤打刻** を開く
2. **「現場に直行」** をタップ
3. 勤務予定の現場を選択
4. 現場に到着したら打刻

**現場にQRコードがある場合:**
- 現場QRコードをスキャンすることでも出勤打刻できます
- 自動的にその現場への出勤として記録されます

**関連**: [現場スタッフの1日の流れマニュアル - 出勤時の流れ](../../manual/scenarios/daily/daily_workflow.md#出勤時の流れ)

## Q4. 道具の持ち出しはどうやって登録しますか?

**A.** QRスキャンまたは道具一覧から持ち出し登録できます。

**方法1: QRコードでスキャン（推奨）**
1. **QRスキャン** をタップ
2. 道具のQRコードをスキャン
3. 「移動」をタップ
4. 移動先で現場を選択
5. 「持ち出す」をタップ

**方法2: 道具一覧から選択**
1. **道具管理** を開く
2. 持ち出す道具を検索
3. 道具詳細画面で「移動」をタップ
4. 移動先で現場を選択
5. 「持ち出す」をタップ

**関連**: [道具管理マニュアル - 道具の移動](../../manual/tool_management.md#道具の移動持ち出し返却)

## Q5. 道具のQRコードが破損して読み取れません。どうすればいいですか?

**A.** 道具管理画面から手動で持ち出し登録してください。

**手順:**
1. **道具管理** を開く
2. 道具名や番号で検索
3. 該当する道具をタップ
4. 「移動」をタップ
5. 移動先で現場を選択
6. 「持ち出す」をタップ

**QRコード再発行について:**
- QRコードの再印刷は管理者に依頼してください
- QRコードIDは変わりません（同じIDで再印刷可能）

**関連**: [QRスキャンのトラブルシューティング](./qr_scan_issues.md)

## Q6. 現場で作業中に写真を撮りました。どこにアップロードすればいいですか?

**A.** 作業報告書に写真を添付してください。

**手順:**
1. **作業報告書** を開く
2. **「+ 新規作成」** をタップ
3. 報告内容を入力
4. **「写真を追加」** をタップ
5. カメラで撮影またはアルバムから選択
6. 最大5枚まで添付可能
7. **「提出」** をタップ

**注意:**
- 1枚あたり10MBまで
- 対応形式: JPG、PNG、HEIC

**関連**: [作業報告書マニュアル - 写真の添付](../../manual/work_reports.md#写真の添付)

## Q7. 作業報告書はいつまでに提出すればいいですか?

**A.** 作業終了後、その日のうちに提出してください。

**一般的な提出タイミング:**
- 退勤前に提出（推奨）
- 遅くとも翌日の午前中まで

**提出期限は組織によって異なります:**
- 管理者に確認してください
- 提出期限を過ぎると、承認されない場合があります

**提出漏れを防ぐコツ:**
- 退勤打刻の前に作業報告書を提出する習慣をつける
- スマホのリマインダーを設定する

**関連**: [作業報告書マニュアル](../../manual/work_reports.md)

## Q8. 道具を返却し忘れて家に持って帰ってしまいました。どうすればいいですか?

**A.** すぐに現場責任者に連絡してください。

**対応手順:**
1. 現場責任者に連絡（電話またはメッセージ）
2. 持ち帰った道具の名前と個数を伝える
3. 翌日、道具を持参して返却
4. 道具管理画面から返却登録

**システム上の処理:**
- 道具の現在地は「現場」のまま
- 翌日、倉庫に返却登録すればOK

**紛失したわけではないので:**
- 慌てず、正直に報告してください
- システム上で「紛失」登録しないでください

**関連**: [現場スタッフの1日の流れマニュアル - 道具返却](../../manual/scenarios/daily/daily_workflow.md#道具返却)

## Q9. 退勤打刻を忘れて帰宅してしまいました。どうすればいいですか?

**A.** 管理者に連絡して、手動で退勤記録を追加してもらってください。

**手順:**
1. 管理者に連絡（翌日朝でもOK）
2. 以下の情報を伝える:
   - 日付
   - 退勤時刻
   - 退勤場所（会社/現場/直帰）
3. 管理者が手動で記録を追加

**予防策:**
- 退勤前に作業報告書を提出する習慣をつける
- スマホのリマインダーを設定

**関連**: [勤怠管理のトラブルシューティング - 出勤打刻を忘れた](./attendance_issues.md#q4-出勤打刻を忘れましたどうすればいいですか)

## Q10. 作業報告書の承認が遅いです。どうすればいいですか?

**A.** 承認者（リーダーまたは管理者）に確認してください。

**確認方法:**
1. **作業報告書** を開く
2. 自分の提出した報告書を確認
3. ステータスが「承認待ち」か確認

**ステータスの意味:**
- **下書き**: まだ提出していない
- **承認待ち**: 提出済み、承認待ち
- **差し戻し**: 修正が必要
- **承認済み**: 承認完了

**承認が遅い理由:**
- 承認者が多忙
- 承認者が報告書を見ていない
- 内容に不明点がある（質問が来ているか確認）

**対処法:**
- 承認者に直接確認する
- 報告書の内容が明確か再確認する

**関連**: [作業報告書マニュアル - 承認フロー](../../manual/work_reports.md#承認フロー)

## Q11. 複数の現場を掛け持ちしています。出勤打刻はどうすればいいですか?

**A.** 最初に到着した現場で出勤打刻してください。

**手順:**
1. 最初の現場に到着したら、「現場に直行」で出勤打刻
2. 2つ目の現場に移動するときは、打刻不要
3. 最後の現場から退勤するときに退勤打刻

**作業報告書は:**
- 各現場ごとに作成してください
- それぞれの現場を選択して提出

**注意:**
- 出勤打刻は1日1回のみ
- 現場移動時は打刻不要

**関連**: [現場スタッフの1日の流れマニュアル](../../manual/scenarios/daily/daily_workflow.md)

## Q12. スマートフォンが充電切れしました。出退勤打刻ができません。

**A.** 充電後、できるだけ早く打刻してください。時刻がずれている場合は管理者に連絡してください。

**対応方法:**

### 1. 充電後すぐに打刻できる場合:
- 通常通り打刻してください
- 多少の遅れは問題ありません

### 2. 充電に時間がかかる場合:
- 同僚のスマホを借りてログイン（推奨しません）
- または、帰宅後に管理者に連絡

### 3. 時刻が大幅にずれている場合:
- 管理者に連絡して正しい時刻で記録してもらう

**予防策:**
- 朝、スマホを充電しておく
- モバイルバッテリーを持参

**関連**: [勤怠管理のトラブルシューティング](./attendance_issues.md)

## Q13. 道具を返却したのに、システム上で「現場にある」と表示されています。

**A.** 返却登録が完了していない可能性があります。再度返却登録してください。

**確認方法:**
1. **道具管理** を開く
2. 返却した道具を検索
3. **「現在の場所」** を確認

**返却登録の手順:**
1. 道具詳細画面で「移動」をタップ
2. 移動先で「倉庫」を選択
3. 保管場所を指定
4. 「持ち出す」をタップ

**それでも解決しない場合:**
- 管理者に連絡してください
- システムの不具合の可能性があります

**関連**: [道具管理マニュアル - 道具の返却](../../manual/tool_management.md#道具の移動持ち出し返却)

## Q14. 現場でWi-Fiがありません。オフラインで使えますか?

**A.** いいえ、ザイロクはオンライン専用です。モバイルデータ通信を使用してください。

**オフラインではできないこと:**
- 出退勤打刻
- 道具の持ち出し・返却登録
- 作業報告書の提出
- QRコードスキャン

**対処法:**
- スマートフォンのモバイルデータ通信をONにする
- 4G/5G回線を使用

**データ通信量は少ないです:**
- 1日の使用で数MB程度
- 写真をアップロードする場合は通信量が増えます

**関連**: [スマートフォン利用のトラブルシューティング](./mobile_usage.md)

## Q15. 1日の作業の流れを簡単に教えてください。

**A.** 以下の流れで作業してください。

**朝（出勤時）:**
1. 出勤打刻（会社または現場）
2. 道具の持ち出し登録
3. 作業開始

**作業中:**
- 道具の使用
- 必要に応じて写真撮影
- メモやメモアプリに報告内容を記録

**夕方（退勤時）:**
1. 道具の返却登録
2. 作業報告書の提出
3. 退勤打刻

**チェックリスト:**
- [ ] 出勤打刻
- [ ] 道具の持ち出し登録
- [ ] 作業実施
- [ ] 道具の返却登録
- [ ] 作業報告書提出
- [ ] 退勤打刻

**関連**: [現場スタッフの1日の流れマニュアル](../../manual/scenarios/daily/daily_workflow.md)

## Q16. 道具を持ち出したのに、別のスタッフが「持ち出し中」と表示されます。

**A.** 前回使用したスタッフが返却登録していない可能性があります。管理者に確認してください。

**原因:**
- 前回使用者が返却登録を忘れた
- システム上の記録ミス

**対処法:**
1. 管理者に連絡
2. 道具名と現在の状況を伝える
3. 管理者が記録を修正

**自分で直せますか?**
- いいえ、スタッフ権限では修正できません
- 管理者に依頼してください

**関連**: [道具管理のトラブルシューティング](./tool_issues.md)

## Q17. 作業報告書の写真を追加し忘れました。後から追加できますか?

**A.** 提出前なら追加できます。提出後は差し戻しを依頼してください。

**提出前の場合:**
1. **作業報告書** を開く
2. 下書きの報告書をタップ
3. 「編集」をタップ
4. 写真を追加
5. 「提出」をタップ

**提出後の場合:**
1. 承認者に連絡
2. 差し戻しを依頼
3. 差し戻されたら、編集して写真を追加
4. 再提出

**注意:**
- 承認済みの報告書は編集できません
- 重要な写真は必ず提出前に確認しましょう

**関連**: [作業報告書マニュアル - 写真の添付](../../manual/work_reports.md#写真の添付)

## Q18. 道具セットをまとめて持ち出すことはできますか?

**A.** はい、道具セット機能を使えば一括で持ち出し登録できます。

**手順:**
1. **道具セット** を開く
2. 持ち出すセットを選択（例: 電動工具セットA）
3. 「一括持ち出し」をタップ
4. 移動先で現場を選択
5. 「持ち出す」をタップ

**道具セットに含まれる道具:**
- セット内のすべての道具が一括で持ち出し登録されます
- 個別にQRスキャンする必要はありません

**返却時も一括可能:**
- セット単位で一括返却もできます

**関連**: [道具セットマニュアル](../../manual/tool_sets.md)

## Q19. 毎日同じ道具を使っています。毎日持ち出し登録が必要ですか?

**A.** 道具の保管場所によって異なります。

**パターン1: 毎日倉庫に返却する場合**
- 毎日持ち出し・返却登録が必要です

**パターン2: 現場に道具を置きっぱなしにする場合**
- 1回持ち出し登録すればOK
- 現場に道具がある状態が続きます
- 最終日に返却登録してください

**推奨:**
- 高額な道具は毎日返却
- 消耗品や軽微な道具は現場保管でもOK（管理者に確認）

**関連**: [現場スタッフの1日の流れマニュアル - 道具持ち出し](../../manual/scenarios/daily/daily_workflow.md#道具持ち出し)

## Q20. 休憩時間の記録は必要ですか?

**A.** 組織設定によって異なります。休憩ボタンが表示される場合は記録してください。

**休憩ボタンがある場合:**
1. 休憩開始時に「休憩開始」をタップ
2. 休憩終了時に「休憩終了」をタップ

**休憩ボタンがない場合:**
- 記録不要です
- 自動控除される場合があります

**確認方法:**
- 出退勤打刻画面で「休憩開始」ボタンが表示されるか確認

**関連**: [勤怠管理のトラブルシューティング - 休憩時間の記録](./attendance_issues.md#q11-休憩時間の記録方法は何がありますか)
