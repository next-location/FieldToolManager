## 概要

ザイロクの出退勤打刻機能は、スタッフの勤務時間を記録するための機能です。

**打刻方法**
- QRコードスキャン（組織設定で有効な場合）
- 手動打刻（組織設定で有効な場合）

**打刻できる場所**
- 🏢 会社（オフィス）
- 🏗️ 現場

**対応デバイス**
- スマートフォン（推奨）
- タブレット
- パソコン

---

## 出勤打刻の方法

### 基本的な流れ

1. サイドメニューから **「勤怠管理」** をタップ
2. **「出退勤打刻」** をタップ
3. 打刻画面が表示される
4. QRスキャンまたは手動打刻で出勤登録

### 出勤打刻画面の構成

**画面上部:**
- 日付と曜日が表示される
- 出勤時刻と退勤時刻の表示エリア

**出勤打刻エリア:**
- **QRコードでスキャン** ボタン（組織設定で有効な場合）
- **手動打刻** セクション（組織設定で有効な場合）

---

## 退勤打刻の方法

### 基本的な流れ

1. サイドメニューから **「勤怠管理」** をタップ
2. **「出退勤打刻」** をタップ
3. 退勤打刻画面が表示される（出勤済みの場合）
4. QRスキャンまたは手動打刻で退勤登録

### 出勤中の画面表示

**勤務時間の表示:**
- 出勤時刻からの経過時間が表示される
- リアルタイムで更新される

**退勤打刻エリア:**
- **QRコードでスキャン** ボタン
- **手動打刻** セクション

## QRコードでの打刻

QRコードスキャンを使用すると、素早く正確に出退勤打刻ができます。

### QRコード打刻の手順

#### 出勤時:

1. 打刻画面で **「QRコードでスキャン」** ボタンをタップ
2. カメラが起動する
3. QRコードをカメラで読み取る
4. 自動的に出勤打刻される
5. 成功メッセージが表示される

#### 退勤時:

1. 打刻画面で **「QRコードでスキャン」** ボタンをタップ
2. カメラが起動する
3. QRコードをカメラで読み取る
4. 自動的に退勤打刻される
5. 成功メッセージが表示される

### 使用できるQRコード

**会社QR:**
- 会社入口に設置されたQRコード
- 管理者が発行・設置
- 有効期限: 組織設定による（デフォルト7日間）

**リーダーQR:**
- リーダーがスマホ画面で表示するQRコード
- 現場で複数人が一斉に打刻する際に使用
- 有効期限: 24時間

**現場QR:**
- 現場に設置されたQRコード
- 現場登録時に発行
- 有効期限: 永続的（現場が完了するまで）

### QRコードスキャンの注意点

**カメラ権限:**
- 初回使用時にカメラ権限の許可が必要
- 拒否した場合は、ブラウザ設定から許可してください

**読み取り環境:**
- 明るい場所で読み取る
- QRコードにピントを合わせる
- 汚れや破損がないか確認

**読み取りできない場合:**
- 手動打刻を使用してください
- QRコードの再発行を管理者に依頼してください

**関連**: [QRスキャンのトラブルシューティング](../qa/staff/qr_scan_issues.md)

## 手動打刻

手動打刻では、打刻場所を選択して出退勤を記録します。

### 手動打刻の手順（出勤時）

1. 打刻画面の **「どこで打刻していますか?」** セクションを確認
2. 打刻場所を選択:
   - **🏢 会社（オフィス）**: 会社に出社する場合
   - **🏗️ 現場**: 現場に直行する場合
3. 現場を選択した場合は、ドロップダウンから現場名を選択
4. **「出勤する」** ボタンをタップ
5. 成功メッセージが表示される

### 手動打刻の手順（退勤時）

1. 打刻画面の **「どこで退勤しますか?」** セクションを確認
2. 退勤場所を選択:
   - **🏢 会社（オフィス）**: 会社で退勤する場合
   - **🏗️ 現場**: 現場で退勤する場合
3. 現場を選択した場合は、ドロップダウンから現場名を選択
4. **「退勤する」** ボタンをタップ
5. 成功メッセージが表示される

### 打刻場所の選択ガイド

**会社（オフィス）を選択する場合:**
- 会社に出社してから業務を開始する
- 会社で業務を終了する

**現場を選択する場合:**
- 自宅から直接現場に向かう（直行）
- 現場から直接帰宅する（直帰）
- 現場で業務を行う

**出勤時と退勤時で場所が異なる場合:**
- 問題ありません。それぞれの場所で打刻してください
- 例: 会社に出勤 → 現場で退勤
- 例: 現場に出勤 → 会社で退勤

## 打刻画面の見方

### 日付と時刻の表示

**画面上部:**
```
1月21日（火）
2026年
```

**出勤時刻の表示:**
- 青色の背景
- 出勤時刻（例: 08:30）
- 出勤場所（🏢 会社 または 🏗️ 現場名）

**退勤時刻の表示:**
- 灰色の背景
- 退勤時刻（例: 17:30）
- 未退勤の場合は「--:--」と表示

### 勤務時間の表示

**出勤中の場合:**
- 緑色の背景で勤務時間が表示される
- リアルタイムで更新される
- 表示形式: 「8時間30分」

**計算方法:**
- 出勤時刻から現在時刻までの経過時間
- 休憩時間は含まれません（別途記録が必要）

### メッセージ表示

**成功メッセージ（緑色）:**
- 「出勤打刻が完了しました」
- 「退勤打刻が完了しました」

**エラーメッセージ（赤色）:**
- 「打刻場所を選択してください」
- 「現場を選択してください」
- 「出勤打刻に失敗しました」

## 注意事項

### 打刻のタイミング

**出勤打刻:**
- 業務開始時に打刻してください
- 遅刻する場合も必ず打刻してください

**退勤打刻:**
- 業務終了時に打刻してください
- 打刻忘れに注意してください

### 打刻忘れの対処

**出勤打刻を忘れた場合:**
- 管理者に連絡して手動で記録を追加してもらう
- 正確な出勤時刻を伝える

**退勤打刻を忘れた場合:**
- 管理者に連絡して手動で記録を追加してもらう
- 正確な退勤時刻を伝える

**予防策:**
- スマホのリマインダーを設定
- 出勤・退勤時のルーチンに組み込む

### 打刻の修正

**自分では修正できません:**
- 打刻ミスや時刻の修正は管理者に依頼
- 管理者が勤怠一覧から修正可能

**修正依頼時に伝える情報:**
- 日付
- 誤った時刻
- 正しい時刻
- 場所（会社/現場）

### 打刻方法の制限

**組織設定による制限:**
- QRコードのみ有効
- 手動打刻のみ有効
- 両方有効
- 両方無効（打刻不可）

**確認方法:**
- 打刻画面にボタンが表示されるか確認
- 表示されない場合は管理者に確認

## トラブルシューティング

### よくある問題と解決方法

#### 1. 出勤打刻ボタンが表示されない

**原因:**
- すでに出勤済み
- 勤怠機能が無効化されている
- 契約プランに含まれていない

**対処法:**
- 画面上部の出勤時刻を確認
- 管理者に勤怠機能の有効化を確認
- 契約プランを確認

#### 2. 現場が選択肢に表示されない

**原因:**
- 現場のステータスが「停止中」または「完了」
- 現場が削除されている

**対処法:**
- 管理者に現場のステータスを確認してもらう
- 現場が稼働中になっているか確認

#### 3. QRコードが読み取れない

**原因:**
- カメラ権限が許可されていない
- QRコードが汚れている・破損している
- 暗い場所で読み取っている

**対処法:**
- ブラウザのカメラ権限を確認
- QRコードを清掃または再発行
- 明るい場所で再度読み取る
- 手動打刻を使用

**詳細**: [QRスキャンのトラブルシューティング](../qa/staff/qr_scan_issues.md)

#### 4. 「出退勤の打刻方法が設定されていません」と表示される

**原因:**
- 組織設定で打刻方法が無効化されている

**対処法:**
- 管理者に連絡して設定を確認してもらう
- 管理者が「出退勤設定」で打刻方法を有効化

#### 5. 打刻後、記録が反映されない

**原因:**
- ネットワーク接続の問題
- ブラウザのキャッシュ問題

**対処法:**
- 画面をリロード（再読み込み）
- ネットワーク接続を確認
- それでも解決しない場合は管理者に連絡
