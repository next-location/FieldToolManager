## 概要

データエクスポート機能では、ザイロクに登録されている組織のデータをCSV形式でダウンロードできます。バックアップやExcelでの分析、外部ツールとの連携などに活用できます。

### 主な用途

- **データバックアップ**: 定期的なデータのバックアップ
- **外部分析**: ExcelやGoogleスプレッドシートでの詳細分析
- **レポート作成**: 経営会議や報告書用のデータ抽出
- **外部連携**: 他システムへのデータ移行・連携
- **監査対応**: 監査や検査時のデータ提出

---

## アクセス方法

### デスクトップ版

1. サイドメニューから **「設定」** をクリック
2. **「データエクスポート」** をクリック

### モバイル版

1. 右上のメニューアイコン（☰）をタップ
2. **「設定」** をタップ
3. **「データエクスポート」** をタップ

**権限**: 管理者（Admin）のみアクセス可能です。他の役割（Manager/Leader/Staff）ではメニューに表示されません。

## エクスポート可能なデータ

データエクスポート画面は、**マスタデータ**と**取引・履歴データ**の2つのセクションに分かれています。

### マスタデータ

基本的な登録データをエクスポートします。

| データ種別 | 説明 | 主な用途 |
|----------|------|----------|
| **取引先マスタ** | 取引先の基本情報、連絡先、支払条件 | 取引先リスト作成、外部会計ソフト連携 |
| **道具マスタ管理** | 道具の名称、カテゴリ、型番、購入情報 | 資産台帳作成、減価償却計算 |
| **消耗品一覧** | 消耗品の名称、在庫数、単価 | 在庫管理表作成、発注計画 |
| **スタッフ管理** | スタッフの氏名、メール、役割、部署 | 社員名簿作成、権限管理表 |
| **現場マスタ** | 現場の名称、住所、工事期間、契約金額 | 現場一覧表作成、売上集計 |
| **重機一覧** | 重機の名称、型番、メンテナンス記録 | 重機台帳作成、保守計画 |

### 取引・履歴データ

見積書、請求書、発注書、操作履歴や移動記録をエクスポートします。

| データ種別 | 説明 | 主な用途 |
|----------|------|----------|
| **見積書管理** | 見積書の基本情報、金額、ステータス | 見積一覧作成、受注分析 |
| **請求書管理** | 請求書の基本情報、金額、入金状況 | 請求一覧作成、入金管理 |
| **発注書一覧** | 発注書の基本情報、金額、承認状況 | 発注一覧作成、支払管理 |
| **在庫移動履歴** | 道具の移動記録（直近1000件） | 移動履歴分析、利用状況確認 |

**注意**: 在庫移動履歴は、データ量が多いため直近1000件のみエクスポートされます。

## エクスポート手順

### 基本的な手順

1. **データエクスポート画面にアクセス**
   - サイドメニュー → 設定 → データエクスポート

2. **エクスポートしたいデータを選択**
   - 各データ種別の右側に件数が表示されます
   - 例: 「道具マスタ管理 (152件)」

3. **「CSVエクスポート」ボタンをクリック**
   - ボタンをクリックすると、CSVファイルのダウンロードが開始されます
   - ファイル名: `{データ種別}_{日付}.csv`
   - 例: `tools_2026-01-22.csv`

4. **ダウンロード完了を確認**
   - 緑色のメッセージで「○○をエクスポートしました」と表示されます
   - ブラウザのダウンロードフォルダにCSVファイルが保存されます

5. **必要に応じて他のデータもエクスポート**
   - 複数のデータを連続してエクスポートできます

### ファイル名の規則

エクスポートされるファイル名は以下の形式です:

```
{データ種別の英名}_{エクスポート日}.csv
```

**例**:
- `clients_2026-01-22.csv` - 取引先マスタ
- `tools_2026-01-22.csv` - 道具マスタ
- `staff_2026-01-22.csv` - スタッフ管理
- `movements_2026-01-22.csv` - 在庫移動履歴

## CSV形式について

### エンコーディング

- **形式**: BOM付きUTF-8
- **互換性**: Excel、Googleスプレッドシート、Numbers等で日本語が正しく表示されます

### 文字コードの特徴

- **BOM（Byte Order Mark）付き**: Excelで開いたときに文字化けしません
- **ダブルクォート対応**: カンマや改行を含むデータも正しく処理されます

### Excelで開く方法

#### Windows

1. ダウンロードしたCSVファイルをダブルクリック
2. Excelで自動的に開きます
3. 日本語が正しく表示されます

#### Mac

1. ダウンロードしたCSVファイルを右クリック
2. **「このアプリケーションで開く」** → **「Excel」** を選択
3. 日本語が正しく表示されます

**注意**: テキストエディタで開くと、BOMが表示される場合があります。ExcelやGoogleスプレッドシートで開くことを推奨します。

### 主なCSVカラム（例: 道具マスタ）

| カラム名 | 説明 |
|---------|------|
| 道具名 | 道具の名称 |
| カテゴリ | 道具のカテゴリ |
| 型番 | 型番（メーカーの製品番号） |
| メーカー | メーカー名 |
| 購入日 | 購入日（YYYY-MM-DD形式） |
| 購入価格 | 購入価格（円） |
| 在庫数 | 現在の在庫数 |
| 最小在庫数 | 最小在庫数（アラート基準） |
| 低在庫アラート | アラートのON/OFF |
| 登録日 | データ登録日（YYYY/MM/DD形式） |

**注意**: データ種別によってカラム構成は異なります。

## 注意事項

### エクスポート対象データ

✅ **含まれるデータ**:
- 有効なデータのみ
- 自組織のデータのみ

❌ **含まれないデータ**:
- 削除済みデータ（論理削除されたデータ）
- 他組織のデータ

### データ件数

- 各データの右側に現在の件数が表示されます
- **在庫移動履歴のみ直近1000件に制限**されています
- 件数が0件の場合、「CSVエクスポート」ボタンは無効化されます

### エクスポート処理時間

- 100件以下: 数秒
- 1000件程度: 10秒前後
- 5000件以上: 30秒〜1分程度

大量データの場合、処理に時間がかかることがあります。エクスポートボタンをクリック後、完了メッセージが表示されるまでお待ちください。

### セキュリティ

- エクスポート機能は**管理者のみ**がアクセス可能です
- エクスポートしたCSVファイルには機密情報が含まれる可能性があります
- ファイルの取り扱いには十分注意してください
- 不要になったCSVファイルは速やかに削除してください

### ブラウザの制限

- ブラウザのポップアップブロック設定によっては、ダウンロードがブロックされる場合があります
- ブロックされた場合は、ブラウザの設定でザイロクのポップアップを許可してください

## トラブルシューティング

### ダウンロードが開始されない

**原因**: ブラウザのポップアップブロック

**対処方法**:
1. ブラウザのアドレスバー右側にポップアップブロックのアイコンが表示されているか確認
2. アイコンをクリックして「常に許可」を選択
3. 再度「CSVエクスポート」ボタンをクリック

### Excelで開くと文字化けする

**原因**: Excelの文字コード設定

**対処方法（Windows）**:
1. CSVファイルを右クリック → **「プログラムから開く」** → **「メモ帳」**
2. メモ帳で **「ファイル」** → **「名前を付けて保存」**
3. エンコーディングを **「UTF-8」** に変更して上書き保存
4. Excelで開く

**対処方法（Mac）**:
1. Excelを開く
2. **「ファイル」** → **「インポート」**
3. **「CSVファイル」** を選択
4. 文字コードを **「UTF-8」** に設定してインポート

**注意**: 通常はダブルクリックで正しく開きますが、一部環境で文字化けする場合があります。

### エクスポートエラーが表示される

**原因1**: ネットワークエラー

**対処方法**:
- インターネット接続を確認
- 再度エクスポートを試す

**原因2**: データ量が多すぎる

**対処方法**:
- 在庫移動履歴以外のデータは通常問題なくエクスポートできます
- エラーが継続する場合は、システム管理者に連絡してください

### CSVファイルが見つからない

**原因**: ブラウザのダウンロードフォルダ設定

**対処方法**:
1. ブラウザの設定で「ダウンロード」フォルダの場所を確認
2. 通常は以下の場所に保存されます:
   - **Windows**: `C:\Users\{ユーザー名}\Downloads`
   - **Mac**: `/Users/{ユーザー名}/Downloads`
3. ブラウザのダウンロード履歴から確認できます

### 件数が0件で エクスポートできない

**原因**: データが登録されていない、または削除済み

**対処方法**:
- データが正しく登録されているか確認してください
- 削除済みデータはエクスポートされません
