# 見積書管理

## 概要

見積書機能は、顧客への見積書を作成・管理するための機能です。
承認フロー、顧客送付、顧客判断（承認/却下）まで、見積書のライフサイクル全体を管理できます。

**主な機能**:
- 見積書の作成・編集（リーダー以上）
- 承認ワークフロー（マネージャー以上が承認）
- 顧客への送付管理
- 顧客判断の記録（承認/却下）
- 承認済み見積書から請求書作成
- 操作履歴の記録
- PDF出力

---

## 対象者

### 作成・編集
- **リーダー（Leader）**: 自分が作成した見積書のみ作成・編集
- **マネージャー（Manager）**: 全ての見積書を作成・編集
- **管理者（Admin）**: 全ての見積書を作成・編集

### 承認
- **マネージャー（Manager）以上**: 提出済み見積書を承認・差し戻しできる

### 顧客送付
- **マネージャー（Manager）以上**: 承認済み見積書を顧客に送付できる

### 顧客判断
- **マネージャー（Manager）以上**: 送付済み見積書の顧客判断（承認/却下）を記録できる

**注意**: スタッフ（Staff）は見積書機能を利用できません。

---

## 前提条件

見積書を作成する前に、以下のマスタデータが登録されている必要があります。

1. **取引先マスタ**: 見積先の取引先が登録されている
   - 取引先タイプが「顧客」または「両方」である必要があります
2. **工事マスタ（任意）**: 見積書に紐づける工事を登録（省略可能）

---

## 見積書のステータス

見積書は以下のステータスで管理されます：

| ステータス | 説明 |
|-----------|------|
| **下書き** | 作成中、編集可能 |
| **提出済み** | 承認待ち（リーダーが提出した場合） |
| **顧客送付済** | 顧客に送付済み、顧客判断待ち |
| **顧客承認** | 顧客が承認（請求書作成可能） |
| **顧客却下** | 顧客が却下（最終状態） |
| **期限切れ** | 有効期限切れ（編集・承認・送付不可） |

### ステータスフロー

```
下書き → 提出済み → 顧客送付済 → 顧客承認 → 請求書作成
                                  ↓
                               顧客却下
```

**注意**:
- マネージャー以上が作成した場合、「提出済み」をスキップして即座に承認されます
- 期限切れになった見積書は、編集・承認・送付ができません

## 見積書の作成

### 基本的な作成手順

1. サイドメニューから **見積書** をクリック
2. 右下の **+** ボタン（FAB）をクリック
3. 基本情報と明細を入力
4. **下書き** または **提出** をクリック

### 入力項目の詳細

#### 基本情報

| 項目 | 必須 | 説明 |
|------|------|------|
| **見積番号** | ○ | 自動生成（EST-年-連番） |
| **取引先** | ○ | 見積先の取引先を選択 |
| **工事** | - | 紐づける工事を選択（任意） |
| **見積日** | ○ | 見積書の発行日 |
| **有効期限** | - | 見積書の有効期限（デフォルト30日後） |
| **件名** | ○ | 見積書の件名 |

**見積番号の自動生成**:
- フォーマット: `EST-2025-0001`
- 年ごとに連番がリセットされます

**件名の自動入力**:
- 工事を選択すると、件名に「〇〇工事 見積書」が自動入力されます

#### 明細

見積書には1つ以上の明細を追加する必要があります。

| 項目 | 必須 | 説明 |
|------|------|------|
| **種別** | ○ | 工事費/材料費/諸経費/その他 |
| **カスタム種別** | - | 種別が「その他」の場合に入力 |
| **項目名** | ○ | 明細の名称 |
| **説明** | - | 明細の詳細説明 |
| **数量** | ○ | 数量 |
| **単位** | ○ | 式/個/台/本/枚/m/m²/m³/kg/t/L/日/時間/人/その他 |
| **カスタム単位** | - | 単位が「その他」の場合に入力 |
| **単価** | ○ | 単価（税抜） |
| **税率** | ○ | 10%/8%/0% |
| **金額** | - | 自動計算（数量×単価） |

**明細の追加・削除**:
- **+ 明細を追加** ボタンで明細を追加
- 各明細の **削除** ボタンで削除（最低1つは必要）

**数値入力**:
- 全角数字を入力しても自動的に半角に変換されます

#### 備考

| 項目 | 説明 |
|------|------|
| **備考（お客様向け）** | 見積書に記載される備考 |
| **社内メモ** | 社内用のメモ（見積書には記載されません） |

## 下書き保存と提出

### 下書き保存

作成途中の見積書を保存する場合は **下書き** ボタンをクリックします。

**下書き状態の特徴**:
- いつでも編集・削除が可能
- 承認者には表示されない
- 一覧画面でステータス「下書き」として表示

### 提出（リーダーの場合）

リーダーが見積書を完成させて提出する場合は **提出** ボタンをクリックします。

**提出時の動作**:
- ステータスが「提出済み」に変更
- マネージャー以上に承認依頼の通知が届く
- 提出後は編集できません（差し戻された場合のみ再編集可能）

### 提出（マネージャー以上の場合）

マネージャー以上が見積書を作成する場合、**提出** ボタンは表示されず、即座に承認されます。

**提出時の動作**:
- ステータスが「顧客送付済」に変更（承認プロセスをスキップ）
- 顧客送付ボタンが表示される

## 見積書の閲覧

### 一覧画面

見積書一覧では、以下の情報がカード形式で表示されます：

- ステータスバッジ
- 見積番号
- 見積日・有効期限
- 取引先名・工事名
- 作成者（リーダーの場合）
- 承認者（マネージャー以上のみ表示）
- 金額

**フィルタリング**:
- ステータスで絞り込み
- キーワード検索（見積番号、取引先名、工事名）

**ソート**:
- 見積日
- 金額

### 詳細画面

見積書をクリックすると、詳細画面が開きます。

**表示内容**:
- 見積書本体（印刷可能なフォーマット）
- 宛先情報（取引先名、住所、担当者）
- 発行者情報（組織名、住所、電話番号、登録番号）
- 件名
- 明細一覧
- 合計金額（小計、消費税、合計）
- 備考
- 社内メモ（印刷時は非表示）
- 承認情報（承認済みの場合）
- 操作履歴

## 承認フロー

### リーダーが作成した場合

1. リーダーが見積書を作成
2. **提出** → ステータス「提出済み」
3. マネージャー以上が承認
4. ステータス「顧客送付済」に変更（顧客送付ボタンが表示される）

### マネージャー以上が作成した場合

1. マネージャー以上が見積書を作成
2. **提出** → ステータス「顧客送付済」（即座に承認）
3. 顧客送付ボタンが表示される

### 承認手順（マネージャー以上）

1. 見積書一覧から「提出済み」の見積書を開く
2. 内容を確認
3. 画面右上の **承認** ボタンをクリック
4. 必要に応じてメモを入力
5. **確定** をクリック

**承認後の動作**:
- ステータスが「顧客送付済」に変更
- 作成者に承認完了の通知が届く
- 顧客送付ボタンが表示される

### 差し戻し手順（マネージャー以上）

承認せずに修正が必要な場合は、差し戻しができます。

1. 見積書一覧から「提出済み」の見積書を開く
2. 内容を確認
3. 画面右上の **差し戻し** ボタンをクリック
4. 差し戻し理由を入力（必須）
5. **確定** をクリック

**差し戻し後の動作**:
- ステータスが「下書き」に戻る
- 作成者に差し戻しの通知が届く
- 作成者は見積書を再編集して再提出できる

---

## 顧客送付

承認済みの見積書は、顧客に送付できます。

### 顧客送付手順（マネージャー以上）

1. 見積書詳細画面を開く
2. 画面右上の **顧客送付** ボタンをクリック
3. 送付先メールアドレスを確認（取引先の連絡先メールアドレスが自動入力）
4. 必要に応じて送付メッセージを入力
5. **送付** をクリック

**送付時の動作**:
- 顧客にメールが送信される（PDF添付）
- ステータスが「顧客送付済」に変更
- 操作履歴に送付記録が追加される
- 顧客判断ボタン（承認/却下）が表示される

---

## 顧客判断の記録

顧客送付済みの見積書は、顧客の判断（承認/却下）を記録できます。

### 顧客承認の記録（マネージャー以上）

1. 見積書詳細画面を開く
2. 画面右上の **顧客承認** ボタンをクリック
3. 必要に応じてメモを入力
4. **確定** をクリック

**顧客承認後の動作**:
- ステータスが「顧客承認」に変更
- 操作履歴に顧客承認記録が追加される
- **請求書作成** ボタンが表示される

### 顧客却下の記録（マネージャー以上）

1. 見積書詳細画面を開く
2. 画面右上の **顧客却下** ボタンをクリック
3. 却下理由を入力（必須）
4. **確定** をクリック

**顧客却下後の動作**:
- ステータスが「顧客却下」に変更
- 操作履歴に顧客却下記録が追加される
- これ以上の操作はできません（最終状態）

---

## 請求書作成

顧客承認済みの見積書から、請求書を作成できます。

### 請求書作成手順

1. 見積書詳細画面を開く（ステータス「顧客承認」）
2. 画面右上の **請求書作成** ボタンをクリック
3. 請求書作成画面が開く
4. 見積書の内容が自動的に引き継がれる
5. 必要に応じて内容を編集
6. 請求書を保存

**関連**: [請求書管理マニュアル](./invoices.md)

---

## PDF出力

承認済みの見積書は、PDF形式でダウンロードできます。

### PDF出力手順

1. 見積書詳細画面を開く
2. 画面右上の **PDF** ボタンをクリック
3. PDFファイルがダウンロードされる

**PDF出力内容**:
- 見積書本体（印刷可能なフォーマット）
- 組織情報（会社名、住所、電話番号、登録番号）
- 取引先情報
- 見積番号、見積日、有効期限
- 件名
- 明細一覧
- 合計金額
- 備考

**注意**:
- PDF出力はマネージャー以上のみ可能です
- 社内メモはPDFに含まれません

## 見積書の編集・削除

### 編集

下書き状態の見積書のみ編集できます。

**編集手順**:
1. 見積書詳細画面を開く（ステータス「下書き」）
2. 画面右上の **編集** ボタンをクリック
3. 編集画面で内容を修正
4. **下書き** または **提出** をクリック

**提出済み・送付済みの見積書は編集できません**。
修正が必要な場合は、マネージャーに差し戻しを依頼してください。

### 削除

未承認の見積書のみ削除できます。

**削除手順**:
1. 見積書詳細画面を開く
2. 画面右上の **削除** ボタンをクリック
3. 確認ダイアログで **削除** をクリック

**削除可能な見積書**:
- 下書き
- 提出済み（承認前）
- 期限切れ

**削除できない見積書**:
- 承認済み
- 顧客送付済
- 顧客承認
- 顧客却下

**注意**: 削除した見積書は復元できません。

## 有効期限と期限切れ

### 有効期限

見積書には有効期限を設定できます（デフォルト30日後）。

**有効期限の扱い**:
- 有効期限を過ぎた見積書は、自動的に「期限切れ」ステータスに変更されます
- 期限切れの見積書は、編集・承認・送付ができません
- 期限切れの見積書は削除できます

### 期限切れの見積書の検索

一覧画面のフィルタで「期限切れ」を選択すると、期限切れの見積書のみ表示されます。

## トラブルシューティング

### 提出ボタンが押せない場合

必須項目が未入力の可能性があります。以下を確認してください：

1. **取引先** が選択されているか
2. **見積日** が入力されているか
3. **件名** が入力されているか
4. **明細** が1つ以上追加されているか
5. 各明細の **項目名**、**数量**、**単位**、**単価** が入力されているか

フォーム内に赤字でエラーメッセージが表示されている項目がないか確認してください。

### 承認ボタンが表示されない場合

以下を確認してください：

1. **権限**: マネージャー以上の権限があるか
2. **ステータス**: 見積書が「提出済み」であるか
3. **既に承認済み**: 既に承認済みの場合、承認ボタンは表示されません

### 顧客送付ボタンが表示されない場合

以下を確認してください：

1. **権限**: マネージャー以上の権限があるか
2. **ステータス**: 見積書が承認済み（顧客送付済）であるか
3. **既に送付済み**: 既に送付済みの場合、ボタンは表示されません

### 期限切れになった見積書を再利用したい場合

期限切れの見積書は編集できませんが、内容をコピーして新しい見積書を作成することができます。

**手順**:
1. 期限切れの見積書を開く
2. 内容をメモ（または画面をスクリーンショット）
3. 新しい見積書を作成
4. 内容を手動で入力
