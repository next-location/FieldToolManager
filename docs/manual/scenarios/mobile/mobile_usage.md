## 概要

ザイロクはスマートフォンに完全対応しています。アプリのインストールは不要で、ブラウザからアクセスするだけですべての機能を使えます。

### 特徴

- ✅ **レスポンシブデザイン**: 画面サイズに自動適応
- ✅ **タッチ操作最適化**: スワイプ、タップに対応
- ✅ **下部固定ナビ**: よく使う機能に素早くアクセス
- ✅ **軽量設計**: データ通信量を抑えた設計
- ✅ **オフライン対応**: 一部機能はオフラインでも使用可能（今後実装予定）

### 推奨環境

**スマートフォン**:
- iPhone（iOS 14以降）
- Android（Android 10以降）

**画面サイズ**:
- 最小: 320px（iPhone SE など）
- 推奨: 375px以上（iPhone 12/13/14 など）

## スマートフォンで使える機能

### 📦 現場資産パック

現場資産パックをご契約の場合、以下の機能がスマートフォンで使えます：

**主要機能**:
- ✅ ダッシュボード（在庫状況の確認）
- ✅ 道具一覧（検索・フィルター）
- ✅ 消耗品一覧（在庫確認）
- ✅ QRスキャン（道具の持ち出し・返却）
- ✅ 道具の移動登録（手動）
- ✅ 倉庫位置管理
- ✅ 現場管理
- ✅ 道具カテゴリ管理（管理者のみ）

### 📋 DX効率化パック

DX効率化パックをご契約の場合、以下の機能がスマートフォンで使えます：

**主要機能**:
- ✅ ダッシュボード
- ✅ 作業報告書作成（写真撮影・アップロード）
- ✅ 作業報告書一覧（閲覧・承認）
- ✅ 出退勤打刻（QRスキャン・手動）
- ✅ 出退勤履歴確認
- ✅ 見積書作成・閲覧
- ✅ 請求書作成・閲覧
- ✅ 発注書作成・閲覧
- ✅ 取引先管理

### 🎯 フル機能統合パック

フル機能統合パックをご契約の場合、上記すべての機能がスマートフォンで使えます。

## モバイル画面の構成

スマートフォンでザイロクにアクセスすると、モバイル専用レイアウトが表示されます。

### 画面の基本構成

```
┌─────────────────────────┐
│ ヘッダー（組織名・通知） │
├─────────────────────────┤
│                         │
│                         │
│    メインコンテンツ      │
│                         │
│                         │
│                         │
├─────────────────────────┤
│  下部ナビゲーション      │
│ [ホーム][機能][QR][メニュー]│
└─────────────────────────┘
```

### ヘッダー

**表示内容**:
- 組織名（左上）
- 通知ベルアイコン（右上）
- ユーザー情報（タップで表示）

**機能**:
- 通知ベルをタップ → 通知一覧を表示
- ヘッダー右端のメニューボタン → ユーザーメニュー表示

### メインコンテンツエリア

各ページの内容が表示されます。スクロールして閲覧できます。

**特徴**:
- タッチスクロール対応
- プルトゥリフレッシュ（一部ページ）
- タップでリンクや詳細を開く

## 下部ナビゲーション

スマートフォンでは、画面下部に固定ナビゲーションバーが表示されます。よく使う機能に素早くアクセスできます。

### 現場資産パック

```
┌──────┬──────┬────────┬──────┬──────┐
│ ホーム│ 道具 │   QR   │消耗品│メニュー│
└──────┴──────┴────────┴──────┴──────┘
```

**各ボタンの機能**:

| ボタン | 遷移先 | 説明 |
|--------|--------|------|
| **ホーム** | ダッシュボード | 在庫状況、最近の移動履歴を表示 |
| **道具** | 道具一覧 | 登録されている道具を一覧表示 |
| **QR** | QRスキャン | 道具のQRコードをスキャン（中央の大きなボタン） |
| **消耗品** | 消耗品一覧 | 消耗品の在庫を確認 |
| **メニュー** | サイドメニュー | すべての機能にアクセス |

### DX効率化パック

```
┌──────┬──────┬────────┬──────┬──────┐
│ ホーム│ 作報 │ 出退勤 │ 一覧 │メニュー│
└──────┴──────┴────────┴──────┴──────┘
```

**各ボタンの機能**:

| ボタン | 遷移先 | 説明 |
|--------|--------|------|
| **ホーム** | ダッシュボード | 作業報告書の承認状況などを表示 |
| **作報** | 作業報告書作成 | 新しい作業報告書を作成 |
| **出退勤** | 出退勤打刻 | QRスキャンまたは手動で打刻（中央の大きなボタン） |
| **一覧** | 作業報告書一覧 | 作業報告書を閲覧・検索 |
| **メニュー** | サイドメニュー | すべての機能にアクセス |

### フル機能統合パック

```
┌──────┬──────┬────────┬──────┬──────┐
│ ホーム│出退勤│   QR   │ 作報 │メニュー│
└──────┴──────┴────────┴──────┴──────┘
```

**各ボタンの機能**:

| ボタン | 遷移先 | 説明 |
|--------|--------|------|
| **ホーム** | ダッシュボード | 全体の状況を表示 |
| **出退勤** | 出退勤打刻 | QRスキャンまたは手動で打刻 |
| **QR** | QRスキャン | 道具のQRコードをスキャン（中央の大きなボタン） |
| **作報** | 作業報告書作成 | 新しい作業報告書を作成 |
| **メニュー** | サイドメニュー | すべての機能にアクセス |

### 中央のクイックアクションボタン

**特徴**:
- 大きな青い円形ボタン
- 画面下部から少し上に浮いたデザイン
- タップしやすい大きさ（48px × 48px）

**機能**:
- 現場資産パック: QRスキャン
- DX効率化パック: 出退勤打刻
- フル機能統合パック: QRスキャン

## サイドメニュー

下部ナビゲーションの「メニュー」ボタンをタップすると、サイドメニューが表示されます。

### サイドメニューの開き方

1. 下部ナビゲーションの **「メニュー」** ボタンをタップ
2. 画面左から右にスライドしてサイドメニューが表示される
3. 背景が暗くなり、メニュー以外をタップすると閉じる

### メニュー構成

**共通メニュー**:
- ダッシュボード
- QRスキャン

**現場資産パック**:
- 道具管理
  - 道具一覧
  - 消耗品一覧
  - 道具セット
  - 道具移動履歴
  - 一括移動
- 倉庫・現場
  - 倉庫位置
  - 現場
- マスター管理（管理者のみ）
  - 道具カテゴリ
  - メーカー

**DX効率化パック**:
- 作業報告書
  - 作業報告書一覧
  - 新規作成
  - 承認待ち（管理者のみ）
- 出退勤管理
  - 出退勤打刻
  - 出退勤履歴
  - 出退勤QR発行（リーダー以上）
  - 出退勤設定（管理者のみ）
- 営業管理
  - 見積書
  - 請求書
  - 発注書
  - 取引先
  - 現場

**設定**:
- アカウント設定
- 運用設定（管理者のみ）
- スタッフ管理（管理者のみ）

**その他**:
- ログアウト

### メニューの使い方

**開く方法**:
1. 下部ナビゲーションの「メニュー」ボタンをタップ
2. または、画面左端から右にスワイプ（一部ブラウザ）

**閉じる方法**:
1. メニュー以外の暗い部分をタップ
2. または、もう一度「メニュー」ボタンをタップ

**アコーディオンメニュー**:
- 「道具管理」「作業報告書」などをタップすると、サブメニューが開きます
- もう一度タップすると閉じます
- 1つずつ開く（排他的な動作）

## モバイルで効率的に使う

### 1. ホーム画面に追加する（ブックマーク）

スマートフォンのホーム画面にザイロクを追加すると、アプリのように素早く起動できます。

**iPhone（Safari）の場合**:
1. ザイロクにアクセス
2. 画面下部の **共有ボタン** をタップ（四角に上向き矢印）
3. **「ホーム画面に追加」** をタップ
4. 名前を編集（例: 「ザイロク」）
5. **「追加」** をタップ

**Android（Chrome）の場合**:
1. ザイロクにアクセス
2. 画面右上の **三点リーダー** をタップ
3. **「ホーム画面に追加」** をタップ
4. 名前を編集（例: 「ザイロク」）
5. **「追加」** をタップ

### 2. 横画面（ランドスケープ）での使用

**特徴**:
- 横画面にも対応
- より広い作業エリア
- 下部ナビゲーションはそのまま表示

**推奨**:
- 一覧画面の閲覧
- 作業報告書の作成
- PDFの確認

### 3. スワイプ操作

**対応している操作**:
- **上下スワイプ**: ページスクロール
- **左右スワイプ**: 一部の画面でタブ切り替え（QRスキャン画面など）
- **プルトゥリフレッシュ**: 一覧ページで下に引っ張って更新（一部ページ）

### 4. 通知を活用する

**通知の確認方法**:
1. ヘッダー右上の **ベルアイコン** をタップ
2. 未読の通知が表示される
3. 通知をタップして詳細を確認

**通知の種類**:
- 道具の低在庫アラート
- 作業報告書の承認依頼
- 見積書・請求書の承認依頼
- 出退勤の打刻忘れ（設定により）

### 5. 検索とフィルターを活用する

**道具一覧での検索**:
1. 道具一覧ページを開く
2. 画面上部の **検索バー** をタップ
3. 道具名、型番、シリアル番号を入力
4. リアルタイムで絞り込まれる

**フィルター機能**:
1. 画面右上の **フィルターアイコン** をタップ
2. カテゴリ、状態、保管場所で絞り込み
3. **「適用」** をタップ
4. 絞り込まれた結果が表示される

## トラブルシューティング

### 画面が正しく表示されない

**症状**: レイアウトが崩れている、ボタンが表示されない

**対処法**:
1. **ブラウザのキャッシュをクリア**
   - iPhone: 設定 > Safari > 履歴とWebサイトデータを消去
   - Android: Chrome > 設定 > プライバシーとセキュリティ > 閲覧履歴データの削除

2. **ページを再読み込み**
   - Safari: 画面を引っ張って更新
   - Chrome: 画面右上の三点リーダー > 更新

3. **ブラウザを最新版に更新**
   - App Store（iOS）または Google Play（Android）でブラウザを更新

### 下部ナビゲーションが表示されない

**症状**: 画面下部の固定ナビゲーションバーが表示されない

**原因と対処法**:

1. **パソコン表示になっている**
   - ブラウザの「デスクトップサイト」設定がオンになっている可能性があります
   - Safari: 設定 > Safari > デスクトップ用Webサイトを表示 > オフ
   - Chrome: 画面右上の三点リーダー > デスクトップサイト > オフ

2. **画面サイズが大きすぎる**
   - タブレット（iPad、大型Androidタブレット）では、画面サイズによりパソコン版レイアウトが表示される場合があります
   - 対処: ブラウザの設定で「モバイルサイト」を要求

### タップしても反応しない

**症状**: ボタンやリンクをタップしても反応しない

**対処法**:

1. **ページの読み込みを待つ**
   - ページが完全に読み込まれるまで待ちます
   - 画面上部のプログレスバーが消えたら操作可能です

2. **JavaScriptが有効か確認**
   - Safari: 設定 > Safari > 詳細 > JavaScript > オン
   - Chrome: 通常はオン（設定で確認可能）

3. **画面の別の場所をタップ**
   - タップした場所が正しいか確認
   - ボタンの中央をタップしてみてください

### QRスキャンが起動しない

**症状**: QRスキャンボタンをタップしてもカメラが起動しない

**対処法**:
- [QRスキャンマニュアル - トラブルシューティング](../../qr_scan.md#トラブルシューティング)を参照

### 写真がアップロードできない

**症状**: 作業報告書で写真を撮影・アップロードできない

**対処法**:

1. **カメラ権限を確認**
   - iPhone: 設定 > Safari > カメラ > 許可
   - Android: 設定 > アプリ > Chrome > 権限 > カメラ > 許可

2. **ストレージ容量を確認**
   - スマートフォンの空き容量が不足していないか確認
   - 不要なアプリやファイルを削除して空き容量を確保

3. **ファイルサイズを確認**
   - 1枚あたり10MB以下の写真を推奨
   - 高解像度の写真は圧縮してアップロード

### インターネット接続が不安定

**症状**: ページが読み込まれない、操作が反映されない

**対処法**:

1. **Wi-Fi接続を確認**
   - Wi-Fiに接続している場合、電波強度を確認
   - モバイルデータ通信に切り替えてみる

2. **機内モードのオンオフ**
   - 機内モードを一度オンにして、数秒後にオフにする
   - ネットワーク接続がリセットされます

3. **ルーターを再起動**
   - 社内Wi-Fiの場合、ルーターを再起動してみる

### ログインできない

**症状**: ログインボタンをタップしても反応しない、エラーが表示される

**対処法**:
- [ログイン・パスワード関連Q&A](../../../qa/staff/login_issues.md)を参照
