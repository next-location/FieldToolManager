## QRコードとは

QRコード（Quick Response Code）は、スマートフォンのカメラで読み取れる2次元バーコードです。ザイロクでは、道具や倉庫位置、現場の情報をQRコードで管理し、スマートフォンで素早くスキャンできます。

### ザイロクでのQRコードのメリット

- ✅ **高速入力**: 手入力より10倍以上速い
- ✅ **正確性**: 入力ミスがゼロ
- ✅ **誤操作防止**: 実物をスキャンするため、別の道具を誤って選択することがない
- ✅ **複数道具の一括処理**: 連続スキャンで効率アップ
- ✅ **スマホで完結**: 特別な機器不要

## ザイロクで使えるQRコード

### 1. 道具QRコード

**用途**: 道具の個別アイテムに貼り付けて使用

**スキャン時の動作**:
- 道具の詳細情報を表示
- 持ち出し・返却の移動登録
- 道具の現在地を確認

**対象**:
- 道具（電動工具、手動工具など）
- 消耗品（在庫管理が必要な場合）
- 重機（ショベルカー、フォークリフトなど）

**印刷方法**:
- 個別印刷: 道具詳細ページから
- 一括印刷: 「道具QR一括印刷」ページから

### 2. 倉庫位置QRコード

**用途**: 倉庫の棚や保管場所に貼り付けて使用

**スキャン時の動作**:
- 倉庫位置の詳細情報を表示
- その倉庫にある道具の一覧を表示

**対象**:
- 倉庫の棚
- 保管エリア
- 保管箱

**印刷方法**:
- 個別印刷: 倉庫位置詳細ページから

### 3. 現場QRコード

**用途**: 現場の入口や事務所に貼り付けて使用

**スキャン時の動作**:
- 現場の詳細情報を表示
- 現場にある道具の一覧を表示

**対象**:
- 建設現場
- 工事現場
- 作業拠点

**印刷方法**:
- 個別印刷: 現場詳細ページから

### 4. 出退勤QRコード（DX効率化パック・フル機能統合パック）

**用途**: 出退勤打刻用

**スキャン時の動作**:
- 出勤・退勤を打刻

**種類**:
- 会社QRコード: 会社での出退勤に使用
- 現場QRコード: 現場での出退勤に使用

**印刷方法**:
- [リーダー用QR発行マニュアル](/manual/attendance_leader_qr)を参照

## QRコード印刷の準備

### 必要な機材

**プリンター**:
- A4サイズ対応の普通紙プリンター
- モノクロ印刷でOK
- 推奨: レーザープリンター（インクジェットも可）

**用紙**:
- A4普通紙
- または、A4ラベルシール（一括印刷時に便利）

**その他**:
- ラミネーター（推奨）: QRコードの保護
- ラミネートフィルム（A4サイズ）
- カッター、ハサミ
- 両面テープ、または接着剤

### QRコードサイズの設定

ザイロクでは、組織設定でQRコードの印刷サイズを設定できます。

**設定方法**:
1. サイドメニュー → **「運用設定」** → **「組織情報」**
2. 「QRコード設定」セクションを探す
3. **「QRコード印刷サイズ」** を選択
4. **「設定を保存」** をクリック

**サイズの選択肢**:

| サイズ | 推奨用途 | 読み取りやすさ |
|--------|----------|----------------|
| **1.2cm × 1.2cm** | 極小の道具（ペン、ドライバーなど） | △ |
| **2cm × 2cm** | 小型道具（ペンチ、スパナなど） | ○ |
| **2.5cm × 2.5cm（推奨）** | 一般的な道具（電動ドリル、グラインダーなど） | ◎ |
| **3cm × 3cm** | 大型道具（発電機、コンプレッサーなど） | ◎ |
| **5cm × 5cm** | 重機、大型機器（ショベルカー、フォークリフトなど） | ◎ |

**推奨設定**:
- 一般的な組織: **2.5cm × 2.5cm**
- 小型道具が多い: **2cm × 2cm**
- 重機管理が中心: **3cm × 3cm** または **5cm × 5cm**

## 道具QRコードの印刷

### 個別印刷（1つの道具のQRコードを印刷）

**手順**:
1. サイドメニュー → **「道具一覧」**
2. 印刷したい道具をクリック
3. 道具詳細ページで **「QRコード」** タブをクリック
4. **「印刷」** ボタンをクリック
5. 印刷プレビュー画面が開く
6. プリンターを選択して **「印刷」** をクリック

**印刷内容**:
```
━━━━━━━━━━━━━━━━━
道具名: 電動ドリル
型番: XYZ-123
管理コード: D-0001

[QRコード画像]

━━━━━━━━━━━━━━━━━
```

### 一括印刷（複数の道具のQRコードを一度に印刷）

**手順**:

1. サイドメニュー → **「道具管理」** → **「道具QR一括印刷」**
2. 印刷したい道具を選択
   - **検索**: 道具名、管理コード、型番で検索
   - **フィルター**: カテゴリで絞り込み
   - **全選択**: すべての道具を選択
3. 選択した道具のチェックボックスをオン
4. 画面下部の **「印刷プレビュー」** ボタンをクリック
5. プレビュー画面で配置を確認
6. **「印刷」** ボタンをクリック

**一括印刷のレイアウト**:

- QRコードサイズに応じて自動レイアウト
- 1枚のA4用紙に複数のQRコードを配置
- 余白を考慮した最適な配置

**レイアウト例**:

| QRコードサイズ | 1ページあたりの個数 | 行×列 |
|----------------|---------------------|-------|
| 1.2cm | 32個 | 8行 × 4列 |
| 2cm | 20個 | 5行 × 4列 |
| 2.5cm（推奨） | 12個 | 4行 × 3列 |
| 3cm | 12個 | 4行 × 3列 |
| 5cm | 6個 | 3行 × 2列 |

**一括印刷のメリット**:
- 時間短縮: 複数の道具を一度に印刷
- 用紙節約: A4用紙1枚に複数のQRコードを配置
- ラベルシール対応: A4ラベルシールに印刷して貼るだけ

**印刷内容**:
```
━━━━━━━━━━━━━━━━━
倉庫位置
```

**推奨サイズ**:
- 倉庫位置QRコードは **5cm × 5cm** 以上を推奨

## QRコードの貼り付けと管理

### 道具へのQRコードの貼り付け

**準備**:
1. QRコードを印刷
2. 切り抜く（カッターまたはハサミ）
3. ラミネート加工（推奨）
4. 両面テープまたは接着剤を準備

**貼り付け手順**:
1. 道具の平らな面を探す
2. 表面を拭いて清掃
3. 両面テープを貼り付け
4. QRコードを貼り付け
5. しっかり押さえて固定

**貼り付け場所の推奨**:
- ✅ 平らな面
- ✅ 目立つ場所
- ✅ 使用時に損傷しない場所
- ❌ 可動部分
- ❌ 熱くなる部分
- ❌ 汚れやすい部分

**耐久性を高める工夫**:
- **ラミネート加工**: 水や汚れから保護
- **クリアテープで補強**: QRコードの四隅を保護
- **定期的な点検**: 剥がれや汚れをチェック

### 倉庫位置へのQRコードの貼り付け

**貼り付け場所**:
- 棚の側面（目の高さ）
- 保管エリアの壁
- 保管箱の表面

**サイズ**:
- **5cm × 5cm** 以上を推奨
- 離れた場所からでも読み取りやすい

**設置例**:
```
┌─────────────────┐
│  倉庫A-1棚      │
│                 │
│  [QRコード]     │
│   5cm × 5cm    │
│                 │
└─────────────────┘
```

### 出退勤QRコードの設置

**設置場所**:
- 会社の入口（受付、玄関）
- 現場の入口（ゲート、事務所）
- タイムカード機の近く

**設置方法**:
- A4用紙に印刷してラミネート
- 壁に貼り付け、またはスタンドに設置
- タブレットに表示（再発行が簡単）

**注意事項**:
- ⚠️ 雨風に濡れない場所
- ⚠️ 直射日光が当たらない場所
- ⚠️ スマートフォンのカメラで撮影しやすい高さ（1.2m〜1.5m）

## QRコード活用のベストプラクティス

### 1. 道具登録時にQRコードを即印刷

**推奨フロー**:
1. 新しい道具を登録
2. 即座にQRコードを印刷
3. ラミネート加工
4. 道具に貼り付け
5. 棚卸時に確認

**メリット**:
- 登録漏れを防ぐ
- すぐに道具管理を開始できる
- QRコード紛失のリスクを減らす

### 2. 定期的なQRコードの点検

**点検項目**:
- [ ] QRコードが剥がれていないか
- [ ] QRコードが汚れていないか
- [ ] QRコードが破損していないか
- [ ] スキャンできるか実際に試す

**推奨頻度**:
- 月次棚卸時に点検
- または、3ヶ月ごとに点検

### 3. 予備QRコードの準備

**予備QRコードの管理**:
1. 道具QRコードを2枚ずつ印刷
2. 1枚は道具に貼り付け
3. もう1枚は予備として保管

**保管場所**:
- 事務所のファイルに保管
- 道具ごとに整理
- 紛失・破損時にすぐ使える

### 4. QRコードのカラー印刷は不要

**推奨**:
- **モノクロ印刷**で十分
- カラー印刷より読み取りやすい
- インク代を節約できる

**理由**:
- QRコードは白黒のコントラストで読み取る
- カラーだと読み取りにくくなる場合がある

### 5. QRコードの管理台帳を作成

**管理台帳の内容**:
- 道具名
- 管理コード
- QRコード番号
- 印刷日
- 貼り付け日
- 点検日

**メリット**:
- QRコードの発行履歴を管理
- 紛失時の再発行がスムーズ
- 棚卸時の確認が簡単

## トラブルシューティング

### QRコードが読み取れない

**原因と対処法**:

#### 1. QRコードが汚れている
- **対処**: QRコードを拭いて汚れを取り除く
- **予防**: ラミネート加工で保護

#### 2. QRコードが小さすぎる
- **対処**: 大きいサイズで再印刷（2.5cm以上を推奨）
- **設定**: 組織設定でQRコードサイズを変更

#### 3. QRコードが破損している
- **対処**: 新しいQRコードを再印刷して貼り替え
- **予防**: ラミネート加工、定期的な点検

#### 4. 照明が暗い
- **対処**: 明るい場所で読み取る
- **対処**: スマートフォンのライトを使用

#### 5. カメラのピントが合っていない
- **対処**: QRコードから20-30cm程度離す
- **対処**: ゆっくりスキャン

### 印刷がうまくいかない

**症状**: QRコードが小さく印刷される、または切れてしまう

**対処法**:

1. **印刷プレビューを確認**
   - ページサイズ: A4
   - 余白: 6mm（標準）
   - 拡大縮小: なし（100%）

2. **ブラウザの印刷設定**
   - 「背景のグラフィック」をオン
   - 「ヘッダーとフッター」をオフ
   - 向き: 縦

3. **PDFに一度保存してから印刷**
   - 印刷画面で「PDFに保存」を選択
   - 保存したPDFを開いて印刷

### QRコードが剥がれやすい

**原因**: 接着力不足、表面が汚れている

**対処法**:

1. **表面を清掃**
   - アルコールで拭いて油分を除去
   - 完全に乾かしてから貼り付け

2. **強力な両面テープを使用**
   - 屋外用の両面テープ
   - 厚手の両面テープ

3. **ラミネート後に貼り付け**
   - ラミネート加工でQRコードを保護
   - 両面テープで貼り付け

### ラミネートフィルムが曇ってQRコードが読み取れない

**原因**: ラミネート時の温度や圧力が不適切

**対処法**:

1. **ラミネーターの温度を調整**
   - 推奨温度: 120-140℃
   - 低すぎると密着不良、高すぎると曇る

2. **ゆっくりラミネート**
   - 速度を遅くして均一に加熱

3. **ラミネート後に再スキャン**
   - ラミネート後にQRコードが読み取れるか確認
   - 読み取れない場合は再度印刷

### QRコードを間違えて印刷してしまった

**症状**: 別の道具のQRコードを印刷してしまった

**対処法**:

1. **正しいQRコードを再印刷**
   - 道具詳細ページから正しいQRコードを印刷

2. **間違ったQRコードは廃棄**
   - 誤って使用しないように、すぐに廃棄

3. **貼り付け前に確認**
   - QRコードをスキャンして、正しい道具かを確認してから貼り付け
