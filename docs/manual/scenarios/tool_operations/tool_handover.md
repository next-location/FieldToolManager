# 道具の引き継ぎ手順マニュアル

## 1. 概要

現場や担当者の変更時に、道具を確実に引き継ぐための手順を説明します。道具の紛失や所在不明を防ぎ、スムーズな業務引き継ぎを実現します。

**このマニュアルで解決できること**:
- 退職者から後任者への道具引き継ぎ
- 現場A→現場Bへの道具移動
- 担当者変更時の道具の所在確認
- チーム間での道具の貸し借り

---

## 2. 引き継ぎが必要なケース

### 2-1. 人の異動・退職

| ケース | 例 | 対応 |
|--------|-----|------|
| **退職** | スタッフAが退職 | 保有道具をすべて倉庫に返却 |
| **部署異動** | 配管工事→電気工事に異動 | 配管工具を返却し、電気工具を貸し出し |
| **現場変更** | 現場A→現場Bに担当変更 | 現場Aの道具を返却し、現場B用の道具を持ち出し |

### 2-2. 現場の完了・閉鎖

| ケース | 例 | 対応 |
|--------|-----|------|
| **工事完了** | 現場Aの工事が完了 | すべての道具を倉庫に返却 |
| **一時中断** | 冬季工事中断 | 道具を倉庫に戻し、春に再開時に持ち出し |

### 2-3. 道具セットの引き継ぎ

| ケース | 例 | 対応 |
|--------|-----|------|
| **セット単位での貸し出し** | 「配管工事セット」を別チームに貸す | 道具セット機能で一括移動 |
| **特定業務の引き継ぎ** | 点検業務を後任に引き継ぎ | 点検用道具セットを一括移動 |

---

## 3. 引き継ぎ前の確認事項

### 3-1. 引き継ぐ道具のリストアップ

**方法1: 個人保有道具を確認**

1. 左サイドメニュー **「道具」** をクリック
2. **「フィルタ」** ボタンをクリック
3. **「場所」** で「現場」を選択
4. **「現場を選択」** で対象現場を選択
5. 該当する道具がリスト表示されます

**方法2: 道具セットで確認**

1. 左サイドメニュー **「道具セット」** をクリック
2. 対象のセットをクリック
3. セット内の道具一覧が表示されます

### 3-2. 道具の状態チェック

**確認項目**:
- ✅ 破損・故障はないか
- ✅ 紛失している道具はないか
- ✅ 清掃・メンテナンスは完了しているか
- ✅ QRコードは読み取り可能か

**破損・紛失があった場合**: 先に [道具の紛失・破損対応マニュアル](./tool_lost_or_damaged.md) を参照してください。

## 4. 引き継ぎ手順

### パターンA: 個別道具の引き継ぎ（1個ずつ移動）

**状況**: 退職者が保有する道具を倉庫に返却する場合

#### 手順1: 返却する道具を確認

1. 左サイドメニュー **「道具」** をクリック
2. フィルタで返却対象の道具を絞り込み
3. 返却する道具をメモまたはスクリーンショット

#### 手順2: 個別移動を実行

1. 返却する道具をクリックして詳細画面を開く
2. **「移動履歴」** タブをクリック
3. **「移動登録」** ボタンをクリック
4. **移動先**: 「🏢 倉庫」を選択
5. **倉庫位置**: 返却先の棚番号を選択（例: A-01）
6. **備考**: 「退職に伴う返却（担当: 山田太郎）」と入力
7. **「登録する」** をクリック

#### 手順3: 返却完了の確認

- 道具一覧画面で「現在位置」が「倉庫」に変わったことを確認
- 道具詳細画面の「移動履歴」に返却記録が追加されたことを確認

---

### パターンB: 一括移動による引き継ぎ（複数個まとめて移動）

**状況**: 現場Aの工事完了に伴い、現場にある10個の道具を倉庫に返却する場合

#### 手順1: 一括移動ページを開く

1. 左サイドメニュー **「移動登録」** をクリック
2. **「一括移動」** タブをクリック

#### 手順2: 移動先を選択

1. **「1. 移動先を選択」** セクションで **「🏢 倉庫」** ボタンをクリック
2. 倉庫位置を選択（オプション）: 「A棟-1階-工具エリア」など

#### 手順3: 移動する道具を選択

**方法1: QRスキャンで選択**

1. **「📷 QRコードをスキャン」** ボタンをクリック
2. カメラが起動するので、返却する道具のQRコードを1つずつスキャン
3. スキャン成功すると「✓ 読み取り成功！」と表示されます
4. 10個すべてをスキャンします

**方法2: 検索して選択**

1. **「道具を検索して追加」** 欄に道具名またはシリアル番号を入力
2. 検索結果から対象の道具をクリック
3. 選択中の道具リストに追加されます

#### 手順4: 選択内容を確認

- **「3. 選択中の道具（10件）」** セクションに10個すべてが表示されることを確認
- 間違って選択した道具があれば **「削除」** ボタンで除外

#### 手順5: 一括移動を実行

1. **「移動を実行（10個）」** ボタンをクリック
2. 「移動中... 5 / 10」のように進捗が表示されます
3. 「10件の道具移動が完了しました」と表示されれば成功

#### 注意事項（一括移動）

⚠️ **同じ場所の道具のみ一括移動可能**

一括移動では、最初に選択した道具の現在位置（例: 現場A）と同じ場所にある道具しか選択できません。

```
✅ OK: 現場Aの道具10個を倉庫に一括移動
❌ NG: 現場Aの道具5個 + 現場Bの道具5個を一括移動
```

現場Aと現場Bの道具を返却する場合は、2回に分けて一括移動してください。

---

### パターンC: 道具セット単位での引き継ぎ

**状況**: 「配管工事セット」（10個の道具）を別の現場に移動する場合

#### 手順1: 移動登録ページを開く

1. 左サイドメニュー **「移動登録」** をクリック
2. **「一括移動」** タブをクリック

#### 手順2: 移動先を選択

1. **「1. 移動先を選択」** セクションで **「🏗️ 現場」** ボタンをクリック
2. **「現場」** ドロップダウンから「現場B」を選択

#### 手順3: 道具セットを選択

1. **「2. 移動する道具の選択方法」** セクションで **「道具セット」** タブをクリック
2. 道具セット一覧から **「配管工事セット」** をクリック
3. セット内の道具（10個）が自動的に選択されます

#### 手順4: 移動を実行

1. **「3. 選択中の道具（10件）」** セクションで内容を確認
2. **「移動を実行（10個）」** ボタンをクリック
3. 完了メッセージが表示されれば成功

#### 道具セット機能のメリット

- **選択ミス防止**: セットに登録された道具が自動選択されるため、取り忘れがない
- **業務標準化**: 「この作業にはこのセット」とテンプレート化できる
- **新人教育**: 新人でも必要な道具を漏れなく持ち出せる

**道具セットの作成方法**: [道具セット管理マニュアル](../../tool_sets.md) を参照

## 5. 引き継ぎ後の確認

### 5-1. 移動履歴の確認

1. 左サイドメニュー **「移動履歴」** をクリック
2. 最新の移動記録が表示されることを確認
3. 「実施者」「移動日時」「移動元」「移動先」が正しいか確認

### 5-2. 道具の現在位置を確認

1. 左サイドメニュー **「道具」** をクリック
2. 引き継いだ道具をクリック
3. **「現在位置」** が正しい場所に更新されているか確認

### 5-3. 後任者への連絡

**引き継ぎメモの例**:

```
【道具引き継ぎ完了】
日時: 2026年1月21日 15:30
引き継ぎ者: 山田太郎（退職）
受領者: 佐藤花子（後任）

【引き継いだ道具】
1. 電動ドリル #001 → 倉庫A-01に返却
2. パイプレンチ #012 → 倉庫A-01に返却
3. 水平器 #005 → 倉庫B-02に返却
（全3点）

【備考】
- 電動ドリルは昨日充電済み
- パイプレンチは若干のサビあり（使用に支障なし）
- 水平器は精密機器棚に保管

【システム記録】
ザイロクの「移動履歴」に記録済み
検索: 「2026-01-21 山田太郎」で確認可能
```

## 6. よくある引き継ぎシナリオ

### シナリオ1: 退職者の道具をすべて返却

**状況**: スタッフAが3月31日で退職。保有道具5個をすべて返却したい。

**手順**:
1. 最終出勤日の前日までに道具を清掃・点検
2. 破損があれば上司に報告
3. 最終出勤日に一括移動機能で5個を倉庫に返却
4. 移動履歴に「退職に伴う返却」と記入
5. 上司に完了報告

---

### シナリオ2: 現場完了時の道具回収

**状況**: 現場Aの工事が完了。現場にある道具15個を倉庫に戻したい。

**手順**:
1. 現場で道具を集める（忘れ物がないか確認）
2. 道具を清掃・点検
3. 倉庫に搬入
4. スマホでQRコードをスキャンしながら一括移動登録
5. 15個すべての返却を確認
6. 現場責任者に完了報告

---

### シナリオ3: チーム間の道具貸し借り

**状況**: チームA（現場A担当）が「配管工事セット」を1週間だけチームB（現場B担当）に貸したい。

**手順**:
1. チームAのリーダーがチームBのリーダーに貸出を連絡
2. 道具セット「配管工事セット」を現場A→現場Bに一括移動
3. 備考欄に「チームAからチームBへ貸出（返却予定: 1/28）」と記入
4. 1週間後、チームBが現場B→現場Aに返却移動
5. 備考欄に「チームBからチームAへ返却」と記入

---

### シナリオ4: 担当者変更時の引き継ぎ

**状況**: 山田さんから佐藤さんに現場Aの担当が変更。現場Aにある道具10個を引き継ぎたい。

**重要**: 道具の現在位置は「現場A」のまま変わりません（担当者情報は道具に紐づいていません）。

**手順**:
1. 山田さんと佐藤さんが一緒に現場Aで道具を確認
2. 道具一覧画面で現場Aの道具10個をリストアップ
3. 実物と照合し、すべて揃っていることを確認
4. 佐藤さんに道具の使い方、注意点を口頭で伝える
5. 引き継ぎ完了報告を上司にメール

**システム操作は不要**: 道具は現場Aにあり続けるため、移動登録は不要です。

## 7. トラブルシューティング

### Q1. 引き継ぐ道具が見つかりません

**A.** 以下を確認してください。

1. **検索してみる**: 道具一覧画面の検索ボックスで道具名・シリアル番号を入力
2. **フィルタを解除**: フィルタが適用されている可能性があります。「フィルタをクリア」をクリック
3. **削除されていないか確認**: 管理者に問い合わせ
4. **紛失の可能性**: [道具の紛失・破損対応マニュアル](./tool_lost_or_damaged.md) を参照

---

### Q2. 一括移動で「現在地が異なるため選択できません」とエラーが出ます

**A.** 一括移動では、最初に選択した道具と同じ場所にある道具しか選択できません。

**解決方法**:
- 現場Aの道具と現場Bの道具を一緒に移動したい場合は、2回に分けて移動してください
- 「すべてクリア」ボタンで選択をリセットし、同じ場所の道具のみ選択し直してください

---

### Q3. 道具セットで移動したいのに、「セット内に道具がありません」と表示されます

**A.** 道具セットに登録されている道具が削除されているか、既に他のセットに登録されている可能性があります。

**解決方法**:
1. 道具セット詳細画面を開く
2. 「セット内の道具」一覧を確認
3. 不足している道具があれば追加登録
4. 管理者権限が必要な場合は上司に相談

---

### Q4. QRコードをスキャンしても反応しません

**A.** 以下を確認してください。

1. **カメラ権限**: ブラウザのカメラ権限を許可してください
2. **QRコードが汚れている**: 清掃するか、再印刷してください
3. **照明が暗い**: 明るい場所でスキャンしてください
4. **手動入力**: QRコードの下に印刷されているUUIDを検索ボックスに入力して検索

**関連**: [QRコードスキャンのトラブルシューティング](../../../qa/staff/qr_scan_issues.md)

---

### Q5. 移動登録後、間違いに気づきました。キャンセルできますか？

**A.** 移動履歴の削除はできませんが、逆方向の移動を登録することで実質的に取り消せます。

**例**:
```
誤って登録: 倉庫 → 現場A
修正方法: 現場A → 倉庫（備考: 「誤登録のため取り消し」）
```

備考欄に「誤登録のため取り消し」と記入することで、後で履歴を確認したときに分かりやすくなります。
