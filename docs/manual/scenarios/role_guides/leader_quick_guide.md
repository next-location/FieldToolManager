**最終更新日**: 2026-01-22
**対象**: leader（リーダー）役割のユーザー

## はじめに

このガイドは、リーダーの方がザイロクを使い始めるための基本操作をまとめたクイックガイドです。

### リーダーができること

リーダーは、現場監督やチームリーダーです。スタッフができることに加えて、以下の操作ができます：

✅ **スタッフの機能に加えて**:
- 自チームの勤怠管理（DX効率化パック・フル機能統合パックの場合）
- 自チームの勤怠手動修正（DX効率化パック・フル機能統合パックの場合）
- 出退勤QRコードの発行（DX効率化パック・フル機能統合パックの場合）
- 現場マスタの閲覧
- 仕入先マスタの閲覧（DX効率化パック・フル機能統合パックの場合）
- 見積書の作成（DX効率化パック・フル機能統合パックの場合）
- 請求書の作成（DX効率化パック・フル機能統合パックの場合）
- 発注書の作成・承認（10万円未満）（DX効率化パック・フル機能統合パックの場合）

❌ **できないこと**:
- 道具の登録・編集・削除
- QRコードの印刷
- スタッフの追加・編集・削除
- システム設定
- 他チームの勤怠管理

## スタッフとの違い

リーダーは、スタッフの機能に加えて、以下の管理機能が使えます：

### 1. チーム管理

**自チームの勤怠管理**:
- 自分のチームメンバーの出退勤履歴を確認
- 打刻忘れの確認
- 勤怠の手動修正

### 2. 現場QR発行

**出退勤QRコードの発行**:
- 現場ごとのQRコードを発行
- 現場の入口に掲示して、スタッフが打刻できるようにする

### 3. 営業管理（DX効率化パック・フル機能統合パック）

**見積書・請求書・発注書の作成**:
- 取引先への見積書作成
- 請求書の作成
- 発注書の作成（10万円未満は自分で承認可能）

## 出退勤QRコードを発行する

**対象**: DX効率化パック・フル機能統合パックをご契約の組織のみ

### 現場QRコードの発行

1. サイドメニュー → **「出退勤管理」** → **「出退勤QR発行」**
2. 「現場QR発行」セクションに移動
3. **現場選択** ドロップダウンをクリック
4. 発行したい現場を選択
5. **「QRコード発行」** ボタンをクリック
6. 成功メッセージが表示される

### 発行したQRコードの確認

画面下部の「有効な現場QRコード」セクションに表示されます。

**表示内容**:
- 現場名
- 発行日時
- 有効期限

**操作ボタン**:
- **表示**: QRコードを画面に表示
- **保存**: PNG画像でダウンロード
- **印刷**: プリンターで印刷

### QRコードを印刷して設置

1. **「印刷」** ボタンをクリック
2. 印刷プレビュー画面が開く
3. プリンターを選択して印刷
4. A4用紙に印刷される
5. 切り抜いて、ラミネート加工（推奨）
6. 現場の入口に掲示

**推奨設置場所**:
- 現場の入口（ゲート）
- 現場事務所入口
- 朝礼場所

**詳細**: [リーダー用QR発行マニュアル](../../attendance_leader_qr.md)を参照

## 自チームの勤怠を確認する

**対象**: DX効率化パック・フル機能統合パックをご契約の組織のみ

### チームメンバーの出退勤履歴

1. サイドメニュー → **「出退勤管理」** → **「出退勤履歴」**
2. 自チームのメンバーの勤怠履歴が表示される

**表示内容**:
- スタッフ名
- 出勤日
- 出勤時刻
- 退勤時刻
- 勤務時間
- 打刻方法（QR/手動）

### フィルター機能

**期間で絞り込み**:
- 今日
- 今週
- 今月
- カスタム期間

**スタッフで絞り込み**:
- 特定のスタッフのみ表示

### 打刻忘れの確認

**アラート通知**:
- 打刻忘れがあると、通知ベルに表示される
- 該当スタッフに連絡して、打刻を促す

## 勤怠を手動修正する

**対象**: DX効率化パック・フル機能統合パックをご契約の組織のみ

### スタッフの打刻忘れを修正

1. サイドメニュー → **「出退勤管理」** → **「出退勤履歴」**
2. 修正したい勤怠記録をクリック
3. **「編集」** ボタンをクリック
4. 出勤時刻または退勤時刻を修正
5. 修正理由を入力（推奨）
6. **「保存」** をクリック

**注意**:
- 手動修正は、監査ログに記録されます
- 修正理由を入力することを推奨します

## 見積書を作成する

**対象**: DX効率化パック・フル機能統合パックをご契約の組織のみ

### 新規見積書の作成

1. サイドメニュー → **「営業管理」** → **「見積書」**
2. **「新規作成」** ボタンをクリック
3. 必要事項を入力
   - 取引先
   - 現場
   - 件名
   - 見積日
   - 有効期限
   - 明細（品目、数量、単価）
4. **「保存」** をクリック

### 見積書を提出（承認依頼）

1. 見積書詳細画面を開く
2. **「提出」** ボタンをクリック
3. 承認者（Manager/Admin）に承認依頼が送信される

### 見積書のステータス

- **下書き**: 作成中
- **提出済み**: 承認待ち
- **承認済み**: 承認完了、顧客に送信可能
- **差戻し**: 修正が必要

**詳細**: [見積書作成マニュアル](../../estimates.md)を参照

## 請求書を作成する

**対象**: DX効率化パック・フル機能統合パックをご契約の組織のみ

### 見積書から請求書を作成

1. サイドメニュー → **「営業管理」** → **「見積書」**
2. 承認済みの見積書をクリック
3. **「請求書に変換」** ボタンをクリック
4. 請求書が自動作成される

### 新規請求書の作成

1. サイドメニュー → **「営業管理」** → **「請求書」**
2. **「新規作成」** ボタンをクリック
3. 必要事項を入力
   - 取引先
   - 現場
   - 請求日
   - 支払期限
   - 明細（品目、数量、単価）
4. **「保存」** をクリック

### 請求書を提出（承認依頼）

1. 請求書詳細画面を開く
2. **「提出」** ボタンをクリック
3. 承認者（Manager/Admin）に承認依頼が送信される

**詳細**: [請求書作成マニュアル](../../invoices.md)を参照

## 発注書を作成・承認する

**対象**: DX効率化パック・フル機能統合パックをご契約の組織のみ

### 新規発注書の作成

1. サイドメニュー → **「営業管理」** → **「発注書」**
2. **「新規作成」** ボタンをクリック
3. 必要事項を入力
   - 仕入先
   - 現場
   - 発注日
   - 納期
   - 明細（品目、数量、単価）
4. **「保存」** をクリック

### 発注書を提出（承認依頼）

1. 発注書詳細画面を開く
2. **「提出」** ボタンをクリック
3. 金額により承認者が異なります

**承認フロー**:
- **10万円未満**: リーダーが自分で承認可能
- **10万円以上100万円未満**: Adminの承認が必要
- **100万円以上**: Adminの承認が必要

### 10万円未満の発注書を承認

1. サイドメニュー → **「営業管理」** → **「発注書」**
2. 提出済みの発注書（10万円未満）をクリック
3. **「承認」** ボタンをクリック
4. 承認完了

**詳細**: [発注書作成マニュアル](../../purchase_orders.md)を参照

## よくある質問

### Q1. 出退勤QRコードの有効期限が切れました

**A.** 再発行してください：
1. 出退勤QR発行画面を開く
2. 該当する現場を選択
3. **「QRコード再発行」** ボタンをクリック
4. 新しいQRコードを印刷・掲示

**注意**: 古いQRコードは無効になります。

### Q2. チームメンバーの勤怠が表示されません

**A.** 以下を確認してください：
- 自分のチームに所属しているか確認
- 期間フィルターを確認
- リーダー権限があるか確認

### Q3. 見積書の承認が遅い

**A.** 承認者（Manager/Admin）に直接連絡してください。

### Q4. 発注書を間違えて承認してしまいました

**A.** 管理者に連絡して、取り消しを依頼してください。

### Q5. 現場QRコードを紛失しました

**A.** 再発行してください。再発行すると、古いQRコードは無効になります。

### Q6. 勤怠の手動修正ができません

**A.** 以下を確認してください：
- 自チームのメンバーか確認
- リーダー権限があるか確認

それでも修正できない場合は、管理者に連絡してください。
