# マネージャー向けクイックガイド

**最終更新日**: 2026-01-22
**対象**: manager（マネージャー）役割のユーザー

---

## はじめに

このガイドは、マネージャーの方がザイロクを使い始めるための基本操作をまとめたクイックガイドです。

### マネージャーができること

マネージャーは、部門長や中間管理職です。リーダーができることに加えて、以下の操作ができます：

✅ **リーダーの機能に加えて**:
- 道具・消耗品・重機の登録・編集
- QRコードの印刷
- 全員の勤怠管理（DX効率化パック・フル機能統合パックの場合）
- 月次勤怠レポート（DX効率化パック・フル機能統合パックの場合）
- 現場の登録・編集
- スタッフの追加・編集（leader/staffのみ）
- 見積書・請求書の承認（DX効率化パック・フル機能統合パックの場合）
- マスタ管理（カテゴリ・道具・消耗品）の閲覧・編集（現場資産パックの場合）
- コスト分析・使用頻度分析

❌ **できないこと**:
- 道具・消耗品・重機の削除（adminのみ）
- スタッフの削除（adminのみ）
- admin役割のスタッフの編集
- 権限変更（adminのみ）
- システム設定（adminのみ）
- 倉庫位置管理（adminのみ）

## リーダーとの違い

マネージャーは、リーダーの機能に加えて、以下の管理機能が使えます：

### 1. 道具管理

**道具・消耗品・重機の登録・編集**:
- 新しい道具を登録
- 既存の道具情報を編集
- QRコードを印刷

### 2. 全社勤怠管理

**全員の勤怠管理**:
- 組織内全員の出退勤履歴を確認
- 全員の勤怠手動修正
- 月次勤怠レポートの確認

### 3. 承認業務

**見積書・請求書の承認**:
- リーダーが提出した見積書を承認
- リーダーが提出した請求書を承認

### 4. スタッフ管理

**leader/staff役割のスタッフ管理**:
- leader/staffの追加
- leader/staffの編集
- leader/staffの削除（無効化）

**注意**: admin役割のスタッフは編集できません。

## 道具を登録する

**対象**: 現場資産パック・フル機能統合パックをご契約の組織のみ

### 新規道具の登録

1. サイドメニュー → **「道具管理」** → **「道具一覧」**
2. **「新規登録」** ボタンをクリック
3. 必要事項を入力
   - 道具名
   - 型番
   - メーカー
   - カテゴリ
   - 管理方式（個別管理/消耗品）
   - 初期保管場所
4. **「登録」** をクリック

### 道具情報の編集

1. 道具一覧から道具をクリック
2. **「編集」** ボタンをクリック
3. 必要な項目を修正
4. **「保存」** をクリック

### 消耗品の登録

1. サイドメニュー → **「道具管理」** → **「消耗品一覧」**
2. **「新規登録」** ボタンをクリック
3. 必要事項を入力
   - 消耗品名
   - カテゴリ
   - 単位（個、箱、セットなど）
   - 最小在庫数
   - 初期在庫数
4. **「登録」** をクリック

**詳細**: [道具管理マニュアル](/manual/tool_management)を参照

## QRコードを印刷する

**対象**: 現場資産パック・フル機能統合パックをご契約の組織のみ

### 個別印刷（1つの道具）

1. 道具詳細ページを開く
2. **「QRコード」** タブをクリック
3. **「印刷」** ボタンをクリック
4. 印刷プレビュー画面が開く
5. プリンターを選択して印刷

### 一括印刷（複数の道具）

1. サイドメニュー → **「道具管理」** → **「道具QR一括印刷」**
2. 印刷したい道具を選択
   - 検索バーで絞り込み
   - カテゴリフィルターで絞り込み
   - チェックボックスで選択
3. **「印刷プレビュー」** ボタンをクリック
4. プレビュー画面で配置を確認
5. **「印刷」** ボタンをクリック

**レイアウト**:
- QRコードサイズに応じて自動レイアウト
- 1枚のA4用紙に複数のQRコードを配置

**詳細**: [QRコード活用ガイド](../mobile/qr_code_guide.md#道具qrコードの印刷)を参照

## 全員の勤怠を確認する

**対象**: DX効率化パック・フル機能統合パックをご契約の組織のみ

### 組織全員の出退勤履歴

1. サイドメニュー → **「出退勤管理」** → **「出退勤履歴」**
2. 組織内全員の勤怠履歴が表示される

**フィルター機能**:
- 期間で絞り込み（今日/今週/今月/カスタム）
- スタッフで絞り込み
- 部署で絞り込み

### 月次勤怠レポート

1. サイドメニュー → **「出退勤管理」** → **「月次レポート」**
2. 対象月を選択
3. レポートが表示される

**表示内容**:
- スタッフごとの勤務日数
- 総勤務時間
- 遅刻・早退の回数
- 欠勤日数

**エクスポート**:
- **「CSV出力」** ボタンで、Excelで編集可能な形式で出力

## 見積書・請求書を承認する

**対象**: DX効率化パック・フル機能統合パックをご契約の組織のみ

### 見積書の承認

1. サイドメニュー → **「営業管理」** → **「見積書」**
2. ステータスを **「提出済み」** で絞り込み
3. 承認したい見積書をクリック
4. 内容を確認
5. **「承認」** ボタンをクリック

**差戻し**:
- 修正が必要な場合は、**「差戻し」** ボタンをクリック
- 差戻し理由を入力

### 請求書の承認

1. サイドメニュー → **「営業管理」** → **「請求書」**
2. ステータスを **「提出済み」** で絞り込み
3. 承認したい請求書をクリック
4. 内容を確認
5. **「承認」** ボタンをクリック

**承認後**:
- 承認済みの見積書・請求書は、顧客に送信できます

## 現場を登録する

### 新規現場の登録

1. サイドメニュー → **「倉庫・現場」** → **「現場」**
2. **「新規登録」** ボタンをクリック
3. 必要事項を入力
   - 現場名
   - 住所
   - 取引先
   - 開始日
   - 終了予定日
   - 現場責任者
4. **「登録」** をクリック

### 現場情報の編集

1. 現場一覧から現場をクリック
2. **「編集」** ボタンをクリック
3. 必要な項目を修正
4. **「保存」** をクリック

### 現場の完了

1. 現場詳細ページを開く
2. **「完了」** ボタンをクリック
3. 確認ダイアログで **「はい」** をクリック

**注意**:
- 完了した現場に道具が残っている場合、警告が表示されます
- 道具を返却してから完了してください

**詳細**: [現場管理マニュアル](/manual/sites)を参照

## スタッフを追加する

### 新規スタッフの追加

1. サイドメニュー → **「設定」** → **「スタッフ管理」**
2. **「スタッフを追加」** ボタンをクリック
3. 必要事項を入力
   - 氏名
   - メールアドレス
   - 役割（**staff** または **leader** のみ選択可能）
   - 所属部署（オプション）
4. **「招待メールを送信」** をクリック

**注意**:
- マネージャーは、**staff** と **leader** 役割のスタッフのみ追加できます
- **manager** や **admin** 役割の追加は、adminのみ可能です

### スタッフ情報の編集

1. スタッフ一覧から対象スタッフをクリック
2. **「編集」** ボタンをクリック
3. 必要な項目を修正
4. **「保存」** をクリック

**編集可能な項目**:
- 氏名
- 所属部署
- 役割（staff/leaderのみ）

**編集できない項目**:
- メールアドレス
- パスワード
- admin役割のスタッフ

**詳細**: [スタッフ管理マニュアル](/manual/staff_management)を参照

## レポートを確認する

### コスト分析

1. サイドメニュー → **「分析」** → **「コスト分析」**
2. 期間を選択
3. レポートが表示される

**表示内容**:
- 現場ごとのコスト
- 道具ごとのコスト
- 月次推移

### 使用頻度分析

1. サイドメニュー → **「分析」** → **「使用頻度分析」**
2. 期間を選択
3. レポートが表示される

**表示内容**:
- 道具ごとの使用回数
- 使用率（利用可能日数に対する使用日数）
- 遊休資産の特定

## よくある質問

### Q1. 道具を削除したい

**A.** マネージャーは道具を削除できません。管理者（admin）に依頼してください。

### Q2. admin役割のスタッフを編集したい

**A.** マネージャーはadmin役割のスタッフを編集できません。他のadminに依頼してください。

### Q3. 見積書の承認権限がありません

**A.** 以下を確認してください：
- Manager権限があるか確認
- DX効率化パックまたはフル機能統合パックを契約しているか確認

### Q4. QRコードが印刷できません

**A.** 以下を確認してください：
- Manager権限があるか確認
- 現場資産パックまたはフル機能統合パックを契約しているか確認
- プリンターが接続されているか確認

### Q5. 勤怠レポートをExcelで編集したい

**A.** CSV出力機能を使用してください：
1. 月次レポート画面を開く
2. **「CSV出力」** ボタンをクリック
3. ダウンロードしたCSVファイルをExcelで開く

### Q6. スタッフを削除したい

**A.** 以下の手順で無効化できます：
1. スタッフ一覧から対象スタッフをクリック
2. **「無効化」** ボタンをクリック
3. 確認ダイアログで **「はい」** をクリック

**注意**: 完全削除はできません。無効化されたスタッフはログインできなくなります。
