# 発注から支払いまでの流れ

## 概要

このマニュアルでは、資材・消耗品を仕入先に発注し、納品を受けて、代金を支払うまでの一連の業務フローを説明します。

## 対象者

- リーダー以上: 発注書の作成・提出
- マネージャー以上: 承認操作、支払記録
- 管理者: すべての操作

## 業務フロー全体図

```
資材・消耗品の必要性確認
    ↓
【STEP 1】発注書作成
    ↓
【STEP 2】発注書提出・承認
    ↓
【STEP 3】発注書送付（仕入先へ）
    ↓
【STEP 4】納品確認
    ↓
【STEP 5】納品書登録（任意）
    ↓
【STEP 6】請求書受領（仕入先から）
    ↓
【STEP 7】支払記録
    ↓
完了
```

---

## STEP 1: 発注書作成

### 目的
仕入先に資材・消耗品を発注するための発注書を作成します。

### 操作手順

#### 1. 発注書作成画面を開く

**方法1: 新規作成**
1. サイドメニューから「発注書」をクリック
2. 「+ 新規発注書」ボタンをクリック

**方法2: 消耗品発注管理から作成**
1. サイドメニューから「消耗品」をクリック
2. 在庫が不足している消耗品を確認
3. 「発注管理」ボタンをクリック
4. 「発注書を作成」ボタンをクリック

#### 2. 基本情報を入力

| 項目 | 説明 | 入力例 |
|------|------|--------|
| **発注番号** | 自動生成（編集不可） | PO-202601-0001 |
| **仕入先** | 仕入先を選択（必須） | B資材株式会社 |
| **件名** | 発注内容の概要（必須） | 養生資材一式発注 |
| **発注日** | 発注書の発行日（必須） | 2026-01-15 |
| **納期** | 納品希望日（必須） | 2026-01-25 |
| **現場** | 紐付ける現場（任意） | 〇〇ビル現場 |

#### 3. 発注明細を入力

「+ 明細を追加」ボタンをクリックして、品目ごとに入力します。

**入力例**：
```
品目: ブルーシート 3.6m×5.4m
数量: 10
単位: 枚
単価: 1,200
金額: 12,000（自動計算）

品目: 養生テープ 50mm×25m
数量: 20
単位: 巻
単価: 180
金額: 3,600（自動計算）

品目: マスキングテープ 18mm×18m
数量: 30
単位: 巻
単価: 120
金額: 3,600（自動計算）
```

#### 4. 合計金額を確認

- 小計: ¥19,200
- 消費税（10%）: ¥1,920
- **合計金額: ¥21,120**

#### 5. 備考を入力（任意）

```
・納品先: 〇〇ビル現場（住所: 東京都〇〇区〇〇1-2-3）
・納品時間: 午前中希望
・運送費: 当社負担
```

#### 6. 保存

- 「下書き保存」: 提出せずに保存（後で編集可能）
- 「提出」: 承認待ちステータスに移行

---

## STEP 2: 発注書提出・承認

### 目的
作成した発注書を上長に承認してもらいます。

### 操作手順

#### 作成者（リーダー）の操作

1. 発注書詳細画面で「提出」ボタンをクリック
2. ステータスが「提出済み」に変更
3. 承認者（マネージャー・管理者）に通知が送信される

#### 承認者（マネージャー・管理者）の操作

1. 通知またはダッシュボードから「承認待ち発注書」を確認
2. 発注書詳細画面を開く
3. 内容を確認
   - 発注先が正しいか
   - 発注数量が適切か
   - 金額が妥当か
   - 納期が適切か
4. **承認する場合**: 「承認」ボタンをクリック
5. **却下する場合**: 「却下」ボタンをクリック → 却下理由を入力

### 承認後のステータス

- 承認: ステータスが「承認済み」に変更
- 却下: ステータスが「却下」に変更 → 作成者が修正して再提出

---

## STEP 3: 発注書送付（仕入先へ）

### 目的
承認済みの発注書を仕入先に送付します。

### 操作手順

#### 1. PDF出力

1. 発注書詳細画面を開く
2. 「PDFダウンロード」ボタンをクリック
3. PDFファイルがダウンロードされる

**PDFに含まれる内容**：
- 会社印（角印）※組織設定で登録している場合のみ
- 発注明細
- 合計金額
- 納期・納品先
- 備考

#### 2. 仕入先へ送付

**方法1: メール送付**
- PDFファイルを添付してメールで送付
- 件名例：「発注書送付のお知らせ（PO-202601-0001）」

**方法2: FAX送付**
- PDFを印刷してFAX送信

**方法3: 郵送**
- PDFを印刷して郵送

#### 3. 送付済みに変更

1. 発注書詳細画面で「仕入先に送付」ボタンをクリック
2. 確認ダイアログで「OK」をクリック
3. ステータスが「送付済み」に変更
4. 送付日が記録される

**注意**: このボタンを押してもメール送信等は行われません。実際に仕入先へ送付した後に、記録のために使用してください。

---

## STEP 4: 納品確認

### 目的
仕入先から納品された資材・消耗品を確認します。

### 操作手順

#### 1. 納品物の確認

納品時に以下を確認します：
- [ ] 納品書が添付されているか
- [ ] 発注した品目がすべて揃っているか
- [ ] 数量が発注数と一致しているか
- [ ] 品質に問題がないか（破損・汚損がないか）

#### 2. 受領確認の記録（任意）

**受領確認ボタンをクリック**
1. 発注書詳細画面を開く
2. 「受領確認」ボタンをクリック
3. 確認ダイアログで「OK」をクリック
4. ステータスが「受領済み」に変更

**または、支払記録に進む**
- 受領確認をスキップして、直接支払記録を登録することも可能です
- 支払記録を登録すると、自動的にステータスが更新されます

**注意**: 納品書のファイル管理は現在システムには実装されていません。紙の納品書は別途ファイリングして保管してください。

---

## STEP 5: 請求書受領（仕入先から）

### 目的
仕入先から請求書を受領します。

### 操作

仕入先から以下の方法で請求書を受領します：
- 郵送
- メール添付
- FAX
- 電子請求書システム

**受領後の確認事項**：
- [ ] 請求書番号
- [ ] 請求日
- [ ] 支払期日
- [ ] 請求金額が発注金額と一致しているか
- [ ] 品目・数量が発注内容と一致しているか

**差異がある場合**：
1. 仕入先に確認
2. 誤りがあれば訂正を依頼
3. 正しい請求書を再発行してもらう

---

## STEP 6: 支払記録

### 目的
仕入先への代金支払いを記録します。

### 操作手順

#### 1. 支払記録画面を開く

**方法1: 発注書詳細画面から**
1. 発注書詳細画面を開く
2. 「支払記録」ボタンをクリック

**方法2: 入金・支払一覧から**
1. サイドメニューから「入金・支払」をクリック
2. 「+ 新規登録」ボタンをクリック
3. 支払タイプで「支払」を選択

#### 2. 支払情報を入力

| 項目 | 説明 | 入力例 |
|------|------|--------|
| **支払日** | 実際に支払った日 | 2026-02-28 |
| **発注書** | 支払対象の発注書を選択 | PO-202601-0001 |
| **支払額** | 今回支払った金額 | 21,120 |
| **支払方法** | 銀行振込、現金など | 銀行振込 |
| **参照番号** | 振込先名など（任意） | B資材(株) |
| **メモ** | 補足情報（任意） | 月末締め翌月末払い |

#### 3. 登録

「登録」ボタンをクリックすると、支払が記録されます。

### 支払後のステータス変化

- **全額支払の場合**: 発注書のステータスが「送付済み」→「完了」に変更
- **部分支払の場合**: ステータスは「送付済み」のまま（残額が表示される）

---

## 部分支払の場合

### シナリオ

発注金額: ¥21,120
1回目支払: ¥10,000
2回目支払: ¥11,120

### 操作手順

#### 1回目の支払記録

1. 支払日: 2026-02-15
2. 支払額: ¥10,000
3. 登録後:
   - 既支払額: ¥10,000
   - 残額: ¥11,120
   - ステータス: 送付済み（変化なし）

#### 2回目の支払記録

1. 支払日: 2026-02-28
2. 支払額: ¥11,120
3. 登録後:
   - 既支払額: ¥21,120
   - 残額: ¥0
   - ステータス: 完了（自動変更）

## よくあるケース

### ケース1: 発注内容と納品内容が異なる場合

**シナリオ**: 一部の品目が欠品で納品されなかった

**対処法**:
1. 仕入先に確認
2. 以下のいずれかを選択:
   - **欠品分を後日納品**: 発注書は継続、納品後に完了
   - **欠品分を削除**: 発注書の明細を修正（ただし承認済みの場合は要再承認）
   - **別途新しい発注書を作成**: 欠品分を新規発注

**支払額の調整**:
- 実際に納品された分のみ支払う
- 支払記録時に金額を調整

**例**:
```
【発注時】
ブルーシート 10枚: ¥12,000
養生テープ 20巻: ¥3,600
マスキングテープ 30巻: ¥3,600
合計: ¥21,120

【納品時】
ブルーシート 10枚: ¥12,000 ← 納品
養生テープ 20巻: ¥3,600 ← 納品
マスキングテープ 0巻: ¥0 ← 欠品
合計: ¥17,160

【支払時】
支払額: ¥17,160（実際の納品分のみ）
```

### ケース2: 緊急発注で承認を省略する場合

**シナリオ**: 現場で急遽資材が必要になった

**通常の対処法（推奨）**:
1. 発注書を作成して「提出」
2. 承認者に電話・チャットで緊急承認を依頼
3. 承認者がシステムで承認
4. 仕入先に発注

**やむを得ない場合**:
1. 口頭またはメール・電話で仕入先に発注
2. 後から発注書を作成（記録として）
3. 承認フローを通す
4. 「この発注書は事後作成です」とメモに記載

### ケース3: 定期発注（毎月同じ資材を発注）

**シナリオ**: 毎月決まった量の消耗品を発注する

**対処法**:
1. 前月の発注書を複製（可能な場合）
2. 発注日・納期を更新
3. 数量を確認・調整
4. 提出・承認フローに進む

**自動化の検討**:
- 定期発注機能がある場合は活用
- 消耗品発注管理から自動生成

## 消耗品発注管理との連携

### 在庫不足アラートから発注

#### 1. 在庫不足を確認

1. ダッシュボードまたは消耗品一覧で「在庫不足」アラートを確認
2. 該当の消耗品をクリック

#### 2. 発注数量を決定

1. 現在の在庫数を確認
2. 最小在庫数を確認
3. 発注数量を決定（最小在庫数の2〜3倍を推奨）

#### 3. 発注書を作成

1. 「発注管理」ボタンをクリック
2. 「発注書を作成」ボタンをクリック
3. 発注明細が自動入力される
4. 内容を確認・修正
5. 提出・承認フローに進む

## チェックリスト

### 発注書作成時

- [ ] 仕入先が正しく選択されているか
- [ ] 件名が具体的に記載されているか
- [ ] 発注日・納期が適切か
- [ ] 明細の品目・数量・単価が正確か
- [ ] 合計金額が妥当か
- [ ] 納品先が明記されているか（現場直送の場合）
- [ ] 備考に必要事項が記載されているか

### 納品確認時

- [ ] 納品書が添付されているか
- [ ] 発注した品目がすべて揃っているか
- [ ] 数量が発注数と一致しているか
- [ ] 品質に問題がないか（破損・汚損がないか）
- [ ] 納品日を記録したか

### 支払記録時

- [ ] 支払日が正しいか（実際に支払った日）
- [ ] 支払額が正確か（請求書と一致しているか）
- [ ] 支払方法が正しく選択されているか
- [ ] 参照番号（振込先名など）が記録されているか

## トラブルシューティング

### Q: 発注書を承認してから内容を変更したい

**A**: 承認済みの発注書は編集できません。以下の対応を取ってください：
1. 発注書を複製して新しい発注書を作成
2. 内容を修正
3. 再度提出・承認フローに進む
4. 仕入先に変更を連絡

### Q: 納品された商品に不良品があった

**A**: 以下の対応を取ってください：
1. 仕入先に連絡して返品・交換を依頼
2. 不良品の写真を撮影
3. 返品伝票を受領
4. 良品が納品されたら、納品確認を完了
5. 支払額を調整（不良品分を減額）

### Q: 支払額を間違えて記録してしまった

**A**: 支払記録は編集できます。以下の手順で修正してください：
1. 入金・支払一覧画面を開く
2. 該当の支払記録をクリック
3. 「編集」ボタンをクリック
4. 正しい金額を入力して保存

**注意**: 編集はマネージャー以上の権限が必要です。

### Q: 発注書の納期に間に合わなかった

**A**: 以下の対応を取ってください：
1. 仕入先に納期遅延の連絡を受ける
2. 新しい納期を確認
3. 現場の工程に影響がないか確認
4. 影響がある場合は、現場責任者に連絡
5. メモ欄に「納期変更: 〇月〇日 → △月△日」と記録
