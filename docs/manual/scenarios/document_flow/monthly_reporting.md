# 月次報告業務

## 概要

このマニュアルでは、月末に行う定例業務（月次報告業務）の流れを説明します。売上・支出の確認、作業報告書の整理、在庫確認、翌月の計画立案などを効率的に行うための手順をまとめています。

## 対象者

- リーダー以上: 担当現場の月次報告
- マネージャー: 複数現場の集計・分析
- 管理者: 全社の月次報告・経営判断

## 月次報告業務の全体像

```
【月末〜翌月初旬の業務】

1. 売上・請求の確認
   - 当月分の請求書送付状況
   - 入金予定・入金済みの確認

2. 支出・支払の確認
   - 当月分の発注書・支払状況
   - 未払金の確認

3. 作業報告書の整理
   - 未提出・未承認の作業報告書確認
   - 月次作業実績の集計

4. 在庫確認
   - 道具・消耗品の在庫確認
   - 在庫不足の発注手配

5. 出退勤記録の確認
   - 打刻漏れの確認
   - 労働時間の集計

6. 翌月の計画立案
   - 次月の工事予定
   - 資材発注計画
   - 人員配置計画
```

## 1. 売上・請求の確認

### 目的
当月の売上を確定し、入金状況を確認します。

### 操作手順

#### 1-1. 当月分の請求書一覧を確認

1. サイドメニューから「請求書」をクリック
2. フィルターで「請求日」を当月に設定
3. ステータスごとに確認:
   - **下書き**: 請求書を完成させて送付
   - **承認待ち**: 承認を進める
   - **送付済み**: 入金待ち（問題なし）
   - **支払済み**: 入金完了（問題なし）

#### 1-2. 未送付の請求書を送付

1. ステータスが「承認済み」の請求書を確認
2. PDFダウンロード → 顧客へ送付
3. 「送付済みに変更」ボタンをクリック

#### 1-3. 入金予定を確認

1. サイドメニューから「売掛金管理」をクリック
2. 「入金予定」タブを開く
3. 当月中に入金予定の請求書を確認
4. 支払期日が近い請求書に注意

#### 1-4. 当月の売上集計

1. サイドメニューから「分析」→「売上分析」をクリック
2. 期間を「当月」に設定
3. 以下を確認:
   - **総売上**: 当月の請求書合計
   - **入金済み**: 実際に入金された金額
   - **未収金**: 請求済みだが未入金の金額

#### 1-5. 月次売上レポート作成

**レポートに含める情報**：
```
【〇月度 売上報告】

1. 請求書発行状況
   - 発行件数: 10件
   - 請求総額: ¥5,200,000
   - 入金済み: ¥3,800,000
   - 未収金: ¥1,400,000

2. 主要案件
   - A案件（〇〇ビル外壁塗装）: ¥1,500,000（入金済み）
   - B案件（△△マンション補修）: ¥1,200,000（入金待ち）
   - C案件（◇◇ビル改修）: ¥900,000（入金済み）

3. 懸念事項
   - D案件（支払期日超過、催促中）
   - E案件（請求書未送付、至急対応予定）
```

## 2. 支出・支払の確認

### 目的
当月の支出を確定し、支払状況を確認します。

### 操作手順

#### 2-1. 当月分の発注書一覧を確認

1. サイドメニューから「発注書」をクリック
2. フィルターで「発注日」を当月に設定
3. ステータスごとに確認:
   - **下書き**: 発注書を完成させて送付
   - **承認待ち**: 承認を進める
   - **送付済み**: 納品・支払待ち
   - **完了**: 支払完了（問題なし）

#### 2-2. 未払いの発注書を確認

1. ステータスが「送付済み」の発注書を確認
2. 納品が完了しているか確認
3. 支払期日を確認
4. 支払予定を立てる

#### 2-3. 当月の支出集計

1. サイドメニューから「分析」→「支出分析」をクリック
2. 期間を「当月」に設定
3. 以下を確認:
   - **総支出**: 当月の発注書合計
   - **支払済み**: 実際に支払った金額
   - **未払金**: 発注済みだが未払いの金額

#### 2-4. カテゴリ別支出を確認

```
資材費: ¥800,000
消耗品費: ¥150,000
外注費: ¥500,000
重機リース: ¥200,000
その他: ¥50,000
──────────────
合計: ¥1,700,000
```

## 3. 作業報告書の整理

### 目的
当月の作業実績を確定し、報告書を整理します。

### 操作手順

#### 3-1. 未提出の作業報告書を確認

1. サイドメニューから「作業報告書」をクリック
2. フィルターで「下書き」を選択
3. 担当者に提出を依頼

#### 3-2. 未承認の作業報告書を確認

1. フィルターで「提出済み」を選択
2. 承認待ちの報告書を確認
3. 内容を確認して承認

#### 3-3. 当月の作業実績を集計

1. フィルターで「報告日」を当月に設定
2. フィルターで「承認済み」を選択
3. 以下を確認:
   - 作業報告書件数
   - 現場別の作業日数
   - 主要作業内容

#### 3-4. 作業報告書を一括PDF出力

1. 「承認済み」の作業報告書一覧を表示
2. 対象の報告書を選択
3. 「一括PDF出力」ボタンをクリック
4. PDFファイルをダウンロード
5. 保管・提出

**用途**：
- 顧客への月次報告
- 社内記録
- 監査資料

## 4. 在庫確認

### 目的
道具・消耗品の在庫を確認し、補充が必要なものを手配します。

### 操作手順

#### 4-1. 在庫不足アラートを確認

1. ダッシュボードで「在庫不足」アラートを確認
2. または、サイドメニューから「道具」「消耗品」をクリック
3. 在庫不足の品目をリストアップ

#### 4-2. 棚卸しを実施（任意）

**物理的な棚卸し**:
1. 実際に倉庫で道具・消耗品を数える
2. システムの在庫数と比較
3. 差異がある場合は在庫調整

**システムでの在庫調整**:
1. 消耗品詳細画面を開く
2. 「在庫調整」ボタンをクリック
3. 調整タイプ「設定」を選択
4. 実際の在庫数を入力
5. 理由「月次棚卸し」と記入

#### 4-3. 補充発注を手配

1. 在庫不足の品目から発注書を作成
2. 適切な発注数量を決定（最小在庫数の2〜3倍）
3. 提出・承認フローに進む

## 5. 出退勤記録の確認

### 目的
スタッフの出退勤記録を確認し、労働時間を集計します。

### 操作手順

#### 5-1. 打刻漏れを確認

1. サイドメニューから「出退勤」→「管理」をクリック
2. フィルターで「当月」を選択
3. 「打刻漏れ」でフィルタリング
4. 該当スタッフに確認・修正を依頼

#### 5-2. 当月の勤務時間を集計

1. サイドメニューから「出退勤」→「レポート」をクリック
2. 期間を「当月」に設定
3. 以下を確認:
   - スタッフ別の勤務時間
   - 残業時間
   - 休日出勤日数

#### 5-3. 異常値をチェック

**確認項目**：
- [ ] 極端に長い勤務時間（12時間以上）
- [ ] 極端に短い勤務時間（4時間未満）
- [ ] 休憩時間が記録されていない
- [ ] 深夜勤務（22:00〜翌5:00）

**異常値がある場合**:
1. 該当スタッフに確認
2. 誤記録であれば修正
3. 実際の勤務であれば、状況を記録

#### 5-4. 給与計算用データを準備

1. 「勤務時間集計」をCSVでエクスポート
2. 給与計算ソフトにインポート
3. または、経理担当者に共有

---

## 6. 翌月の計画立案

### 目的
翌月の工事予定を確認し、必要な準備を行います。

### 操作手順

#### 6-1. 翌月の工事予定を確認

1. サイドメニューから「現場」をクリック
2. ステータスが「進行中」「開始予定」の現場を確認
3. 翌月の工程表を確認

#### 6-2. 必要資材をリストアップ

1. 各現場の工程から必要資材を洗い出す
2. 在庫を確認
3. 不足分を発注リストに追加

#### 6-3. 人員配置を計画

1. 各現場の必要人数を確認
2. スタッフの予定を確認
3. 人員配置表を作成

**人員配置表の例**：
```
【2026年2月 人員配置計画】

A現場（〇〇ビル外壁塗装）:
- リーダー: 山田太郎
- スタッフ: 佐藤花子、鈴木一郎
- 期間: 2/1〜2/28

B現場（△△マンション補修）:
- リーダー: 田中次郎
- スタッフ: 高橋三郎、伊藤四郎
- 期間: 2/10〜2/20
```

#### 6-4. 翌月の目標を設定

```
【2026年2月 月間目標】

1. 売上目標: ¥6,000,000
   - A現場: ¥3,500,000
   - B現場: ¥1,500,000
   - C現場: ¥1,000,000

2. 原価率目標: 70%以下
   - 資材費の削減
   - 外注費の最適化

3. 安全目標: 無事故・無災害
   - 毎朝の安全ミーティング実施
   - 月1回の安全パトロール

4. 品質目標: クレームゼロ
   - 作業報告書の徹底
   - 完工検査の厳格化
```

## 月次報告会議

### 目的
管理者・マネージャーが集まり、月次実績を報告・共有します。

### 会議の流れ

#### 1. 月次実績報告（各リーダー）

- 担当現場の進捗状況
- 売上・支出の実績
- 課題・問題点
- 翌月の予定

#### 2. 全社実績報告（管理者）

- 全社の売上・利益
- 予算との対比
- 主要指標（売上高、原価率、利益率）

#### 3. 課題の共有と対策検討

- 赤字案件の原因分析
- 在庫不足の再発防止
- 品質問題の対策

#### 4. 翌月の重点施策決定

- 売上目標
- コスト削減目標
- 安全・品質目標

## 効率化のヒント

### 1. 月次業務スケジュールを作成

```
【月次業務スケジュール】

毎月25日: 当月分の作業報告書提出締切
毎月28日: 請求書作成・送付
毎月末日: 在庫確認・棚卸し
翌月3日: 出退勤記録確認
翌月5日: 月次報告会議
翌月10日: 翌月計画確定
```

### 2. テンプレートを活用

- 月次売上レポートテンプレート
- 月次支出レポートテンプレート
- 人員配置表テンプレート
- 月間目標シートテンプレート

### 3. リマインダーを設定

- 出退勤設定で「月次棚卸しリマインド」を有効化
- カレンダーに月次業務の予定を登録
- 管理者向け日次レポートを有効化

### 4. 役割分担を明確にする

| 役割 | 担当者 | 業務内容 |
|------|--------|---------|
| **売上管理** | 管理者 | 請求書送付、入金確認、催促 |
| **支出管理** | マネージャー | 発注書承認、支払手配 |
| **作業報告** | リーダー | 作業報告書提出、承認依頼 |
| **在庫管理** | 資材担当 | 在庫確認、補充発注 |
| **出退勤管理** | 管理者 | 打刻漏れ確認、勤務時間集計 |

## トラブルシューティング

### Q: 月末に作業報告書が大量に未提出になってしまう

**A**: 以下の対策を取ってください：
1. **定期的な提出を促す**: 週1回のリマインド
2. **提出期限を設定**: 作業完了後3日以内など
3. **未提出者に個別連絡**: 25日時点で未提出者をリストアップして連絡
4. **提出率を可視化**: ダッシュボードで提出率を表示

### Q: 在庫数がシステムと実物で合わない

**A**: 以下の原因が考えられます：
1. **移動記録漏れ**: 道具を持ち出したが記録していない
2. **返却記録漏れ**: 道具を返却したが記録していない
3. **在庫調整漏れ**: 消耗品を使用したが記録していない

**対策**：
- 月次棚卸しで定期的に実物確認
- QRスキャン必須設定を有効化
- 在庫調整の記録を徹底

### Q: 支払期日を過ぎた請求書が多数ある

**A**: 以下の対策を取ってください：
1. **売掛金管理画面で一覧確認**: 「支払期日超過」でフィルタリング
2. **優先順位をつける**: 金額が大きい順、期日超過日数が長い順
3. **顧客に催促**: 電話またはメールで入金を依頼
4. **支払計画を確認**: 分割払いなどの相談
5. **再発防止**: 請求書送付を早める、支払条件を見直す
