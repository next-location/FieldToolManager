# 新入社員の受け入れマニュアル

## 1. 概要

新入社員がザイロクを使い始めるために必要な、アカウント作成から初期設定までの手順を説明します。管理者が実施すべき作業と、新入社員に伝えるべき情報を網羅しています。

**このマニュアルで解決できること**:
- 新入社員のアカウントを素早く作成
- 適切な権限を設定
- 初回ログインの案内

**対象者**: Admin（管理者）のみ

---

## 2. 受け入れフロー全体像

### 2-1. 受け入れステップ（全4ステップ）

| ステップ | 作業内容 | 担当者 | 所要時間 |
|---------|---------|--------|----------|
| **1. アカウント作成** | ザイロクに新入社員を登録 | 管理者 | 3分/人 |
| **2. 初期パスワード連絡** | 新入社員にログイン情報を伝達 | 管理者 | 2分/人 |
| **3. 初回ログイン** | 新入社員が実際にログイン | 新入社員 | 5分 |
| **4. 基本操作の説明** | 基本的な使い方を説明 | 管理者または先輩社員 | 30分 |

**合計所要時間**: 1人あたり約40分

---

## 3. ステップ1: アカウント作成

### 3-1. アカウント作成手順

#### 手順

1. **スタッフ一覧ページを開く**
   - 左サイドメニュー **「スタッフ」** をクリック

2. **新規登録モーダルを開く**
   - 画面右上の **「+ 新規スタッフを追加」** ボタンをクリック

3. **必須項目を入力**

| 項目 | 必須 | 説明 | 例 |
|------|------|------|-----|
| **名前** | ✅ | フルネーム（50文字以内） | 山田太郎 |
| **メールアドレス** | ✅ | ログインIDとして使用（100文字以内） | yamada@company.com |
| **初期パスワード** | ✅ | 8文字以上、大文字・小文字・数字必須 | TempPass123! |
| **権限** | ✅ | Staff / Leader / Manager / Admin | Staff |

4. **オプション項目を入力**

| 項目 | 必須 | 説明 | 例 |
|------|------|------|-----|
| **部署** | - | 所属部署（50文字以内） | 工事部 |
| **社員番号** | - | 社内の社員番号（20文字以内） | EMP-001 |
| **電話番号** | - | 連絡先（20文字以内） | 090-1234-5678 |

5. **初期パスワードの生成**
   - 手動で入力するか、**「🔄 ランダム生成」** ボタンをクリック
   - ランダム生成されたパスワードは12文字で、大文字・小文字・数字・記号を含みます
   - **必ずメモしておいてください**（後で新入社員に伝える必要があります）

6. **登録を実行**
   - **「追加する」** ボタンをクリック
   - 成功メッセージが表示されます

**複数名の登録について**:
- 新入社員が複数名いる場合は、上記の手順を繰り返してください
- 大量採用の場合は、システム管理者にデータ挿入代行を依頼してください

---

### 4-1. 新入社員に伝える情報

新入社員には以下の情報を伝えてください：

```
【ザイロクアカウント情報】

アクセスURL: https://zairoku.com/login

メールアドレス: yamada@company.com
初期パスワード: TempPass123!

【初回ログイン後の手順】
1. 上記URLにアクセスしてログイン
2. パスワード変更を求められたら、新しいパスワードに変更してください
3. ログイン後、基本操作マニュアルを確認してください

【サポート】
不明点があれば、○○（管理者名）まで連絡してください。
```

### 4-2. 伝達方法の選択肢

| 方法 | メリット | デメリット | 推奨度 |
|------|---------|-----------|--------|
| **対面で伝える** | 確実、質問に即答 | 時間がかかる | ⭐⭐⭐ |
| **メールで送る** | 記録に残る、一斉送信可能 | セキュリティリスク（平文） | ⭐⭐ |
| **チャットで送る** | 迅速 | 記録が流れやすい | ⭐ |
| **紙で渡す** | オフライン環境でもOK | 紛失リスク | ⭐⭐ |

**推奨**: 対面で伝え、補足としてメールで送る（ただし「初回ログイン後すぐにパスワード変更してください」と明記）

### 4-3. セキュリティ上の注意

⚠️ **初期パスワードの取り扱い注意**:
- 初期パスワードはセキュリティが低いため、初回ログイン後すぐに変更するよう指導してください
- メールで送る場合、件名に「パスワード」と書かない
- Slackなどのパブリックチャンネルには絶対に投稿しない

---

## 5. ステップ3: 初回ログイン（新入社員が実施）

### 5-1. 新入社員への説明内容

以下の手順を新入社員に説明してください：

1. **ログインページにアクセス**
   - ブラウザで https://zairoku.com/login を開く
   - Chrome、Edge、Safari推奨

2. **ログイン情報を入力**
   - メールアドレス: 受け取ったメールアドレスを入力
   - パスワード: 初期パスワードを入力
   - **「ログイン」** ボタンをクリック

3. **パスワード変更（推奨）**
   - 初回ログイン後、すぐにパスワードを変更
   - 右上のアイコン → **「アカウント設定」** → **「パスワード変更」**
   - 新しいパスワード: 8文字以上、大文字・小文字・数字を含む

4. **動作確認**
   - 左サイドメニューから各機能にアクセスできるか確認
   - 道具一覧、移動履歴などを開いてみる

## 6. ステップ4: 基本操作の説明

### 6-1. 説明内容（所要時間: 30分）

新入社員に以下の基本操作を説明してください：

**1. ログインとダッシュボード（5分）**
- ログイン方法の確認
- ダッシュボードの見方

**2. 道具の検索と確認（10分）**
- 道具一覧の見方
- フィルタ機能の使い方
- 道具詳細画面の見方

**3. 道具の移動登録（10分）**
- 道具の持ち出し手順
- 道具の返却手順
- QRコードスキャン機能の使い方

**4. 参考資料の案内（5分）**

新入社員に以下のマニュアルを案内してください：

- [はじめにマニュアル](../../getting_started.md) - 基本操作全般
- [道具管理マニュアル](../../tool_management.md) - 道具の操作方法

## 7. 受け入れ後のフォローアップ

### 7-1. 1週間後のチェックリスト

新入社員の受け入れから1週間後、以下を確認してください：

- [ ] ログインできているか（最終ログイン日時を確認）
- [ ] 道具の移動登録ができているか
- [ ] 作業報告書を提出できているか
- [ ] 不明点や質問がないか（直接ヒアリング）

**確認方法**:
1. スタッフ一覧ページを開く
2. 対象スタッフをクリック
3. **「履歴」** ボタンで操作履歴を確認

### 8-2. よくある質問と回答

#### Q1. 新入社員がログインできません

**A.** 以下を確認してください：

1. **メールアドレスは正しいか**
   - スペースや全角文字が入っていないか確認
2. **パスワードは正しいか**
   - 大文字・小文字を区別します
3. **アカウントが有効か**
   - 管理者がスタッフ一覧で「有効」になっているか確認
4. **パスワードリセット**
   - 管理者がパスワードをリセットして再発行

**パスワードリセット手順**:
1. スタッフ一覧ページを開く
2. 対象スタッフをクリック
3. **「パスワードリセット」** ボタンをクリック
4. 新しいパスワードが自動生成されます
5. 新入社員に伝える

#### Q2. 新入社員に権限を後から変更できますか？

**A.** はい、いつでも変更できます。

**手順**:
1. スタッフ一覧ページを開く
2. 対象スタッフをクリック
3. **「編集」** ボタンをクリック
4. **「権限」** ドロップダウンで新しい権限を選択
5. **「保存」** をクリック

**注意**: 権限を変更すると、即座に反映されます。次回ログイン時から新しい権限で操作できます。
