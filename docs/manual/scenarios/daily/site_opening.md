## 概要

新しい工事現場を開始する際の立ち上げ手順を説明します。

**立ち上げの流れ**
1. 事前準備（取引先情報確認）
2. 現場登録
3. 現場QRコード発行・設置
4. 工事情報登録
5. 道具配置
6. スタッフアサイン
7. 初日の準備確認

**所要時間の目安**
- システム登録: 20〜30分
- QRコード印刷・設置: 10分
- 道具配置: 30分〜1時間
- 合計: 1〜2時間程度

---

## 事前準備

### 必要な情報を集める

**現場情報**
- 現場名
- 住所
- 取引先（顧客）情報
- 工事期間（開始日・完了予定日）
- 契約金額・予算

**スタッフ情報**
- 現場責任者
- アサインするスタッフのリスト
- 各スタッフの役割

**道具情報**
- 必要な道具のリスト
- 道具セット（定型パターンがある場合）
- 消耗品の必要数

### 取引先の確認

**取引先が未登録の場合**
1. サイドメニューから「取引先管理」をクリック
2. 「+ 新規登録」をクリック
3. 取引先情報を入力して登録

**参照**: [取引先管理マニュアル](../../clients.md)

## 現場登録

### 登録手順

1. サイドメニューから「現場管理」をクリック
2. 「+ 新規登録」ボタンをクリック
3. 現場登録フォームが表示される

### 入力項目

**必須項目**
- **現場名**: 工事名や建物名（例: 〇〇ビル建設工事）
  - 最大100文字
  - スタッフが識別しやすい名称にする

**推奨項目**
- **取引先**: プルダウンから顧客を選択
  - 顧客または顧客兼仕入先の取引先のみ表示されます
- **住所**: 現場の所在地
- **現場コード**: 自動生成（変更可能）
- **現場責任者**: プルダウンからスタッフを選択
- **工期**: 開始日・完了予定日
- **備考**: 現場の特記事項

### 登録完了

「登録」ボタンをクリックすると、現場が登録され、現場詳細画面に移動します。

**自動的に行われること**
- 現場IDが発行される
- 現場QRコードが生成される
- 現場コードが採番される（S-YYYY-NNNN形式）

**参照**: [現場管理マニュアル](../../sites.md)

## 現場QRコード発行

### QRコードの役割

現場QRコードは以下の用途で使用されます：
- 出退勤打刻（QR打刻の場合）
- 道具の持ち出し先指定
- 現場情報の確認

### QRコード発行手順

1. 現場詳細画面で「QRコード」タブをクリック
2. 「QRコードを表示」ボタンをクリック
3. QRコードが表示される
4. 「印刷」ボタンをクリックして印刷

**印刷設定**
- 推奨サイズ: 10cm × 10cm 以上
- カラー・白黒どちらでも可
- ラミネート加工推奨（屋外使用のため）

### QRコード設置

**設置場所**
- 現場の入口付近
- スタッフが毎日通る場所
- 雨や直射日光が当たりにくい場所
- 道具置き場の近く

**設置方法**
- ラミネート加工して掲示
- 防水ケースに入れて設置
- 壁やボードに貼り付け

**注意事項**
- QRコードが汚れたり破損したりしないよう保護する
- 定期的に読み取りテストを行う
- 破損した場合は再発行（いつでも可能）

## 工事情報登録

### 工事登録の目的

現場と工事を紐づけることで、以下が可能になります：
- 発注書の工事別集計
- 予算管理
- 原価計算

### 工事登録手順

1. サイドメニューから「工事管理」をクリック
2. 「+ 新規登録」ボタンをクリック
3. 工事登録フォームが表示される

### 入力項目

**必須項目**
- 工事名
- ステータス（計画中 / 進行中 / 完了 / キャンセル）

**推奨項目**
- 発注者（取引先）
- 関連現場（先ほど登録した現場を選択）
- 工事責任者
- 工事開始日・完了予定日
- 契約金額
- 予算金額

**参照**: [工事管理マニュアル](../../projects.md)

## 道具配置

### 道具配置の方法

**パターン1: 道具セットを使う（推奨）**
1. サイドメニューから「道具セット」をクリック
2. 現場に適した道具セットを選択
3. 「一括持ち出し」をクリック
4. 移動先で登録した現場を選択
5. 「持ち出す」をクリック

**パターン2: 個別に持ち出し登録**
1. サイドメニューから「道具管理」をクリック
2. 持ち出す道具を検索
3. 道具詳細画面で「移動」をクリック
4. 移動先で登録した現場を選択
5. 「持ち出す」をクリック

### 現場に配置する道具の例

**基本工具**
- 電動ドリル
- インパクトドライバー
- 丸ノコ
- 電動ノコギリ
- グラインダー

**測定器具**
- 水平器
- メジャー
- レーザー距離計

**安全用具**
- ヘルメット
- 安全帯
- 保護メガネ
- 手袋

**消耗品**
- ビス・釘各種
- ドリルビット
- 養生シート
- 清掃用具

**参照**:
- [道具管理マニュアル](../../tool_management.md)
- [道具セットマニュアル](../../tool_sets.md)

## スタッフアサイン

### スタッフへの周知事項

**初日までに伝えるべき情報**
- 現場名・住所
- 集合時間・集合場所
- 現場責任者
- 駐車場の有無
- 作業内容の概要
- 必要な装備・持ち物

### 現場責任者の設定

1. 現場詳細画面で「編集」をクリック
2. 現場責任者をプルダウンから選択
3. 「更新」をクリック

**現場責任者の役割**
- スタッフの出退勤管理
- 作業報告書の確認・承認
- 道具の管理
- 安全管理

### 勤怠設定の確認

**QR打刻を使用する場合**
1. 現場QRコードの設置完了を確認
2. スタッフにQRコードの場所を周知

**リーダーQRを使用する場合**
1. サイドメニューから「勤怠管理」→「リーダーQR」をクリック
2. 現場責任者用のQRコードを発行
3. QRコードを印刷または画面で表示

**参照**: [勤怠管理マニュアル](../../attendance.md)

## 初日の準備

### 前日までに確認すること

**システム面**
- [ ] 現場登録完了
- [ ] 現場QRコード設置完了
- [ ] 工事情報登録完了
- [ ] 道具配置完了
- [ ] スタッフへの周知完了

**現場面**
- [ ] 現場の鍵の受け渡し
- [ ] 駐車スペースの確認
- [ ] トイレ・休憩場所の確認
- [ ] 電源・水道の確認
- [ ] 道具置き場の確保

### 初日の流れ

**朝のミーティング（キックオフ）**
1. スタッフ全員集合
2. 自己紹介
3. 現場の説明
   - 工事内容
   - 工期
   - 注意事項
4. 役割分担の確認
5. 安全確認
6. 質疑応答

**システム操作の説明**
1. 出退勤打刻の方法
   - QRコードの場所
   - 打刻の手順
2. 道具管理の方法
   - 持ち出し・返却の手順
   - QRコードの読み方
3. 作業報告書の書き方
   - 提出期限
   - 記載項目

**初日の作業開始**
1. 出勤打刻
2. 道具確認
3. 安全確認
4. 作業開始
