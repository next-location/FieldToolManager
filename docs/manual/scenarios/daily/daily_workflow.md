## 概要

このマニュアルでは、現場スタッフの典型的な1日の業務フローを順を追って説明します。

**1日の基本的な流れ**
1. 出勤打刻
2. 道具持ち出し
3. 現場での作業
4. 道具返却
5. 退勤打刻
6. 作業報告書提出

**所要時間の目安**
- 出退勤打刻: 各1分
- 道具持ち出し・返却: 各5〜10分
- 作業報告書作成: 10〜15分

## 出勤時の流れ

### 1. ログイン

**スマートフォンから**
1. ブラウザでザイロクにアクセス
2. メールアドレスとパスワードを入力
3. 「ログイン」をタップ

**パソコンから**
1. ブラウザでザイロクにアクセス
2. メールアドレスとパスワードを入力
3. 「ログイン」をクリック

### 2. 出勤打刻

**手順**
1. サイドメニューから「出退勤」をタップ
2. 現場を選択（プルダウンメニューから）
3. 「出勤」ボタンをタップ
4. 確認メッセージが表示されたら完了

**打刻方法の種類**

組織の設定により、以下のいずれかの方法で打刻します：

| 方法 | 説明 |
|------|------|
| 手動打刻 | ボタンを押すだけで打刻 |
| QR打刻 | 現場のQRコードまたはリーダーQRコードをスキャン |
| 位置情報打刻 | GPS位置情報を記録 |

**ポイント**
- 出勤打刻は現場到着時に行ってください
- 打刻を忘れた場合は、できるだけ早く打刻してください
- QR打刻の場合、リーダーが発行したQRコードまたは現場QRコードが必要です

**参照**: [勤怠管理マニュアル](../../attendance.md)

## 道具持ち出し

### 準備

**必要なもの**
- スマートフォン（カメラ機能使用）
- 持ち出す道具のQRコードラベル

### 持ち出し手順

#### 方法1: QRスキャンから持ち出し（推奨）

1. サイドメニューから「QRスキャン」をタップ
2. カメラが起動するので、道具のQRコードをスキャン
3. 道具情報が表示される
4. 「移動」ボタンをタップ
5. 移動先（現場）を選択
6. 備考があれば入力（任意）
7. 「持ち出す」をタップ

#### 方法2: 道具一覧から持ち出し

1. サイドメニューから「道具管理」をタップ
2. 持ち出す道具を検索または一覧から選択
3. 道具詳細画面で「移動」ボタンをタップ
4. 移動先（現場）を選択
5. 備考があれば入力（任意）
6. 「持ち出す」をタップ

### 複数の道具を持ち出す場合

**効率的な方法**
1. 道具セット機能を使用（事前に登録が必要）
2. 1つずつQRスキャンして持ち出し登録

**道具セット機能**
- よく使う道具をセットとして登録しておくと、まとめて持ち出し・返却が可能
- 管理者またはリーダーに道具セットの登録を依頼してください

**参照**: [道具セットマニュアル](../../tool_sets.md)

### 持ち出し時の注意事項

**確認すべきこと**
- 道具の状態を確認（破損していないか）
- 正しい現場に持ち出しているか
- 数量に間違いがないか（消耗品の場合）

**記録が残ること**
- 誰が持ち出したか
- どの現場に持ち出したか
- 持ち出した日時

**参照**: [道具管理マニュアル](../../tool_management.md)

## 現場での作業

### 作業中の注意事項

**道具の管理**
- 道具を紛失しないよう注意してください
- 破損した場合は、すぐにリーダーまたは管理者に報告してください
- 他のスタッフと道具を共有する場合は、引き継ぎ記録を残してください

**作業記録のメモ**

作業報告書作成のため、以下をメモしておくと便利です：
- 作業内容（何をしたか）
- 作業時間（何時間かかったか）
- 使用した資材
- 特記事項（トラブルや気づいた点）
- 写真（ビフォー・アフター、作業中の様子）

### 作業中のトラブル

**道具が故障・破損した場合**
1. 作業を中断し、安全を確保
2. リーダーまたは管理者に連絡
3. 可能であれば写真を撮影
4. 帰社後、故障・破損報告を行う

**参照**: [道具の紛失・破損対応マニュアル](../tool_operations/tool_lost_or_damaged.md)

## 道具返却

### 返却のタイミング

**当日返却が基本**
- 作業終了後、会社倉庫に戻った際に返却
- 現場に置いたままにしない（盗難防止）

**現場保管する場合**
- 現場に道具置き場がある場合のみ
- リーダーの承認が必要
- 道具の移動先を「現場」から「現場保管場所」に変更

### 返却手順

#### 方法1: QRスキャンから返却（推奨）

1. サイドメニューから「QRスキャン」をタップ
2. 返却する道具のQRコードをスキャン
3. 「移動」ボタンをタップ
4. 移動先で「会社倉庫」または「倉庫位置」を選択
5. 備考があれば入力（任意）
6. 「返却する」をタップ

#### 方法2: 道具一覧から返却

1. サイドメニューから「道具管理」をタップ
2. 「自分が持ち出し中」でフィルター
3. 返却する道具を選択
4. 「移動」ボタンをタップ
5. 移動先で「会社倉庫」または「倉庫位置」を選択
6. 備考があれば入力（任意）
7. 「返却する」をタップ

### 返却時の確認事項

**チェックリスト**
- [ ] 道具に破損・汚れがないか確認
- [ ] 付属品（充電器、ケースなど）が揃っているか
- [ ] 正しい保管場所に戻しているか
- [ ] 次に使う人のために清掃・整理しているか

**返却忘れを防ぐコツ**
- 朝、持ち出した道具をメモしておく
- 道具セット機能を使う（全数確認しやすい）
- 車に「返却チェックリスト」を貼っておく

## 退勤打刻

### 退勤打刻のタイミング

**退勤打刻は、すべての業務終了後に行う**
- 道具返却完了後
- 作業報告書提出後（または作成中）
- 片付け完了後

### 退勤打刻手順

1. サイドメニューから「出退勤」をタップ
2. 「退勤」ボタンをタップ
3. 確認メッセージが表示されたら完了

**ポイント**
- 退勤打刻後に作業報告書を提出することも可能です
- 残業する場合も、通常の退勤時刻に打刻し、別途残業打刻を行ってください（組織の運用による）

### 打刻の修正が必要な場合

打刻時刻を間違えた場合：
1. 自分では修正できません
2. リーダーまたは管理者に連絡
3. 修正を依頼してください

**参照**: [勤怠管理マニュアル](../../attendance.md)

## 作業報告書提出

### 作業報告書とは

**目的**
- 日々の作業内容を記録
- 進捗状況を共有
- 顧客への報告資料として使用

**提出タイミング**
- 原則、作業当日中に提出
- 遅くとも翌営業日の午前中まで

### 作業報告書作成手順

1. サイドメニューから「作業報告書」をタップ
2. 「+ 新規作成」をタップ
3. 以下の項目を入力

**必須項目**
- 報告日
- 現場（プルダウンから選択）
- 作業内容（テキストエリアに詳しく記入）
- 作業時間または進捗率

**任意項目**
- 使用資材
- 天候
- 作業人数
- 写真（複数枚アップロード可能）
- 特記事項

4. 「下書き保存」または「提出」をタップ

**参照**: [作業報告書マニュアル](../../work_reports.md)

### 写真添付のポイント

**撮影すべき写真**
- 作業前（ビフォー）
- 作業中の様子
- 作業後（アフター）
- 使用した資材
- 安全対策の実施状況

**写真のコツ**
- 明るい場所で撮影
- ピントを合わせる
- 全体が写るように
- 1枚の写真に複数の情報を入れすぎない

### 下書き保存と提出の違い

| 状態 | 説明 |
|------|------|
| 下書き | 自分だけが見られる。後で編集可能 |
| 提出済み | リーダー・管理者が閲覧可能。編集不可（差し戻しされた場合を除く） |

**ポイント**
- 入力途中で保存したい場合は「下書き保存」
- 内容が確定したら「提出」
- 提出後は編集できないので、内容を確認してから提出してください
