## 概要

工事現場が完了する際の片付けと完了処理の手順を説明します。

**完了処理の流れ**
1. 完了日の決定・スタッフへの周知
2. 道具の回収確認（現場→倉庫）
3. 消耗品・在庫の整理
4. 作業報告書の最終確認
5. 現場QRコードの撤去
6. システムでの完了処理
7. 取引先への完了報告
8. データの整理・保管

**所要時間の目安**
- 道具回収: 30分〜1時間
- 書類確認: 30分〜1時間
- システム処理: 10〜20分
- 合計: 1.5〜3時間程度

## 道具の回収確認

### 現場にある道具の確認

**確認方法1: システムで確認**
1. サイドメニューから「道具管理」をクリック
2. フィルタで対象の現場を選択
3. 現場にある道具の一覧が表示される
4. リストを印刷またはスマホで確認

**確認方法2: QRコードでスキャン**
1. サイドメニューから「QRスキャン」をクリック
2. 現場にある道具のQRコードをスキャン
3. 道具情報画面で「現在の場所」を確認

### 道具の返却登録

**パターン1: QRコードで返却**
1. サイドメニューから「QRスキャン」をクリック
2. 返却する道具のQRコードをスキャン
3. 「移動」をクリック
4. 移動先で「倉庫」を選択
5. 「持ち出す」をクリック

**パターン2: 一括返却**
1. サイドメニューから「道具セット」をクリック
2. 「一括返却」機能を使用（該当する場合）
3. 返却先で倉庫の保管場所を選択
4. 「返却する」をクリック

**パターン3: 個別に返却登録**
1. サイドメニューから「道具管理」をクリック
2. 返却する道具を検索
3. 道具詳細画面で「移動」をクリック
4. 移動先で「倉庫」を選択
5. 保管場所を指定
6. 「持ち出す」をクリック

### 紛失・破損道具の処理

**紛失した場合**
1. 道具詳細画面で「紛失登録」をクリック
2. 紛失日と状況を入力
3. 「登録」をクリック

**破損した場合**
1. 道具詳細画面で「破損登録」をクリック
2. 破損日と状況を入力
3. 修理可能か廃棄かを選択
4. 「登録」をクリック

**参照**: [道具の紛失・破損対応マニュアル](../../tool_lost_or_damaged.md)

## 消耗品・在庫の整理

### 残った消耗品の処理

**使用可能な消耗品**
- 倉庫に返却
- 在庫数の更新
- 次の現場で使用可能

**使用不可能な消耗品**
- 廃棄処理
- 在庫から削除

### 在庫数の確認

1. サイドメニューから「消耗品管理」をクリック
2. 現場で使用した消耗品を検索
3. 在庫数が実数と一致しているか確認
4. 差異がある場合は「在庫調整」で修正

**参照**: [消耗品管理マニュアル](../../consumable_management.md)

## 作業報告書の確認

### 未提出報告書の確認

1. サイドメニューから「作業報告書」をクリック
2. フィルタで対象の現場を選択
3. ステータスで「未提出」を選択
4. 未提出の報告書がないか確認

**未提出がある場合**
- スタッフに提出を依頼
- 提出期限を設定
- リマインドを送信

### 承認待ち報告書の処理

1. ステータスで「承認待ち」を選択
2. 各報告書を確認
3. 内容に問題がなければ「承認」をクリック
4. 修正が必要な場合は「差し戻し」をクリック

### 最終報告書の作成

**最終日の報告書**
- 片付け作業の内容
- 道具返却の記録
- 現場の最終状態
- 引き渡し完了の記録

**参照**: [作業報告書マニュアル](../../work_reports.md)

## 現場の完了処理

### 現場QRコードの撤去

**作業内容**
1. 現場に設置したQRコードを取り外す
2. ラミネートやケースを回収
3. QRコードは廃棄またはファイリング

**注意事項**
- QRコード自体に個人情報は含まれていません
- 同じQRコードは再利用できません（現場ごとに固有）
- 記念に保管する場合もあります

### システムでの完了処理

1. サイドメニューから「現場管理」をクリック
2. 完了する現場をクリック
3. 現場詳細画面が表示される
4. 「完了にする」ボタンをクリック
5. 確認ダイアログが表示される
6. 「OK」をクリック

**完了処理で実行されること**
- 現場のステータスが「完了」になる
- 完了日が自動記録される
- 出退勤打刻ができなくなる
- 道具の持ち出し先として選択できなくなる
- 一覧画面で「完了」フィルタに表示される

**注意事項**
- 完了処理後も現場情報は参照可能です
- 道具の返却は完了前に完了してください
- 作業報告書は完了後も参照・承認可能です
- 完了処理は取り消せません（管理者に依頼が必要）

### 工事情報の完了処理

1. サイドメニューから「工事管理」をクリック
2. 対象の工事をクリック
3. 「編集」をクリック
4. ステータスを「完了」に変更
5. 実際の完了日を入力
6. 「更新」をクリック

**参照**: [工事管理マニュアル](../../projects.md)

## 完了後の対応

### データの整理・保管

**保存すべきデータ**
- 作業報告書（PDF出力）
- 写真データ
- 請求書・発注書
- 図面・設計書
- 契約書類

**保存場所**
- システム内に保存（自動）
- クラウドストレージに保存
- 社内ファイルサーバに保存

### 完了報告

**取引先への報告**
- 工事完了報告書の送付
- 請求書の送付
- 保証書の交付
- お礼状の送付（任意）

**社内報告**
- 完了報告書の作成
- 原価計算・損益確認
- 反省会の実施
- ノウハウの共有

### 原価計算

1. サイドメニューから「工事管理」をクリック
2. 完了した工事をクリック
3. 「原価」タブをクリック
4. 発注書・人件費の集計を確認
5. 予算との差異を確認

**確認項目**
- 契約金額（売上）
- 発注額（原価）
- 人件費（労務費）
- 粗利益・利益率
