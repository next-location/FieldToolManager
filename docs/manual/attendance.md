# 勤怠管理マニュアル

## 対象者

このマニュアルは、出退勤管理を行うすべてのユーザーを対象としています。

- **Staff（スタッフ）**: 自分の出退勤打刻、休憩登録、自分の勤怠記録閲覧
- **Leader（リーダー）**: スタッフの業務 + 現場QR発行
- **Manager（マネージャー）**: リーダーの業務 + 全員の勤怠管理、月次集計
- **Admin（管理者）**: 全機能 + 勤怠設定、手動修正

---

## 前提条件

勤怠管理を使用する前に、以下を確認してください。

1. **契約プラン**: 現場DX業務効率化パック、または フル機能統合パックが契約済みであること
2. **勤怠設定**: 管理者が勤怠設定を完了していること

---

## 勤怠管理の基本概念

### 1. 出退勤打刻とは

出退勤打刻は、スタッフが**いつ出勤して、いつ退勤したか**を記録する機能です。

**記録される情報:**
- 出勤時刻
- 出勤場所（会社 / 現場）
- 退勤時刻
- 退勤場所（会社 / 現場）
- 休憩時間（設定による）

### 2. 打刻方法の種類

ザイロクでは、以下の打刻方法が利用できます（組織設定による）。

| 打刻方法 | 説明 | 使用場面 |
|---------|------|---------|
| **手動打刻** | ボタンをタップして打刻 | いつでも、どこでも |
| **QRスキャン** | QRコードをスキャンして打刻 | 会社入口、現場 |
| **リーダーQR** | リーダーが発行したQRをスキャン | 現場での一斉打刻 |

### 3. 出勤パターン

出勤パターンは以下の2種類があります。

**パターン1: 会社出勤**
```
会社に出勤 → 会社で退勤
会社に出勤 → 現場へ移動 → 現場で退勤
会社に出勤 → 直帰
```

**パターン2: 現場直行**
```
現場に直行 → 現場で退勤
現場に直行 → 会社へ移動 → 会社で退勤
現場に直行 → 直帰
```

## 出勤打刻

### 手動打刻

#### 出勤打刻の手順

1. **勤怠管理 > 出退勤打刻** を開く
2. **「どこで打刻していますか？」** で出勤場所を選択
   - **🏢 会社（オフィス）**: 会社に出社する場合
   - **🏗️ 現場**: 現場に直行する場合
3. 現場を選択した場合は、**現場を選択**ドロップダウンから現場を選ぶ
4. **「出勤する」** ボタンをタップ
5. 成功メッセージが表示されます

**ポイント:**
- 出勤場所はラジオボタンで選択します
- 単一の「出勤する」ボタンで打刻します
- 退勤予定先の選択機能はありません（退勤時にその場所を選択します）

### QRスキャン（組織設定で有効な場合）

#### 会社QRコードでの出勤

1. 会社入口のQRコードをスマートフォンでスキャン
2. 自動的に会社出勤として記録されます
3. 完了画面が表示されます

#### 現場QRコードでの出勤

1. 現場入口のQRコードをスマートフォンでスキャン
2. 自動的に現場出勤として記録されます
3. 完了画面が表示されます

#### 方法2: リーダーQR（Leader/Manager/Admin限定）

**リーダーの操作:**
1. **勤怠管理 > リーダー用QR** を開く
2. **現場を選択**
3. **「QR発行」** をタップ
4. QRコードが表示される
5. スタッフに見せる

**スタッフの操作:**
1. スマートフォンのQRスキャンを開く
2. リーダーのQRコードをスキャン
3. 自動的にその現場への出勤として記録されます

**注意:** リーダー自身は手動で出勤打刻する必要があります。

## 退勤打刻

### 手動打刻

#### 退勤打刻の手順

1. **勤怠管理 > 出退勤打刻** を開く
2. **「どこで退勤しますか？」** で退勤場所を選択
   - **🏢 会社（オフィス）**: 会社で退勤する場合
   - **🏗️ 現場**: 現場で退勤する場合
3. 現場を選択した場合は、**現場を選択**ドロップダウンから現場を選ぶ
4. **「退勤する」** ボタンをタップ
5. 成功メッセージが表示されます

**ポイント:**
- 退勤場所は出勤場所と異なっても問題ありません
- 例: 会社に出勤 → 現場で退勤
- 例: 現場に出勤 → 会社で退勤

### QRスキャン（組織設定で有効な場合）

#### 会社QRコードでの退勤

1. 会社入口のQRコードをスマートフォンでスキャン
2. 自動的に会社退勤として記録されます
3. 完了画面が表示されます

#### 現場QRコードでの退勤

1. 現場入口のQRコードをスマートフォンでスキャン
2. 自動的に現場退勤として記録されます
3. 完了画面が表示されます

## 休憩時間の登録

休憩時間の記録方法は、組織設定により異なります。

### モード1: none（記録しない）

休憩時間の記録は行いません。自動控除設定がある場合は、6時間超過時に自動的に控除されます。

### モード2: simple（1日1回）

**休憩開始:**
1. **勤怠管理 > 出退勤打刻** を開く
2. **「休憩開始」** をタップ

**休憩終了:**
1. **勤怠管理 > 出退勤打刻** を開く
2. **「休憩終了」** をタップ

### モード3: detailed（複数回可能）

複数回の休憩を記録できます。

**休憩開始:**
1. **勤怠管理 > 出退勤打刻** を開く
2. **「休憩開始」** をタップ

**休憩終了:**
1. **勤怠管理 > 出退勤打刻** を開く
2. **「休憩終了」** をタップ

**例:**
```
1回目: 12:00-12:45（45分）
2回目: 15:00-15:15（15分）
合計: 1時間0分
```

## 勤怠記録の確認

### 自分の勤怠記録（全スタッフ）

1. **勤怠管理 > 自分の勤怠** を開く
2. 自分の出退勤記録が一覧表示されます

**表示内容:**
- 日付
- 出勤時刻・出勤場所
- 退勤時刻・退勤場所
- 勤務時間
- 休憩時間
- 実労働時間

### 全員の勤怠記録（Manager/Admin限定）

1. **勤怠管理 > 勤怠一覧** を開く
2. 全スタッフの出退勤記録が一覧表示されます

**フィルター機能:**
- **日付範囲**: 期間を指定
- **スタッフ**: 特定のスタッフで絞り込み
- **出勤先**: 会社 / 現場で絞り込み
- **ステータス**: 勤務中 / 退勤済み / 未出勤

**操作:**
- **編集**: 記録を修正（打刻忘れ対応）
- **削除**: 記録を削除
- **CSVエクスポート**: データをダウンロード

## 月次集計レポート（Manager/Admin限定）

### スタッフ別集計

1. **勤怠管理 > 月次レポート** を開く
2. **対象月**を選択
3. **スタッフを選択**
4. レポートが表示されます

**集計内容:**
- 出勤日数
- 欠勤日数
- 遅刻回数
- 早退回数
- 総勤務時間
- 平均勤務時間
- 残業時間

### 現場別集計

1. **勤怠管理 > 月次レポート** を開く
2. **対象月**を選択
3. **現場を選択**
4. レポートが表示されます

**集計内容:**
- 稼働日数
- 延べ人数
- 総稼働時間
- 平均人数/日

## 手動修正（Admin限定）

### 打刻忘れの対応

スタッフが打刻を忘れた場合、管理者が手動で記録を追加できます。

1. **勤怠管理 > 勤怠一覧** を開く
2. **「+ 手動追加」** をクリック
3. 以下の情報を入力
   - スタッフ
   - 日付
   - 出勤時刻・出勤場所
   - 退勤時刻・退勤場所
   - 休憩時間
4. **「追加」** をクリック

### 記録の修正

1. **勤怠管理 > 勤怠一覧** を開く
2. 修正したい記録の **「編集」** をクリック
3. 時刻や場所を修正
4. **「保存」** をクリック

## リーダー用QR発行（Leader/Manager/Admin限定）

リーダーは、現場でスタッフが一斉に出勤打刻できるQRコードを発行できます。

### QR発行手順

1. **勤怠管理 > リーダー用QR** を開く
2. **現場を選択**
3. **「QR発行」** をクリック
4. QRコードが表示されます

### QRの有効期限

リーダーQRは、**発行時から24時間有効**です。

- 有効期限が切れた場合は、再度発行してください
- 既存のQRがある場合は、一覧に表示されます

### 会社QR（Admin/Manager限定）

会社入口に設置するQRコードを発行できます。

1. **勤怠管理 > リーダー用QR** を開く
2. **「会社QR」** タブを開く
3. **「QR発行」** をクリック
4. QRコードが表示されます

**有効期限:** 組織設定により異なります（デフォルト: 7日間）

## 勤怠設定（Admin限定）

### 基本設定

1. **設定 > 勤怠設定** を開く
2. 以下の項目を設定

#### 会社出勤機能
- **使用する / 使用しない**: 会社出勤を有効化

#### 会社打刻方法
- **手動**: 手動打刻を許可
- **QRスキャン**: QRスキャンを許可

#### 現場出勤機能
- **使用する / 使用しない**: 現場出勤を有効化

#### 現場打刻方法
- **手動**: 手動打刻を許可
- **QRスキャン**: QRスキャンを許可
- **リーダーQR**: リーダーQR発行を許可

#### 休憩時間記録
- **none**: 記録しない
- **simple**: 1日1回
- **detailed**: 複数回可能

#### 自動休憩控除
- **有効 / 無効**: 6時間超過時に自動控除
- **控除時間**: 自動控除する時間（分単位）

3. **「保存」** をクリック

### アラート設定

1. **設定 > 勤怠設定 > アラート** を開く
2. 以下の項目を設定

#### 出勤忘れ通知
- **有効 / 無効**: 出勤忘れ通知を有効化
- **通知時刻**: 通知を送る時刻（例: 10:00）

#### 退勤忘れ通知
- **有効 / 無効**: 退勤忘れ通知を有効化
- **通知時刻**: 通知を送る時刻（例: 20:00）

#### QR期限切れ通知
- **有効 / 無効**: QR期限切れ通知を有効化

#### 長時間労働アラート
- **時間数**: アラートを送る勤務時間（時間）

3. **「保存」** をクリック

## トラブルシューティング

### 出勤打刻ボタンが表示されない

以下を確認してください：

1. **すでに出勤済みでないか確認**
   - 出勤済みの場合は、「退勤する」ボタンが表示されます

2. **勤怠設定を確認**
   - 管理者に勤怠機能が有効化されているか確認を依頼

3. **契約プランを確認**
   - 現場DXパックまたはフル機能パックが契約されているか確認

### QRスキャンで打刻できない

以下を確認してください：

1. **QRコードが有効か確認**
   - リーダーQRは24時間で期限切れになります
   - 会社QRは設定された期限で期限切れになります
   - 期限切れの場合は、再発行してください

2. **カメラ権限を確認**
   - ブラウザのカメラ権限が許可されているか確認

3. **正しいQRコードか確認**
   - 勤怠用のQRコードか確認（道具や倉庫位置のQRコードではないか）

### 休憩ボタンが表示されない

組織設定で休憩時間記録が「none」に設定されている可能性があります。管理者に確認してください。

## 注意事項

### 打刻時の注意

- **打刻忘れに注意**: 打刻忘れは管理者に修正を依頼する必要があります
- **正確な時刻に打刻**: 勤務時間の記録に影響します
- **退勤場所の変更**: 出勤時の予定と異なる場合は、退勤時に変更してください

### 運用上の注意

- **QRコードの管理**: リーダーQRは期限切れに注意
- **月次レポートの確認**: 給与計算の基礎データとなるため、確認が重要
- **アラート通知**: 出退勤忘れ通知を活用して打刻忘れを防止

### セキュリティ

- **他人の打刻は禁止**: 必ず本人が打刻してください
- **QRコードの共有に注意**: リーダーQRは信頼できるスタッフのみに共有
