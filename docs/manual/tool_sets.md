# 道具セット管理マニュアル

## 対象者

このマニュアルは、道具セットの登録・管理を行う以下の権限のユーザーを対象としています。

- **Manager（マネージャー）**: 道具セットの登録・編集が可能
- **Admin（管理者）**: 道具セットの登録・編集・削除が可能

※ Staff、Leaderは道具セットの閲覧とQRスキャンのみ可能です。登録・編集はできません。

---

## 前提条件

道具セット管理を始める前に、以下が完了していることを確認してください。

1. **道具の登録**: セットに含める道具が登録済みであること
2. **個別アイテムの確認**: セットに含める個別アイテム（#001、#002など）が存在すること

**関連**: [道具管理マニュアル](/manual/tool_management)

---

## 道具セットとは

### 1. 道具セットの目的

道具セットは、**よく一緒に使う道具の組み合わせ**をテンプレートとして登録する機能です。

**活用例:**
```
配管工事セット
├── 電動ドリル #001
├── パイプレンチ #012
├── 水平器 #005
└── 配管カッター #003

内装工事セット
├── 電動ドライバー #007
├── メジャー #015
├── カッターナイフ #022
└── 水平器 #006
```

### 2. メリット

道具セットを使うと、以下のメリットがあります：

- **移動登録が簡単**: 複数の道具をまとめて移動登録
- **忘れ物防止**: セット内容を確認して、必要な道具を漏れなく持ち出し
- **業務の標準化**: よく使う道具の組み合わせを共有
- **新人教育**: 「この作業にはこのセット」と明確化

### 3. 個別アイテムの明示的な指定

道具セットには、**個別アイテム（#001、#002など）を明示的に登録**します。

**なぜ個別アイテムを指定するのか:**

```
❌ 悪い例（道具マスタのみ登録）
「電動ドリル」と登録
→ 現場に持ち出す際、#001、#002、#003のどれを持っていくか不明確

✅ 良い例（個別アイテムを明示）
「電動ドリル #001」と登録
→ 現場に持ち出す際、#001を持っていくことが明確
```

**メリット:**
- 移動時の間違いを防ぐ
- 誰がどの個別アイテムを使っているか追跡可能
- セット内容が明確

## 道具セットの新規登録

### ステップ1: 基本情報の入力

1. **道具セット管理 > セット一覧** を開く
2. **「+ セットを追加」** をクリック
3. 以下の情報を入力

#### 基本情報（必須）
- **セット名**: セットの名称（例: 配管工事セット）
- **説明**: セットの用途や説明（例: 配管工事に必要な道具一式）

4. **「次へ」** をクリック

### ステップ2: 道具の追加

セットに含める道具を追加します。

1. **「+ 道具を追加」** をクリック
2. 道具マスタを選択（例: 電動ドリル）
3. **個別アイテム**を選択（例: #001）
4. **「追加」** をクリック
5. 繰り返して複数の道具を追加

**登録例:**
```
配管工事セット
├── 電動ドリル #001
├── パイプレンチ #012
├── 水平器 #005
└── 配管カッター #003
```

**注意:**
- 同じ道具マスタから複数の個別アイテムを追加できます
  - 例: 電動ドリル #001、電動ドリル #002
- 個別アイテムを明示的に指定することで、移動時の間違いを防ぎます

### ステップ3: 登録完了

1. 道具の追加が完了したら **「登録」** をクリック
2. 道具セットが作成されます

## 道具セットを使った移動登録

### QRスキャンによる一括移動

道具セットを使うと、複数の道具をまとめて移動登録できます。

1. スマートフォンで **QRスキャン** を開く
2. セット内の道具のQRコードを順番にスキャン
3. または、**道具セット選択** から登録済みのセットを選択
4. **移動先**を選択（倉庫 / 現場）
5. 現場の場合は、現場名を選択
6. **「一括移動」** をクリック

**メリット:**
- セット内の全道具を一度に移動登録
- スキャン漏れを防ぐチェックリスト機能
- 移動履歴が一括で記録される

### 手動での一括移動

1. **道具管理 > 一括移動** を開く
2. **道具セット**を選択
3. セット内の道具が自動的にリストに追加される
4. **移動元・移動先**を選択
5. **「一括移動」** をクリック

## 道具セットの編集

### セット名・説明の編集

1. **道具セット管理 > セット一覧** を開く
2. 編集したいセットをクリック
3. **「編集」** をクリック
4. セット名または説明を修正
5. **「保存」** をクリック

### セット内容（道具）の編集

セットに含まれる道具を追加・削除できます。

#### 道具を追加する場合

1. セット詳細画面を開く
2. **「+ 道具を追加」** をクリック
3. 道具マスタと個別アイテムを選択
4. **「追加」** をクリック

#### 道具を削除する場合

1. セット詳細画面を開く
2. 削除したい道具の **「削除」** ボタンをクリック
3. 確認ダイアログで **「削除」** をクリック

**注意:** セットから削除しても、道具マスタや個別アイテムは削除されません。

---

## 道具セットの削除

### 削除手順

1. **道具セット管理 > セット一覧** を開く
2. 削除したいセットをクリック
3. **「削除」** をクリック
4. 確認ダイアログで **「削除」** をクリック

**注意:**
- セットを削除しても、セット内の道具（道具マスタや個別アイテム）は削除されません
- 削除は論理削除（データは残るが非表示）です

## 道具セットの活用例

### 1. 工事種類別セット

工事の種類ごとにセットを作成します。

**例:**
- 配管工事セット
- 電気工事セット
- 内装工事セット
- 塗装工事セット

**メリット:** 工事の種類が決まれば、必要な道具が明確

### 2. 現場別セット

特定の現場専用のセットを作成します。

**例:**
- A現場用セット（Aビル建設）
- B現場用セット（道路工事）

**メリット:** 現場ごとに必要な道具が明確、持ち出し忘れ防止

### 3. 作業者別セット

作業者ごとによく使う道具をセット化します。

**例:**
- 田中さん専用セット
- 鈴木さん専用セット

**メリット:** 作業者が決まれば、その人が使う道具が明確

### 4. 緊急対応セット

緊急時に必要な道具をセット化します。

**例:**
- 緊急修理セット
- 応急処置セット

**メリット:** 緊急時でも必要な道具を素早く準備

## 道具セットのチェックリスト機能

### チェックリストとは

道具セットを使って移動登録する際、セット内の道具をチェックリスト形式で確認できます。

**使い方:**

1. **道具セット管理 > セット一覧** で対象のセットをクリック
2. **「チェックリスト表示」** をクリック
3. セット内の道具がリスト表示される
4. 実際に持ち出す道具にチェックを入れる
5. チェック済みの道具のみ移動登録

**メリット:**
- 持ち出し忘れ防止
- 確認しながら準備できる
- 不要な道具を除外できる

## 検索とフィルタリング

### 基本的な検索

1. **道具セット管理 > セット一覧** を開く
2. 画面上部の検索ボックスにキーワードを入力
3. セット名、説明で検索できます

### フィルタリング

**「フィルタ」** ボタンをクリックすると、以下の条件で絞り込みができます：

- **作成者**: セットを作成したユーザーで絞り込み
- **作成日**: 作成日の範囲で絞り込み

## トラブルシューティング

### 道具セットが見つからない場合の検索方法

道具セット一覧で目的のセットが見つからない場合は、以下の手順で検索してください。

1. **検索ボックスを使用**
   - 画面上部の検索ボックスにセット名を入力
   - 部分一致で検索可能

2. **フィルタリング機能を使用**
   - **「フィルタ」** ボタンをクリック
   - 作成者、作成日で絞り込み

3. **削除済みセットの確認**
   - 削除された道具セットは一覧に表示されません
   - 管理者に確認を依頼してください

### セット内の道具が削除された場合

セットに含まれている個別アイテムが削除された場合の動作：

1. セット詳細画面で、削除された道具には **「削除済み」** マークが表示されます
2. 削除済みの道具は移動登録時にスキップされます
3. セット編集画面で削除済みの道具を削除し、別の個別アイテムを追加してください

### セットから道具を削除する際の注意点

セットから道具を削除しても、以下は削除されません：

- ❌ 道具マスタは削除されません
- ❌ 個別アイテムは削除されません
- ❌ 移動履歴は削除されません

セットの構成が変わるだけです。安心して削除できます。

## 注意事項

### 登録時の注意

- **個別アイテムを明示的に指定**: 道具マスタだけでなく、個別アイテム（#001など）を指定してください
- **セット名をわかりやすく**: 用途や工事種類がわかる名前にすると便利
- **説明を記入**: セットの用途を説明に記載すると、他のユーザーにも分かりやすい

### 運用上の注意

- **定期的に見直し**: 使わない道具がセットに含まれていないか定期的に確認
- **セット数は適度に**: 多すぎると管理が大変なので、本当に必要なセットのみ作成
- **チーム内で共有**: 道具セットをチーム内で共有すると、業務の標準化に役立つ

### セキュリティ

- **削除は慎重に**: セットを削除すると復元できません（管理者に依頼すれば復元可能）
- **他社のセットは見えません**: マルチテナント型のため、他社データは完全分離
