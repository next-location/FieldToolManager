# はじめに - ザイロク（Zairoku）ユーザーマニュアル

## ザイロクへようこそ

ザイロク（Zairoku）は、建設現場や工事業向けの現場管理システムです。
道具・消耗品・重機の在庫管理から、出退勤管理、作業報告書、見積・請求書まで、現場業務を一元管理できます。

---

## このマニュアルの使い方

### 対象者

このマニュアルは、ザイロクを初めて使う方、または基本的な使い方を確認したい方を対象としています。

- **現場スタッフ（Staff）**: QRスキャン、出退勤、作業報告書の作成
- **リーダー（Leader）**: スタッフの業務 + 見積・請求・発注書作成、チーム管理
- **マネージャー（Manager）**: リーダーの業務 + 承認、道具・消耗品登録、分析レポート
- **管理者（Admin）**: 全機能 + 組織設定、スタッフ管理

### マニュアルの構成

各機能ごとにマニュアルが分かれています。必要な機能のマニュアルを参照してください。

## ザイロクの基本概念

### 1. 組織とマルチテナント

ザイロクは**マルチテナント型**のシステムです。
各企業（組織）のデータは完全に分離されており、他社のデータは一切見えません。

- **組織**: あなたの会社
- **サブドメイン**: あなたの会社専用のURL（例: `https://your-company.zairoku.com`）

### 2. 権限（ロール）

ザイロクには4つの権限があります。

| 権限 | 説明 | 主な業務 |
|------|------|---------|
| **Staff（スタッフ）** | 現場作業員 | QRスキャン、出退勤、作業報告書作成 |
| **Leader（リーダー）** | 現場監督、チームリーダー | スタッフの業務 + 見積・請求・発注書作成 |
| **Manager（マネージャー）** | 部門長、中間管理職 | リーダーの業務 + 承認、道具・消耗品登録、分析 |
| **Admin（管理者）** | IT管理者、経営層 | 全機能 + 組織設定、スタッフ管理 |

### 3. パッケージ（契約プラン）

ザイロクには2つのパッケージがあります。

#### 現場資産パック（¥18,000/月）
- 道具管理（QRコード、在庫管理）
- 消耗品管理
- 重機管理
- 移動履歴
- 倉庫位置管理

#### 現場DX業務効率化パック（¥22,000/月）
- 出退勤管理（QRコード打刻）
- 作業報告書（写真添付、PDF出力）
- 見積書・請求書・発注書
- 売上管理、入出金管理

#### フル機能統合パック（¥32,000/月）
- 現場資産パック + 現場DXパックの全機能
- 消耗品発注管理（フル機能パック限定）

**契約中のパッケージによって、使える機能が異なります。**

### 4. QRコードの活用

ザイロクの最大の特徴は**QRコード**を使った現場管理です。

- **道具・消耗品・重機**: 各アイテムに固有のQRコード
- **倉庫位置**: 倉庫内の場所にもQRコード
- **出退勤**: QRコードをスキャンして打刻

スマートフォンやタブレットでQRコードをスキャンするだけで、移動登録や出退勤が完了します。

## 初期設定の流れ（管理者向け）

ザイロクを使い始めるには、以下の初期設定が必要です。

### ステップ1: 組織情報の登録

1. **設定 > 組織情報** を開く
2. 会社名、住所、電話番号、代表者名を入力
3. インボイス登録番号（適格請求書発行事業者の場合）を入力
4. 保存

### ステップ2: スタッフの追加

1. **スタッフ管理** を開く
2. 「+ スタッフを追加」をクリック
3. 名前、メールアドレス、権限（Staff/Leader/Manager/Admin）を選択
4. 初期パスワードを設定（または自動生成）
5. 追加

**複数人を一度に登録する場合は、CSVファイルで一括登録も可能です。**

### ステップ3: 取引先・現場の登録

#### 取引先マスタ（顧客・仕入先）

1. **取引先管理** を開く
2. 「+ 取引先を追加」をクリック
3. 取引先名、住所、電話番号、取引先分類（顧客/仕入先）を入力
4. 保存

#### 現場（工事）の登録

1. **現場管理** を開く
2. 「+ 現場を追加」をクリック
3. 現場名、住所、発注者（取引先）を選択
4. 工期、予算を入力
5. 保存

### ステップ4: 倉庫位置の設定（現場資産パック契約時）

1. **設定 > 倉庫位置設定 > 階層設定** を開く
2. 倉庫の階層構造を設定（例: レベル1=エリア、レベル2=棚、レベル3=段）
3. **倉庫位置マスタ** で具体的な位置を登録
   - 例: 「Aエリア 1番棚 上段」
4. 各位置のQRコードを印刷して、実際の場所に貼り付け

**詳細は [倉庫位置管理マニュアル](./warehouse_locations.md) を参照**

### ステップ5: 道具・消耗品の登録（現場資産パック契約時）

1. **道具管理 > 道具一覧** を開く
2. 「+ 道具を追加」をクリック
3. 道具名、型番、メーカー、カテゴリ、個数を入力
4. 保存
5. 自動生成された個別アイテム（#001, #002...）のQRコードを印刷
6. 実際の道具に貼り付け

**詳細は [道具管理マニュアル](./tool_management.md) を参照**

### ステップ6: 運用設定

1. **設定 > 運用設定** を開く
2. 以下の設定を確認・変更
   - QRスキャン必須設定（移動時・返却時）
   - 紛失承認フロー
   - 月次棚卸し通知
3. 保存
