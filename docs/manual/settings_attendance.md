# 出退勤設定マニュアル

## 概要

出退勤設定（勤怠管理設定）では、打刻方法、休憩時間の記録方式、リマインダー通知、管理者向けレポート・アラートなど、出退勤管理に関する組織全体の設定を変更できます。

## 対象者

- 管理者（Admin）
- マネージャー（Manager）

※ リーダーおよび一般スタッフは設定ページにアクセスできません。

## アクセス方法

1. ダッシュボードまたはサイドメニューから「設定」をクリック
2. 「出退勤設定」または「勤怠管理設定」をクリック

または、サイドメニューから「設定」→「出退勤設定」を直接選択

## 設定項目一覧

出退勤設定は以下のセクションに分かれています：

1. 打刻方法設定
2. 休憩時間設定
3. リマインダー設定
4. 管理者向け設定

---

## 1. 打刻方法設定

スタッフが出退勤打刻で利用可能な方法を選択します。複数選択可能です。

### 設定項目

#### 手動打刻（ボタンタップ）

**説明**: 場所を選択して打刻ボタンをタップする方法です。

**効果**:
- ✅ 有効: スタッフは打刻画面で場所を選択し、「出勤」「退勤」ボタンをタップできます
- ❌ 無効: 手動打刻ボタンが非表示になります

**推奨**: 必ず有効にすることを推奨（基本的な打刻方法）

#### QRコード打刻

**説明**: QRコードをスキャンして場所を自動判別する方法です。

**効果**:
- ✅ 有効: スタッフはQRコードをスキャンして打刻できます。場所が自動判別されます
- ❌ 無効: QRコード打刻機能が非表示になります

**推奨**: 固定場所（オフィス、現場）での打刻を厳密に管理したい場合に有効化

### QRコード更新頻度

QRコード打刻を有効にした場合、QRコードの更新頻度を設定します。

| 更新頻度 | 説明 | 推奨シーン |
|---------|------|-----------|
| **1日** | 毎日更新（最高セキュリティ） | 厳格な労務管理が必要な組織 |
| **3日** | 3日ごとに更新 | セキュリティと運用負荷のバランス |
| **7日（推奨）** | 週1回更新 | 多くの組織に最適 |
| **30日** | 月1回更新（運用負荷最小） | QRコードの掲示替えが困難な現場 |

**デフォルト**: 7日（週1回更新）

### 注意事項

- 少なくとも1つの打刻方法を有効にする必要があります
- 手動打刻のみを有効にする運用が最もシンプルです
- QRコード打刻を追加すると、固定場所での打刻を厳密に管理できます

## 2. 休憩時間設定

休憩時間の記録方法を設定します。

### 休憩時間モード

| モード | 説明 | 使用シーン |
|--------|------|-----------|
| **休憩時間を記録しない** | 休憩時間は記録せず、勤務時間のみを管理 | 休憩時間の記録が不要な組織 |
| **簡易モード（休憩開始・終了ボタン）** | 1回の休憩のみを記録。開始・終了ボタンで記録 | 昼休憩のみを記録する組織 |
| **詳細モード（複数回の休憩を記録）** | 複数回の休憩を記録可能 | 複数回の休憩を正確に記録したい組織 |

**デフォルト**: 簡易モード

### 休憩時間を自動差し引き

**説明**: 勤務時間から休憩時間を自動的に差し引きます。

**効果**:
- ✅ 有効: スタッフが休憩開始・終了ボタンを押さなくても、指定した時間を自動で差し引きます
- ❌ 無効: 休憩時間は記録されません（または手動で記録）

**推奨**: 休憩時間が固定の組織（例：毎日45分の昼休憩）

### 自動差し引き時間（分）

休憩時間を自動差し引きを有効にした場合、差し引く時間を設定します。

- **設定範囲**: 0〜120分
- **入力単位**: 15分単位を推奨
- **デフォルト**: 45分

**例**：
- 45分: 一般的な昼休憩
- 60分: 1時間の昼休憩
- 90分: 1時間30分の昼休憩

## 3. リマインダー設定

打刻忘れを防ぐためのリマインダー通知を設定します。

### 出勤リマインダー

**説明**: 指定した時刻に、まだ出勤打刻していないスタッフに通知を送信します。

**効果**:
- ✅ 有効: 設定した時刻に出勤リマインダーを送信
- ❌ 無効: 出勤リマインダーを送信しない

**デフォルト**: 有効（10:00）

**推奨設定時刻**: 始業時刻の30分〜1時間後

**例**：
- 始業時刻が9:00の場合: 10:00に設定
- 始業時刻が8:00の場合: 9:00に設定

### 退勤リマインダー

**説明**: 指定した時刻に、まだ退勤打刻していないスタッフに通知を送信します。

**効果**:
- ✅ 有効: 設定した時刻に退勤リマインダーを送信
- ❌ 無効: 退勤リマインダーを送信しない

**デフォルト**: 有効（20:00）

**推奨設定時刻**: 定時終業時刻の1〜2時間後

**例**：
- 定時が18:00の場合: 20:00に設定
- 定時が17:00の場合: 19:00に設定

## 4. 管理者向け設定

レポートとアラートの設定です。管理者とマネージャーに送信されます。

### 日次レポート

**説明**: 毎日指定した時刻に、出退勤状況のレポートを送信します。

**効果**:
- ✅ 有効: 設定した時刻に日次レポートを送信
- ❌ 無効: 日次レポートを送信しない

**デフォルト**: 有効（10:00）

**レポート内容**:
- 前日の出退勤状況
- 打刻忘れのスタッフ一覧
- 長時間労働のスタッフ一覧

#### 送信時刻

日次レポートを送信する時刻を設定します。

**推奨時刻**: 始業時刻の30分〜1時間後

**例**: 9:00始業の場合は10:00に設定

#### メールで送信

**説明**: 日次レポートをメールでも送信します。

**効果**:
- ✅ 有効: システム内通知とメールの両方で送信
- ❌ 無効: システム内通知のみ

**推奨**: 有効（メールで確認できると便利）

### 長時間労働アラート

**説明**: 指定した時間を超えて勤務しているスタッフに対してアラートを送信します。

**効果**:
- ✅ 有効: 設定した基準時間を超えると管理者に通知
- ❌ 無効: 長時間労働アラートを送信しない

**デフォルト**: 無効

**推奨**: 有効（長時間労働を防ぐため）

#### アラート基準時間（時間）

長時間労働アラートの基準となる勤務時間を設定します。

- **設定範囲**: 8〜24時間
- **デフォルト**: 12時間

**例**：
- 12時間: 残業を含めて12時間を超えたら通知
- 10時間: 残業2時間を超えたら通知（8時間勤務の場合）
- 15時間: 深夜勤務が発生したら通知

## 設定の保存

### 手順

1. 各セクションで設定を変更
2. ページ下部の「設定を保存」ボタンをクリック
3. 成功メッセージが表示されます

### 保存後の確認

- 成功メッセージがページ上部に緑色の背景で表示されます
- エラーがある場合は、赤色の背景でエラーメッセージが表示されます

---

## 設定のベストプラクティス

### 初期設定時の推奨設定

新規導入時に推奨する設定：

| 設定項目 | 推奨値 | 理由 |
|---------|--------|------|
| 手動打刻 | ✅ 有効 | 基本的な打刻方法 |
| QRコード打刻 | ❌ 無効（運用に慣れたら有効化） | 最初はシンプルに |
| QR更新頻度 | 7日 | バランスが良い |
| 休憩時間モード | 簡易モード | 1回の休憩記録 |
| 自動休憩差し引き | ❌ 無効（運用に慣れたら有効化） | 最初は手動記録 |
| 出勤リマインダー | ✅ 有効（10:00） | 打刻忘れを防ぐ |
| 退勤リマインダー | ✅ 有効（20:00） | 打刻忘れを防ぐ |
| 日次レポート | ✅ 有効（10:00） | 管理者が状況把握 |
| 長時間労働アラート | ✅ 有効（12時間） | 長時間労働を防ぐ |

### 運用に慣れたら

運用に慣れてきたら、以下の変更を検討：

1. **QRコード打刻を有効化**: 固定場所での打刻を厳密に管理
2. **自動休憩差し引きを有効化**: 休憩時間が固定の場合、運用が楽になる
3. **休憩時間モードを詳細モードに変更**: 複数回の休憩を正確に記録
4. **長時間労働アラートの基準時間を調整**: 組織の実態に合わせて調整

## よくある設定パターン

### パターン1: シンプル運用（小規模組織向け）

```
打刻方法: 手動打刻のみ
休憩時間: 簡易モード + 自動差し引き45分
リマインダー: 出勤・退勤ともに有効
日次レポート: 有効
長時間労働アラート: 無効
```

**特徴**: 最小限の設定で運用開始。打刻忘れを防ぎつつ、シンプルな管理。

### パターン2: 厳密管理（中規模〜大規模組織向け）

```
打刻方法: 手動打刻 + QRコード打刻（7日更新）
休憩時間: 詳細モード（自動差し引きなし）
リマインダー: 出勤・退勤ともに有効
日次レポート: 有効 + メール送信
長時間労働アラート: 有効（12時間）
```

**特徴**: QRコード打刻で場所を厳密に管理。複数回の休憩を正確に記録。長時間労働を監視。

### パターン3: 完全自動化（運用負荷最小）

```
打刻方法: 手動打刻のみ
休憩時間: 簡易モード + 自動差し引き60分
リマインダー: 出勤・退勤ともに有効
日次レポート: 有効
長時間労働アラート: 有効（10時間）
```

**特徴**: 休憩時間を自動差し引きで管理。リマインダーとレポートで自動化。

## トラブルシューティング

### 設定ページにアクセスできない

**原因**: リーダーまたは一般スタッフでログインしている

**対処法**:
- 管理者またはマネージャーの権限が必要です
- 権限が必要な場合は、管理者に権限変更を依頼してください

### 設定を保存できない

**原因**: 必須項目が未入力、または入力値が不正

**対処法**:
1. すべての時刻が正しく入力されているか確認（HH:MM形式）
2. 自動差し引き時間が0〜120分の範囲内か確認
3. 長時間労働アラートの基準時間が8〜24時間の範囲内か確認
4. ネットワーク接続を確認
5. ブラウザをリフレッシュ（F5）して再度試してください

### 設定変更後もスタッフに反映されない

**対処法**:
1. スタッフにアプリをリフレッシュしてもらってください（アプリを閉じて再起動）
2. ブラウザ版を使用している場合は、ブラウザをリフレッシュ（F5）してもらってください
3. それでも反映されない場合は、一度ログアウトして再ログインしてもらってください

### リマインダー通知が送信されない

**原因1**: リマインダーが無効になっている

**対処法**: 出勤リマインダー・退勤リマインダーが有効（チェックボックスがON）になっているか確認してください

**原因2**: 通知設定の時刻が正しくない

**対処法**: 時刻が正しく入力されているか確認してください（HH:MM形式）

**原因3**: スタッフの通知設定がオフになっている

**対処法**: スタッフに通知設定を確認してもらってください（ブラウザまたはアプリの通知設定）

### 日次レポートが送信されない

**原因1**: 日次レポートが無効になっている

**対処法**: 日次レポートが有効（チェックボックスがON）になっているか確認してください

**原因2**: メールアドレスが登録されていない

**対処法**: 管理者・マネージャーのメールアドレスが登録されているか確認してください

### QRコードが読み取れない

**原因1**: QRコードが古い（有効期限切れ）

**対処法**: QRコード更新頻度の設定に応じて、QRコードを再生成してください

**原因2**: QRコード打刻が無効になっている

**対処法**: 出退勤設定で「QRコード打刻」が有効になっているか確認してください
