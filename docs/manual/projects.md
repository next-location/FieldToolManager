## 概要

工事管理機能では、建設プロジェクト（工事案件）の情報を一元管理できます。取引先・現場との紐づけ、工期や契約金額の管理、発注書の集計など、工事全体の進捗や予算を把握するための機能です。

**主な用途**
- 工事案件の基本情報管理（工事番号、工事名、発注者など）
- 工期・契約金額・予算の管理
- 発注書との紐づけによる予算消化状況の可視化
- 工事責任者の設定とステータス管理

**アクセス権限**
- **管理者、マネージャーのみアクセス可能**
- リーダーおよびスタッフは工事管理ページにアクセスできません

## 工事一覧の確認

### 一覧画面へのアクセス

サイドメニューから「工事管理」をクリックすると、工事一覧画面が表示されます。

### 表示内容

一覧画面では、各工事について以下の情報が表示されます。

- **工事番号**: 自動生成される一意の識別番号（例: PRJ-2026-0001）
- **工事名**: 工事の名称（例: 〇〇ビル新築工事）
- **ステータス**: 計画中 / 進行中 / 完了 / キャンセル
- **取引先**: 発注者（顧客）
- **現場**: 紐づく現場名
- **契約金額**: 発注者との契約金額
- **工期**: 工事開始日～完了予定日

## 新規工事の登録

### 登録手順

1. 工事一覧画面の右上にある「+ 新規工事登録」ボタンをクリックします（PCのみ表示）。
2. モバイルの場合は、画面右下の青いFABボタン（+アイコン）をタップします。
3. 工事登録フォームが表示されます。

### 入力項目

#### 基本情報

| 項目 | 必須 | 説明 |
|------|------|------|
| 工事番号 | - | 自動生成（PRJ-YYYY-NNNN形式）。編集不可 |
| ステータス | ✅ | 計画中 / 進行中 / 完了 / キャンセル から選択 |
| 工事名 | ✅ | 工事の名称（例: 〇〇ビル新築工事） |
| 発注者（取引先） | - | 検索ボックスで顧客を選択 |
| 関連現場 | - | 取引先を選択すると、その取引先に紐づく現場が表示されます |
| 工事責任者 | - | 工事を管理する責任者を選択 |

**ポイント**
- **工事番号**: 年度ごとに連番で自動採番されます（例: PRJ-2026-0001, PRJ-2026-0002...）
- **発注者（取引先）**: 顧客（customer）または顧客兼仕入先（both）のみが選択肢に表示されます
- **関連現場**: 取引先を選択すると、その取引先に紐づく現場のみが表示されます
  - 紐づく現場が1件だけの場合、自動的に選択されます
  - 紐づく現場がない場合、選択できません
  - 取引先を未選択の場合、現場選択欄は無効（グレーアウト）になります

#### 工期・金額

| 項目 | 必須 | 説明 |
|------|------|------|
| 工事開始日 | - | 工事の開始予定日 |
| 工事完了予定日 | - | 工事の完了予定日（開始日以降のみ選択可能） |
| 契約金額 | - | 発注者との契約金額（円） |
| 予算金額 | - | 工事の予算金額（円） |

**ポイント**
- **契約金額**: 発注者から受け取る金額
- **予算金額**: 工事にかける予算。発注書の承認済金額と比較して予算消化率が計算されます
- 金額は半角数字で入力してください（カンマ不要）

### 登録の完了

すべての必須項目を入力後、「登録」ボタンをクリックすると工事が登録され、一覧画面に戻ります。

## 工事詳細の確認

### 詳細画面へのアクセス

工事一覧から任意の工事カードの「詳細」ボタンをクリックすると、詳細画面が表示されます。

### 表示内容

#### 基本情報

- ステータス（バッジ表示）
- 工事番号
- 工事名
- 発注者（取引先）
- 工事責任者（役割も表示）
- 関連現場（現場名・住所・取引先名）

#### 工期・金額

- 工事開始日
- 工事完了予定日
- 契約金額
- 予算金額

#### 発注情報

工事に紐づく発注書の集計情報が表示されます。

| 項目 | 説明 |
|------|------|
| 発注書件数 | この工事に紐づく発注書の総数 |
| 発注総額 | すべての発注書の合計金額 |
| 承認済件数 | 承認済み以降のステータス（承認済み / 発注済み / 一部納品 / 納品済み / 支払済み）の発注書件数 |
| 承認済金額 | 承認済み以降のステータスの発注書の合計金額 |

**予算消化率**
- 予算金額が設定されている場合、予算消化率が進捗バーで表示されます
- 消化率80%超で黄色、100%超で赤色に変化します
- 予算残額も表示されます

**発注書一覧へのリンク**
- 「発注書一覧を見る →」をクリックすると、この工事に絞り込まれた発注書一覧が表示されます

#### システム情報

- 作成日時
- 更新日時

## 工事情報の編集

### 編集手順

1. 工事詳細画面の「編集」ボタンをクリックします（管理者・リーダーのみ表示）。
2. 編集フォームが表示されます。
3. 変更したい項目を修正します。
4. 「更新」ボタンをクリックすると変更が保存されます。

**注意事項**
- 工事番号は編集できません
- ステータスを「完了」または「キャンセル」に変更しても、紐づく発注書には影響しません

## 検索・フィルター機能

### PC版

検索・フィルターエリアに以下の条件を設定できます。

| 項目 | 説明 |
|------|------|
| キーワード検索 | 工事番号・工事名・取引先名で部分一致検索 |
| ステータス | すべて / 計画中 / 進行中 / 完了 / キャンセル から選択 |

「クリア」ボタンをクリックすると、すべての検索条件がリセットされます。

### モバイル版

- 画面上部の検索ボックスでキーワード検索が可能です
- 検索ボックス右側のフィルターアイコン（スライダー）をタップすると、フィルターモーダルが開きます
- フィルター適用中は、フィルターアイコンに数字（適用中の条件数）が表示されます

### 検索結果の表示

- 検索結果の件数は「全 XX 件中 XX 件を表示」のように表示されます
- 該当データがない場合、「検索条件に一致する工事がありません」と表示されます

## 関連機能との連携

### 発注書との連携

- 発注書作成時に工事を選択することで、発注書と工事を紐づけられます
- 工事詳細画面で承認済金額や予算消化率を確認できます
- 発注書一覧で工事でフィルタリングして表示できます

### 取引先管理との連携

- 工事の発注者として、顧客または顧客兼仕入先の取引先を選択できます
- 仕入先（supplier）や協力会社（partner）のみの取引先は選択肢に表示されません

### 現場管理との連携

- 工事に現場を紐づけることで、工事と現場の関係を明確化できます
- 取引先を選択すると、その取引先に紐づく現場のみが表示されます
- 工事詳細画面から現場詳細画面へ直接リンクできます
