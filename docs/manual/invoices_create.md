# 請求書作成マニュアル

## 📋 概要

請求書作成機能では、取引先への請求書を作成し、提出・承認フローに進むことができます。見積書から自動変換することも、新規作成することも可能です。

## 👥 対象者

- **リーダー以上**: 請求書の作成・提出が可能
- **マネージャー以上**: 請求書の作成・提出・承認が可能
- **管理者**: すべての請求書操作が可能

**注意**: スタッフは請求書を作成できません。

## 📍 アクセス方法

1. 左メニューから「請求書」をクリック
2. 「+ 新規請求書」ボタンをクリック

または

1. 見積書詳細画面から「請求書に変換」ボタンをクリック（見積書承認後）

## 🔄 請求書作成の流れ

### 1️⃣ 新規作成の場合

```
基本情報入力 → 明細入力 → 備考入力 → 保存・提出
```

### 2️⃣ 見積書から変換の場合

```
見積書選択 → 自動データ読込 → 内容確認・修正 → 保存・提出
```

## 📝 基本情報の入力

### 請求番号（自動生成）

- **形式**: `INV-年月-連番`
- **例**: `INV-202601-0001`（2026年1月の1件目）
- 自動で採番されるため、編集不可

### 取引先（必須）

- **選択対象**: 「顧客」または「顧客兼仕入先」として登録済みの取引先のみ
- **表示形式**: `取引先名 (取引先コード)`
- 取引先を選択すると、支払期日が自動計算されます

**支払期日の自動計算**:
- 取引先マスタに「支払条件（◯日後）」が登録されている場合
- 請求日 + 支払条件日数 = 支払期日として自動設定
- 例: 請求日が1月10日、支払条件が30日後 → 支払期日は2月9日

### 工事（任意）

- **選択対象**: 「計画中」または「進行中」の工事のみ
- **表示形式**: `工事名 (工事コード)`
- 工事に紐づけることで、工事別の売上管理が可能

### 請求日（必須）

- デフォルト: 本日の日付
- 手動で変更可能

### 支払期日（必須）

- デフォルト: 請求日 + 30日後
- 取引先選択時に支払条件から自動計算
- 手動で変更可能

### 件名（必須）

- **例**: `〇〇工事 請求書`
- 最大200文字
- 見積書から変換した場合、「見積書」→「請求書」に自動変換されます

### 適格請求書発行事業者

- 組織情報に登録されている場合、自動で表示されます
- **表示内容**: 登録番号（T + 13桁の数字）
- 適格請求書（インボイス制度対応）として発行されます

## 📊 明細の入力

明細はカード形式で1件ずつ入力します。

### 種別

以下から選択します：

| 種別 | 説明 |
|------|------|
| 工事費 | 建設工事費、施工費など |
| 材料費 | 建設資材、部品など |
| 諸経費 | 交通費、管理費など |
| その他 | カスタム種別を入力可能 |

**「その他」を選択した場合**:
- 「カスタム種別」入力欄が表示されます
- 例: 運搬費、設計費、調査費など（最大50文字）

### 項目名（必須）

- 明細の名称を入力
- 最大100文字
- 例: 基礎工事一式、コンクリート打設、足場設置など

### 説明（任意）

- 明細の詳細説明
- 最大200文字
- 例: 鉄筋コンクリート造、厚さ15cm、面積50m²

### 数量（必須）

- 数値を入力
- 小数点も入力可能
- 例: 1、10.5、100

### 単位（必須）

以下から選択、または「その他」でカスタム単位を入力：

| 単位 | 用途 |
|------|------|
| 式 | 一式の工事、まとまった作業 |
| 個 | 個数 |
| 台 | 機械・設備の台数 |
| 本 | 長尺物の本数 |
| 枚 | 板材などの枚数 |
| m | 長さ |
| m² | 面積 |
| m³ | 体積 |
| kg | 重量（キログラム） |
| t | 重量（トン） |
| L | 容量（リットル） |
| 日 | 日数 |
| 時間 | 時間数 |
| 人 | 人数 |
| その他 | カスタム単位を入力 |

**「その他」を選択した場合**:
- 「カスタム単位」入力欄が表示されます
- 例: セット、組、巻、袋など（最大20文字）

### 単価（必須）

- 1単位あたりの価格（税抜）
- 数値を入力
- 例: 10000、50000、100000

### 税率

以下から選択：

| 税率 | 用途 |
|------|------|
| 10% | 標準税率（デフォルト） |
| 8% | 軽減税率（飲食料品など） |
| 0% | 非課税（特定条件下） |

### 金額（自動計算）

- **計算式**: 数量 × 単価 = 金額
- 編集不可（自動計算）
- 例: 数量10 × 単価50,000円 = 金額500,000円

### 明細の追加・削除

- **追加**: 「+ 明細を追加」ボタンをクリック
- **削除**: 各明細カードの「削除」ボタンをクリック
- **注意**: 明細は最低1件必要（すべて削除不可）

## 💰 合計金額の確認

画面右下に自動計算された金額が表示されます：

| 項目 | 説明 |
|------|------|
| 小計 | 全明細の金額合計（税抜） |
| 消費税 | 各明細の消費税合計（小数点以下切り捨て） |
| 合計 | 小計 + 消費税（請求金額） |

**計算式**:
```
小計 = Σ(数量 × 単価)
消費税 = Σ(金額 × 税率 / 100) ※切り捨て
合計 = 小計 + 消費税
```

## 📝 備考の入力

### 備考（請求書に記載）

- 請求書PDFに記載される備考
- 最大2000文字
- **例**:
  - お振込手数料はお客様ご負担でお願いいたします。
  - 支払期日を過ぎた場合、遅延損害金が発生します。
  - 不明点がございましたらお気軽にお問い合わせください。

### 社内メモ

- 請求書には記載されない、社内用のメモ
- 最大2000文字
- **例**:
  - 前回の請求で指摘があったため、明細を詳しく記載
  - 支払い遅延の常習取引先のため注意
  - 次回は値上げ交渉予定

## 💾 保存・提出

画面下部に3つのボタンが表示されます：

### キャンセル

- 入力内容を破棄して請求書一覧に戻ります
- 入力内容は保存されません

### 下書き

- **ステータス**: 「下書き」として保存
- **誰が使う**: リーダー、マネージャー、管理者
- **用途**: 一旦保存して後で編集したい場合
- 下書きは何度でも編集可能

### 提出 / 承認

**リーダーの場合**:
- ボタン表示: 「提出」
- **ステータス**: 「提出済み」として保存
- マネージャー以上に承認を依頼

**マネージャー・管理者の場合**:
- ボタン表示: 「承認」
- **ステータス**: 「承認済み」として保存
- 承認フローをスキップして直接承認

## 📋 見積書から変換する場合

### 変換できる見積書

- **条件**: 「顧客承認」ステータスの見積書のみ
- 見積書詳細画面に「請求書に変換」ボタンが表示されます

### 自動コピーされる内容

以下の情報が見積書から自動的にコピーされます：

| 項目 | 変換内容 |
|------|----------|
| 取引先 | そのままコピー |
| 工事 | そのままコピー |
| 件名 | 「見積書」→「請求書」に自動変換 |
| 明細 | すべての明細がコピー |
| 備考 | そのままコピー |
| 支払期日 | 取引先の支払条件から自動計算 |

### 変換後の編集

- コピーされた内容は自由に編集可能
- 明細の追加・削除も可能
- 金額の修正も可能

## ⚠️ 注意事項

### 権限について

- **スタッフ**: 請求書を作成できません
- **リーダー**: 作成・提出は可能、承認はできません
- **マネージャー以上**: 作成・提出・承認がすべて可能

### 取引先の登録

- 請求書を作成する前に、取引先マスタに「顧客」として登録が必要
- 支払条件も事前に登録しておくと、支払期日が自動計算されます

### 請求番号

- 自動採番のため、欠番が発生する可能性があります
  - 例: 下書き保存後に削除した場合
- 請求番号は変更できません

### 適格請求書（インボイス）

- 組織情報で「適格請求書発行事業者」として登録済みの場合のみ有効
- 登録番号が請求書に自動記載されます
- 登録していない場合は通常の請求書として発行されます

### 見積書から変換

- 見積書を変換しても、元の見積書は残ります（削除されません）
- 変換後に見積書が変更されても、請求書には反映されません
- 請求書と見積書は関連付けられますが、独立して管理されます
