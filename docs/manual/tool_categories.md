# 道具カテゴリ管理マニュアル

## 概要

道具カテゴリ管理機能は、道具や消耗品を分類するためのカテゴリを作成・編集・削除する機能です。カテゴリを設定することで、道具や消耗品を整理しやすくなり、検索やフィルタリングが効率的になります。

## 対象者

- マネージャー（Manager）: カテゴリの作成・編集が可能
- 管理者（Admin）: カテゴリの作成・編集・削除が可能

※ リーダーおよび一般スタッフはカテゴリ管理機能にアクセスできません。

## アクセス方法

1. ダッシュボードまたはサイドメニューから「道具マスタ」をクリック
2. 「カテゴリ管理」タブをクリック

## カテゴリの新規作成

### 手順

#### 1. 新規作成画面を開く

1. 「カテゴリ管理」タブを開く
2. 右上の「新規作成」ボタンをクリック
3. カテゴリ登録フォームのモーダルが表示されます

#### 2. カテゴリ情報を入力

##### カテゴリ名（必須）

- 最大50文字
- カテゴリの名称を入力します
- 例：「養生材」「保護具」「接着・固定材」「電動工具」「測定器」

**入力例**：
```
✓ 良い例:
- 養生材
- 電動工具
- 測定器
- 保護具
- 接着・固定材

✗ 悪い例:
- カテゴリ1（具体的でない）
- その他（分類として不明確）
```

##### 説明（任意）

- 最大500文字
- カテゴリに含まれる道具の種類や用途を説明します
- 例：「養生テープ、ブルーシート、マスキングテープ等」

**入力例**：
```
カテゴリ名: 養生材
説明: 養生テープ、ブルーシート、マスキングテープ、ガムテープなど、作業現場を保護するための資材

カテゴリ名: 電動工具
説明: ドリル、インパクトドライバー、グラインダー、丸ノコなど、電動で動作する工具類

カテゴリ名: 測定器
説明: レーザー距離計、水平器、メジャー、ノギスなど、計測・測定を行う道具
```

#### 3. 登録

- すべての必須項目を入力後、「登録」ボタンをクリック
- 成功メッセージが表示され、カテゴリ一覧に追加されます

## カテゴリの編集

### 手順

1. カテゴリ一覧から編集したいカテゴリを探す
2. カテゴリの右側にある「編集」ボタンをクリック
3. カテゴリ編集フォームのモーダルが表示されます
4. カテゴリ名または説明を修正
5. 「更新」ボタンをクリック

### 注意事項

- システムカテゴリ（「消耗品」など）は編集できません
- カテゴリ名を変更しても、既存の道具・消耗品との紐付けは維持されます

## カテゴリの削除

### 対象者

- 管理者（Admin）のみ

マネージャーはカテゴリを削除できません。削除が必要な場合は管理者に依頼してください。

### 手順

1. カテゴリ一覧から削除したいカテゴリを探す
2. カテゴリの右側にある「削除」ボタンをクリック
3. 確認ダイアログが表示されます：
   ```
   このカテゴリを削除してもよろしいですか？
   ※このカテゴリを使用している道具・消耗品は削除されません。
   ```
4. 「OK」をクリックすると削除されます

### 削除時の影響

- **カテゴリのみが削除されます**
- このカテゴリを使用している道具・消耗品は削除されません
- 既存の道具・消耗品のカテゴリ設定は「未設定」になります
- 削除されたカテゴリは論理削除され、完全には消えません（管理者が復元可能）

## カテゴリの活用方法

### 推奨カテゴリ例

建設・工事現場で一般的に使用されるカテゴリの例：

| カテゴリ名 | 説明 | 含まれる道具の例 |
|-----------|------|-----------------|
| **養生材** | 現場保護用資材 | 養生テープ、ブルーシート、マスキングテープ |
| **電動工具** | 電動で動作する工具 | ドリル、インパクトドライバー、グラインダー |
| **手工具** | 手動で使用する工具 | ハンマー、スパナ、ドライバー |
| **測定器** | 計測・測定用機器 | レーザー距離計、水平器、メジャー |
| **保護具** | 安全保護用品 | ヘルメット、安全帯、保護メガネ |
| **接着・固定材** | 接着剤・固定用資材 | 接着剤、ビス、釘、アンカー |
| **塗装用品** | 塗装関連資材 | ハケ、ローラー、塗料、シンナー |
| **配線・電気** | 電気工事用資材 | ケーブル、コンセント、スイッチ、端子 |
| **配管・水道** | 配管工事用資材 | パイプ、継手、バルブ、シール材 |
| **消耗品** | 使い捨て資材 | 軍手、タオル、ゴミ袋、マスク |

### カテゴリ設計のポイント

#### 1. 具体的で明確な名称

- ❌ 悪い例：「カテゴリA」「その他」「雑貨」
- ✅ 良い例：「電動工具」「測定器」「保護具」

#### 2. 適切な粒度

- **細かすぎる分類は避ける**：カテゴリが100個以上になると管理が煩雑
- **大まかすぎる分類も避ける**：「工具」だけでは分類の意味がない
- **推奨：10〜30カテゴリ程度**

#### 3. 現場のニーズに合わせる

- 使用頻度の高い道具種別を優先してカテゴリ化
- 現場の作業内容に応じてカスタマイズ
- 例：電気工事が多い現場では「配線・電気」を細分化

#### 4. 統一性のある命名

- 名詞で統一：「電動工具」「測定器」「保護具」
- 用途で統一：「塗装用品」「配管用品」「配線用品」

## カテゴリによる検索・フィルタリング

カテゴリを設定すると、以下の画面で活用できます：

### 1. 道具マスタ画面

- カテゴリでフィルタリングして道具マスタを表示
- PC版：検索・フィルター欄でカテゴリを選択
- モバイル版：フィルターアイコンからカテゴリを選択

### 2. 道具一覧画面

- カテゴリでフィルタリングして道具を表示
- 特定のカテゴリの道具だけを素早く見つけられる

### 3. 消耗品一覧画面

- カテゴリでフィルタリングして消耗品を表示
- 在庫確認時にカテゴリごとに整理できる

### 4. 道具・消耗品の新規登録画面

- カテゴリをドロップダウンから選択
- 登録時に適切なカテゴリを設定することで、後の管理が楽になる

## システムカテゴリについて

### 「消耗品」カテゴリ

システムによって自動作成される特別なカテゴリです。

#### 特徴

- カテゴリ一覧に「システム」バッジが表示されます
- 編集・削除ができません
- 消耗品管理機能で自動的に使用されます

#### 使用目的

- 道具（個別管理）と消耗品（在庫管理）を明確に分類するため
- 消耗品登録時にデフォルトで設定される場合があります

## 注意事項

### 権限について

| 操作 | リーダー | マネージャー | 管理者 |
|------|---------|-------------|--------|
| カテゴリ閲覧 | ❌ | ✅ | ✅ |
| カテゴリ作成 | ❌ | ✅ | ✅ |
| カテゴリ編集 | ❌ | ✅ | ✅ |
| カテゴリ削除 | ❌ | ❌ | ✅ |

### カテゴリ名の重複

- 同じ組織内で同一のカテゴリ名を複数作成できます
- ただし、混乱を避けるため重複は避けることを推奨します

### カテゴリ削除の影響

- カテゴリを削除しても、道具・消耗品は削除されません
- 削除されたカテゴリに紐付いていた道具・消耗品は「カテゴリ未設定」になります
- 必要に応じて、別のカテゴリを再設定してください

### カテゴリの復元

- 削除されたカテゴリは論理削除（deleted_at）されます
- システム管理者がデータベースから復元することが可能です
- 誤って削除した場合は、管理者に連絡してください

## トラブルシューティング

### カテゴリ管理タブが表示されない

**原因**：リーダーまたは一般スタッフでログインしている

**対処法**：
- マネージャーまたは管理者の権限が必要です
- 権限が必要な場合は、管理者に権限変更を依頼してください

### カテゴリを削除できない

**原因1**：マネージャーでログインしている

**対処法**：
- カテゴリの削除は管理者（Admin）のみが可能です
- 削除が必要な場合は、管理者に依頼してください

**原因2**：システムカテゴリ（「消耗品」など）を削除しようとしている

**対処法**：
- システムカテゴリは削除できません
- 「削除」ボタンが表示されません

### カテゴリ登録時にエラーが表示される

**エラー例**：「登録に失敗しました」

**対処法**：
1. カテゴリ名が入力されているか確認
2. カテゴリ名が50文字以内か確認
3. 説明が500文字以内か確認
4. ネットワーク接続を確認
5. ブラウザをリフレッシュして再度試す
6. 問題が解決しない場合は、管理者に連絡

### カテゴリを編集しても道具に反映されない

**原因**：カテゴリ名を変更しても、道具との紐付けはカテゴリIDで管理されています

**対処法**：
- カテゴリ名を変更すると、そのカテゴリを使用している道具の表示名も自動的に変更されます
- 道具詳細ページやフィルター欄で、新しいカテゴリ名が表示されるか確認してください
- 表示が更新されない場合は、ブラウザをリフレッシュ（F5）してください
