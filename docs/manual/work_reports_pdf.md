# 作業報告書PDF出力マニュアル

## 1. 概要

作業報告書をPDF形式で出力する機能です。取引先への提出、社内保管、印刷など、様々な用途で活用できます。

**主な機能**:
- 個別PDF出力（報告書詳細画面から）
- 一括PDF出力（承認待ち一覧から複数選択）
- 角印・担当印・承認印の自動押印
- 写真付きPDF生成
- カスタムフィールド対応

---

## 2. 対象者

| 権限レベル | 個別PDF出力 | 一括PDF出力 |
|------------|-------------|-------------|
| 一般スタッフ | ✅ | ❌ |
| リーダー | ✅ | ✅ |
| マネージャー | ✅ | ❌ |
| 管理者 | ✅ | ✅ |

すべての権限レベルで個別PDF出力が可能です。一括PDF出力はリーダーと管理者のみです。

---

## 3. 個別PDF出力の手順

### 3-1. アクセス方法

1. 左サイドメニューから **「作業報告書」** をクリック
2. 報告書一覧から対象の報告書をクリック
3. 報告書詳細画面が表示されます
4. 画面右上の **「PDF出力」** ボタンをクリック
5. PDFが自動的にダウンロードされます

**URL**: `/work-reports/[id]` → PDF出力ボタンをクリック

### 3-2. ダウンロードされるファイル

**ファイル名の形式**:
```
作業報告書_現場名_報告日.pdf
```

**例**:
```
作業報告書_A工事現場_2026-01-15.pdf
作業報告書_B建設プロジェクト_2026-01-20.pdf
```

### 3-3. 保存場所

ブラウザのダウンロードフォルダに保存されます。

**主な保存場所**:
- Windows: `C:\Users\ユーザー名\Downloads`
- Mac: `/Users/ユーザー名/Downloads`
- Chrome: 設定 → ダウンロード → 保存先で確認可能

## 4. 一括PDF出力の手順

### 4-1. アクセス方法（承認待ち一覧から）

1. 左サイドメニューから **「作業報告書」** → **「承認待ち」** をクリック
2. 承認待ち報告書一覧が表示されます
3. PDF出力したい報告書の左側のチェックボックスにチェックを入れます
4. 複数選択する場合は、各報告書のチェックボックスをクリック
5. または **「全選択」** をクリックしてすべて選択
6. 画面右上の **「一括PDF出力」** ボタン（青色）をクリック
7. 選択したすべての報告書が個別にPDFとしてダウンロードされます

### 4-2. 一括PDF出力の注意点

- **各報告書が個別のPDFファイル**としてダウンロードされます（ZIPファイルではありません）
- ブラウザによっては、複数ファイルのダウンロードを許可するか確認されます
- ダウンロードには時間がかかる場合があります（報告書数×数秒）
- 一度に大量の報告書を選択すると、ブラウザが重くなる可能性があります

### 4-3. 推奨ダウンロード数

| 報告書数 | 推奨 |
|---------|------|
| 1〜10件 | ✅ 問題なし |
| 11〜30件 | △ 時間がかかる |
| 31件以上 | ❌ 複数回に分けて実行 |

## 5. PDF出力される内容

### 5-1. ヘッダー部分

**左上**:
- 作業報告書タイトル（中央）
- 作成日
- 報告書番号

**右上**:
- 担当印（作成者の印鑑）
- 承認印（承認者の印鑑、承認済みの場合のみ）

**右側**:
- 会社情報（社名、住所、電話番号、FAX）
- 角印（会社の印鑑）

**左側**:
- 取引先情報（社名、現場名、作業場所）

### 5-2. 本文部分

**基本情報**:
- 作業日（曜日付き）
- 作業人数
- 開始時間・終了時間
- 天気
- 進捗率

**作業員情報**（複数人の場合）:
- No.
- 作業員名
- 作業時間
- 時間外労働

**作業内容**:
- 作業の詳細説明（改行対応）

**使用資材**:
- 使用した資材のリスト

**特記事項・備考**:
- 特記事項（左側）
- 備考（右側）

**カスタムフィールド**（設定している場合）:
- 組織・現場ごとに設定したカスタム項目

### 5-3. 写真部分

報告書に添付された写真が自動的に埋め込まれます。

**写真の表示形式**:
- 1ページあたり2枚の写真を配置
- 各写真の下にキャプション（説明文）を表示
- 写真が多い場合は複数ページに分割

### 5-4. フッター部分

- ページ番号（右下）
- 例: ページ 1 / 3

## 6. 印鑑の表示

### 6-1. 角印（会社印）

PDF右上に会社の角印が自動的に表示されます。

**角印の仕様**:
- 赤色の正方形印鑑
- 会社名が印鑑内に表示されます
- 設定で保存した角印、または自動生成された角印が使用されます

### 6-2. 担当印（作成者印）

PDF右上の「担当印」枠に作成者の印鑑が表示されます。

**担当印の条件**:
- 作成者が個人印鑑を登録している場合のみ表示
- 未登録の場合は枠のみ表示

### 6-3. 承認印（承認者印）

PDF右上の「承認印」枠に承認者の印鑑が表示されます。

**承認印の条件**:
- 報告書が承認済み（approved）の場合のみ表示
- 承認者が個人印鑑を登録している場合のみ表示
- 未承認または未登録の場合は枠のみ表示

## 7. PDF出力のタイミング

### 7-1. いつでも出力可能

作業報告書はステータスに関わらず、いつでもPDF出力できます。

| ステータス | PDF出力 | 印鑑表示 |
|-----------|---------|----------|
| 下書き（draft） | ✅ | 担当印のみ |
| 提出済み（submitted） | ✅ | 担当印のみ |
| 承認済み（approved） | ✅ | 担当印 + 承認印 |
| 却下（rejected） | ✅ | 担当印のみ |

### 7-2. 推奨出力タイミング

**下書き時**:
- 内容確認用として出力
- 印刷してチェック
- 上司に見せて修正点を確認

**提出後**:
- 社内保管用として出力
- 現場担当者への配布

**承認後**:
- 取引先への提出用として出力
- 最終版として保管
- 承認印が押された正式版

## 8. PDF出力の活用例

### 8-1. 取引先への提出

1. 報告書を作成し、提出・承認を完了
2. 承認後にPDF出力
3. メールまたは郵送で取引先に提出

**メリット**:
- 承認印が押された正式な報告書
- 印刷してそのまま提出可能
- 電子ファイルとして保管も可能

### 8-2. 社内保管

1. 月末に承認済み報告書を一括PDF出力
2. 共有フォルダに保存
3. 経営会議資料として活用

**ファイル管理例**:
```
作業報告書/
├── 2026年/
│   ├── 01月/
│   │   ├── 作業報告書_A現場_2026-01-15.pdf
│   │   ├── 作業報告書_B現場_2026-01-16.pdf
│   │   └── 作業報告書_C現場_2026-01-20.pdf
│   └── 02月/
```

### 8-3. 現場配布

1. 報告書をPDF出力
2. 印刷して現場事務所に掲示
3. 作業員に配布して情報共有

### 8-4. 経営会議資料

1. 週次または月次で報告書を一括PDF出力
2. 役員会議資料として配布
3. プロジェクト進捗報告として活用

## 9. PDFの印刷

### 9-1. 印刷手順

1. PDF出力してファイルを開く
2. Adobe Acrobat ReaderやブラウザのPDFビューアーで開く
3. **「印刷」** をクリック（Ctrl+P / Command+P）
4. プリンターを選択
5. 印刷設定を確認
6. **「印刷」** をクリック

### 9-2. 推奨印刷設定

| 項目 | 推奨設定 |
|------|----------|
| 用紙サイズ | A4 |
| 向き | 縦 |
| カラー | カラー（角印・承認印が赤色） |
| 両面印刷 | 任意 |
| ページ範囲 | すべて |

### 9-3. 印刷時の注意点

- **カラー印刷推奨**: 角印・承認印が赤色で印刷されます
- **白黒印刷**: 赤色が濃いグレーになります
- **写真付き報告書**: インク消費量が多くなります

## 10. トラブルシューティング

| 問題 | 原因 | 解決方法 |
|------|------|----------|
| PDF出力ボタンが表示されない | 権限不足またはページが読み込まれていない | ページをリロードしてください |
| PDF生成中から進まない | サーバーエラーまたは報告書データの問題 | ページをリロードして再度試してください |
| ダウンロードされたファイルが開けない | ダウンロードが途中で中断された | 再度PDF出力を実行してください |
| 一括PDF出力に失敗する | 選択した報告書が多すぎる | 10件ずつに分けて出力してください |
| 角印が表示されない | 会社印が未設定 | 設定画面から角印を登録してください |
| 担当印・承認印が表示されない | 個人印鑑が未登録 | プロフィール設定から個人印鑑を登録してください |
| 写真が表示されない | 写真の読み込みエラー | 報告書に写真が正しく添付されているか確認してください |
| ファイル名が文字化けする | ブラウザの文字エンコード問題 | 別のブラウザで試してください |

## 11. よくある質問

### Q1: PDFは何度でも出力できますか？
**A**: はい、同じ報告書を何度でもPDF出力できます。ただし、内容を変更した場合は最新の内容でPDFが生成されます。

### Q2: 下書き状態でもPDF出力できますか？
**A**: はい、可能です。下書き時はプレビュー確認用として活用できます。

### Q3: PDFに変更履歴は記録されますか？
**A**: いいえ、PDFは出力時点の内容が記録されます。変更履歴はシステム内で管理されます。

### Q4: PDF出力に時間がかかります
**A**: 写真が多い報告書や、一括PDF出力の場合は時間がかかることがあります。数十秒〜数分お待ちください。

### Q5: 一括PDF出力でZIPファイルにならないのはなぜですか？
**A**: 現在のシステムでは個別PDFファイルとして連続ダウンロードされます。ブラウザのダウンロードフォルダに個別ファイルとして保存されます。

## 12. 注意事項とベストプラクティス

### 12-1. 注意事項

1. **個人情報の取り扱い**: PDF内に個人情報が含まれる場合、管理に注意してください
2. **ファイルサイズ**: 写真が多い報告書は、PDFファイルサイズが大きくなります（数MB〜数十MB）
3. **印鑑の登録**: 角印・個人印鑑を登録しないと、印鑑枠のみ表示されます
4. **ブラウザの互換性**: Chrome、Firefox、Safari、Edgeで動作確認済みです

### 12-2. ベストプラクティス

1. **承認後に出力**: 取引先提出用は必ず承認後に出力
2. **定期的なバックアップ**: 月次で一括PDF出力して共有フォルダに保存
3. **ファイル名の統一**: ダウンロード後に必要に応じてファイル名を整理
4. **印刷版の保管**: 重要な報告書は印刷して紙でも保管
