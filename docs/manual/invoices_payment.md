# 入金記録マニュアル

## 概要

入金記録機能は、請求書に対する顧客からの入金を記録する機能です。部分入金や全額入金に対応し、入金が完了すると請求書のステータスが自動的に「支払済み」に更新されます。

## 対象者

- リーダー（Leader）
- マネージャー（Manager）
- 管理者（Admin）

※ 一般スタッフは入金記録機能にアクセスできません。

## アクセス方法

### 1. 請求書詳細ページから

1. ダッシュボードまたはサイドメニューから「請求書」をクリック
2. 入金を記録したい請求書をクリック
3. 請求書詳細ページの右上にある「入金登録」ボタンをクリック

※ 「入金登録」ボタンは以下の条件を満たす場合にのみ表示されます：
- 請求書のステータスが「送付済み（sent）」である
- 請求額が未払い（または一部入金）である
- ユーザーの権限が管理者またはマネージャーである

### 2. 入金一覧ページから

1. サイドメニューから「入金・支払」をクリック
2. 右上の「新規登録」ボタンをクリック
3. 支払タイプで「入金」を選択
4. 入金を記録したい請求書を選択

## 入金登録の詳細手順

### 基本情報の入力

#### 1. 入金日（必須）

- デフォルトで今日の日付が設定されます
- カレンダーから入金があった実際の日付を選択してください

#### 2. 請求書の選択（必須）

- ドロップダウンメニューから入金対象の請求書を選択します
- 請求書は「請求日 - 取引先名 - 請求金額」の形式で表示されます
- 選択すると、請求金額・既入金額・残額が自動表示されます

#### 3. 入金額（必須）

- 今回入金された金額を入力します
- 部分入金の場合は一部の金額のみを入力可能です
- 入力後、「入金後残額」が自動計算されて表示されます
- 残額を超える金額は入力できません

**入金額の表示例**：
```
請求金額: ¥1,000,000
既入金額: ¥300,000
残額: ¥700,000

入金額を入力: 500,000
→ 入金後残額: ¥200,000
```

#### 4. 支払方法（必須）

以下から選択してください：

| 支払方法 | 説明 |
|----------|------|
| **銀行振込** | 最も一般的な方法。振込手数料は通常顧客負担 |
| **現金** | 現金での直接支払い |
| **小切手** | 小切手による支払い |
| **クレジットカード** | クレジットカード決済 |
| **その他** | 上記以外の支払方法（手形など） |

#### 5. 参照番号（任意）

- 銀行振込の場合：振込依頼人名や取引番号
- クレジットカードの場合：決済ID
- その他：管理用の識別番号

参照番号を記録しておくと、後で入金の照合作業がスムーズになります。

#### 6. メモ（任意）

- 入金に関する補足情報を記録できます
- 例：「請求書No.2024-001に対する2回目の入金」「振込手数料440円引かれて入金」

### 登録の完了

1. すべての必須項目を入力後、「登録」ボタンをクリック
2. 成功メッセージが表示され、入金一覧ページにリダイレクトされます
3. 登録した入金記録が一覧に表示されます

## 入金後のステータス変化

入金登録後、請求書のステータスは自動的に更新されます：

### 部分入金の場合

- **ステータス**: 「送付済み（sent）」のまま（変化なし）
- **既入金額**: 今回の入金額が加算されます
- **残額**: 自動再計算されます

**例**：
```
請求金額: ¥1,000,000
1回目入金: ¥300,000 → ステータス: 送付済み、残額: ¥700,000
2回目入金: ¥500,000 → ステータス: 送付済み、残額: ¥200,000
```

### 全額入金の場合

- **ステータス**: 「送付済み（sent）」→「支払済み（paid）」に自動変更
- **既入金額**: 請求金額と一致
- **残額**: ¥0

**例**：
```
請求金額: ¥1,000,000
既入金額: ¥800,000
残額: ¥200,000

最終入金: ¥200,000
→ ステータス: 支払済み、残額: ¥0
→ 「入金登録」ボタンが非表示になります
```

## 入金記録の確認

### 請求書詳細ページで確認

1. 請求書詳細ページを開く
2. 「入金履歴」セクションで入金記録を確認できます
3. 各入金記録には以下が表示されます：
   - 入金日
   - 入金額
   - 支払方法
   - 参照番号
   - メモ
   - 登録者名
   - 登録日時

### 入金一覧ページで確認

1. サイドメニューから「入金・支払」をクリック
2. すべての入金記録が一覧表示されます
3. 請求書ごとにフィルタリングや検索が可能です

## 注意事項

### 入金記録の修正・削除

- 入金記録は登録後の編集・削除ができません
- 誤って登録した場合は、管理者に連絡して手動で修正する必要があります
- 正確な金額と日付を入力してから登録してください

### 権限について

- 入金記録機能はリーダー以上が利用できます
- 一般スタッフは入金記録を閲覧することもできません
- 権限が不足している場合は、管理者に権限付与を依頼してください

### 入金額の注意点

- 振込手数料が引かれて入金された場合：
  - 実際に入金された金額を記録してください
  - メモ欄に「振込手数料○○円引き」と記録することを推奨します
- 端数処理による誤差：
  - 端数が合わない場合は、実際に入金された金額を正確に記録してください
  - メモ欄に「端数調整 -○円」などと記録してください

### 複数回の部分入金

- 同じ請求書に対して複数回の入金記録が可能です
- 各入金は独立した記録として保存されます
- 既入金額は自動的に累積されます

### 入金日について

- 入金日は実際に銀行口座に入金された日付を記録してください
- 振込日と入金日が異なる場合があります（土日祝日の場合など）
- 正確な入金日を記録することで、入金管理がスムーズになります

## トラブルシューティング

### 「入金登録」ボタンが表示されない

**原因と対処法**：

1. **権限不足**: 一般スタッフの場合
   - → リーダー以上の権限を持つユーザーに入金記録を依頼してください

2. **請求書のステータスが不適切**: ステータスが「下書き」「承認待ち」の場合
   - → 請求書を「送付済み」にしてください

3. **すでに全額入金済み**: 既入金額 = 請求金額の場合
   - → 入金記録は完了しています

### 入金額が残額を超えてしまう場合

**原因**：請求金額以上の金額を入力した

**対処法**：
1. 入力金額を確認してください
2. 過入金（overpayment）の場合は、管理者に相談してください

### 誤った金額で登録してしまった

**対処法**：
1. 現状では入金記録の編集・削除機能がありません
2. 管理者またはシステム管理者に連絡して、データベースから手動で修正を依頼してください
3. 今後のために、登録前に金額を必ず確認してください
