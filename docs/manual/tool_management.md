# 道具管理マニュアル

## 対象者

このマニュアルは、道具の登録・管理を行う以下の権限のユーザーを対象としています。

- **Manager（マネージャー）**: 道具の登録・編集が可能
- **Admin（管理者）**: 全機能が利用可能

※ Staff、Leaderは道具情報の閲覧とQRスキャンのみ可能です。

---

## 前提条件

道具管理を始める前に、以下の設定が完了していることを確認してください。

1. **倉庫位置の設定**（オプション）
   - 倉庫内で道具を管理する場合は、[倉庫位置管理マニュアル](./warehouse_locations.md)を参照して倉庫位置を設定してください

2. **道具カテゴリの設定**（オプション）
   - カテゴリで道具を分類したい場合は、事前にカテゴリを登録してください

---

## 道具管理の基本概念

ザイロクの道具管理は、**個別アイテム管理**が特徴です。

### 1. 道具マスタと個別アイテムの違い

```
道具マスタ（Tool）
├── 名前: "電動ドリル"
├── 型番: "DRL-5000"
├── メーカー: "マキタ"
├── 全体の個数: 3個
│
└── 個別アイテム（ToolItem）
    ├── #001（QRコード: xxx-xxx-xxx-001）
    ├── #002（QRコード: xxx-xxx-xxx-002）
    └── #003（QRコード: xxx-xxx-xxx-003）
```

**重要:**
- **道具マスタ**: 道具の種類を表す（例: 電動ドリル DRL-5000）
- **個別アイテム**: 実際の物理的な道具（例: 電動ドリル #001、#002、#003）
- **各個別アイテムに固有のQRコードが割り当てられます**

### 2. なぜ個別管理するのか?

高価な工具や重機は、個別に追跡することで以下のメリットがあります：

- **紛失防止**: どの個体がどこにあるか正確に把握
- **メンテナンス管理**: 個別にメンテナンス履歴を記録
- **移動履歴**: 誰が、いつ、どこに移動したか追跡可能
- **責任の明確化**: 現在の保管責任者が明確

## 道具の新規登録

### 登録方法の選択

道具を登録する際、以下の2つの方法から選択できます：

1. **🏢 共通マスタから選択**（推奨）
   - 業界で一般的な道具のテンプレート
   - 道具名・型番・メーカーが事前設定済み
   - 入力の手間を削減

2. **👥 自社マスタから選択**
   - 過去に登録した道具を再利用
   - 同じ道具を追加購入した場合に便利

3. **新規作成**
   - 共通マスタにない道具を登録
   - 道具・消耗品マスタページから作成

### ステップ1: 道具マスタの選択

1. **道具管理 > 道具一覧** を開く
2. **「+ 道具を追加」** をクリック
3. **「道具マスタ」** ドロップダウンを開く

**選択肢:**
- **🏢 共通マスタ**: 電動ドリル、測定器、重機などの一般的な道具
- **👥 自社マスタ**: 過去に登録した道具マスタ

### ステップ2: 登録情報の入力

選択した道具マスタに基づいて、以下の情報を入力します。

#### 基本情報（必須）
- **道具マスタ**: 共通マスタまたは自社マスタから選択（必須）
- **個数**: 登録する個数（例: 3）

#### 詳細情報（推奨）
- **購入日**: 購入した日付
- **購入価格**: 1個あたりの購入価格（円）
- **備考**: メモや特記事項

**注意:**
- 道具名、型番、メーカーは道具マスタから自動設定されます
- 共通マスタを使用すると入力の手間が省けます

### ステップ3: 登録完了と個別アイテムの自動生成

**「登録」** をクリックすると、道具マスタと**個数分の個別アイテムが自動生成**されます。

**例:** 個数「3」で登録した場合
```
電動ドリル DRL-5000
├── #001（利用可能）
├── #002（利用可能）
└── #003（利用可能）
```

各個別アイテムには、**固有のQRコードが自動生成**されます。

## QRコードの印刷と貼り付け

### QRコードの印刷方法

1. **道具管理 > 道具一覧** を開く
2. 登録した道具をクリックして詳細画面を開く
3. **「個別アイテム一覧」** セクションで、印刷したいアイテムの **「QR印刷」** をクリック
4. PDFがダウンロードされます
5. A4用紙またはラベル用紙に印刷

**推奨設定:**
- **用紙サイズ**: A4
- **QRコードサイズ**: 3cm x 3cm 以上
- **ラベル用紙**: シール型ラベルを使うと貼り付けが簡単

### 一括QRコード印刷

複数の道具のQRコードをまとめて印刷したい場合:

1. **道具管理 > QR一括印刷** を開く
2. カテゴリやメーカーでフィルタリング
3. **「選択したアイテムのQR印刷」** をクリック
4. PDFがダウンロードされます

### QRコードの貼り付け

印刷したQRコードを実際の道具に貼り付けます。

**注意点:**
- **見やすい場所に貼る**: スキャンしやすい位置
- **汚れにくい場所**: 油や水がかからない場所
- **剥がれにくいラベル**: 耐久性のあるラベル用紙を使用
- **複数箇所に貼る**: 大型機械の場合、複数面に貼ると便利

## 道具情報の編集

### 道具マスタの編集

1. **道具管理 > 道具一覧** を開く
2. 編集したい道具をクリック
3. **「編集」** をクリック
4. 情報を修正
5. **「保存」** をクリック

**編集可能な項目:**
- 道具名、型番、メーカー
- カテゴリ
- 購入日、購入価格
- 最小在庫数
- 低在庫アラート設定

**注意:**
- **個数は編集できません**。個数を変更したい場合は、個別アイテムを追加・削除してください。

### 個別アイテムの追加

既存の道具マスタに個別アイテムを追加する場合:

1. 道具詳細画面を開く
2. **「個別アイテムを追加」** をクリック
3. 追加する個数を入力
4. **「追加」** をクリック

新しい個別アイテムが自動生成され、連番が割り当てられます。

**例:**
```
既存: #001, #002, #003
追加後: #001, #002, #003, #004, #005（2個追加した場合）
```

### 個別アイテムのメモ編集

各個別アイテムには、個別のメモを記録できます。

1. 道具詳細画面の **「個別アイテム一覧」** を開く
2. 対象アイテムの **「メモ編集」** をクリック
3. メモを入力（例: 「バッテリー交換済み」「グリップ破損」）
4. **「保存」** をクリック

**活用例:**
- メンテナンス履歴
- 不具合情報
- 特記事項

## 道具の削除

### 削除の制限事項

道具を削除する際は、以下の条件を確認してください：

- **移動履歴がある道具は削除できません**
- **現在使用中（貸出中）の道具は削除できません**

誤って登録した道具のみ削除可能です。

### 削除手順

1. **道具管理 > 道具一覧** を開く
2. 削除したい道具をクリック
3. **「削除」** をクリック
4. 確認ダイアログで **「削除」** をクリック

**注意:**
- 削除は論理削除（データは残るが非表示）です
- 復元が必要な場合は、管理者に連絡してください

## 道具の検索とフィルタリング

### 基本的な検索

1. **道具管理 > 道具一覧** を開く
2. 画面上部の検索ボックスにキーワードを入力
3. 道具名、型番、メーカー名で検索できます

### フィルタリング

**「フィルタ」** ボタンをクリックすると、以下の条件で絞り込みができます：

- **カテゴリ**: 道具カテゴリで絞り込み
- **メーカー**: メーカーで絞り込み
- **在庫状態**: 「在庫あり」「在庫なし」「低在庫」で絞り込み
- **場所**: 「倉庫」「現場」「修理中」「紛失」で絞り込み

## 道具の状態管理

### 個別アイテムの状態

各個別アイテムには、以下の状態があります：

| 状態 | 説明 | 表示色 |
|------|------|--------|
| **利用可能** | 正常に使用できる状態 | 緑 |
| **使用中** | 現在、誰かが使用中 | 青 |
| **修理中** | メンテナンス・修理中 | 黄 |
| **紛失** | 紛失した状態 | 赤 |

状態は、QRスキャンや移動登録によって自動的に更新されます。

### 状態の手動変更

状態を手動で変更することもできます：

1. 道具詳細画面の **「個別アイテム一覧」** を開く
2. 対象アイテムの **「状態変更」** をクリック
3. 新しい状態を選択
4. 理由を入力（オプション）
5. **「変更」** をクリック

## カスタムフィールド（拡張機能）

道具マスタには、カスタムフィールドを追加できます。

### 使用例

業種や業務に応じて、以下のような情報を記録できます：

- **校正日・校正期限**（測定器の場合）
- **レンタル返却期限**（レンタル品の場合）
- **減価償却率**（経理管理の場合）
- **絶縁検査日**（電気工事の場合）

### カスタムフィールドの設定

1. 道具編集画面の **「カスタムフィールド」** セクションを開く
2. **「+ フィールドを追加」** をクリック
3. フィールド名と値を入力
4. **「保存」** をクリック

## トラブルシューティング

### 道具が見つからない場合の検索方法

道具一覧で目的の道具が見つからない場合は、以下の手順で検索してください。

1. **検索ボックスを使用**
   - 画面上部の検索ボックスに道具名、型番、メーカー名を入力
   - 部分一致で検索可能

2. **フィルタリング機能を使用**
   - **「フィルタ」** ボタンをクリック
   - カテゴリ、メーカー、在庫状態、場所で絞り込み
   - 複数条件を組み合わせて検索可能

3. **個別アイテムの状態を確認**
   - 道具詳細画面の **「個別アイテム一覧」** を確認
   - 「紛失」状態になっていないかチェック

4. **削除済み道具の確認**
   - 削除された道具は一覧に表示されません
   - 管理者に確認を依頼してください

### QRコードの再印刷方法

QRコードが汚れて読めなくなった場合や、破損した場合は再印刷できます。

1. 道具詳細画面を開く
2. **「個別アイテム一覧」** セクションを確認
3. 対象アイテムの **「QR印刷」** をクリック
4. PDFをダウンロードして印刷
5. 新しいQRコードを貼り直す

**注意:** QRコード（UUID）自体は変わらないため、既存の移動履歴は保持されます。

### 個別アイテムの削除方法

登録した道具の個数を減らしたい場合は、個別アイテムを削除します。

1. 道具詳細画面の **「個別アイテム一覧」** を開く
2. 削除したいアイテムの **「削除」** をクリック
3. 確認ダイアログで **「削除」** をクリック

**削除の条件:**
- 移動履歴がないアイテムのみ削除可能
- 現在使用中のアイテムは削除不可
- 誤って登録した直後のみ削除を推奨

## 注意事項

### 登録時の注意

- **個数は慎重に入力**: 後から減らすのは面倒です
- **型番・メーカーは正確に**: 検索や管理に影響します
- **購入価格は記録推奨**: 資産管理・減価償却に必要

### 運用上の注意

- **QRコードは必ず貼る**: 個別管理の基本です
- **状態を正しく更新**: 紛失・修理中などは手動で更新
- **メモを活用**: 個別アイテムのメモは保守に役立ちます

### セキュリティ

- **削除は慎重に**: 論理削除ですが、復元には管理者権限が必要
- **他社のQRコードは読めません**: マルチテナント型のため、他社データは完全分離
