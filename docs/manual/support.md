## 概要

システムに不具合が発生した際の対応方法をご案内します。問い合わせ前に、よくある不具合と対処法を確認することで、迅速に問題を解決できる場合があります。

### このマニュアルで分かること

- サポートへの問い合わせ方法
- 問い合わせ時に必要な情報
- よくある不具合の自己解決方法
- 緊急時の対応手順

## 問い合わせ前の確認事項

サポートに問い合わせる前に、以下を確認してください。多くの問題は、簡単な対処法で解決できます。

### 基本的な確認項目

1. **インターネット接続を確認**
   - 他のウェブサイトにアクセスできるか確認してください
   - Wi-Fiが接続されているか確認してください

2. **ブラウザを最新版に更新**
   - Chrome、Edge、Safariの最新版を使用してください
   - 古いバージョンでは動作しない場合があります

3. **ブラウザのキャッシュをクリア**
   - **Chrome/Edge**: Ctrl+Shift+Delete（Windows）、Cmd+Shift+Delete（Mac）
   - **Safari**: Cmd+Option+E

4. **別のブラウザで試す**
   - Chrome、Edge、Safariなど複数のブラウザで試してください

5. **[よくある不具合と対処法](#よくある不具合と対処法)を確認**
   - このマニュアル内の対処法を試してください

## ログインできない場合の問い合わせ方法

ログイン画面にアクセスできない、またはログインができない場合は、以下の方法で運営にお問い合わせください。

### 手順

1. **ログイン画面にアクセス**
   - `https://zairoku.com` にアクセス
   - または `https://あなたの会社名.zairoku.com` にアクセス

2. **「運営への問い合わせ」リンクをクリック**
   - ログイン画面の下部にあります

3. **問い合わせフォームに必要事項を入力**
   - 下記の「問い合わせ時に記載する情報」を参照

4. **送信ボタンをクリック**
   - 送信完了画面が表示されます

### アクセス方法

**PC**: ログイン画面下部の「運営への問い合わせ」

**スマートフォン**: ログイン画面を下にスクロールして「運営への問い合わせ」

## ログインできる場合の問い合わせ方法

ログインはできるが、システム内で不具合が発生している場合は、サポートチケット機能をご利用ください。

### サポートチケットでの問い合わせ手順

1. **ログイン後、画面右上のユーザーメニューをクリック**
   - プロフィールアイコンをクリック

2. **「サポートチケット」を選択**
   - メニューから「サポートチケット」を選択

3. **新規チケットを作成**
   - 「+ 新規チケット」ボタンをクリック

4. **不具合の詳細を入力して送信**
   - タイトル、詳細、優先度を入力
   - 必要に応じてスクリーンショットを添付

**⚠️ 注意**: サポートチケット機能は現在準備中です。実装されるまでは、ログイン画面の「運営への問い合わせ」リンクからお問い合わせください。

### 代替方法（現在）

サポートチケット機能が実装されるまでは、以下の方法でお問い合わせください:

1. 一度ログアウト
2. ログイン画面の「運営への問い合わせ」からお問い合わせ
3. または、組織の管理者に連絡して代理で問い合わせを依頼

## 問い合わせ時に記載する情報

スムーズに対応するため、以下の情報を記載してください。

### 必須情報

| 項目 | 説明 | 例 |
|------|------|-----|
| **組織名** | あなたの会社名またはサブドメイン | 株式会社サンプル |
| **メールアドレス** | 登録しているメールアドレス | user@example.com |
| **発生している問題** | できるだけ具体的に | ログイン画面でパスワードを入力しても「メールアドレスまたはパスワードが正しくありません」と表示される |
| **使用環境** | PC/スマホ、ブラウザの種類 | PC、Google Chrome |

### 推奨情報（可能であれば）

| 項目 | 説明 | 例 |
|------|------|-----|
| **発生日時** | いつから発生しているか | 2026年1月22日 14:30頃 |
| **発生箇所** | どのページ・機能で発生しているか | 道具移動ページ |
| **操作手順** | 何をした時に発生したか | QRコードをスキャンして「移動を実行」ボタンを押した |
| **エラーメッセージ** | 表示されている場合はそのまま記載 | 「移動に失敗しました」 |
| **スクリーンショット** | 可能であれば添付 | 添付ファイル |

### 問い合わせ例

#### 例1: ログインできない場合

```
【組織名】株式会社サンプル
【メールアドレス】user@example.com
【問題】
ログイン画面でメールアドレスとパスワードを入力しても
「メールアドレスまたはパスワードが正しくありません」と
表示されてログインできません。
パスワードは間違っていないはずです。

【使用環境】PC、Google Chrome
```

#### 例2: システム内で不具合が発生している場合

```
【組織名】株式会社サンプル
【メールアドレス】user@example.com
【発生日時】2026年1月22日 14:30頃
【発生箇所】道具移動ページ
【操作手順】
QRコードをスキャンして「移動を実行」ボタンを押したところ、
エラーが表示されました。

【エラーメッセージ】「移動に失敗しました」
【使用環境】スマートフォン、Safari（iOS）
```

## よくある不具合と対処法

サポートに問い合わせる前に、以下の対処法をお試しください。

### 1. ページが表示されない・真っ白になる

#### 症状
- ページにアクセスすると真っ白な画面が表示される
- ページの一部が表示されない

#### 対処法

1. **ブラウザのキャッシュをクリア**
   - **Chrome/Edge**: Ctrl+Shift+Delete → 「キャッシュされた画像とファイル」にチェック → クリア
   - **Safari**: Cmd+Option+E

2. **ページを再読み込み**
   - **Windows**: Ctrl+R または F5
   - **Mac**: Cmd+R

3. **別のブラウザで試す**
   - Chrome、Edge、Safariなど

4. **JavaScriptが有効か確認**
   - ブラウザの設定でJavaScriptが有効になっているか確認

### 2. ボタンが反応しない

#### 症状
- ボタンをクリックしても何も起きない
- 保存ボタンが押せない

#### 対処法

1. **ページを再読み込み**
   - Ctrl+R（Windows）、Cmd+R（Mac）

2. **JavaScriptが有効か確認**
   - ブラウザの設定を確認

3. **別のブラウザで試す**
   - Chrome、Edge、Safariなど

4. **広告ブロッカーを無効にする**
   - 広告ブロッカーが機能を妨げている可能性があります

### 3. 画像が表示されない

#### 症状
- プロフィール画像が表示されない
- アップロードした画像が表示されない

#### 対処法

1. **インターネット接続を確認**
   - 他のウェブサイトの画像が表示されるか確認

2. **ページを再読み込み**
   - Ctrl+R（Windows）、Cmd+R（Mac）

3. **ブラウザのキャッシュをクリア**
   - Ctrl+Shift+Delete → キャッシュをクリア

4. **画像のファイル形式を確認**
   - 対応形式: JPEG、PNG、GIF
   - ファイルサイズ: 10MB以下

### 4. データが保存されない

#### 症状
- 入力したデータが保存されない
- 保存ボタンを押してもエラーが表示される

#### 対処法

1. **必須項目がすべて入力されているか確認**
   - 赤い枠で表示されている項目を入力してください

2. **ブラウザのCookieが有効か確認**
   - ブラウザの設定でCookieが有効になっているか確認

3. **ページを再読み込みしてもう一度試す**
   - Ctrl+R（Windows）、Cmd+R（Mac）

4. **入力内容をコピーしてから再度試す**
   - データが消えないように、メモ帳などにコピーしておく

### 5. スマートフォンで操作がしづらい

#### 症状
- ボタンが小さくて押しにくい
- 文字が読みにくい

#### 対処法

1. **ブラウザを最新版に更新**
   - App Store（iOS）またはGoogle Play（Android）から更新

2. **画面の向きを変えてみる**
   - 縦向き ↔ 横向き

3. **別のブラウザで試す**
   - Safari（iOS）、Chrome（Android）

4. **拡大表示を使う**
   - ピンチアウトで拡大できます

### 6. QRコードがスキャンできない

#### 症状
- カメラが起動しない
- QRコードを読み取れない

#### 対処法

1. **カメラ権限を確認**
   - ブラウザの設定でカメラのアクセスを許可してください

2. **QRコードにピントを合わせる**
   - QRコードから10〜30cm離れた位置で試す

3. **明るい場所で試す**
   - 暗い場所ではQRコードを読み取れません

4. **QRコードが汚れていないか確認**
   - 汚れや傷がある場合は再印刷してください

詳細は [QRスキャン](qr_scan.md) を参照してください。

## 緊急時の対応

### システム全体が使えない場合

システム全体に障害が発生している可能性があります。

#### 確認方法

1. **他のユーザーにも同じ問題が発生しているか確認**
   - 組織内の他のメンバーに確認してください

2. **別のネットワークで試す**
   - Wi-Fiからモバイル通信に切り替えて試す
   - または、別のWi-Fiに接続して試す

3. **システムステータスページを確認（実装予定）**
   - 現在準備中です

#### 対応

1. **運営への問い合わせリンクから速やかに報告**
   - 「緊急」と明記してください

2. **復旧まで時間がかかる場合**
   - 一時的に別の方法で業務を継続してください
   - 紙の台帳などで記録し、復旧後に入力してください

3. **復旧後の確認**
   - データが正しく保存されているか確認してください

### データが消えた・表示されない場合

#### 対応

1. **ページを再読み込み**
   - Ctrl+R（Windows）、Cmd+R（Mac）

2. **フィルター設定を確認**
   - 一覧画面でフィルターがかかっていないか確認

3. **削除されていないか確認**
   - 削除済みデータは表示されません
   - 管理者に確認してください

4. **それでも見つからない場合**
   - 速やかに運営に問い合わせてください
   - 「データ消失」と明記してください

## サポート対応時間

### 対応時間

- **平日**: 9:00〜18:00
- **休業日**: 土日祝日、年末年始

### 返信までの時間

運営からの返信は、通常**1〜2営業日以内**を目安としています。

- 緊急性の高い問題の場合は、問い合わせ時に「緊急」と明記してください
- 優先的に対応いたします

### 対応の優先順位

以下の順で優先的に対応します:

1. **緊急**: システム全体が使えない、データ消失など
2. **高**: ログインできない、重要機能が使えない
3. **中**: 一部機能が使えない、操作がしづらい
4. **低**: 改善要望、質問
