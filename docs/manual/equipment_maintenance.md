# 重機メンテナンス記録マニュアル

## 1. 概要

重機（ショベルカー、クレーン、ダンプカーなど）の点検・修理記録を管理する機能です。車検、保険更新、修理、その他のメンテナンス履歴を記録し、次回予定日を管理できます。

**主な機能**:
- 4種類のメンテナンスタイプ（車検、保険更新、修理、その他）
- 点検記録の登録（実施日、実施者、費用、次回予定日）
- 領収書・報告書URLの保存
- 車検・保険更新時の自動マスタ更新
- 点検履歴の一覧表示

---

## 2. 対象者

| 権限レベル | 点検記録登録 | 点検履歴閲覧 |
|------------|--------------|--------------|
| 一般スタッフ | ✅ | ✅ |
| リーダー | ✅ | ✅ |
| マネージャー | ✅ | ✅ |
| 管理者 | ✅ | ✅ |

すべての権限レベルで点検記録を登録・閲覧できます。

---

## 3. アクセス方法

### 3-1. 点検記録登録ページへのアクセス

1. 左サイドメニューから **「重機」** をクリック
2. 重機一覧から対象の重機をクリック
3. **「メンテナンス」** タブをクリック
4. **「点検記録追加」** ボタンをクリック
5. 点検記録登録フォームが表示されます

**URL**: `/equipment/[id]/maintenance`

### 3-2. 点検履歴の閲覧

1. 左サイドメニューから **「重機」** をクリック
2. 重機一覧から対象の重機をクリック
3. **「メンテナンス」** タブをクリック
4. 過去の点検記録一覧が表示されます

---

## 4. メンテナンスタイプ

### 4-1. 4種類のメンテナンスタイプ

| タイプ | 説明 | 使用例 |
|--------|------|--------|
| **車検** | 車両の法定点検 | 2年に1回の車検実施 |
| **保険更新** | 保険の更新手続き | 年1回の保険更新 |
| **修理** | 故障・不具合の修理 | エンジン修理、油圧系統修理 |
| **その他** | 上記以外の点検・メンテナンス | 定期点検、オイル交換、タイヤ交換 |

### 4-2. 車検と保険更新の特別機能

**車検**または**保険更新**を記録し、**次回予定日**を入力すると、重機マスタの対応する日付が自動的に更新されます。

- **車検**: 次回予定日 → 重機マスタの「次回車検日」を自動更新
- **保険更新**: 次回予定日 → 重機マスタの「保険終了日」を自動更新

これにより、重機一覧画面で車検・保険期限のアラート表示が正確になります。

---

## 5. 点検記録登録の詳細手順

### 5-1. 基本情報の入力

| 項目 | 必須 | 説明 | 例 |
|------|------|------|-----|
| **点検タイプ** | ✅ | 車検、保険更新、修理、その他から選択 | 車検 |
| **実施日** | ✅ | 点検・修理を実施した日付 | 2026-01-15 |
| **実施者** | - | 点検・修理を行った業者または担当者 | 株式会社○○整備工場 |
| **費用（円）** | - | 点検・修理にかかった費用 | 150000 |
| **次回予定日** | - | 次回の点検予定日 | 2028-01-15 |
| **領収書URL** | - | 領収書のクラウドストレージURL | https://... |
| **報告書URL** | - | 点検報告書のクラウドストレージURL | https://... |
| **備考** | - | 点検内容の詳細、交換部品、注意事項など | エンジンオイル交換、ブレーキパッド交換 |

### 5-2. 点検タイプの選択

フォーム上部に4つのボタンが表示されます：

- **車検** （紫色ボタン）
- **保険更新** （紫色ボタン）
- **修理** （紫色ボタン）
- **その他** （紫色ボタン）

選択したボタンが紫色にハイライトされます。

### 5-3. 実施日の入力

- デフォルトで今日の日付が設定されます
- カレンダーから過去の日付も選択可能です
- 必須項目です

### 5-4. 実施者の入力

実施者名または業者名を入力します（任意）。

**入力例**:
- 「株式会社○○整備工場」
- 「△△メンテナンス」
- 「自社メンテナンス部門」

### 5-5. 費用の入力

点検・修理にかかった費用を円単位で入力します（任意）。

**入力例**:
- 車検: 150000
- 保険更新: 80000
- 修理: 250000
- オイル交換: 15000

### 5-6. 次回予定日の入力

次回の点検予定日を入力します（任意）。

**重要**:
- 車検または保険更新を選択している場合、次回予定日を入力すると**重機マスタが自動更新**されます
- 修理やその他の場合は、自動更新されません（記録のみ）

**自動更新の詳細**:
```
点検タイプ: 車検 + 次回予定日: 2028-01-15
→ 重機マスタの「次回車検日」が 2028-01-15 に更新されます

点検タイプ: 保険更新 + 次回予定日: 2027-01-20
→ 重機マスタの「保険終了日」が 2027-01-20 に更新されます
```

### 5-7. 領収書URL・報告書URLの入力

領収書や点検報告書をクラウドストレージ（Google Drive、Dropboxなど）にアップロードしている場合、そのURLを入力します（任意）。

**使い方**:
1. 領収書をクラウドストレージにアップロード
2. 共有リンクを取得
3. URLをコピーして入力

**メリット**:
- 領収書をシステムから直接参照できる
- 税務調査時の証憑管理が簡単
- 紙の領収書を紛失しても安心

### 5-8. 備考の入力

点検内容の詳細、交換した部品、今後の注意事項などを自由に記入します（任意）。

**記入例**:
```
【実施内容】
- エンジンオイル交換
- ブレーキパッド交換（前輪）
- タイヤ空気圧調整

【次回までの注意事項】
- 後輪ブレーキパッドの摩耗が進んでいるため、次回交換推奨
- 油圧ホースに若干の劣化あり、1年後に交換検討
```

### 5-9. 登録の実行

1. すべての項目を入力
2. **「登録する」** ボタンをクリック
3. 成功メッセージが表示され、重機詳細ページにリダイレクトされます
4. 「メンテナンス」タブに記録が追加されます

## 6. 点検履歴の確認

### 6-1. 点検記録一覧の見方

重機詳細画面の「メンテナンス」タブに、過去の点検記録が新しい順に表示されます。

**表示内容**:
- 点検タイプ（車検、保険更新、修理、その他）
- 実施日
- 実施者
- 費用
- 次回予定日
- 備考

### 6-2. 点検記録の詳細

各記録には以下の情報が表示されます：

```
[車検] 2026-01-15
実施者: 株式会社○○整備工場
費用: ¥150,000
次回予定: 2028-01-15
備考: エンジンオイル交換、ブレーキパッド交換
```

### 6-3. 領収書・報告書へのアクセス

領収書URLまたは報告書URLが登録されている場合、リンクをクリックして直接アクセスできます。

## 7. 車検・保険管理との連携

### 7-1. 車検管理

重機マスタに「車検が必要」として登録されている場合：

- 重機詳細画面の「基本情報」タブに「車検管理」セクションが表示されます
- 次回車検日が表示されます
- 車検期限が近づくとアラートが表示されます
  - **30日前**: 警告アラート（黄色）
  - **14日前**: エラーアラート（赤色）
  - **期限切れ**: エラーアラート（赤色）「○○日過ぎています」

**車検記録登録時の自動更新**:
- 点検タイプ「車検」を選択し、次回予定日を入力すると、重機マスタの「次回車検日」が自動的に更新されます

### 7-2. 保険管理

すべての重機に保険情報を登録できます：

- 保険会社
- 保険証券番号
- 保険期間（開始日〜終了日）

**保険更新記録登録時の自動更新**:
- 点検タイプ「保険更新」を選択し、次回予定日を入力すると、重機マスタの「保険終了日」が自動的に更新されます

### 7-3. アラート機能との連携

重機マスタが自動更新されると、重機一覧画面のアラート表示も自動的に更新されます。

**例**:
```
車検記録登録前:
[アラート] 車検期限まであと10日です

車検記録登録後（次回予定日: 2年後）:
[アラート表示なし] 車検期限まで十分余裕があります
```

---

## 8. コスト管理との連携

### 8-1. 点検・修理コストの自動集計

登録した点検記録の費用は、重機詳細画面の「コスト」タブで自動的に集計されます。

**集計項目**:
- 点検・修理コスト合計
- 今年の点検・修理コスト
- 今月の点検・修理コスト

### 8-2. コストレポートへの反映

点検・修理コストは、重機コスト分析レポートにも反映されます：

- 所有形態別コスト集計
- 重機別コスト比較
- 月次・年次トレンド分析

## 9. 注意事項とベストプラクティス

### 9-1. 注意事項

1. **点検記録は削除・編集できません**: 誤って登録した場合、備考に「誤記録」と記入し、正しい記録を新規登録してください
2. **次回予定日は慎重に入力**: 車検・保険更新の次回予定日を入力すると、重機マスタが自動更新されます
3. **領収書・報告書は外部保存**: システム内に直接アップロード機能はありません。クラウドストレージを使用してください
4. **費用は税込金額を推奨**: 消費税込みの金額を入力することをお勧めします

### 9-2. ベストプラクティス

1. **定期点検を記録する習慣**: オイル交換、タイヤ交換なども「その他」として記録しましょう
2. **備考を充実させる**: 交換部品、次回の推奨事項を詳しく記入すると、後で役立ちます
3. **領収書は必ず保存**: 税務調査や保証請求時に必要になります。クラウドストレージに保存しURLを記録しましょう
4. **車検・保険更新後は必ず記録**: 次回予定日を入力することで、アラート機能が正しく動作します
5. **修理内容を詳細に記録**: 同じ箇所が故障した場合、過去の修理履歴から原因を推測できます

---

## 10. トラブルシューティング

| 問題 | 原因 | 解決方法 |
|------|------|----------|
| 次回予定日を入力したのに重機マスタが更新されない | 点検タイプが「修理」または「その他」 | 車検または保険更新を選択してください |
| 点検記録が表示されない | ページがキャッシュされている | ページをリロードしてください |
| 登録ボタンを押してもエラーになる | 実施日が未入力 | 実施日は必須項目です。入力してください |
| 領収書URLをクリックしてもアクセスできない | URLが正しくないか、権限が不足 | 共有リンクのアクセス権限を「リンクを知っている全員」に設定してください |
| 過去の点検記録を修正したい | 点検記録は編集・削除できません | 備考に「○月○日の記録は誤り」と記入し、正しい記録を新規登録してください |

## 11. よくある使用例

### 11-1. 車検実施時

```
点検タイプ: 車検
実施日: 2026-01-15
実施者: 株式会社○○整備工場
費用: 150000
次回予定日: 2028-01-15
備考:
【実施内容】
- 法定24ヶ月点検
- エンジンオイル交換
- ブレーキパッド交換（前輪）
- タイヤローテーション

【指摘事項】
- 後輪ブレーキパッド残量: 3mm（次回交換推奨）
- バッテリー劣化あり（1年以内に交換推奨）
```

### 11-2. 保険更新時

```
点検タイプ: 保険更新
実施日: 2026-03-01
実施者: △△保険会社
費用: 80000
次回予定日: 2027-03-01
備考: 年間保険料80,000円（昨年比-5,000円）
```

### 11-3. 修理実施時

```
点検タイプ: 修理
実施日: 2026-02-10
実施者: 自社メンテナンス部門
費用: 35000
次回予定日: （空欄）
備考:
【故障内容】
- 油圧ホース亀裂による油漏れ

【修理内容】
- 油圧ホース交換（品番: HH-12345）
- 作動油補充（20L）

【今後の対応】
- 同型ホースの予防交換を検討
```

### 11-4. 定期点検時

```
点検タイプ: その他
実施日: 2026-01-05
実施者: 現場スタッフ（山田太郎）
費用: 0
次回予定日: 2026-04-05
備考:
【定期点検（3ヶ月ごと）】
✅ エンジンオイルレベル確認 → 正常
✅ 冷却水レベル確認 → 正常
✅ タイヤ空気圧確認 → 調整済み
✅ グリスアップ → 実施
⚠️ 運転席シート破れあり → 張替え検討
```
