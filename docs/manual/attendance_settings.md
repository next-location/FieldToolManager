# 勤怠設定マニュアル

## 1. 概要

出退勤管理システムの動作を組織に合わせてカスタマイズする機能です。打刻方法、休憩時間管理、リマインダー通知など、組織の運用に合わせて詳細に設定できます。

**主な設定項目**:
- 打刻方法（手動打刻、QRコード打刻）
- QRコード更新頻度（1日、3日、7日、30日）
- 休憩時間モード（なし、簡易、詳細）
- 休憩時間自動差し引き
- 出勤・退勤リマインダー
- 管理者向け日次レポート
- 長時間労働アラート

---

## 2. 対象者

| 権限レベル | 勤怠設定変更 | 設定閲覧 |
|------------|--------------|----------|
| 一般スタッフ | ❌ | ❌ |
| リーダー | ❌ | ❌ |
| マネージャー | ✅ | ✅ |
| 管理者 | ✅ | ✅ |

マネージャーと管理者のみが勤怠設定を変更できます。

---

## 3. アクセス方法

1. 左サイドメニューから **「出退勤」** をクリック
2. サブメニューから **「設定」** をクリック
3. 出退勤管理設定画面が表示されます

**URL**: `/attendance/settings`

---

## 4. 打刻方法設定

### 4-1. 利用可能な打刻方法

組織で利用する打刻方法を選択します（複数選択可）。

| 打刻方法 | 説明 | 推奨利用シーン |
|----------|------|----------------|
| **手動打刻（ボタンタップ）** | 場所を選択してボタンをタップ | すべての組織 |
| **QRコード打刻** | QRコードで場所を自動判別 | 不正打刻防止、タブレット設置 |

### 4-2. 手動打刻

- スマートフォンやPCから、会社または現場を選択してボタンをタップします
- 最もシンプルで、すべての組織にお勧めです
- GPS位置情報は記録されますが、必須ではありません

### 4-3. QRコード打刻

- 会社や現場に設置されたQRコードをスマートフォンで読み取って打刻します
- 場所が自動的に判別されるため、不正打刻を防止できます
- タブレットを設置して常時QRコードを表示することも可能です

**QRコード打刻のメリット**:
- 不正打刻防止（実際にその場所にいないと打刻できない）
- 操作が簡単（QRをスキャンするだけ）
- 場所の選択ミスを防止

**QRコード打刻の運用方法**:
1. **会社QRコード**: 受付やタイムカードの位置に掲示
2. **現場QRコード**: 現場事務所や工具保管場所に掲示
3. **タブレット設置**: 専用タブレットでQRコードを常時表示

---

## 5. QRコード更新頻度

### 5-1. 更新頻度の選択

セキュリティのため、QRコードは定期的に自動更新されます。

| 更新頻度 | 説明 | 推奨シーン |
|----------|------|-----------|
| **1日** | 毎日更新（最高セキュリティ） | 厳格な労務管理が必要な組織 |
| **3日** | 3日ごとに更新 | バランス重視 |
| **7日（推奨）** | 週1回更新 | 多くの組織に最適 |
| **30日** | 月1回更新（運用負荷最小） | QRコードの掲示替えが困難な現場 |

### 5-2. 更新頻度の決め方

**高頻度（1日・3日）が適している場合**:
- セキュリティを最優先したい
- QRコードをタブレットで表示している（自動更新される）
- 不正打刻のリスクが高い

**低頻度（7日・30日）が適している場合**:
- QRコードを印刷して掲示している
- 現場が多く、頻繁な掲示替えが困難
- スタッフの信頼性が高い

## 6. 休憩時間設定

### 6-1. 休憩時間モード

組織の運用に合わせて3つのモードから選択できます。

| モード | 説明 | 使い方 |
|--------|------|--------|
| **休憩時間を記録しない** | 休憩時間を管理しない | 休憩時間が不定期、または管理不要 |
| **簡易モード（推奨）** | 休憩開始・終了ボタンで記録 | 昼休憩など1回の休憩を記録 |
| **詳細モード** | 複数回の休憩を記録 | 小休憩が複数回ある場合 |

### 6-2. 簡易モードの使い方

- 休憩開始時に **「休憩開始」** ボタンをタップ
- 休憩終了時に **「休憩終了」** ボタンをタップ
- 休憩時間が自動的に計算され、実働時間から差し引かれます

**例**:
```
出勤: 09:00
休憩開始: 12:00
休憩終了: 13:00
退勤: 18:00

勤務時間: 9時間
休憩時間: 1時間
実働時間: 8時間
```

### 6-3. 詳細モードの使い方

- 小休憩ごとに休憩開始・終了を記録できます
- 複数回の休憩時間が合算されます

**例**:
```
出勤: 09:00
休憩1: 10:30 - 10:45（15分）
休憩2: 12:00 - 13:00（60分）
休憩3: 15:00 - 15:15（15分）
退勤: 18:00

勤務時間: 9時間
休憩時間: 1時間30分
実働時間: 7時間30分
```

### 6-4. 休憩時間自動差し引き

休憩時間を手動で記録しない場合、設定した時間を自動的に差し引くことができます。

**設定方法**:
1. **「休憩時間を自動差し引き」** にチェックを入れる
2. **「自動差し引き時間」** を入力（例: 45分、60分）

**使用例**:
```
設定: 自動差し引き 60分

出勤: 09:00
退勤: 18:00

勤務時間: 9時間
自動差し引き: 1時間
実働時間: 8時間（自動計算）
```

**注意**:
- 実際に休憩を取ったかどうかに関わらず、自動的に差し引かれます
- 労働基準法に基づく適切な時間を設定してください
- 休憩時間モードが「簡易」「詳細」の場合、手動記録が優先されます

## 7. リマインダー設定

### 7-1. 出勤リマインダー

指定した時刻に、まだ出勤打刻していないスタッフに通知を送ります。

**設定方法**:
1. **「出勤リマインダーを有効にする」** にチェックを入れる
2. リマインダー送信時刻を設定（例: 10:00）

**動作例**:
```
設定: 出勤リマインダー 10:00

10:00時点で出勤打刻していないスタッフに通知:
「まだ出勤打刻していません。打刻をお願いします。」
```

**推奨設定**:
- 始業時刻の30分〜1時間後
- 例: 始業9:00の場合、リマインダー10:00

### 7-2. 退勤リマインダー

指定した時刻に、まだ退勤打刻していないスタッフに通知を送ります。

**設定方法**:
1. **「退勤リマインダーを有効にする」** にチェックを入れる
2. リマインダー送信時刻を設定（例: 20:00）

**動作例**:
```
設定: 退勤リマインダー 20:00

20:00時点で退勤打刻していないスタッフに通知:
「退勤打刻を忘れていませんか？打刻をお願いします。」
```

**推奨設定**:
- 定時終業時刻の2〜3時間後
- 例: 終業18:00の場合、リマインダー20:00

## 8. 管理者向けレポート設定

### 8-1. 管理者日次レポート

毎日、管理者に勤怠状況のサマリーをメール送信する機能です。

**レポート内容**:
- 当日の出勤人数
- 未出勤者一覧
- 未退勤者一覧
- 長時間労働者一覧（12時間超など）

**設定方法**:
1. **「管理者日次レポートを有効にする」** にチェックを入れる
2. レポート送信時刻を設定（例: 10:00）
3. **「メール送信」** にチェックを入れる（推奨）

**推奨設定**:
- 朝礼後の時刻（例: 10:00）
- メール送信を有効にする

---

## 9. アラート設定

### 9-1. QR期限切れアラート

QRコードの更新期限が近づいた際に管理者に通知します。

**設定方法**:
1. **「QR期限切れアラートを有効にする」** にチェックを入れる
2. **「メール送信」** にチェックを入れる

**通知タイミング**:
- QRコード更新の1日前

### 9-2. 長時間労働アラート

設定した時間を超えて勤務しているスタッフがいる場合、管理者に通知します。

**設定方法**:
1. **「長時間労働アラートを有効にする」** にチェックを入れる
2. アラート基準時間を設定（例: 12時間）

**動作例**:
```
設定: 長時間労働アラート 12時間

スタッフAが12時間勤務した場合:
管理者に通知「スタッフAが12時間勤務しています。退勤確認をお願いします。」
```

**推奨設定**:
- 12時間（労働基準法の限度を考慮）
- 残業が多い組織では10時間

---

## 10. 設定の保存

すべての設定を変更したら、画面下部の **「設定を保存」** ボタンをクリックします。

**保存後の動作**:
- 設定がすぐに反映されます
- 次回の打刻から新しい設定が適用されます
- QRコード更新頻度の変更は、次回の更新タイミングから適用されます

## 12. 注意事項とベストプラクティス

### 12-1. 注意事項

1. **設定変更は即座に反映**: スタッフが打刻中の場合、混乱を避けるため業務時間外に変更することを推奨します
2. **少なくとも1つの打刻方法を有効化**: すべての打刻方法をオフにすると、打刻できなくなります
3. **休憩時間の法的要件**: 労働基準法に基づく適切な休憩時間を設定してください
4. **QRコード更新頻度**: 印刷掲示の場合、頻繁な更新は運用負荷が高くなります

### 12-2. ベストプラクティス

1. **段階的導入**: 最初は手動打刻のみで開始し、慣れてからQRコード打刻を追加
2. **リマインダーの活用**: 打刻忘れを防ぐため、リマインダーは有効化を推奨
3. **管理者レポートの確認**: 毎日のレポートを確認し、異常な勤務パターンを早期発見
4. **定期的な設定見直し**: 3ヶ月〜半年ごとに設定を見直し、組織に合わせて調整

## 13. トラブルシューティング

| 問題 | 原因 | 解決方法 |
|------|------|----------|
| 設定が保存できない | 権限不足 | マネージャーまたは管理者でログインしてください |
| QRコード打刻ができない | QRコード打刻が無効 | 「打刻方法設定」でQRコード打刻を有効にしてください |
| リマインダーが届かない | 通知設定が無効 | リマインダーを有効にし、時刻を設定してください |
| 管理者レポートが届かない | メール送信が無効 | 「メール送信」にチェックを入れてください |
| 休憩時間が自動差し引きされない | 休憩モードが「簡易」「詳細」で手動記録がある | 手動記録が優先されます。自動差し引きのみにする場合は、休憩モードを「なし」にしてください |
