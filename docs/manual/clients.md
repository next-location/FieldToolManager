# 取引先管理

## 概要

取引先管理機能は、顧客・仕入先・協力会社などの取引先情報を一元管理するための機能です。
見積書、請求書、発注書などすべての帳票機能の基盤となる重要なマスタデータです。

**主な機能**:
- 取引先の登録・編集・削除
- 取引先分類（顧客/仕入先/協力会社/両方）
- 連絡先・担当者情報の管理
- 支払条件・銀行情報の管理
- CSV一括インポート・エクスポート
- 取引先統計・分析

---

## 対象者

取引先管理機能は **管理者（Admin）のみ** が利用できます。

- **管理者（Admin）**: 取引先の登録・編集・削除、CSV操作、統計閲覧

**注意**: スタッフ、リーダー、マネージャーは取引先管理機能を利用できません。

---

## 取引先分類

取引先は以下の4つの分類で管理されます：

| 分類 | 説明 | 使用される機能 |
|------|------|--------------|
| **顧客** | 見積書・請求書を発行する取引先 | 見積書、請求書、工事 |
| **仕入先** | 発注書を発行する取引先（材料・道具調達先） | 発注書、消耗品発注 |
| **協力会社** | 外注・下請けの協力会社 | 発注書 |
| **顧客兼仕入先** | 顧客でもあり仕入先でもある取引先 | 見積書、請求書、発注書 |

**重要**:
- 取引先分類により、各機能で選択可能な取引先が自動的にフィルタリングされます
- 見積書・請求書作成時: 「顧客」または「顧客兼仕入先」のみ選択可能
- 発注書作成時: 「仕入先」、「協力会社」、「顧客兼仕入先」のみ選択可能

## 取引先の登録

### 基本的な登録手順

1. サイドメニューから **取引先管理** をクリック
2. 右下の **+** ボタン（FAB）をクリック
3. 取引先情報を入力
4. **登録** をクリック

### 入力項目の詳細

#### 基本情報

| 項目 | 必須 | 説明 |
|------|------|------|
| **取引先コード** | ○ | 自動生成（CL-0001形式）、手動変更可能 |
| **正式名称** | ○ | 取引先の正式な会社名・店舗名 |
| **フリガナ** | - | 検索用のフリガナ（カタカナ） |
| **略称** | - | 社内で使用する略称 |
| **取引先分類** | ○ | 顧客/仕入先/協力会社/顧客兼仕入先 |
| **業種** | - | 業種・業態 |

**取引先コードの自動生成**:
- フォーマット: `CL-0001`（連番）
- 手動で変更することもできます
- 一度登録すると変更できません

#### 連絡先情報

| 項目 | 必須 | 説明 |
|------|------|------|
| **郵便番号** | - | 7桁の郵便番号（ハイフンあり・なし両方可） |
| **住所** | - | 郵便番号入力で自動検索 |
| **電話番号** | - | 代表電話番号 |
| **FAX番号** | - | FAX番号 |
| **メールアドレス** | - | 代表メールアドレス |
| **ウェブサイト** | - | 公式サイトURL |

**郵便番号自動検索**:
- 7桁の郵便番号を入力すると、自動的に住所が検索・入力されます
- 住所は手動で編集できます

#### 担当者情報

| 項目 | 説明 |
|------|------|
| **担当者名** | 先方の担当者名 |
| **部署** | 担当者の所属部署 |
| **担当者電話番号** | 担当者の直通電話 |
| **担当者メールアドレス** | 担当者のメールアドレス |

#### 取引条件

| 項目 | 説明 |
|------|------|
| **支払条件** | 月末締め翌月末払いなど |
| **支払方法** | 銀行振込/現金/小切手/掛売り/その他 |
| **支払期日（日数）** | 請求日・発注日からの支払期日（デフォルト30日） |
| **与信限度額** | 取引可能な上限金額 |

**支払期日の自動計算**:
- 請求書・発注書作成時に、ここで設定した日数が自動的に適用されます
- 例: 支払期日30日 → 請求日の30日後が支払期日

#### 銀行情報

| 項目 | 説明 |
|------|------|
| **銀行名** | 振込先銀行名 |
| **支店名** | 支店名 |
| **口座種別** | 普通預金/当座預金/その他 |
| **口座番号** | 口座番号 |
| **口座名義** | 口座名義人 |

#### 税務情報

| 項目 | 説明 |
|------|------|
| **法人番号・事業者番号** | 法人番号または事業者番号 |
| **インボイス登録番号** | 適格請求書発行事業者の登録番号（Tから始まる13桁） |
| **非課税事業者** | チェックONで非課税事業者として扱う |

#### 評価・メモ

| 項目 | 説明 |
|------|------|
| **評価** | 1〜5段階の評価（任意） |
| **備考** | 取引先に関する一般的なメモ |
| **社内用メモ** | 社内用のメモ（取引先には表示されません） |

#### ステータス

| 項目 | 説明 |
|------|------|
| **有効** | チェックONで有効、OFFで無効（デフォルトON） |

**無効化**:
- 取引を停止した取引先は「無効」にします
- 無効にしても削除されず、データは残ります
- 無効化した取引先は、見積書・請求書・発注書で選択できなくなります

## 取引先の閲覧

### 一覧画面

取引先一覧では、タブで取引先分類ごとに表示を切り替えられます：

- **すべて**: すべての取引先
- **顧客**: 顧客のみ
- **仕入先**: 仕入先のみ
- **協力会社**: 協力会社のみ
- **顧客兼仕入先**: 両方の取引先のみ

**表示内容**:
- 取引先コード
- 正式名称
- 略称
- 取引先分類
- 電話番号
- 住所
- ステータス（有効/無効）

**検索**:
- キーワード検索（名称、コード、住所、電話番号）

### 詳細画面

取引先をクリックすると、詳細画面が開きます。

**表示内容**:
- 基本情報
- 連絡先情報
- 担当者情報
- 取引条件・銀行情報
- 税務情報
- 評価・メモ
- 関連現場一覧（この取引先に紐づいた現場）

---

## 取引先の編集・削除

### 編集

1. 取引先一覧または詳細画面から **編集** ボタンをクリック
2. 情報を修正
3. **更新** をクリック

**注意**:
- 取引先コードは変更できません
- 取引先分類を変更すると、既存の見積書・請求書・発注書との関連に影響する可能性があります

### 削除

1. 取引先詳細画面を開く
2. **削除** ボタンをクリック
3. 確認ダイアログで **削除** をクリック

**注意**:
- 削除は論理削除（ソフトデリート）です
- 削除した取引先はデータベースに残りますが、一覧には表示されません
- 見積書・請求書・発注書などに紐づいている取引先は削除できません

## CSV一括操作

### CSVインポート

複数の取引先を一括登録できます。

**インポート手順**:
1. 取引先一覧画面右上の **インポート** ボタンをクリック
2. CSVテンプレートをダウンロード
3. Excelなどで取引先情報を入力
4. CSVファイルを選択してアップロード
5. プレビューで内容を確認
6. **インポート実行** をクリック

**CSVフォーマット**:
```csv
code,name,name_kana,short_name,client_type,industry,postal_code,address,phone,email,...
CL-0001,株式会社サンプル,カブシキガイシャサンプル,サンプル社,customer,建設業,100-0001,東京都千代田区,03-1234-5678,info@sample.co.jp,...
```

**注意**:
- 必須項目: code, name, client_type
- client_typeの値: customer, supplier, partner, both
- エラーがある行はスキップされます

### CSVエクスポート

登録されている取引先をCSV形式でダウンロードできます。

**エクスポート手順**:
1. 取引先一覧画面右上の **エクスポート** ボタンをクリック
2. CSVファイルがダウンロードされる

**エクスポート内容**:
- 有効な取引先のみ
- すべての項目（基本情報、連絡先、取引条件、銀行情報、税務情報など）

## 取引先統計

取引先の統計情報を確認できます。

### 統計の表示

1. 取引先一覧画面右上の **統計** ボタンをクリック

**表示内容**:
- 総取引先数
- 有効な取引先数
- 取引先分類別の件数（顧客/仕入先/協力会社）
- 平均評価

## トラブルシューティング

### 登録ボタンが押せない場合

必須項目が未入力の可能性があります。以下を確認してください：

1. **取引先コード** が入力されているか
2. **正式名称** が入力されているか
3. **取引先分類** が選択されているか

フォーム内に赤字でエラーメッセージが表示されている項目がないか確認してください。

### 郵便番号から住所が自動入力されない場合

以下を確認してください：

1. **郵便番号が7桁** であるか（ハイフンあり・なし両方可）
2. **インターネット接続** があるか
3. **有効な郵便番号** であるか

郵便番号検索は外部APIを使用しているため、タイムラグがある場合があります。
自動入力されない場合は、手動で住所を入力してください。

### 取引先が見積書・請求書・発注書で選択できない場合

以下を確認してください：

1. **取引先が有効** になっているか
2. **取引先分類が正しい** か
   - 見積書・請求書: 「顧客」または「顧客兼仕入先」
   - 発注書: 「仕入先」、「協力会社」、「顧客兼仕入先」

取引先分類を変更する場合は、取引先編集画面から変更してください。

### CSVインポート時にエラーが発生する場合

以下を確認してください：

1. **CSVフォーマット** が正しいか（テンプレートを使用）
2. **文字コード** がUTF-8であるか
3. **必須項目** が入力されているか（code, name, client_type）
4. **client_typeの値** が正しいか（customer, supplier, partner, both）

エラーがある行はスキップされ、正常な行のみインポートされます。
インポート後にエラーメッセージを確認してください。
