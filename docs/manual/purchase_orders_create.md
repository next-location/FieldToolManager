# 発注書作成マニュアル

## 📋 概要

発注書作成機能では、仕入先への発注書を作成し、提出・承認フローに進むことができます。消耗品発注管理から自動変換することも、新規作成することも可能です。

## 👥 対象者

- **リーダー以上**: 発注書の作成・提出が可能
- **マネージャー以上**: 発注書の作成・提出・承認が可能
- **管理者**: すべての発注書操作が可能

**注意**: スタッフは発注書を作成できません。

## 📍 アクセス方法

1. 左メニューから「発注書」をクリック
2. 「+ 新規発注書」ボタンをクリック

または

1. 消耗品発注管理から「発注書を作成」ボタンをクリック

## 🔄 発注書作成の流れ

### 1️⃣ 新規作成の場合

```
基本情報入力 → 発注明細入力 → 備考入力 → 保存・提出
```

### 2️⃣ 消耗品発注から変換の場合

```
消耗品発注選択 → 自動データ読込 → 内容確認・修正 → 保存・提出
```

## 📝 基本情報の入力

### 発注番号（自動生成）

- **形式**: `PO-年月-連番`
- **例**: `PO-202601-0001`（2026年1月の1件目）
- 自動で採番されるため、編集不可

### 仕入先（必須）

- **選択対象**: 「仕入先」または「顧客兼仕入先」として登録済みの取引先のみ
- **表示形式**: `仕入先名 (取引先コード)`
- 仕入先を選択すると、支払条件が自動設定されます

**支払条件の自動設定**:
- 取引先マスタに「支払条件」が登録されている場合
- 仕入先選択時に自動で「支払条件」欄に反映されます
- 例: 月末締め翌月払い、翌月末払い、当月末払いなど

### 工事（任意）

- **選択対象**: すべての工事
- **表示形式**: `工事名 (工事コード)`
- 工事に紐づけることで、工事別の原価管理が可能

**工事選択時の自動設定**:
- 工事に紐づく現場の住所が「納品場所」に自動設定されます
- 手動で変更も可能

### 発注日（必須）

- デフォルト: 本日の日付
- 手動で変更可能

### 希望納期（必須）

- デフォルト: 発注日 + 7日後
- 手動で変更可能
- 仕入先に納品を希望する日付を設定

### 支払条件（任意）

- 仕入先選択時に自動設定（取引先マスタから）
- 手動で変更・追加も可能
- **例**: 月末締め翌月払い、翌月末払い、当月末払い、納品時現金払い

### 納品場所（任意）

- 工事選択時に現場住所が自動設定
- 手動で入力・変更も可能
- **例**: 〇〇現場（住所）、本社倉庫、資材置き場

## 📊 発注明細の入力

明細はカード形式で1件ずつ入力します。

### 種別

以下から選択します：

| 種別 | 説明 |
|------|------|
| 材料 | 建設資材、部品など |
| 機材 | 建設機械、工具など |
| 外注 | 外注作業、下請け工事 |
| 労務 | 人工、作業員の手配 |
| その他 | 上記以外 |

### 品名（必須）

- 発注する品物の名称を入力
- 最大100文字
- 例: セメント、鉄筋、足場材、コンクリートミキサー

### 仕様・規格（任意）

- 品物の詳細仕様や規格を入力
- 最大200文字
- 例: 25kg袋、直径13mm、長さ4m、型番ABC-123

### 数量（必須）

- 数値を入力
- 小数点も入力可能
- 例: 1、10、100、10.5

### 単位（必須）

以下から選択：

| 単位 | 用途 |
|------|------|
| 個 | 個数 |
| 袋 | 袋詰め材料 |
| 箱 | 箱入り製品 |
| 台 | 機械・設備 |
| 本 | 長尺物 |
| 枚 | 板材 |
| m | 長さ |
| m² | 面積 |
| m³ | 体積 |
| kg | 重量（キログラム） |
| t | 重量（トン） |
| L | 容量（リットル） |
| 式 | 一式 |
| 日 | 日数 |
| 時間 | 時間数 |
| 人 | 人数 |

### 単価（必須）

- 1単位あたりの価格（税抜）
- 数値を入力
- 例: 500、1000、50000

### 税率

以下から選択：

| 税率 | 用途 |
|------|------|
| 10% | 標準税率（デフォルト） |
| 8% | 軽減税率（飲食料品など） |
| 0% | 非課税（特定条件下） |

### 金額（自動計算）

- **計算式**: 数量 × 単価 = 金額
- 編集不可（自動計算）
- 例: 数量10 × 単価500円 = 金額5,000円

### 明細の追加・削除

- **追加**: 「+ 明細を追加」ボタンをクリック
- **削除**: 各明細カードの「削除」ボタンをクリック
- **注意**: 明細は最低1件必要（すべて削除不可）

## 💰 合計金額の確認

画面右下に自動計算された金額が表示されます：

| 項目 | 説明 |
|------|------|
| 小計 | 全明細の金額合計（税抜） |
| 消費税 | 各明細の消費税合計（小数点以下切り捨て） |
| 合計 | 小計 + 消費税（発注金額） |

**計算式**:
```
小計 = Σ(数量 × 単価)
消費税 = Σ(金額 × 税率 / 100) ※切り捨て
合計 = 小計 + 消費税
```

## 📝 備考の入力

### 発注備考（仕入先向け）

- 発注書PDFに記載される備考
- 最大2000文字
- **例**:
  - 納品時間: 午前中を希望します。
  - 配送業者を手配してください。
  - 荷下ろしは当方で行います。
  - 雨天時は翌日に延期してください。

### 社内メモ

- 発注書には記載されない、社内用のメモ
- 最大2000文字
- **例**:
  - 急ぎの発注のため、納期厳守を依頼済み
  - 前回の発注で遅延があったため注意
  - 消耗品発注管理から作成 (発注番号: ORD-20260104-0001)

## 💾 保存・提出

画面下部に3つのボタンが表示されます：

### キャンセル

- 入力内容を破棄して発注書一覧に戻ります
- 入力内容は保存されません

### 下書き

- **ステータス**: 「下書き」として保存
- **誰が使う**: リーダー、マネージャー、管理者
- **用途**: 一旦保存して後で編集したい場合
- 下書きは何度でも編集可能

### 提出

- **ステータス**: 「提出済み」として保存
- **用途**: 承認フローに進む
- 提出後は編集不可（承認または差し戻しまで）

**注意**: 発注金額により、承認フローが異なります：
- 10万円未満: 自動承認（承認フローなし）
- 10万円以上100万円未満: マネージャー承認が必要
- 100万円以上: マネージャー＋管理者の2段階承認が必要

## 🔄 消耗品発注から変換する場合

### 変換できる消耗品発注

- **条件**: 「発注済み」ステータスの消耗品発注のみ
- 消耗品発注詳細画面に「発注書を作成」ボタンが表示されます

### 自動コピーされる内容

以下の情報が消耗品発注から自動的にコピーされます：

| 項目 | 変換内容 |
|------|----------|
| 仕入先 | そのままコピー（取引先IDで関連付け） |
| 発注日 | そのままコピー |
| 希望納期 | 納品予定日をコピー |
| 明細 | 消耗品の名称・数量・単価・金額がコピー |
| 備考 | そのままコピー |
| 社内メモ | 「消耗品発注管理から作成 (発注番号: ...)」が自動追加 |

発注番号は新規に生成されます。

### 変換後の編集

- コピーされた内容は自由に編集可能
- 明細の追加・削除も可能
- 金額の修正も可能

### 消耗品発注との連携

- 発注書を作成しても、元の消耗品発注は削除されません
- 消耗品発注と発注書は独立して管理されます
- 発注書の社内メモに消耗品発注番号が記録されます

## ⚠️ 注意事項

### 権限について

- **スタッフ**: 発注書を作成できません
- **リーダー**: 作成・提出は可能、承認はできません
- **マネージャー以上**: 作成・提出・承認がすべて可能

### 仕入先の登録

- 発注書を作成する前に、取引先マスタに「仕入先」として登録が必要
- 支払条件も事前に登録しておくと、自動設定されます

### 発注番号

- 自動採番のため、欠番が発生する可能性があります
  - 例: 下書き保存後に削除した場合
- 発注番号は変更できません

### 承認フロー（金額別）

発注金額により、承認フローが自動的に決定されます：

| 発注金額 | 承認フロー | 所要時間目安 |
|---------|-----------|-------------|
| 10万円未満 | 自動承認 | 即時 |
| 10万円以上100万円未満 | マネージャー承認 | 数時間〜1日 |
| 100万円以上 | マネージャー＋管理者承認 | 1〜3日 |

金額が大きい発注は、複数段階の承認が必要になります。

### 納品場所の自動設定

- 工事を選択すると、現場住所が納品場所に自動設定されます
- 倉庫など別の場所に納品したい場合は、手動で変更してください

### 消耗品発注から変換

- 消耗品発注を変換しても、元の消耗品発注は残ります（削除されません）
- 変換後に消耗品発注が変更されても、発注書には反映されません
- 発注書と消耗品発注は独立して管理されます
