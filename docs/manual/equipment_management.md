# 重機管理マニュアル

## 対象者

このマニュアルは、重機（建設機械）の登録・管理を行う以下の権限のユーザーを対象としています。

- **Manager（マネージャー）**: 重機の登録・編集・メンテナンス記録が可能
- **Admin（管理者）**: 全機能が利用可能

※ Staff、Leaderは重機情報の閲覧とQRスキャン、移動登録のみ可能です。重機の新規登録・編集はできません。

---

## 前提条件

重機管理を始める前に、以下を確認してください。

1. **契約プラン**: 現場資産パック、または フル機能統合パックが契約済みであること
2. **重機機能の有効化**: 管理者が設定で重機管理機能を有効にしていること

---

## 重機管理の基本概念

### 1. 重機とは

重機（Heavy Equipment）は、建設現場で使用する大型機械のことです。

**対象となる機械:**
- バックホウ（油圧ショベル）
- ホイールローダー
- ダンプトラック
- クレーン車
- 高所作業車
- フォークリフト
- ローラー・締固め機械
- その他建設機械

### 2. 道具管理との違い

| 項目 | 道具（Tool） | 重機（Heavy Equipment） |
|------|-------------|------------------------|
| **管理対象** | 電動工具、手工具、測定器 | 建設機械、車両 |
| **価格帯** | 数千円〜数十万円 | 数百万円〜数千万円 |
| **管理内容** | 在庫、所在、状態 | 所有形態、車検、保険、メンテナンス |
| **QRコード** | 個別アイテムごと | 1台ごと |
| **特有の管理** | なし | 車検、保険、稼働時間、燃料 |

### 3. 所有形態の種類

重機は、以下の所有形態で管理できます。

| 所有形態 | 説明 | 管理ポイント |
|---------|------|-------------|
| **自社所有** | 購入した重機 | 減価償却、車検、保険 |
| **リース** | リース契約で使用 | リース期間、返却期限 |
| **レンタル** | 短期レンタル | レンタル期間、返却期限、レンタル料 |

## 重機の新規登録

### ステップ1: 基本情報の入力

1. **重機管理 > 重機一覧** を開く
2. **「+ 重機を追加」** をクリック
3. 以下の情報を入力

#### 基本情報（必須）
- **重機コード**: 管理用コード（例: HE-001）
  - 自動採番またはカスタムコード
- **機械名**: 重機の名称（例: バックホウ 0.25m³）
- **カテゴリ**: バックホウ、ホイールローダーなどを選択

#### 詳細情報（推奨）
- **メーカー**: 製造メーカー（例: コマツ、日立建機）
- **型番**: メーカーの型番（例: PC30MR-5）
- **製造番号**: 車体番号（シリアルナンバー）
- **購入日**: 購入した日付（自社所有の場合）
- **購入価格**: 購入価格（円）

### ステップ2: 所有形態の設定

**所有形態**を選択します。

#### 自社所有の場合
- **所有形態**: 「自社所有」を選択
- 減価償却、車検、保険の管理が有効になります

#### リースの場合
- **所有形態**: 「リース」を選択
- **リース会社**: リース会社名を入力
- **リース開始日**: リース契約開始日
- **リース終了日**: リース契約終了日
- **月額リース料**: 月額料金（円）

#### レンタルの場合
- **所有形態**: 「レンタル」を選択
- **レンタル会社**: レンタル会社名を入力
- **レンタル開始日**: レンタル開始日
- **レンタル終了日**: レンタル返却期限
- **日額レンタル料**: 1日あたりの料金（円）

### ステップ3: 車検・保険の設定（自社所有の場合）

#### 車検が必要な重機の場合

- **車検が必要**: チェックを入れる
- **車検満了日**: 次回の車検満了日を入力
- **車検実施業者**: 車検を受ける業者名

**車検が必要な重機:**
- 3トン以上の建設機械
- 公道を走行する車両
- クレーン車、高所作業車など

#### 保険の設定

- **保険契約番号**: 保険証券番号
- **保険会社**: 保険会社名
- **保険開始日**: 保険契約開始日
- **保険終了日**: 保険満期日
- **保険種類**: 自賠責、任意保険など

### ステップ4: オプション設定

#### 稼働時間メーター管理

重機の稼働時間を記録したい場合:

- **メーター管理を有効にする**: チェックを入れる
- **現在のメーター値**: 現在の稼働時間（時間）

**活用例:**
- 定期メンテナンスの時期管理
- オイル交換のタイミング
- 減価償却の参考データ

#### 燃料管理

燃料消費を記録したい場合:

- **燃料管理を有効にする**: チェックを入れる
- **燃料タンク容量**: タンク容量（リットル）

4. **「登録」** をクリック

### ステップ5: QRコードの印刷と貼り付け

重機登録時に、自動的にQRコードが生成されます。

1. 重機詳細画面を開く
2. **「QR印刷」** をクリック
3. PDFをダウンロードして印刷
4. 重機の見やすい場所に貼り付け

**推奨貼付位置:**
- 運転席のドア付近
- 機体の側面
- 操作パネル付近

**ラベルサイズ:** 5cm x 5cm 以上（重機は屋外のため大きめ推奨）

## 重機の移動登録

### QRスキャンによる移動登録

1. スマートフォンで **QRスキャン** を開く
2. 重機のQRコードをスキャン
3. **移動先**を選択（倉庫 / 現場）
4. 現場の場合は、現場名を選択
5. **メモ**（オプション）を入力
6. **「移動登録」** をクリック

### 手動での移動登録

1. **重機管理 > 移動登録** を開く
2. **重機**を選択
3. **移動元・移動先**を選択
4. **移動日時**を入力（過去の移動も記録可能）
5. **メモ**を入力
6. **「登録」** をクリック

---

## メンテナンス記録

### 点検記録の種類

重機のメンテナンス記録は、以下の種類があります。

| 点検タイプ | 説明 | 記録内容 |
|-----------|------|---------|
| **車検** | 法定車検の実施記録 | 車検日、次回車検日、業者名、費用 |
| **保険更新** | 保険の更新記録 | 更新日、満期日、保険会社、保険料 |
| **定期点検** | 定期的なメンテナンス | 点検日、点検内容、費用 |
| **修理** | 故障時の修理記録 | 修理日、修理内容、修理業者、費用 |
| **部品交換** | 消耗品・部品の交換 | 交換日、部品名、費用 |

### 点検記録の登録方法

1. **重機管理 > 重機一覧** で対象の重機をクリック
2. **「メンテナンス」** タブを開く
3. **「+ 記録を追加」** をクリック
4. 以下を入力

#### 基本情報
- **点検タイプ**: 車検 / 保険更新 / 定期点検 / 修理 / 部品交換
- **点検日**: 実施日
- **次回点検予定日**（オプション）

#### 実施内容
- **実施業者**: 業者名（例: ○○整備工場）
- **点検内容**: 実施した内容を記録
- **費用**: 点検・修理費用（円）

#### その他
- **稼働時間**: 点検時のメーター値（メーター管理有効時）
- **添付ファイル**（オプション）: 点検記録票のPDFなど
- **メモ**: 特記事項

5. **「登録」** をクリック

### 車検・保険の更新

車検または保険を更新した際は、以下の手順で記録します。

1. メンテナンス記録を追加
2. **点検タイプ**: 「車検」または「保険更新」を選択
3. **次回点検予定日**: 次回の車検満了日または保険満期日を入力
4. 登録すると、重機マスタの車検日・保険満期日が自動更新されます

## 稼働時間の記録（メーター管理）

### 稼働時間の入力

メーター管理を有効にしている重機は、稼働時間を記録できます。

1. 重機詳細画面を開く
2. **「稼働時間を更新」** をクリック
3. **現在のメーター値**を入力（例: 1250時間）
4. **記録日時**を入力
5. **「更新」** をクリック

### 稼働時間の履歴確認

1. 重機詳細画面の **「稼働時間履歴」** タブを開く
2. 過去の稼働時間の推移をグラフで確認
3. メンテナンス時期の目安として活用

## アラート機能

### 車検満了日アラート

車検満了日の60日前になると、重機一覧で警告マークが表示されます。

**表示例:**
```
⚠️ バックホウ HE-001 (車検満了: 2026-03-15 まであと45日)
```

### 保険満期アラート

保険満期日の60日前になると、警告マークが表示されます。

**表示例:**
```
⚠️ クレーン車 HE-005 (保険満期: 2026-04-20 まであと30日)
```

### リース・レンタル返却期限アラート

リース終了日・レンタル返却期限の30日前になると、警告マークが表示されます。

**表示例:**
```
⚠️ ホイールローダー HE-003 (レンタル返却: 2026-02-28 まであと20日)
```

### アラート設定のカスタマイズ

アラート通知期間は、組織設定で変更できます。

1. **設定 > 運用設定 > 重機管理設定** を開く
2. **車検アラート期間**: 60日（デフォルト）を変更
3. **保険アラート期間**: 60日（デフォルト）を変更
4. **レンタル返却アラート期間**: 30日（デフォルト）を変更

## 重機情報の編集

### 編集手順

1. **重機管理 > 重機一覧** を開く
2. 編集したい重機をクリック
3. **「編集」** をクリック
4. 情報を修正
5. **「保存」** をクリック

**編集可能な項目:**
- 基本情報（機械名、型番、メーカー）
- カテゴリ
- 所有形態、リース・レンタル情報
- 車検、保険情報
- メーター管理設定
- 燃料管理設定

## 重機の削除

### 削除の制限事項

重機を削除する際は、以下の条件を確認してください：

- **使用履歴がある重機は削除できません**
- **メンテナンス記録がある重機は削除できません**

誤って登録した重機で、まだ使用していないもののみ削除可能です。

### 削除手順

1. **重機管理 > 重機一覧** を開く
2. 削除したい重機をクリック
3. **「削除」** をクリック
4. 確認ダイアログで **「削除」** をクリック

**注意:**
- 削除は論理削除（データは残るが非表示）です
- 復元が必要な場合は、管理者に連絡してください

## 検索とフィルタリング

### 基本的な検索

1. **重機管理 > 重機一覧** を開く
2. 画面上部の検索ボックスにキーワードを入力
3. 機械名、型番、メーカー名、重機コードで検索できます

### フィルタリング

**「フィルタ」** ボタンをクリックすると、以下の条件で絞り込みができます：

- **カテゴリ**: バックホウ、ホイールローダーなど
- **メーカー**: メーカー名で絞り込み
- **所有形態**: 自社所有 / リース / レンタル
- **場所**: 倉庫 / 現場 / 修理中
- **車検期限**: 車検期限が近い重機
- **保険期限**: 保険満期が近い重機

## コスト分析（Manager/Admin限定）

### コストレポートの表示

重機の運用コストを分析できます。

1. **重機管理 > コストレポート** を開く
2. **期間**を選択（月次 / 四半期 / 年次）
3. レポートが表示されます

**レポート内容:**
- 重機ごとのコスト内訳（購入費 / リース料 / レンタル料 / メンテナンス費）
- 月別コスト推移グラフ
- 所有形態別コスト比較
- 稼働率（使用日数 / 総日数）

### コストの内訳

レポートには以下のコストが集計されます：

- **購入費**: 自社所有の重機の減価償却費
- **リース料**: 月額リース料の合計
- **レンタル料**: 日額レンタル料の合計
- **メンテナンス費**: 点検・修理・部品交換の合計
- **保険料**: 保険料
- **燃料費**: 燃料管理を有効にしている場合

## トラブルシューティング

### 重機が見つからない場合の検索方法

重機一覧で目的の重機が見つからない場合は、以下の手順で検索してください。

1. **検索ボックスを使用**
   - 画面上部の検索ボックスに機械名、型番、重機コードを入力
   - 部分一致で検索可能

2. **フィルタリング機能を使用**
   - **「フィルタ」** ボタンをクリック
   - カテゴリ、メーカー、所有形態、場所で絞り込み

3. **削除済み重機の確認**
   - 削除された重機は一覧に表示されません
   - 管理者に確認を依頼してください

### QRコードの再印刷方法

QRコードが汚れて読めなくなった場合や、破損した場合は再印刷できます。

1. 重機詳細画面を開く
2. **「QR印刷」** をクリック
3. PDFをダウンロードして印刷
4. 新しいQRコードを貼り直す

**注意:** QRコード（UUID）自体は変わらないため、既存の使用履歴は保持されます。

### 車検・保険満期日の修正方法

車検日または保険満期日に誤りがある場合は、編集画面で修正できます。

1. 重機詳細画面を開く
2. **「編集」** をクリック
3. **車検満了日**または**保険終了日**を修正
4. **「保存」** をクリック

または、メンテナンス記録で車検・保険更新を登録すると、自動的に更新されます。

## 注意事項

### 登録時の注意

- **製造番号は正確に**: 車体番号は資産管理に必須
- **所有形態を明確に**: リース・レンタルは返却期限を必ず記録
- **車検・保険情報は漏れなく**: 法令遵守のため必須

### 運用上の注意

- **QRコードは必ず貼る**: 移動登録の基本です
- **メンテナンス記録を徹底**: 故障予防と資産管理のため
- **アラートを確認**: 車検・保険切れを防ぐ

### セキュリティ

- **削除は慎重に**: 論理削除ですが、復元には管理者権限が必要
- **他社のQRコードは読めません**: マルチテナント型のため、他社データは完全分離
