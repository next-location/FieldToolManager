## 見積書作成の概要

ザイロクの見積書作成機能は、顧客への見積書を簡単に作成できる機能です。

### 見積書の用途

- 顧客への工事費用の提示
- 資材・労務費の見積もり
- プロジェクト単位での見積書管理
- 見積書から請求書への変換

### 作成できるユーザー

| 権限 | 見積書作成 |
|------|-----------|
| **Staff** | ❌ |
| **Leader** | ✅ |
| **Manager** | ✅ |
| **Admin** | ✅ |

**Leader以上**の権限で見積書を作成できます。一般スタッフは作成できません。

---

## 見積書作成の流れ

### 全体の流れ

```
1. 見積書作成画面を開く
   ↓
2. 基本情報を入力（取引先、工事、見積日など）
   ↓
3. 明細を入力（品目、数量、単価など）
   ↓
4. 備考を入力（顧客向け・社内用）
   ↓
5. 合計金額を確認
   ↓
6. 下書き保存 または 提出
```

### 所要時間

- **簡単な見積書**: 5分
- **通常の見積書**: 10〜15分
- **複雑な見積書**: 20分〜

## 基本情報の入力

### アクセス方法

```
見積管理 > 新規見積書
```

### 見積番号（自動生成）

**見積番号は自動的に生成されます。**

**フォーマット**: `EST-年-連番`

**例**:
```
EST-2026-0001
EST-2026-0002
EST-2026-0003
```

**特徴**:
- 年ごとに連番がリセットされます
- 自動採番されるため、入力不要です
- 変更はできません（読み取り専用）

### 取引先（必須）

**見積書を提出する顧客を選択します。**

**入力方法**:

1. 取引先フィールドをクリック
2. 検索ボックスが表示される
3. 取引先名を入力（部分一致で検索）
4. 候補から選択

**検索機能**:
```
検索対象:
- 取引先名（漢字）
- 取引先名（カナ）※ひらがな入力でも検索可能
- 取引先コード
```

**表示例**:
```
┌────────────────────────┐
│ 取引先を検索...        │
└────────────────────────┘
  ↓ 「山田」と入力
┌────────────────────────┐
│ 山田建設株式会社        │
│ CLI-0001 / ヤマダケンセツ│
├────────────────────────┤
│ 山田工業株式会社        │
│ CLI-0002 / ヤマダコウギョウ│
└────────────────────────┘
```

**選択後**:
- 選択をクリア（青いリンク）で選択を解除できます
- 取引先を変更すると、工事の選択もクリアされます

**注意事項**:
- ⚠️ 取引先は必須項目です
- ⚠️ 取引先タイプが「顧客」または「顧客兼仕入先」の取引先のみ選択可能
- ⚠️ 取引先を後から変更すると、工事の選択もリセットされます

### 工事（任意）

**取引先に紐づいた工事を選択します。**

**入力方法**:

1. 取引先を先に選択
2. 工事フィールドをクリック
3. 検索ボックスが表示される
4. 工事名を入力（部分一致で検索）
5. 候補から選択

**検索機能**:
```
検索対象:
- 工事名
- 工事コード
```

**表示例**:
```
┌────────────────────────┐
│ 工事を検索...          │
└────────────────────────┘
  ↓ 「ビル」と入力
┌────────────────────────┐
│ 新宿ビル建設工事        │
│ PRJ-0001              │
├────────────────────────┤
│ 渋谷ビル改修工事        │
│ PRJ-0002              │
└────────────────────────┘
```

**工事選択時の自動入力**:
- 工事を選択すると、**件名が自動的に設定**されます
- 例: 「新宿ビル建設工事」を選択 → 件名: 「新宿ビル建設工事 見積書」

**注意事項**:
- ⚠️ 取引先を選択していないと、工事は選択できません
- ⚠️ 選択した取引先に紐づいた工事のみ表示されます
- ⚠️ ステータスが「計画中」または「進行中」の工事のみ表示されます

### 見積日（必須）

**見積書の発行日を指定します。**

**デフォルト値**: 今日の日付

**入力方法**:
- カレンダーから選択
- または、手動で入力（YYYY-MM-DD形式）

**注意事項**:
- ⚠️ 見積日は必須項目です
- ⚠️ 通常は今日の日付を使用します

### 有効期限（任意）

**見積書の有効期限を指定します。**

**デフォルト値**: 見積日から30日後

**入力方法**:
- カレンダーから選択
- または、手動で入力（YYYY-MM-DD形式）

**有効期限の目安**:
```
通常の工事: 30日
大規模工事: 60日
小規模工事: 14日
```

**注意事項**:
- ⚠️ 有効期限を過ぎると、見積書は「期限切れ」ステータスになります
- ⚠️ 期限切れ後は編集・承認・送付ができなくなります
- ⚠️ 余裕を持った期限設定を推奨します

### 件名（必須）

**見積書のタイトルを入力します。**

**入力方法**:
- 手動で入力
- または、工事を選択すると自動入力

**入力例**:
```
✅ 良い例:
- 新宿ビル建設工事 見積書
- ○○マンション改修工事 見積書
- 電気設備工事 見積書

❌ 悪い例:
- 見積もり
- 工事
- お見積り
```

**文字数制限**: 最大200文字

**注意事項**:
- ⚠️ 件名は必須項目です
- ⚠️ 顧客に表示される重要な項目のため、わかりやすく記載してください

## 明細の入力

見積書の明細（品目、数量、単価など）を入力します。

### 明細の追加

**初期状態**: 1行の明細が表示されています

**明細を追加する方法**:
1. 明細セクションの下部にある**「明細を追加」**ボタンをクリック
2. 新しい明細行が追加される

**明細の削除**:
- 各明細の右上にある**「削除」**ボタンをクリック
- 最後の1行は削除できません

### 明細の入力項目

#### 1. 種別

**明細の種類を選択します。**

**選択肢**:
```
- 工事: 工事費用
- 材料: 資材・材料費
- 経費: 交通費、宿泊費など
- その他: 上記に当てはまらないもの
```

**「その他」を選択した場合**:
- カスタム種別入力フィールドが表示されます
- 自由に種別名を入力できます（例: 「設計費」「調査費」）

#### 2. 品目名（必須）

**見積もりする品目の名前を入力します。**

**入力例**:
```
工事の場合:
- 基礎工事
- 内装工事
- 電気設備工事

材料の場合:
- コンクリート
- 鉄筋
- 配管材料

経費の場合:
- 交通費
- 宿泊費
- 車両費
```

**注意事項**:
- ⚠️ 品目名は必須項目です
- ⚠️ 顧客に表示されるため、わかりやすく記載してください

#### 3. 摘要・説明

**品目の詳細説明を入力します（任意）。**

**入力例**:
```
- RC造、深さ3m
- φ25mm、SD345、10m
- 東京-現場往復、3日分
```

**用途**:
- 品目の詳細仕様
- 作業内容の補足
- 材料の規格

#### 4. 数量（必須）

**品目の数量を入力します。**

**入力方法**:
- 数値を入力（小数点も可能）
- 全角数字を入力しても、自動的に半角に変換されます

**入力例**:
```
10
100.5
0.5
```

**注意事項**:
- ⚠️ 数量は必須項目です
- ⚠️ 0以上の数値を入力してください
- ⚠️ 全角数字（０、１、２...）は自動的に半角（0、1、2...）に変換されます

#### 5. 単位（必須）

**品目の単位を選択します。**

**選択肢**:
```
よく使う単位:
- 式: 工事一式、設備一式など
- 個: 部品、器具など
- 枚: 板材、シートなど
- m: 長さ
- m²: 面積
- m³: 体積
- kg: 重量
- 本: パイプ、柱など
- 人工: 作業員の工数
- 日: 日数
- その他: カスタム単位
```

**「その他」を選択した場合**:
- カスタム単位入力フィールドが表示されます
- 自由に単位を入力できます（例: 「台」「セット」「組」）

**注意事項**:
- ⚠️ 単位は必須項目です
- ⚠️ 適切な単位を選択してください

#### 6. 単価（必須）

**品目1単位あたりの金額を入力します。**

**入力方法**:
- 数値を入力（小数点も可能）
- 全角数字を入力しても、自動的に半角に変換されます
- カンマ区切りは不要（自動的に表示されます）

**入力例**:
```
50000
100000.5
12345
```

**注意事項**:
- ⚠️ 単価は必須項目です
- ⚠️ 0以上の数値を入力してください
- ⚠️ 全角数字は自動的に半角に変換されます

#### 7. 税率（必須）

**消費税率を選択します。**

**選択肢**:
```
- 10%（標準税率）← デフォルト
- 8%（軽減税率）
- 0%（非課税）
```

**デフォルト**: 10%

**注意事項**:
- ⚠️ 通常は10%を使用します
- ⚠️ 食品など軽減税率対象の場合は8%を選択
- ⚠️ 非課税取引の場合は0%を選択

#### 8. 金額（自動計算）

**金額は自動的に計算されます。**

**計算式**:
```
金額 = 数量 × 単価
```

**例**:
```
数量: 10
単価: 50,000円
↓
金額: 500,000円（自動計算）
```

**注意事項**:
- ⚠️ 金額は手動で入力できません（自動計算のみ）
- ⚠️ 数量または単価を変更すると、即座に再計算されます

### 明細入力のショートカット

**効率的な入力方法**:

1. **Tab キー**: 次の入力フィールドに移動
2. **Enter キー**: フォーム送信（注意）
3. **コピー&ペースト**: Excel から数値をコピー可能

**注意事項**:

- ⚠️ PDF出力時も印刷されません

## 合計金額の確認

見積書の下部に、合計金額が自動的に表示されます。

### 表示内容

```
┌─────────────────────────┐
│ 小計: ¥500,000         │
│ 消費税（10%）: ¥50,000  │
│ 合計: ¥550,000         │
└─────────────────────────┘
```

### 計算方法

**小計**:
```
小計 = 各明細の金額の合計
```

**消費税**:
```
消費税 = 各明細の（金額 × 税率 / 100）の合計
```

**合計**:
```
合計 = 小計 + 消費税
```

**例**:
```
明細1: 金額 300,000円、税率 10%
明細2: 金額 200,000円、税率 10%

小計: 500,000円
消費税: 50,000円（300,000 × 0.1 + 200,000 × 0.1）
合計: 550,000円
```

## 保存と提出

見積書を保存または提出します。

### 下書き保存

**見積書を下書きとして保存します。**

**操作方法**:
1. 画面下部の**「下書き保存」**ボタン（灰色）をクリック
2. 見積書が保存される
3. 見積書一覧画面に戻る

**下書き保存後の状態**:
- ステータス: 「下書き」（灰色）
- 後から編集可能
- 提出前に何度でも修正可能

**用途**:
- 見積書を途中まで作成して保存
- 上司に確認してもらう前に保存
- 後で修正する予定の見積書

### 提出

**見積書をManager/Adminに提出して承認を依頼します。**

**操作方法**:
1. 画面下部の**「提出する」**ボタン（青色）をクリック
2. 見積書が提出される
3. 見積書一覧画面に戻る

**提出後の状態**:
- ステータス: 「提出済み」（オレンジ色）
- **編集不可**
- Manager/Adminが承認またはし差戻しを行う

**注意事項**:
- ⚠️ 提出後は編集できません
- ⚠️ 差し戻された場合のみ、再度編集可能になります
- ⚠️ 提出前に内容を十分に確認してください

### 保存・提出時のバリデーション

**必須項目のチェック**:
- ✅ 取引先
- ✅ 見積日
- ✅ 件名
- ✅ 明細の品目名（各行）
- ✅ 明細の数量（各行）
- ✅ 明細の単位（各行）
- ✅ 明細の単価（各行）

**未入力の場合**:
```
❌ エラーメッセージが表示されます
「必須項目を入力してください」
```
