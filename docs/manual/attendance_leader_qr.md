## QR発行の概要

ザイロクでは、スタッフが出退勤時にQRコードをスキャンして簡単に打刻できる機能があります。リーダー・管理者は、**会社QR**と**現場QR**の2種類のQRコードを発行できます。

### QRコードの種類

| QRコード | 発行権限 | 用途 | 設置場所 |
|---------|---------|------|----------|
| **会社QR** | Manager / Admin のみ | 会社での出退勤打刻 | 会社の入口・受付 |
| **現場QR** | Leader / Manager / Admin | 現場での出退勤打刻 | 各現場の入口・事務所 |

### 機能の特徴

- ✅ **簡単発行**: ボタン1つでQRコード生成
- ✅ **有効期限管理**: 自動的に期限を設定（1日/3日/7日/30日）
- ✅ **再発行可能**: 期限切れや紛失時に即座に再発行
- ✅ **印刷・保存**: PNG画像で保存、直接印刷も可能
- ✅ **セキュリティ**: 期限切れQRは使用不可

## 会社QR発行（管理者専用）

会社での出退勤に使用するQRコードです。**Manager**または**Admin**権限のみ発行できます。

### アクセス方法

```
出退勤 > 出退勤QR発行
```

画面上部に「会社QR発行（管理者専用）」セクションが表示されます。

### 初回発行手順

**ステップ1: 発行ボタンをクリック**

1. 「出退勤QR発行」画面を開く
2. **「会社QR発行」**ボタン（青色）をクリック
3. 数秒待つ（QRコード生成中）

**ステップ2: 発行完了**

発行が完了すると、以下の情報が表示されます：

```
会社QRコード
有効期限: 2026-02-20 23:59:59
残り 30 日
```

**表示されるボタン**:
- 🔵 **表示** - QRコードを画面に表示
- ⚪ **保存** - PNG画像としてダウンロード
- ⚪ **印刷** - プリンターで直接印刷
- 🟠 **再発行** - 新しいQRコードを発行

### 会社QRの表示

1. **「表示」**ボタンをクリック
2. 大きなQRコードが画面に表示される
3. もう一度クリックすると非表示

**画面イメージ**:
```
┌──────────────────────┐
│                      │
│    [QRコード画像]    │
│    320px × 320px     │
│                      │
└──────────────────────┘
```

### 会社QRの保存

1. **「保存」**ボタンをクリック
2. ファイル名: `office-qr.png`
3. ダウンロードフォルダに保存される

**保存後の使い方**:
- 印刷して会社の入口に掲示
- タブレットに表示して受付に設置
- 社内共有フォルダに保存

### 会社QRの印刷

1. **「印刷」**ボタンをクリック
2. 印刷プレビュー画面が自動的に開く
3. プリンターを選択して印刷

**印刷内容**:
```
━━━━━━━━━━━━━━━━━━━━
会社QRコード

[QRコード画像]

有効期限: 2026-02-20 23:59:59
残り 30 日
━━━━━━━━━━━━━━━━━━━━
```

### 会社QRの再発行

有効期限が近づいた場合や、紛失した場合に再発行できます。

**再発行手順**:

1. **「再発行」**ボタン（オレンジ色）をクリック
2. 確認なしで即座に新しいQRコードを発行
3. **古いQRコードは無効化**される

**注意事項**:
- ⚠️ 再発行すると、古いQRコードは使用できなくなります
- ⚠️ 既に印刷・掲示している場合は、新しいQRコードに貼り替えてください
- ⚠️ スタッフに再発行を周知してください

### 有効期限について

会社QRの有効期限は、**出退勤設定の「QR更新頻度」**で設定されます。

**設定方法**:
```
出退勤 > 出退勤設定 > QR更新頻度
```

**選択肢**:
- 1日（毎日再発行）
- 3日
- 7日（1週間）
- **30日**（デフォルト、推奨）

**残り日数の確認**:
- 会社QRコード情報に「残り ○○ 日」と表示されます
- 期限が近づいたら早めに再発行してください

## 現場QR発行（リーダー・管理者）

各現場での出退勤に使用するQRコードです。**Leader**、**Manager**、**Admin**権限で発行できます。

### 現場QR発行手順

**ステップ1: 現場を選択**

1. 「出退勤QR発行」画面を開く
2. 「現場QR発行」セクションに移動
3. **現場選択**ドロップダウンをクリック
4. 発行したい現場を選択

**ドロップダウン表示例**:
```
現場を選択してください
─────────────────
A現場
B現場（発行済み）
C現場
新宿ビル建設現場（発行済み）
```

**発行済みの現場**:
- 「（発行済み）」と表示される
- 再選択すると再発行される

**ステップ2: QRコード発行**

1. 現場を選択
2. **「QRコード発行」**ボタンをクリック
3. 発行中は「発行中...」と表示
4. 緑色の成功メッセージが表示される

**成功メッセージ例**:
```
✓ QRコードを発行しました
```

**ステップ3: 発行済みQRコードを確認**

画面下部の「有効な現場QRコード」セクションに、発行した現場が表示されます。

### 現場QRの表示・操作

発行済みの現場QRコードは、以下の情報が表示されます：

```
A現場
発行日時: 2026-01-21 10:30:00
有効期限: 2026-02-20 23:59:59

[表示] [保存] [印刷]
```

**ボタンの説明**:

| ボタン | 機能 | 説明 |
|--------|------|------|
| **表示** | QRコード表示/非表示 | クリックでQRコードを表示 |
| **保存** | PNG画像で保存 | ファイル名: `site-qr-現場名.png` |
| **印刷** | 直接印刷 | 印刷プレビュー画面を開く |

### 現場QRの表示

1. **「表示」**ボタンをクリック
2. QRコードが画面に表示される（320px × 320px）
3. もう一度クリックすると非表示

**表示イメージ**:
```
A現場
発行日時: 2026-01-21 10:30:00
有効期限: 2026-02-20 23:59:59

[非表示] [保存] [印刷]

┌──────────────────────┐
│                      │
│    [QRコード画像]    │
│    320px × 320px     │
│                      │
└──────────────────────┘
```

### 現場QRの保存

1. **「保存」**ボタンをクリック
2. ファイル名: `site-qr-A現場.png`（現場名が自動的に付与）
3. ダウンロードフォルダに保存される

**保存後の使い方**:
- 印刷して現場の入口に掲示
- タブレットに表示して現場事務所に設置
- クラウドストレージに保存して共有

### 現場QRの印刷

1. **「印刷」**ボタンをクリック
2. 印刷プレビュー画面が自動的に開く
3. プリンターを選択して印刷

**印刷内容**:
```
━━━━━━━━━━━━━━━━━━━━
A現場 - 現場QRコード

[QRコード画像]

発行日時: 2026-01-21 10:30:00
有効期限: 2026-02-20 23:59:59
期限切れ後は新しいQRコードを発行してください
━━━━━━━━━━━━━━━━━━━━
```

### 現場QRの再発行

**再発行が必要な場合**:
- 有効期限が切れた
- QRコードを紛失した
- 印刷物が破損・汚損した
- セキュリティ上、定期的に更新したい

**再発行手順**:

1. 画面上部の「現場選択」で、再発行したい現場を選択
2. ボタンが **「QRコード再発行」** と表示される
3. クリックすると即座に新しいQRコードを発行
4. **古いQRコードは無効化**される

**注意事項**:
- ⚠️ 再発行すると、古いQRコードは使用できなくなります
- ⚠️ 現場に掲示しているQRコードを新しいものに貼り替えてください

## QR更新頻度の設定

QRコードの有効期限は、**出退勤設定**で変更できます。

### 設定変更方法

**ステップ1: 出退勤設定を開く**

```
出退勤 > 出退勤設定
```

**ステップ2: QR更新頻度を変更**

1. 「QRコード設定」セクションに移動
2. 「QR更新頻度」ドロップダウンをクリック
3. 希望の頻度を選択
4. **「設定を保存」**ボタンをクリック

**選択肢**:

| 頻度 | 有効期限 | 推奨用途 |
|------|----------|----------|
| **1日** | 翌日まで | 高セキュリティが必要な現場 |
| **3日** | 3日後まで | 短期プロジェクト |
| **7日** | 1週間後まで | 週単位で管理したい場合 |
| **30日** | 30日後まで | 長期プロジェクト（推奨） |

### 推奨設定

**通常の現場**: **30日**（デフォルト）
- 毎月1回の再発行で管理が簡単
- スタッフの混乱を防げる

**セキュリティ重視**: **7日**
- 週次で新しいQRコードに更新
- 紛失リスクを低減

**高セキュリティ**: **1日**
- 毎日新しいQRコードを発行
- 不正使用を防止
- ただし、毎日の再発行作業が必要

### 設定変更時の注意

**既存QRへの影響**:
- ✅ 既に発行されているQRコードの有効期限は変わりません
- ✅ 次回発行から新しい設定が適用されます

**例**:
```
現在の設定: 30日
既存のQRコード: 2026-02-20まで有効
↓ 設定を7日に変更
既存のQRコード: 引き続き2026-02-20まで有効
新規発行: 7日後まで有効
```

## QRコードの印刷と設置

### 印刷の推奨サイズ

**A4用紙に印刷する場合**:
- QRコードサイズ: 10cm × 10cm 以上
- タイトル: 大きな文字で「出退勤QR」
- 有効期限を明記

**印刷例**:
```
┏━━━━━━━━━━━━━━━━━┓
┃                   ┃
┃   出退勤QR        ┃
┃   A現場           ┃
┃                   ┃
┃  ┌───────────┐   ┃
┃  │             │   ┃
┃  │  [QR画像]   │   ┃
┃  │  10×10cm    │   ┃
┃  │             │   ┃
┃  └───────────┘   ┃
┃                   ┃
┃ 有効期限:         ┃
┃ 2026-02-20        ┃
┃                   ┃
┗━━━━━━━━━━━━━━━━━┛
```

### 設置場所の推奨

**会社QRの設置場所**:
- ✅ 会社の入口（正面玄関）
- ✅ 受付カウンター
- ✅ タイムカード機の近く
- ✅ 社員食堂入口

**現場QRの設置場所**:
- ✅ 現場の入口（ゲート）
- ✅ 現場事務所入口
- ✅ 朝礼場所
- ✅ 休憩所入口

**設置時の注意**:
- ⚠️ 雨風に濡れない場所
- ⚠️ 直射日光が当たらない場所
- ⚠️ スマートフォンのカメラで撮影しやすい高さ（1.2m〜1.5m）
- ⚠️ ラミネート加工を推奨（耐久性向上）

### タブレット表示の場合

**推奨タブレット**:
- iPad（第7世代以降）
- Android タブレット（10インチ以上）

**表示方法**:
1. QRコード画像を保存
2. タブレットに転送
3. 画像を全画面表示
4. タブレットを固定（スタンド使用）

**メリット**:
- ✅ 印刷不要
- ✅ 再発行時も画像を更新するだけ
- ✅ 雨風に強い（屋外用ケース使用）

**デメリット**:
- ❌ タブレットのバッテリー管理が必要
- ❌ 盗難リスク

## トラブルシューティング

### QRコードが発行できない

**症状**: 「QRコード発行」ボタンをクリックしても発行されない

**原因と対処法**:

1. **権限不足**
   - 確認: 自分の権限が Leader / Manager / Admin か確認
   - 対処: 権限がない場合は管理者に連絡

2. **現場が選択されていない**
   - 確認: 現場選択ドロップダウンで現場を選択したか確認
   - 対処: 現場を選択してから発行ボタンをクリック

3. **ネットワークエラー**
   - 確認: インターネット接続を確認
   - 対処: ページを再読み込みして再試行

### 会社QRが表示されない

**症状**: 会社QR発行セクションが表示されない

**原因と対処法**:

1. **権限不足**
   - 確認: 自分の権限が Manager または Admin か確認
   - 対処: Leader権限では会社QRを発行できません。Manager/Adminに依頼

2. **出退勤QR機能が無効**
   - 確認: 出退勤設定で「QRスキャン許可」がONか確認
   - 対処: 管理者に設定変更を依頼

### QRコードが期限切れと表示される

**症状**: スタッフがQRコードをスキャンすると「期限切れ」エラー

**原因と対処法**:

**有効期限を確認**:
```
出退勤 > 出退勤QR発行 > 有効な現場QRコード
```

**期限切れの場合**:
1. 該当する現場を選択
2. 「QRコード再発行」をクリック
3. 新しいQRコードを印刷・設置

**予防策**:
- 有効期限の1週間前にリマインダーを設定
- カレンダーに再発行日を登録
- 長めの有効期限（30日）を設定

### 印刷がうまくいかない

**症状**: QRコードが小さく印刷される、または切れてしまう

**対処法**:

1. **印刷プレビューを確認**
   - ページサイズ: A4
   - 余白: 2cm（標準）
   - 拡大縮小: なし（100%）

2. **ブラウザの印刷設定**
   - 「背景のグラフィック」をON
   - 「ヘッダーとフッター」をOFF
   - 向き: 縦

3. **PDFに一度保存してから印刷**
   - 印刷画面で「PDFに保存」を選択
   - 保存したPDFを開いて印刷

### 保存ボタンを押してもダウンロードされない

**症状**: 「保存」ボタンをクリックしても画像がダウンロードされない

**原因と対処法**:

1. **ブラウザのポップアップブロック**
   - 確認: ブラウザのアドレスバーにブロックアイコンがないか確認
   - 対処: ポップアップを許可

2. **ダウンロードフォルダの確認**
   - 確認: ブラウザの設定でダウンロードフォルダを確認
   - 対処: フォルダが存在するか、書き込み権限があるか確認

3. **別の方法で保存**
   - 「表示」ボタンでQRコードを表示
   - QRコード画像を右クリック > 「名前を付けて画像を保存」

### QRコードをスキャンしても反応しない

**症状**: スタッフがQRコードをスキャンしても打刻されない

**原因と対処法**:

1. **QRコードが不鮮明**
   - 確認: 印刷が鮮明か、汚れていないか確認
   - 対処: 新しく印刷し直す、ラミネート加工

2. **カメラ権限がない**
   - 確認: スマートフォンのカメラ権限を確認
   - 対処: 出退勤打刻マニュアルの「カメラ権限の許可」を参照

3. **期限切れ**
   - 確認: QRコードの有効期限を確認
   - 対処: 新しいQRコードを再発行

4. **組織設定でQRスキャンが無効**
   - 確認: 出退勤設定で「QRスキャン許可」がONか確認
   - 対処: 管理者に設定変更を依頼
