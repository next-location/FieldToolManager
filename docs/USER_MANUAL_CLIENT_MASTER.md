## 1. 取引先マスタとは

取引先マスタは、顧客・仕入先・協力会社などの取引先情報を一元管理する機能です。

### 1.1 主な用途

- **顧客管理**: 見積書・請求書発行の基礎情報
- **仕入先管理**: 発注先情報、支払い管理
- **協力会社管理**: 外注先情報、工事委託先管理
- **現場との紐付け**: 各現場がどの取引先のものかを記録

### 1.2 管理できる情報

- 基本情報（名称、略称、業種）
- 連絡先情報（住所、電話、メール、ウェブサイト）
- 担当者情報（氏名、部署、連絡先）
- 取引条件（支払条件、支払方法、与信限度額）
- 銀行情報（振込先口座）
- 税務情報（法人番号、インボイス登録番号）
- 評価とメモ

## 2. アクセス方法

### 2.1 メニューからのアクセス

1. 左側のサイドバーを開く
2. **「マスタ管理」** をクリック
3. **「取引先マスタ」** をクリック

### 2.2 権限について

⚠️ **重要**: 取引先マスタは**管理者（admin）権限のみ**がアクセス可能です。

リーダーやスタッフ権限のユーザーはメニューに表示されません。

取引先一覧画面では、登録されている全ての取引先が表示されます。

```

│ 取引先マスタ                 [CSVエクスポート]   │

│ フィルター: [すべて▼] [有効のみ☑] [検索...]    │
├─────────────────────────────────────────────────┤
│ 📊 統計情報（グラフ表示）                       │

│ CL-0001 | 株式会社サンライズ開発  [顧客] [有効] │
│         📍 東京都渋谷区...                      │
│                                                 │
│ CL-0002 | 有限会社グリーンホーム  [顧客] [有効] │
│         📍 神奈川県横浜市...                    │

```

### 3.2 フィルター機能

#### 取引先分類フィルター
- **すべて**: 全ての取引先を表示
- **顧客**: 顧客のみ表示
- **仕入先**: 仕入先のみ表示
- **協力会社**: 協力会社のみ表示
- **顧客兼仕入先**: 両方の役割を持つ取引先のみ表示

#### 有効/無効フィルター
- **有効のみ**: 現在取引中の取引先のみ表示
- **すべて**: 無効化された取引先も含めて表示

#### 検索機能
検索ボックスで以下の項目を検索できます:
- 取引先名
- 取引先コード
- 住所
- 電話番号

### 3.3 統計ダッシュボード

一覧画面の上部に表示される統計情報:

- **総取引先数**: 登録されている取引先の総数
- **有効/無効数**: 現在取引中の取引先数
- **平均評価**: 取引先の評価平均
- **分類別統計**: 顧客・仕入先・協力会社の内訳
- **評価分布**: ★5〜★1の分布グラフ
- **取引実績**: 累計取引額、取引回数（将来機能）

## 4. 取引先の新規登録

### 4.1 登録手順

1. 一覧画面右上の **[+ 新規登録]** ボタンをクリック
2. 必要な情報を入力
3. 画面下部の **[登録]** ボタンをクリック

### 4.2 入力項目

#### ⭐ 必須項目
- **取引先名**: 正式な会社名・屋号
- **取引先分類**: 顧客 / 仕入先 / 協力会社 / 顧客兼仕入先

#### 📝 基本情報（任意）
- **フリガナ**: 取引先名のカナ読み
- **略称**: 社内で使う短縮名
- **業種**: 建設業、塗装業など
- **有効フラグ**: 現在取引中かどうか（デフォルト: 有効）

#### 📞 連絡先情報（任意）
- **郵便番号**: 123-4567形式
- **住所**: 本社所在地
- **電話番号**: 代表電話
- **FAX番号**: FAX番号
- **メールアドレス**: 代表メール
- **ウェブサイト**: 公式サイトURL

#### 👤 担当者情報（任意）
- **担当者名**: 窓口担当者の氏名
- **部署**: 担当者の所属部署
- **電話番号**: 担当者直通電話
- **メールアドレス**: 担当者メール

#### 💰 取引条件（任意）
- **支払条件**: 例「月末締め翌月末払い」
- **支払方法**: 銀行振込 / 現金 / 小切手 / 掛売り / その他
- **支払期日（日数）**: 締日からの支払期日（デフォルト: 30日）
- **与信限度額**: 掛取引の上限額（円）

#### 🏦 銀行情報（任意）
- **銀行名**: 例「みずほ銀行」
- **支店名**: 例「渋谷支店」
- **口座種別**: 普通預金 / 当座預金 / その他
- **口座番号**: 7桁の口座番号
- **口座名義**: カタカナの口座名義

#### 📋 税務情報（任意）
- **法人番号**: 13桁の法人番号
- **インボイス登録番号**: T + 13桁の番号
- **免税事業者**: チェックボックス

#### ⭐ 評価とメモ（任意）
- **評価**: ★1〜★5の5段階評価
- **備考**: 取引先に関する一般的なメモ
- **社内用メモ**: 社内のみで共有する機密メモ（🔒マーク付き）

### 4.3 自動設定項目

以下の項目は自動的に設定されます:

- **取引先コード**: CL-0001, CL-0002... と自動採番
- **登録日**: 登録した日時
- **更新日**: 最後に更新した日時

### 4.4 登録時の注意点

✅ **取引先名の重複はできません**
- 同じ名前の取引先は登録できません
- 略称や（株）（有）の有無に注意

❌ **削除はできません（無効化のみ）**
- 一度登録した取引先は削除できません
- 取引終了時は「有効フラグ」を外して無効化してください

## 6. 取引先の編集

### 6.1 編集手順

1. 詳細画面右上の **[編集]** ボタンをクリック
2. 変更したい項目を修正
3. 画面下部の **[更新]** ボタンをクリック

### 6.2 編集時の注意点

⚠️ **取引先コードは変更できません**
- 自動採番されたコードは変更不可

⚠️ **取引先名の重複チェック**
- 他の取引先と同じ名前には変更できません

✅ **更新日が自動更新されます**
- 編集すると「更新日」が現在日時に更新されます

## 7. 取引先の削除

### 7.1 削除の制限

取引先は**論理削除**（無効化）のみ可能で、物理的な削除はできません。

❌ **削除できない場合**
- 現場に紐づいている取引先は削除できません
- エラーメッセージ: 「この取引先には◯件の現場が紐づいているため削除できません」

### 7.2 削除手順

1. 詳細画面右上の **[削除]** ボタンをクリック
2. 確認ダイアログで **[削除]** をクリック
3. 削除された取引先は一覧に表示されなくなります

### 7.3 削除後の復旧

削除された取引先は完全に消えるわけではなく、「削除済み」状態になります。

復旧が必要な場合は、システム管理者に連絡してください。

## 8. CSV一括登録

### 8.1 CSVインポート機能

複数の取引先を一度に登録できる機能です。

### 8.2 インポート手順

1. 一覧画面右上の **[CSVインポート]** ボタンをクリック
2. CSVファイルを選択
3. インポート結果が表示されます
   - 成功: ◯件
   - エラー: ◯件（エラー詳細が表示されます）

### 8.3 CSVファイル形式

#### ヘッダー行（1行目）

```csv
取引先名,フリガナ,略称,取引先分類,業種,郵便番号,住所,電話番号,FAX番号,メールアドレス,ウェブサイト,担当者名,担当者部署,担当者電話,担当者メール,支払条件,支払方法,支払期日（日数）,与信限度額,銀行名,支店名,口座種別,口座番号,口座名義,法人番号,インボイス登録番号,免税事業者,評価,備考,社内用メモ,有効フラグ
```

#### データ行（2行目以降）

```csv
株式会社サンプル,カブシキガイシャサンプル,サンプル,顧客,建設業,100-0001,東京都千代田区,03-1234-5678,03-1234-5679,info@sample.co.jp,https://sample.co.jp,山田太郎,営業部,03-1234-5680,yamada@sample.co.jp,月末締め翌月末払い,銀行振込,30,5000000,みずほ銀行,渋谷支店,普通預金,1234567,カ)サンプル,1234567890123,T1234567890123,いいえ,5,優良顧客,機密メモ,有効
```

### 8.4 CSVインポートの注意点

✅ **文字コード**: UTF-8（BOM付き）を推奨
✅ **区切り文字**: カンマ（,）
✅ **引用符**: 値にカンマや改行を含む場合はダブルクォート（"）で囲む

❌ **エラーになる場合**
- 必須項目（取引先名、取引先分類）が空の場合
- 取引先名が重複している場合
- 取引先分類が「顧客」「仕入先」「協力会社」「顧客兼仕入先」以外の場合

### 8.5 取引先分類の指定方法

以下のいずれかの形式で指定できます:

- **日本語**: 顧客 / 仕入先 / 協力会社 / 顧客兼仕入先
- **英語**: customer / supplier / partner / both

### 8.6 エラー時の対応

インポート結果画面にエラー詳細が表示されます:

```
インポート結果
成功: 5件 / エラー: 2件

エラー詳細:
行3: 取引先名と取引先分類は必須です
行7: 同じ名前の取引先がすでに登録されています
```

エラーがあった行のみCSVファイルを修正して、再度インポートしてください。

## 9. CSVエクスポート

### 9.1 エクスポート機能

登録されている取引先情報をCSVファイルとしてダウンロードできます。

### 9.2 エクスポート手順

1. 一覧画面右上の **[CSVエクスポート]** ボタンをクリック
2. 自動的にCSVファイルがダウンロードされます

ファイル名: `clients_2025-12-05.csv`（日付は実行日）

### 9.3 エクスポートされる項目

全ての取引先情報（33項目）がエクスポートされます:

- 基本情報、連絡先、担当者、取引条件
- 銀行情報、税務情報、評価、メモ
- 登録日

### 9.4 フィルター適用

エクスポート時、一覧画面で適用しているフィルターがそのまま反映されます。

例:
- 「顧客」のみフィルター → 顧客のみエクスポート
- 「有効のみ」フィルター → 有効な取引先のみエクスポート

### 9.5 Excelでの開き方

1. CSVファイルをダウンロード
2. Excelで開く（自動的にBOM付きUTF-8として認識されます）
3. 日本語が正しく表示されます

## 10. 現場との紐付け

### 10.1 現場登録時の紐付け

現場を登録する際に、その現場がどの取引先のものかを指定できます。

#### 手順

1. **現場マスタ** → **[+ 新規登録]** をクリック
2. 「取引先」欄で取引先を選択
3. 現場情報を入力して登録

### 10.2 紐付けのメリット

✅ **一覧性の向上**
- 取引先詳細画面から、その取引先の全現場を一覧表示

✅ **将来の機能拡張**
- 取引先別の売上集計
- 見積書・請求書発行時の自動入力
- 取引先別の作業報告書

### 10.3 紐付けの変更

現場の編集画面から、紐付けている取引先を変更できます。

### 10.4 紐付けの解除

現場編集画面で「取引先」欄を「選択してください」に戻すと、紐付けが解除されます。

## 11. 統計情報の見方

### 11.1 統計ダッシュボードの表示条件

統計情報は以下の条件で表示されます:

✅ フィルターが「すべて」の場合
✅ 検索キーワードが空の場合
✅ 「有効のみ」が選択されている場合

### 11.2 統計項目の説明

#### 📊 取引先サマリー
- **総取引先数**: 登録されている全取引先
- **有効**: 現在取引中の取引先
- **無効**: 取引終了した取引先
- **平均評価**: 評価の平均値（★1〜★5）

#### 📈 取引先分類
- 顧客、仕入先、協力会社、顧客兼仕入先の内訳
- 各分類の割合を棒グラフで表示

#### ⭐ 評価分布
- ★5〜★1の評価別の取引先数
- 棒グラフで視覚的に表示

#### 💰 取引実績（将来機能）
- 累計取引額、取引回数、平均取引額
- ※現在は準備中

#### 📋 その他統計
- **与信限度額合計**: 全取引先の与信限度額の合計
- **インボイス登録事業者**: インボイス登録番号を持つ取引先数
- **免税事業者**: 免税事業者の取引先数

## 12. よくある質問

### Q1. 取引先コードは変更できますか？

**A:** いいえ、取引先コードは自動採番されるため変更できません。

---

### Q2. 取引先名を変更したら、紐付いている現場も自動更新されますか？

**A:** はい、現場には取引先のIDで紐付けているため、取引先名を変更しても現場との紐付けは維持されます。

---

### Q3. 削除した取引先を復活できますか？

**A:** 削除は論理削除のため、データベースには残っています。復活が必要な場合は、システム管理者に連絡してください。

---

### Q4. CSVインポート時にエラーが出ました。どうすればいいですか？

**A:** インポート結果画面にエラー詳細が表示されます。エラーがあった行のみCSVファイルを修正して、再度インポートしてください。

---

### Q5. 取引先の評価は必須ですか？

**A:** いいえ、評価は任意項目です。未評価の取引先は「未評価」として扱われます。

---

### Q6. 社内用メモは誰に見えますか？

**A:** 社内用メモは管理者のみが閲覧できます（🔒マーク付き）。

---

### Q7. インボイス登録番号とは何ですか？

**A:** 適格請求書発行事業者に発行される番号です（T + 13桁）。インボイス制度に対応した請求書を発行するために必要な情報です。

---

### Q8. 与信限度額を超えた場合、警告が出ますか？

**A:** 現在は与信限度額を記録するのみで、警告機能は実装されていません。将来のバージョンで対応予定です。

---

### Q9. 取引先を一括削除できますか？

**A:** いいえ、一括削除機能はありません。取引先は1件ずつ削除してください。

---

### Q10. 取引先に紐づく現場を一括で確認できますか？

**A:** はい、取引先の詳細画面の「関連現場」タブで、その取引先に紐づく全現場を確認できます。
