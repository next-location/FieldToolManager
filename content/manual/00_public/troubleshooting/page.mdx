システムに不具合が発生した際の対応方法をご案内します。

---

## ログインできない場合

ログイン画面にアクセスできない、またはログインができない場合は、以下の方法で運営にお問い合わせください。

### 問い合わせ方法

1. ログイン画面（`https://zairoku.com` または `https://あなたの会社名.zairoku.com`）にアクセス
2. ページ下部にある「**運営への問い合わせ**」リンクをクリック
3. 問い合わせフォームに必要事項を入力
4. 送信ボタンをクリック

### 問い合わせ時に記載する情報

以下の情報を記載いただくと、スムーズに対応できます：

- **組織名**: あなたの会社名またはサブドメイン
- **メールアドレス**: 登録しているメールアドレス
- **発生している問題**: できるだけ具体的に
- **エラーメッセージ**: 表示されている場合はそのまま記載
- **使用環境**: PC/スマホ、ブラウザの種類

**例:**
```
組織名: 株式会社サンプル
メールアドレス: user@example.com
問題: ログイン画面でメールアドレスとパスワードを入力しても
     「メールアドレスまたはパスワードが正しくありません」と
     表示されてログインできません。
使用環境: PC、Google Chrome
```

## ログインできる場合

ログインはできるが、システム内で不具合が発生している場合は、サポートチケット機能をご利用ください。

### サポートチケットでの問い合わせ方法

1. ログイン後、画面右上のユーザーメニューをクリック
2. 「**サポートチケット**」を選択
3. 新規チケットを作成
4. 不具合の詳細を入力して送信

**⚠️ 注意**: サポートチケット機能は現在準備中です。実装されるまでは、ログイン画面の「運営への問い合わせ」リンクからお問い合わせください。

### 問い合わせ時に記載する情報

- **発生日時**: いつから発生しているか
- **発生箇所**: どのページ・機能で発生しているか
- **操作手順**: 何をした時に発生したか
- **エラーメッセージ**: 表示されている場合
- **スクリーンショット**: 可能であれば添付

**例:**
```
発生日時: 2025年1月13日 14:30頃
発生箇所: 道具移動ページ
操作手順: QRコードをスキャンして「移動を実行」ボタンを
         押したところ、エラーが表示されました
エラーメッセージ: 「移動に失敗しました」
```

## よくある不具合と対処法

サポートに問い合わせる前に、以下の対処法をお試しください。

### 1. ページが表示されない・真っ白になる

**対処法:**
- ブラウザのキャッシュをクリア
- ページを再読み込み（Ctrl+R または Command+R）
- 別のブラウザで試す

### 2. ボタンが反応しない

**対処法:**
- ページを再読み込み
- ブラウザの JavaScript が有効か確認
- 別のブラウザで試す

### 3. 画像が表示されない

**対処法:**
- インターネット接続を確認
- ページを再読み込み
- ブラウザのキャッシュをクリア

### 4. データが保存されない

**対処法:**
- 必須項目がすべて入力されているか確認
- ブラウザのCookieが有効か確認
- ページを再読み込みしてもう一度試す

### 5. スマホで操作がしづらい

**対処法:**
- ブラウザを最新版に更新
- 画面の向きを変えてみる（縦↔横）
- 別のブラウザで試す

## 緊急時の対応

### システム全体が使えない場合

システム全体に障害が発生している可能性があります。

**確認方法:**
- 他のユーザーにも同じ問題が発生しているか確認
- 別のネットワーク（モバイル通信など）で試す

**対応:**
- 運営への問い合わせリンクから速やかに報告
- 復旧まで時間がかかる場合は、一時的に別の方法で業務を継続

---

## サポート対応時間

運営からの返信は、通常1〜2営業日以内を目安としています。

- **対応時間**: 平日 9:00〜18:00
- **休業日**: 土日祝日、年末年始

緊急性の高い問題の場合は、問い合わせ時に「緊急」と明記してください。
